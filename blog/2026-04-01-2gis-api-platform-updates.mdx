## Map Tiles API — теперь по запросам {#map-titles-api}

**Map Tiles API** — сервис для интеграции карт в проекты с позапросной тарификацией. Раньше он назывался Map GL JS API и работал по безлимитной модели: мы предоставляли JavaScript-библиотеку, которая под капотом обращалась к тайловому серверу.

Во всех продуктах API Платформы используется позапросная тарификация. Мы привели карты к этой же модели — вместо доступа к библиотекам теперь оплачиваются запросы к серверу.

Для работы с тайловым сервером мы по-прежнему предлагаем собственную JavaScript-библиотеку с широким набором возможностей.

**Особенно выигрывают от обновления стартапы и проекты с небольшим потреблением** — порог входа значительно снизился. Прозрачная ценовая политика, статистика и контроль расходов — всё, как любят наши пользователи.

Подробнее про Map Tiles API можно прочитать [в документации](https://docs.2gis.com/maps/others/maptiles/overview).

## Raster Tiles API {#raster-tiles-api}

Универсальные растровые карты по стандартному протоколу для встраивания. Сервис работает с любыми библиотеками — можно подключить карты 2ГИС к любым геоинформационным системам напрямую или через OpenSource-библиотеки вроде Leaflet.

Раньше карты были доступны только через библиотеку Raster JS, что требовало изменений в коде самих геоинформационных систем для интеграции. Теперь это ограничение снято.

Подробнее — [в документации](https://docs.2gis.com/maps/others/rastertiles/overview).

## Static API {#static-api}

Статические изображения карт 2ГИС для случаев, когда нужно просто отобразить картинку карты:

- простая неинтерактивная карта на сайте
- в печатном документе
- на умных устройствах с экраном, но без поддержки JavaScript
- в ИИ-чатботах

Подробнее о возможностях Static API — в [документации](https://docs.2gis.com/maps/others/static/overview).

## Обновлённая тарифная сетка {#updated-tariff-grid}

Теперь облачные сервисы API Платформы можно купить по отдельности. Ранее часть сервисов можно было купить отдельно, но основная масса продуктов продавалась по единой подписке «котловым методом».

Новая схема позволяет купить подписку и получить доступ именно к тому API-сервису, который нужен на вашем проекте.

Получить бесплатный демоключ и ознакомиться с обновлёнными тарифами на сервисы API-Платформы можно [по этой ссылке](https://platform.2gis.ru/ru/tariffs).

## Оплата картой {#pay-by-card}

В этом месяце в кабинете по управлению API появилась долгожданная оплата картой. Теперь кроме классической оплаты счёта через бухгалтерию или банк можно оплатить подписку на API-сервисы банковской картой — как физическое или юридическое лицо.

Система стала гибкой: можно купить подписку на юрлицо по счёту, а продлить картой.

После релиза **10% пользователей** воспользовались новым способом оплаты. Доля оплаты картой среди новых пользователей за две недели достигла почти **15% и продолжает расти**.

## Вход и регистрация — проще {#login-and-registration}

Разделили вход и регистрацию в кабинете API. Эксперимент с объединением входа и регистрации в одном окне не оправдался — это создавало путаницу, приводило к ложным регистрациям и увеличивало нагрузку на техподдержку.

Вернулись к классическому решению с раздельными сценариями. **Просто и логично**.
