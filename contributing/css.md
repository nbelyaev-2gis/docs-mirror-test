# CSS

- Для написания стилей используй [CSS Modules](https://docusaurus.io/docs/styling-layout#css-modules). Также желательно использовать PostCSS.
- Для написания классов используем camelCase, например `myClassName`.
- Старайся использовать уже существующие css-переменные. Как минимум, потому что есть темы: светлая и темная.
