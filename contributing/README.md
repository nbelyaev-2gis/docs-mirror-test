## Технологии

- NodeJS. Актуальную версию см. в [docker/Dockerfile.app](/docker/Dockerfile.app), в директиве `FROM`.
- Docusaurus

## Установка

```
$ npm ci
```

## Локальная разработка

Создай в корне файл `.env.local` и пропиши внутри env-переменные. Значения возьми из Gitlab CI.

```
APIKEY_MAPGL=
APIKEY_MAPGL_JSFIDDLE=
APIKEY_NAVIGATION=
APIKEY_PLACES=

APIKEY_DOCS_SEARCH=
```

Запускай проект через команду

```
$ npm start
```

## Внесение изменений

Коротко о самом важном:

- Тексты на русском языке хранятся в виде файлов `.mdx` внутри `docs` в подпапке продукта (например, `docs/pro`).
- Тесты на английском языке хранятся в виде файлов `.mdx` внутри `i18n/en/docusaurus-plugin-content-docs/current` в подпапке продукта (например, `i18n/en/docusaurus-plugin-content-docs/current/pro`).
- Картинки хранятся внутри `static/img`.
- Картинки следует оформлять в разметке Markdown, например: `![Внешний вид деревьев](/img/mapgl/models/tree_intro.png)`. HTML-теги можно использовать, только если необходимо указать дополнительные атрибуты типа ширины.
- Для картинок в формате svg реализована логика инвертации цвета при включении темной темы. Чтобы избежать этого (например если svg используется в виде схемы, а не как иконка) - оберни свою картинку в компонент `<NoInvert />`.
- Ссылки имеют новый формат: без указания локали (`ru/` или `/en`). Пример: `/maps/styles/overview`.
- Интерактивные демо карт прописываются внутри файлов `.mdx` и не хранятся в виде отдельных файлов.
- Добавить бренд (2ГИС / Urbi) в mdx можно с помощью компонента `<Brand />`.
- CTA-кнопку в MDX можно добавить через `<CtaButton />`. Подробнее: `contributing/mdx/button.md`.

Чтобы добавить/удалить страницу из структуры документа или изменить её заголовок в меню, отредактируйте нужный файл в папке `sidebars` (например `pro.sidebar.ts`).

## Полезное

Для удобной смены темы на отдельной вкладке используйте query param `?docusaurus-theme=dark` или `?docusaurus-theme=light`. Например, `http://localhost:3000/?theme=dark`. Подробнее см. [документацию Docusaurus](https://docusaurus.io/docs/styling-layout#dark-mode).
Если какая-то информация не была найдена в оригинальной доке, можно попробовать поискать в [community документации](https://docusaurus.community/).
