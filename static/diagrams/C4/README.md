# Как обновлять диаграммы

1. Откройте нужный файл `.dsl` и скопируйте его содержимое в редактор https://structurizr.com/dsl.
1. Внесите необходимые изменения. При необходимости обращайтесь Structurizr DSL language reference (верхний левый угол редактора).
1. Чтобы сгенерировать диаграмму:

    1. Над полем для редактирования кода нажмите **Render**.
    1. Наведите курсор на сгенерированную диаграмму. Нажмите на значок PNG, появившийся в правом нижнем углу.
    1. Выберите опцию **Crop diagrams** и нажмите **Export current diagram**.
    1. Нажмите на предложенную диаграмму (ниже показана легенда для этой диаграммы в формате PNG, ее генерировать не нужно).
    1. Сохраните PNG-файл в `digital-docs\public\img\c4`.
