# Обзор решения 

<Brand /> Про для сервера — сервис для управления геоданными и их анализа, который может работать в вашей инфраструктуре.

<Brand /> Про позволяет:

* управлять наборами геоданных и фильтровать их по атрибутам;
* работать со слоями данных по зданиям, организациям, населению и инфраструктуре;
* визуализировать геоданные в виде тепловых карт, кластеров, гексагонов и т. д.

Сервис может работать с геоданными по организациям, зданиям и населению, собранными, обработанными и опубликованными <Brand /> Про, а также с загруженными наборами пользовательских геоданных.

Для работы <Brand /> Про установите [API-платформу](/on-premise-api-platform/overview/summary).
