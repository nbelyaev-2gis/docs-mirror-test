# iOS SDK

iOS SDK позволяет добавить [карту <Brand />](https://2gis.ru/) в ваше iOS-приложение. С помощью SDK можно отобразить карту на экране, получать информацию об объектах карты, строить маршруты, использовать навигатор и т. д.

Примеры использования SDK см. в разделе [Примеры](/ios/sdk/examples/base). Описание всех классов и методов см. в разделе [Справочник API](/ios/sdk/reference/stable/class).

Картографические данные поддерживают [стандарты OGC](https://ru.wikipedia.org/wiki/Open_Geospatial_Consortium).

## Версии SDK {#versions}

Пакеты iOS SDK доступны в двух версиях:

* **Full** — полная версия, которая позволяет:
  * Добавлять [карту](/ios/sdk/examples/map) в приложение и управлять ей: добавлять маркеры и геометрические фигуры, управлять движением камеры, отображать слои (например, пробки), работать со стилями карты и т. д.
  * Искать объекты в [справочнике](/ios/sdk/examples/directory): организации, здания, места и другие.
  * Работать с [навигатором](/ios/sdk/examples/navigation) в приложении в трёх режимах: свободная навигация, ведение по маршруту и симуляция ведения.
  * Строить оптимальные [маршруты](/ios/sdk/examples/routing) на карте, в том числе внутри зданий.
  * Использовать [офлайн-данные](/ios/sdk/start#offline-mode).
* **Map** — облегчённая версия, которая позволяет:
  * Добавлять [карту](/ios/sdk/examples/map) в приложение и управлять ей: добавлять маркеры и геометрические фигуры, управлять движением камеры, отображать слои (например, пробки), работать со стилями карты и т. д.
  * Искать объекты в [справочнике](/ios/sdk/examples/directory): организации, здания, места и другие.

:::warning

Не используйте версии **Full** и **Map** вместе: это приведёт к ошибке сборки.

:::

## API для работы SDK {#api-list}

В зависимости от нужных функциональностей, для работы SDK требуются ключи доступа к API карт, поиска и навигации. Доступные возможности зависят от [версии SDK](#versions): **Full** или **Map**.

При использовании ключа оплачивается только доступ к используемым API, в зависимости от количества отправленных запросов (и в некоторых случаях — полученных ответов). Вы можете работать с ключами в личном кабинете [Менеджер Платформы](https://platform.2gis.ru/ru/?utm_source=docs&utm_medium=referral&utm_campaign=sdk_ios).

## Требования к устройствам {#system-requirements}

* Xcode 14.
* iOS 16 и выше / iPadOS 16 и выше (это обусловлено использованием [SwiftUI](https://developer.apple.com/documentation/swiftui) в проекте).

## Варианты размещения {#distribution}

* **Облако**: iOS SDK доступен для работы в облачной инфраструктуре <Brand />.
* **On-Premise**: iOS SDK доступен при установке API-платформы <Brand /> в закрытом контуре.

   * Для версий SDK выше 12.10 установите API-платформу версии 2.54.0 и выше и включите поддержку тайлов типа `native-v4`.
   * Для версий SDK 12.10 и ниже установите API-платформу любой версии и включите поддержку тайлов типа `native`.

   Подробнее см. в разделе [API-платформа для сервера](/on-premise-api-platform/overview/summary).
