# UIKit

## Карта

Все варианты поставки SDK содержат стандартный набор UI-элементов для управления картой, которые вы можете добавить в иерархию и использовать для управления параметрами карты.

Процесс создания и добавления `Map` подробно описан в разделе [Создание карты](/ios/sdk/examples/map#creating-map).

Все перечисленные ниже элементы для удобства объединены в [IMapUIControlsFactory](/ios/sdk/reference/stable/interface#imap-uicontrols-factory) и [IRoadEventCardViewFactory](/ios/sdk/reference/stable/IRoadEventCardViewFactory).

Стандартный набор включает в себя:

<img src="/img/mobile-sdk/ios_map_controls_1.png" width="200" /> <img src="/img/mobile-sdk/ios_map_controls_2.png" width="200" /> <img src="/img/mobile-sdk/ios_map_controls_3.png" width="200" />

1. [IndoorUIControl](/ios/sdk/reference/stable/class#indoor-uicontrol) для переключения этажей.
2. [TrafficUIControl](/ios/sdk/reference/stable/class#traffic-uicontrol) для управления видимостью пробок на карте.
3. [ZoomUIControl](/ios/sdk/reference/stable/class#zoom-uicontrol) для масштабирования.
4. [CreateRoadEventControl](/ios/sdk/reference/stable/CreateRoadEventControl) для добавления дорожного события.
5. [CompassControl](/ios/sdk/reference/stable/CompassControl) для отображения текущего угла поворота карты относительно севера.
6. [CurrentLocationControl](/ios/sdk/reference/stable/CurrentLocationControl) для перелёта к текущему местоположению пользователя.
7. [CreateRoadEventView](/ios/sdk/reference/stable/IRoadEventCardViewFactory) для описания нового дорожного события.
8. [RoadEventCardView](/ios/sdk/reference/stable/IRoadEventCardViewFactory) для просмотра информации о дорожном событии.

## Навигатор

Полный вариант поставки SDK содержит набор UI-элементов для отображения и управления навигацией. Для удобства все элементы объединены в [INavigationMapUIControlsFactory](/ios/sdk/reference/stable/interface#inavigation-map-uicontrols-factory) и [INavigationUIControlsFactory](/ios/sdk/reference/stable/interface#inavigation-uicontrols-factory). Способ их использования и добавления описан в разделе [Навигатор](/ios/sdk/examples/navigation).

Набор UI-элементов навигации состоит из:

<img src="/img/mobile-sdk/ios_navi_controls_1.png" width="200" /> <img src="/img/mobile-sdk/ios_navi_controls_2.png" width="200" />

1. [ThermometerUIControl](/ios/sdk/reference/stable/interface#inavigation-uicontrols-factory/make-thermometer-uicontrol) для отображения уровня пробок на маршруте.
2. [NextManeuverUIControl](/ios/sdk/reference/stable/interface#inavigation-uicontrols-factory/make-next-maneuver-uicontrol) для отображения информации о следующем манёвре.
3. [SpeedUIControl](/ios/sdk/reference/stable/interface#inavigation-uicontrols-factory/make-next-maneuver-uicontrol) для отображения текущей скорости и скоростного ограничения.
4. [TrafficAndParkingMapControl](/ios/sdk/reference/stable/INavigationMapControlsFactory#nav-lvl1--makeTrafficAndParkingMapControl) для управления видимостью пробок и парковок на карте.
5. [ZoomUIControl](/ios/sdk/reference/stable/class#zoom-uicontrol) для масштабирования (приближения и удаления) карты.
6. [CompassControl](/ios/sdk/reference/stable/INavigationMapControlsFactory#nav-lvl1--makeCompassControl) для отображения текущего угла поворота карты относительно севера.
7. [NavigationFollowingControl](/ios/sdk/reference/stable/INavigationMapControlsFactory#nav-lvl1--makeNavigationFollowingControl) для управления слежением за маркером местоположения.
8. [RemainingRouteInfoUIControl](/ios/sdk/reference/stable/interface#inavigation-uicontrols-factory/make-next-maneuver-uicontrol) для отображения информации в режиме [ведения по маршруту](/ios/sdk/examples/navigation#turn-by-turn-mode).

<img src="/img/mobile-sdk/ios_navi_controls_3.png" width="200" /> <img src="/img/mobile-sdk/ios_navi_controls_4.png" width="200" /> <img src="/img/mobile-sdk/ios_navi_controls_5.png" width="200" />

9. [RemainingRouteInfoUIControl](/ios/sdk/reference/stable/interface#inavigation-uicontrols-factory/make-next-maneuver-uicontrol) для отображения информации в режиме [свободной навигации](/ios/sdk/examples/navigation#free-drive-mode).
10. [RemainingRouteInfoUIControl](/ios/sdk/reference/stable/interface#inavigation-uicontrols-factory/make-next-maneuver-uicontrol) для отображения информации в режиме [навигации внутри зданий](/ios/sdk/examples/navigation#indoor-navigation).
11. [IndoorControl](/ios/sdk/reference/stable/INavigationMapControlsFactory#nav-lvl1--makeIndoorControl) для выделения этажей, через которые проходит маршрут.
