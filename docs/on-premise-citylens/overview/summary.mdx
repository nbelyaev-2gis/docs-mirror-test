
# Обзор решения 

<Brand /> Ситискан (CityLens) для сервера — сервис для получения и обработки изображений, который может работать в вашей инфраструктуре.

<Brand /> Ситискан позволяет:

* получать изображение в режиме вождения через мобильное приложение;
* обрабатывать изображения, обнаруживать инциденты и классифицировать их с использованием алгоритмов компьютерного зрения;
* планировать задачи для объезда территорий, анализировать результаты и составлять отчеты.

Для работы <Brand /> Ситискан установите [API-платформу](/on-premise-api-platform/overview/summary) и [<Brand /> Про](/on-premise-pro/overview/summary).
