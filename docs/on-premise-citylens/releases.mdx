# Релизы <Brand /> Ситискан

Чтобы установить <Brand /> Ситискан версии 2.0.0 и выше, установите [утилиту 2GIS CLI версии 3](/on-premise-citylens/overview/2gis-cli/releases#v3) и следуйте [инструкциям по обновлению](/on-premise-citylens/update).

## 2.2.0 {#v2.2.0}

Дата релиза: 2025-12-05

<h3 id="dependencies_v2.2.0">Зависимости</h3>

Перед обновлением <Brand /> Ситискан до версии 2.2.0 убедитесь, что установлены компоненты-зависимости нужных версий:

| <div style={{"width":"170px"}}>Компонент</div> | Минимальная версия |
| ---------------------------------------- | ------ |
| Утилита 2GIS CLI                         | 3      |
| Базовые сервисы                          | 2.1.0  |
| API-платформа                            | 2.47.0 |
| <Brand /> Про                            | 2.1.0  |

Подробнее см. в разделе [Обновление <Brand /> Ситискан](/on-premise-citylens/update).

<h3 id="new_v2.2.0">Новое</h3>

| <div style={{"width":"170px"}}>Сервис</div> | Изменения |
| ------------------------------------------- | --------- |
| CityLens API и CityLens Routes API | Добавлена возможность [cтроить маршруты по точкам на карте](/citylens/administrator/planner/tasks#creating-task) в Планировщике задач. |

<h3 id="breaking_v2.2.0">Изменения, влияющие на обратную совместимость</h3>

| <div style={{"width":"170px"}}>Сервис</div> | Изменения |
| ------------------------------------------- | --------- |
| CityLens API и CityLens Routes API | Добавлены обязательные параметры для доступа к Планировщику задач и внутренним сервисам:<ul><li>`routes.adminCredentials.password`,</li><li>`routes.adminCredentials.username`,</li><li>`routes.endpointInternalApiKey.dataScience`,</li><li>`routes.endpointInternalApiKey.companies`.</li></ul>См. [пример values.yaml на GitHub](https://github.com/2gis/on-premise-helm-charts/blob/Citylens-2.2.0/charts/citylens/values.yaml) и обновлённый [конфигурационный файл](/on-premise-citylens/install/installation#install-citylens-api).|

## 2.1.0 {#v2.1.0}

Дата релиза: 2025-10-16

<h3 id="dependencies_v2.1.0">Зависимости</h3>

Перед обновлением <Brand /> Ситискан до версии 2.1.0 убедитесь, что установлены компоненты-зависимости нужных версий:

| <div style={{"width":"170px"}}>Компонент</div> | Минимальная версия |
| ---------------------------------------- | ------ |
| Утилита 2GIS CLI                         | 3      |
| Базовые сервисы                          | 2.0.0  |
| API-платформа                            | 2.44.0 |
| <Brand /> Про                            | 2.1.0  |

Подробнее см. в разделе [Обновление <Brand /> Ситискан](/on-premise-citylens/update).

<h3 id="new_v2.1.0">Новое</h3>

| <div style={{"width":"170px"}}>Сервис</div> | Изменения |
| ------------------------------------------- | --------- |
| CityLens API и CityLens Routes API    | <ul><li>Добавлена возможность логировать SQL-запросы: см. параметры `routes.api.logging.extendedSqlLogging`, `routes.worker.logging.extendedSqlLogging`, `routes.realtimeDataApi.logging.extendedSqlLogging`.</li><li>Добавлена возможность просматривать информацию о маршруте и точку последней активности водителя на карте в Планировщике задач.</li></ul>См. [пример values.yaml на GitHub](https://github.com/2gis/on-premise-helm-charts/blob/Citylens-2.1.0/charts/citylens/values.yaml). |
| CityLens Routes UI                    | Добавлена переменная окружения `env.METRICS_API_URL`, которая определяет адрес сервиса для сбора метрик: см. [пример values.yaml на GitHub](https://github.com/2gis/on-premise-helm-charts/blob/Citylens-2.1.0/charts/citylens-routes-ui/values.yaml). |

<h3 id="breaking_v2.1.0">Изменения, влияющие на обратную совместимость</h3>

| <div style={{"width":"170px"}}>Сервис</div> | Изменения                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| CityLens API и CityLens Routes API    | <ul><li>Параметр `routes.api.logging.extended` для логирования REST-запросов переименован в `routes.api.logging.extendedRestLogging`.</li><li>Все настройки для подключения к базам данных вынесены в отдельный блок `routes.postgres`:<ul><li>Группа параметров `routes.postgres` перемещена в `routes.postgres.api`.</li><li>Группа параметров `routes.hangfire.postgres` перемещена в `routes.postgres.hangfire`.</li><li>Группа параметров `routes.realtimeDataApi.postgres` перемещена в `routes.postgres.realtimeDataApi`.</li><li>Параметры `routes.postgres.maxPoolSize` и `routes.postgres.pooling` для настройки пула соединений с PostgreSQL объединены в группу `routes.postgres.api.poolSize`.</li></ul></li></ul>См. [пример values.yaml на GitHub](https://github.com/2gis/on-premise-helm-charts/blob/Citylens-2.1.0/charts/citylens/values.yaml), а также обновлённый [конфигурационный файл](/on-premise-citylens/install/installation#install-citylens-api). |
| CityLens Routes UI                    | Добавлена обязательная переменная окружения `env.MAPGL_DEFAULT_CENTER`, которая определяет координаты центра карты по умолчанию: см. [пример values.yaml на GitHub](https://github.com/2gis/on-premise-helm-charts/blob/Citylens-2.1.0/charts/citylens-routes-ui/values.yaml), а также обновлённый [конфигурационный файл](/on-premise-citylens/install/installation#install-citylens-ui). |
