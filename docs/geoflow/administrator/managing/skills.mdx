# Навыки

**Навыки** определяют, какие задачи может выполнять агент. Задача автоматически назначается агенту в смене, если он обладает всеми навыками, которые требуются для её выполнения.

Например, вы можете назначить навык `Холодильная камера` [агентам](/geoflow/administrator/managing/agents) (автомобилям) и указать его при создании [задачи](/geoflow/administrator/tasks) для перевозки скоропортящихся продуктов. Такая задача автоматически назначится агентам, у которых указан навык `Холодильная камера`.

Агент может посмотреть список задач в [мобильном приложении](/geoflow/operator/tasks).

В роли **Главный администратор**, **Администратор компании**  и **Диспетчер** вы можете работать с навыками на вкладке **Управление → Навыки**:

* [создавать навыки](#creating-skill),
* [редактировать навыки](#editing-skill),
* [удалять навыки](#removing-skill).

В роли **Руководитель группы агентов** вы можете только просматривать список навыков и информацию о них.

![Работа с навыками](/img/geoflow/managing-skills.png)

## Создание навыка {#creating-skill}

![Создание навыка](/img/geoflow/creating-skill.png)

1. Перейдите на вкладку **Управление → Навыки**.
2. Нажмите **Добавить навык**.
3. Укажите параметры навыка:

   * **Название** (обязательный параметр): название навыка.
   * **Описание**: описание навыка.

4. Нажмите **Сохранить**.

При создании [шаблона агента](/geoflow/administrator/managing/templates) и [шаблона задачи](/geoflow/administrator/managing/templates) вы можете добавить в него атрибут с типом данных `Навыки`. При создании агента или задачи из этого шаблона вы сможете выбрать в соответствующем поле виды навыков. Для одного агента и одной задачи можно указать до 50 навыков.

## Редактирование навыка {#editing-skill}

Вы можете изменять название навыка и его описание.

1. Перейдите на вкладку **Управление → Навыки**.
2. В списке навыков нажмите значок ![Меню навыка](/img/menu-dots-horizontal.svg) у навыка и выберите **Изменить**.
3. Внесите изменения в навык.
4. Нажмите **Сохранить**.

## Удаление навыка {#removing-skill}

:::warning

При удалении навыка он будет удалён из всех агентов и задач, в которых был указан.

:::

1. Перейдите на вкладку **Управление → Навыки**.
2. В строке навыка нажмите значок ![Меню навыка](/img/menu-dots-horizontal.svg) и выберите **Удалить**.
3. Подтвердите удаление.
