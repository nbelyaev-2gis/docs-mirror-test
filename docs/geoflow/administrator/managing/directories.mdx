# Справочники

**Справочники** — это группы признаков для агентов, которые используются для дополнительной справочной информации.

Например, вы можете создать справочник **Тип транспорта** с признаками `Автомобиль`, `Велосипед`, `Пешком` и добавить его в качестве атрибута в [шаблон агента](/geoflow/administrator/managing/templates). При создании агента из этого шаблона вы сможете выбрать в соответствующем поле, каким транспортом может управлять агент, используя значения из справочника.

В роли **Главный администратор** и **Администратор компании** вы можете работать со справочниками на вкладке **Управление → Справочники**:

* [создавать справочники](#creating-directory),
* [редактировать справочники](#editing-directory),
* [удалять справочники](#removing-directory).

Пользователи с ролями **Диспетчер** и **Руководитель группы агентов** могут только просматривать список справочников.

![Работа со справочниками](/img/geoflow/managing-directories.png)

## Создание справочника {#creating-directory}

![Создание справочника](/img/geoflow/creating-directory.png)

1. Перейдите на вкладку **Управление → Справочники**.
2. Нажмите **Добавить справочник**.
3. Укажите параметры справочника:

   * **Название**: название справочника.
   * **Навык**: чтобы создать справочник навыков, отметьте чекбокс.
   * **Объекты**: введите названия объектов справочника.

   Чтобы добавить объект, нажмите **Добавить объект**. Чтобы удалить объект, нажмите значок ![Удалить](/img/pro/trash-icon.svg) рядом с ним. Чтобы изменить порядок объектов, зажмите значок ![Перемещение](/img/pro/drag-n-drop-icon.svg) у объекта и перетащите его.

4. Нажмите **Сохранить**.

При создании [шаблона агента](/geoflow/administrator/managing/templates) вы можете добавить в него атрибут со справочником. При создании агента из этого шаблона вы сможете выбрать в соответствующем поле значения из справочника.

## Редактирование справочника {#editing-directory}

1. Перейдите на вкладку **Управление → Справочники**.
2. В списке справочников нажмите значок ![Меню справочника](/img/menu-dots-horizontal.svg) у справочника и выберите **Изменить**.
3. Внесите изменения в справочник.
4. Нажмите **Сохранить**.

## Удаление справочника {#removing-directory}

:::warning

При удалении справочника он будет удалён из всех шаблонов агентов, для которых был добавлен.

:::

1. Перейдите на вкладку **Управление → Справочники**.
2. В строке справочника нажмите значок ![Меню справочника](/img/menu-dots-horizontal.svg) и выберите **Удалить**.
3. Подтвердите удаление.
