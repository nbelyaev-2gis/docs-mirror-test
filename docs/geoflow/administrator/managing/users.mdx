# Пользователи

В веб-панели администратора поддерживается [ролевая модель](/geoflow/administrator/overview#role-model) для разграничения прав доступа.

В роли **Главный администратор** и **Администратор компании** вы можете работать с пользователями на вкладке **Управление → Пользователи**:

* [добавлять новых пользователей](#adding-user),
* [просматривать информацию о пользователях](#viewing-user),
* [изменять роли пользователей](#changing-role),
* [редактировать информацию о пользователях](#editing-user),
* [блокировать пользователей](#blocking-user),
* [удалять пользователей](#removing-user).

Назначить роль **Администратор компании** может только **Главный администратор**. В роли **Диспетчер** вы можете только просматривать список пользователей и информацию о них.

![Работа с пользователями](/img/geoflow/managing-users-609.png)

## Добавление пользователя {#adding-user}

![Добавление пользователя](/img/geoflow/creating-user-609.png)

1. Перейдите на вкладку **Управление → Пользователи**.
2. Нажмите **Добавить пользователя**.
3. Укажите данные пользователя:

   * **Способ авторизации**: способ авторизации пользователя:

      * **Почта**: выберите этот способ, чтобы выдать пользователю доступ к веб-панели и мобильному приложению. Укажите email пользователя, на который будет отправлено письмо со ссылкой для регистрации. При регистрации пользователь создаст пароль для входа и укажет свои имя и фамилию.
      
         Приглашение действует 8 часов. Если пользователь не успел принять приглашение, нажмите **Пригласить повторно** в его карточке.

      * **Логин**: выберите этот способ, чтобы выдать пользователю доступ только к мобильному приложению. Укажите фамилию, имя, отчество пользователя и логин. Передайте пользователю логин и временный пароль для первого входа в приложение. При следующем входе пользователь сможет задать постоянный пароль.

   * **Номер телефона**: номер телефона пользователя.
   * **Роль**: роль пользователя. Доступные роли: **Администратор компании** (вы можете назначить её пользователям, только если вы главный администратор), **Диспетчер**, **Руководитель группы агентов**, **Сотрудник**. Подробнее о ролях см. в разделе [Роли пользователей](/geoflow/administrator/overview#role-model).

4. Нажмите **Добавить**.

## Просмотр информации о пользователях {#viewing-user}

1. Перейдите на вкладку **Управление → Пользователи**.
2. Откройте карточку пользователя. Также вы можете использовать поиск по имени пользователя в списке и фильтровать пользователей по статусу приглашения.

<br />

В карточке пользователя отображается:

* **ФИО**: ФИО пользователя.
* **Основная информация**:

   * **Email / Логин**: email или логин пользователя в зависимости от способа авторизации.
   * **Роль**: роль пользователя.

      Здесь могут отображаться другие настройки (например, номер телефона), если они были указаны при добавлении пользователя или отредактированы позже.

* **Настройки**:

   * **Способ авторизации**: почта или логин.
   * **Сбросить пароль**: нажмите, чтобы сбросить пароль пользователя. Если способ авторизации — почта, пользователю будет отправлено письмо со ссылкой для сброса пароля. Если способ авторизации — логин, нажмите **Скопировать пароль** и передайте пользователю новый временный пароль. При следующем входе пользователь сможет задать новый постоянный пароль.

* **Дата создания**: дата создания пользователя.
* **Последнее изменение**: дата последнего изменения информации о пользователе.

## Изменение роли пользователя {#changing-role}

Роли пользователей могут изменять только пользователи с ролями **Главный администратор** или **Администратор компании**. **Администратор компании** может назначать только роли **Диспетчер**, **Руководитель группы агентов** и **Сотрудник**.

Чтобы назначить пользователю другую роль:

1. Перейдите на вкладку **Управление → Пользователи**.
2. В строке пользователя нажмите значок ![Меню пользователя](/img/menu-dots-horizontal.svg) и выберите **Изменить информацию**.
3. Выберите новую роль в выпадающем списке.
4. Нажмите **Сохранить**.

## Редактирование информации о пользователе {#editing-user}

Вы можете изменить ФИО пользователя (если они не указаны самим пользователем) и номер телефона. Редактировать информацию могут только пользователи с ролями **Главный администратор** или **Администратор компании**.

1. Перейдите на вкладку **Управление → Пользователи**.
2. В строке пользователя нажмите значок ![Меню пользователя](/img/menu-dots-horizontal.svg) и выберите **Изменить информацию**.
3. Внесите изменения в данные.
4. Нажмите **Сохранить**.

## Блокировка пользователя {#blocking-user}

:::warning

Пользователь не сможет входить в систему (веб-панель или мобильное приложение) и будет исключён из смен. Данные пользователя сохранятся, и вы сможете его разблокировать.

:::

1. Перейдите на вкладку **Управление → Пользователи**.
2. В строке пользователя нажмите значок ![Меню пользователя](/img/menu-dots-horizontal.svg) и выберите **Заблокировать**.
3. Подтвердите блокировку.

Чтобы разблокировать пользователя, нажмите значок ![Меню пользователя](/img/menu-dots-horizontal.svg), выберите **Разблокировать** и подтвердите действие.

## Удаление пользователя {#removing-user}

:::warning

Пользователь будет исключён из смен. Восстановить пользователя после удаления нельзя.

:::

1. Перейдите на вкладку **Управление → Пользователи**.
2. В строке пользователя нажмите значок ![Меню пользователя](/img/menu-dots-horizontal.svg) и выберите **Удалить**.
3. Подтвердите удаление.
