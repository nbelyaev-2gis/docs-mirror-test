# Шаблоны

**Шаблоны** — это наборы атрибутов, которые нужно указать при создании [агентов](/geoflow/administrator/managing/agents) или [задач](/geoflow/administrator/tasks). Шаблоны позволяют:

* создавать типовых агентов (например, курьеров) и типовые задачи (например, доставка);
* фильтровать агентов на дашборде [мониторинга](/geoflow/administrator/monitoring) по типу;
* использовать данные при планировании смен.

Шаблон необходимо создать перед [созданием агента](/geoflow/administrator/managing/agents#creating-agent) или [созданием задачи](/geoflow/administrator/tasks#importing-tasks).

В роли **Главный администратор** и **Администратор компании** вы можете работать с шаблонами на вкладке **Управление → Шаблоны**:

* [создавать шаблоны агентов](#creating-agent-template),
* [создавать шаблоны задач](#creating-task-template),
* [редактировать шаблоны](#editing-template),
* [дублировать шаблоны](#duplicating-template).

В роли **Диспетчер** и **Руководитель группы агентов** вы можете только просматривать список шаблонов и информацию о них.

## Создание шаблона агента {#creating-agent-template}

![Создание шаблона агента](/img/geoflow/creating-agent-template.png)

1. Перейдите на вкладку **Управление → Шаблоны**.
2. Нажмите **Добавить шаблон**.
3. Выберите в выпадающем списке **Агент**.
4. Укажите параметры шаблона:

   * **Предзаполнить из набора**: выберите тип агентов. Параметры ниже заполнятся автоматически для выбранного типа, при необходимости вы можете отредактировать их. Доступные типы агентов:

      * Оператор: сотрудник, выполняющий задачи.
      * Курьер: сотрудник, доставляющий документы или посылки.
      * Спецтехника: транспортные средства или машины, выполняющие работы. Например, курьерский автомобиль, бензовоз, экскаватор и т. д.

      Чтобы создать собственный шаблон, оставьте это поле пустым.

   * **Название** (обязательный параметр): общее название для типа агентов.
   * **Вид на карте**: отображение иконки агентов на карте.

      * **Иконка**: иконка агента.
      * **Цвет**: цвет иконки.

   * **Атрибуты** (обязательный параметр): список полей для создания агентов этого типа.

      * **Заголовок поля**: название атрибута.
      * **Тип данных**: тип данных атрибута.
      * **Обязательное поле**: чтобы сделать поле обязательным для заполнения при создании агента, отметьте чекбокс.
      * **Разрешить множественный выбор**: чтобы разрешить выбирать несколько значений из справочника, отметьте чекбокс. Доступно только для атрибутов с типом данных `Справочник`.

      Для каждого шаблона по умолчанию добавляется обязательный атрибут для идентификации агента: для предзаполенных шаблонов — `Идентификатор`, для собственных — `Наименование`. В качестве значения атрибута можно будет указать любой уникальный набор символов (например, табельный номер или ФИО).

      Вы можете переименовать этот атрибут. Чтобы добавить дополнительный атрибут, нажмите **Добавить атрибут**. Чтобы удалить атрибут, нажмите значок ![Удалить](/img/pro/trash-icon.svg) рядом с ним. Чтобы изменить порядок атрибутов, зажмите значок ![Перемещение](/img/pro/drag-n-drop-icon.svg) у атрибута и перетащите его.

5. Нажмите **Сохранить**.

## Создание шаблона задачи {#creating-task-template}

![Создание шаблона задачи](/img/geoflow/creating-task-template.png)

1. Перейдите на вкладку **Управление → Шаблоны**.
2. Нажмите **Добавить шаблон**.
3. Выберите в выпадающем списке **Задача**.
4. Укажите параметры шаблона:

   * **Название** (обязательный параметр): общее название для типа задач.
   * **Атрибуты** (обязательный параметр): список полей для создания задач этого типа.

      * **Заголовок поля**: название атрибута.
      * **Тип данных**: тип данных атрибута.
      * **Обязательное поле**: чтобы сделать поле обязательным для заполнения при создании задачи, отметьте чекбокс.
      * **Разрешить множественный выбор**: чтобы разрешить выбирать несколько значений из справочника, отметьте чекбокс. Доступно только для атрибутов с типом данных `Справочник`.
      * **Ограничивать время на задачу**: чтобы ограничить время на выполнение задачи, отметьте чекбокс. При создании задачи из этого шаблона добавятся поля для указания времени на выполнение задачи.

      Для каждого шаблона по умолчанию добавляются два обязательных атрибута:
      
      * `Название` — идентификатор задачи. В качестве значения атрибута можно будет указать любой уникальный набор символов (например, номер задачи или название).
      * `Статус` — статус для отображения текущего состояния задачи.

      Вы можете переименовать эти атрибуты. Чтобы добавить дополнительный атрибут, нажмите **Добавить атрибут**. Чтобы удалить атрибут, нажмите значок ![Удалить](/img/pro/trash-icon.svg) рядом с ним. Чтобы изменить порядок атрибутов, зажмите значок ![Перемещение](/img/pro/drag-n-drop-icon.svg) у атрибута и перетащите его.

5. Нажмите **Сохранить**.

## Редактирование шаблона {#editing-template}

:::warning

Редактирование шаблона (например, его атрибутов) изменяет настройки всех агентов или задач, созданных по этому шаблону.

:::

Вы можете редактировать шаблон:

* переименовать шаблон;
* изменить список атрибутов, их названия, тип и обязательность для заполнения;
* дополнительно для шаблонов агентов: выбрать готовый тип агентов и изменить вид иконки агента на карте.

<br />

Чтобы отредактировать шаблон:

1. Перейдите на вкладку **Управление → Шаблоны**.
2. В списке шаблонов нажмите значок ![Меню шаблона](/img/menu-dots-horizontal.svg) у шаблона и выберите **Изменить**.
3. Внесите изменения в шаблон.
4. Нажмите **Сохранить**.

## Дублирование шаблона {#duplicating-template}

1. Перейдите на вкладку **Управление → Шаблоны**.
2. В списке шаблонов нажмите значок ![Меню шаблона](/img/menu-dots-horizontal.svg) у шаблона и выберите **Дублировать**.
3. (Опционально) Укажите новые параметры шаблона.
4. Нажмите **Сохранить**.
