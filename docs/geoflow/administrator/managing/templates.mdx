# Шаблоны агентов

**Шаблоны агентов** — это шаблоны с атрибутами, которые нужно указать при создании [агентов](/geoflow/administrator/managing/agents). Шаблоны позволяют создавать типовых агентов (например, курьеров), фильтровать агентов на дашборде [мониторинга](/geoflow/administrator/monitoring) по типу, а также использовать эти данные при планировании смен. Шаблон необходимо создать перед [созданием агента](/geoflow/administrator/managing/agents#creating-agent).

В роли **Главный администратор** и **Администратор компании** вы можете работать с шаблонами агентов на вкладке **Управление → Шаблоны агентов**:

* [создавать шаблоны агентов](#creating-template),
* [редактировать шаблоны](#editing-template),
* [дублировать шаблоны](#duplicating-template).

Пользователи с ролями **Диспетчер** и **Руководитель группы агентов** могут только просматривать список шаблонов и информацию о них.

![Работа с шаблонами агентов](/img/geoflow/managing-templates.png)

## Создание шаблона агентов {#creating-template}

![Создание шаблона агентов](/img/geoflow/creating-template.png)

1. Перейдите на вкладку **Управление → Шаблоны агентов**.
2. Нажмите **Добавить шаблон**.
3. Укажите параметры шаблона:

   * **Предзаполнить из набора**: выберите тип агентов. Параметры ниже заполнятся автоматически для выбранного типа, при необходимости вы можете отредактировать их. Доступные типы агентов:

      * Оператор: сотрудник, выполняющий задачи.
      * Курьер: сотрудник, доставляющий документы или посылки.
      * Спецтехника: транспортные средства или машины, выполняющие работы. Например, курьерский автомобиль, бензовоз, экскаватор и т. д.

      Чтобы создать собственный набор, оставьте это поле пустым.

   * **Название** (обязательный параметр): общее название для типа агентов.
   * **Вид на карте**: отображение иконки агентов на карте.

      * **Иконка**: иконка агента.
      * **Цвет**: цвет иконки.

   * **Атрибуты** (обязательный параметр): список полей для создания агентов этого типа.

      * **Заголовок поля**: название атрибута.
      * **Тип данных**: тип данных атрибута.
      * **Обязательное поле**: чтобы сделать поле обязательным для заполнения при создании агента, отметьте чекбокс.

      Для каждого шаблона необходимо добавить один обязательный атрибут — идентификатор объекта. Это может быть любой уникальный набор символов для идентификации агента (например, табельный номер или ФИО).

      Чтобы добавить атрибут, нажмите **Добавить атрибут**. Чтобы удалить атрибут, нажмите значок ![Удалить](/img/pro/trash-icon.svg) рядом с ним. Чтобы изменить порядок атрибутов, зажмите значок ![Перемещение](/img/pro/drag-n-drop-icon.svg) у атрибута и перетащите его.

4. Нажмите **Сохранить**.

## Редактирование шаблона {#editing-template}

:::warning

Редактирование шаблона (например, его атрибутов) изменяет настройки всех агентов, созданных по этому шаблону.

:::

Вы можете редактировать шаблоны:

* выбрать готовый тип агентов;
* переименовать шаблон;
* изменить вид иконки агента на карте;
* изменить список атрибутов, их названия, тип и обязательность для заполнения.

<br />

Чтобы отредактировать шаблон:

1. Перейдите на вкладку **Управление → Шаблоны агентов**.
2. В списке шаблонов нажмите значок ![Меню шаблона](/img/menu-dots-horizontal.svg) у шаблона и выберите **Изменить**.
3. Внесите изменения в шаблон.
4. Нажмите **Сохранить**.

## Дублирование шаблона {#duplicating-template}

1. Перейдите на вкладку **Управление → Шаблоны агентов**.
2. В списке шаблонов нажмите значок ![Меню шаблона](/img/menu-dots-horizontal.svg) у шаблона и выберите **Дублировать**.
3. (Опционально) Укажите новые параметры шаблона.
4. Нажмите **Сохранить**.
