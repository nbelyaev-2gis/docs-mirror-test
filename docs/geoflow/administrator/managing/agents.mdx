# Агенты

**Агенты** выполняют рабочие задачи, учитываются при планировании смен и отслеживаются при мониторинге работы. Агентами могут быть мобильные сотрудники (например, операторы и курьеры), спецтехника и т. д.

Перед созданием агентов [создайте шаблон агентов](/geoflow/administrator/managing/templates#creating-agent-template) или выберите тип агента из предзаполненных шаблонов. Шаблон содержит набор атрибутов, которые указываются при создании агента.

В роли **Главный администратор** и **Администратор компании** вы можете работать с агентами на вкладке **Управление → Агенты**:

* [создавать агентов](#creating-agent),
* [просматривать информацию об агентах](#viewing-agent),
* [редактировать информацию об агентах](#editing-agent),
* [дублировать агентов](#duplicating-agent),
* [удалять агентов](#deleting-agent).

В роли **Диспетчер** вы можете только просматривать список агентов и информацию о них. В роли **Руководитель группы агентов** — агентов из своей группы.

## Создание агента {#creating-agent}

![Создание агента](/img/geoflow/creating-agent-609.png)

1. Перейдите на вкладку **Управление → Агенты**.
2. Нажмите **Добавить агента**.
3. Если вы создали несколько [шаблонов агента](/geoflow/administrator/managing/templates#creating-agent-template), выберите один из них.
4. Укажите параметры агента, которые зависят от выбранного шаблона.
5. Нажмите **Добавить источник** и выберите один из типов источников локации:

   * **Приложение**: выберите пользователя, который использует мобильное приложение и будет привязан к агенту.
   * **EGTS-трекер**: выберите [устройство](/geoflow/administrator/managing/devices), которое будет привязано к агенту и с которого будут собираться данные о местоположении агента.

   Вы можете выбрать несколько источников локации для одного агента, но каждый источник может быть одновременно привязан только к одному агенту.

   Чтобы добавить источник, нажмите **Добавить источник**. Чтобы удалить источник, нажмите значок ![Удалить](/img/pro/cross-icon.svg) рядом с ним.

6. Нажмите **Сохранить**.

## Просмотр информации об агенте {#viewing-agent}

1. Перейдите на вкладку **Управление → Агенты**.
2. Откройте карточку агента. Вы можете использовать поиск по имени агента и сортировку по типу (шаблону) агента.

<br />

В карточке агента отображается:

* **Идентификатор**: уникальный идентификатор, указанный при создании. Поле может иметь другое название в зависимости от шаблона агента, например, `Наименование`.
* **Основные данные**: данные, которые были указаны при создании агента. Зависят от шаблона агента.
* **Источник локации**: источники, с помощью которых собираются данные о местоположении агента (мобильное приложение или трекер).

   * **Приложение**: имя пользователя, который использует мобильное приложение и привязан к агенту.
   * **EGTS-трекер**: название и уникальный идентификатор устройства.
   * **Сигнал**: текущий статус GPS-сигнала на источнике локации агента.
   * **Последняя активность**: время, когда агент в последний раз был активен. Для агентов, использующих мобильное приложение, это время последнего входа в мобильное приложение. Для агентов, использующих трекер, это время последнего получения данных о местоположении.
   * **Посмотреть историю перемещений**: нажмите, чтобы открыть на карте текущую геопозицию агента и историю его перемещений.

* **Дата создания**: дата и время создания агента.
* **Последнее изменение**: дата и время последнего обновления информации об агенте.

## Редактирование информации об агенте {#editing-agent}

Вы можете редактировать основные данные агентов и источники их локации.

Чтобы отредактировать информацию об агенте:

1. Перейдите на вкладку **Управление → Агенты**.
2. В списке агентов нажмите значок ![Меню агента](/img/menu-dots-horizontal.svg) у агента и выберите **Изменить**.
3. Внесите изменения в информацию об агенте.
4. Нажмите **Сохранить**.

## Дублирование агента {#duplicating-agent}

1. Перейдите на вкладку **Управление → Агенты**.
2. В списке агентов нажмите значок ![Меню агента](/img/menu-dots-horizontal.svg) у агента и выберите **Дублировать**.
3. (Опционально) Укажите новые параметры агента.
4. Нажмите **Сохранить**.

## Удаление агента {#deleting-agent}

:::warning

Агент будет исключён из смен. Восстановить агента после удаления нельзя.

:::

1. Перейдите на вкладку **Управление → Агенты**.
2. В строке агента нажмите значок ![Меню агента](/img/menu-dots-horizontal.svg) и выберите **Удалить**.
3. Подтвердите удаление.
