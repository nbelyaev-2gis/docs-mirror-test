# Группы агентов

Вы можете объединять [агентов](/geoflow/administrator/managing/agents) в группы для управления доступом: пользователи с ролью **Руководитель группы агентов** смогут управлять агентами только из своей группы.

Вы можете объединять в группы агентов одного или разных типов, например, операторов и курьеров. Агенты из одной группы могут видеть информацию и местоположение друг друга в мобильном приложении.

В роли **Главный администратор**, **Администратор компании** и **Диспетчер** вы можете работать с группами на вкладке **Управление → Группы агентов**:

* [создавать группы](#creating-group),
* [редактировать группы](#editing-group),
* [удалять группы](#deleting-group).

В роли **Руководитель группы агентов** вы можете просматривать агентов из своей группы и редактировать её.

![Работа с группами](/img/geoflow/managing-agent-groups-609.png)

## Создание группы {#creating-group}

![Создание группы агентов](/img/geoflow/creating-agent-group-609.png)

1. Перейдите на вкладку **Управление → Группы агентов**.
2. Нажмите **Добавить группу**.
3. Укажите параметры группы:

   * **Название группы** (обязательный параметр): название группы.
   * **Цвет агентов на карте**: цвет иконок агентов из группы на карте. Если агент состоит в нескольких группах, иконка агента будет разноцветной.
   * **Геозона**: геозона, в которой работают агенты из группы. По умолчанию список геозон импортируется в веб-панель администратора при интеграции сервиса 2ГИС ГеоПоток.
   * **Руководитель**: пользователь с ролью **Руководитель группы агентов**, который будет руководить группой. Он сможет просматривать агентов из группы и редактировать её.
   * **Агенты**: агенты, которые входят в группу.
   
      Чтобы добавить агента, нажмите **Добавить агентов** и выберите его. Вы можете добавлять агентов, созданных из разных шаблонов. Чтобы удалить агента из группы, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с ним и подтвердите удаление.

4. Нажмите **Сохранить**.

## Редактирование группы {#editing-group}

Вы можете редактировать группы:

* переименовать группу;
* изменить цвет иконки агентов на карте;
* изменить геозону, в которой работают агенты из группы;
* изменить руководителя группы;
* добавить или удалить агентов из группы (недоступно пользователям с ролью **Руководитель группы агентов**).

<br />

Чтобы отредактировать группу:

1. Перейдите на вкладку **Управление → Группы агентов**.
2. В списке групп нажмите значок ![Меню группы](/img/menu-dots-horizontal.svg) у группы и выберите **Изменить**.
3. Внесите изменения в группу.
4. Нажмите **Сохранить изменения**.

## Удаление группы {#deleting-group}

При удалении группы агенты из неё не удаляются из системы. Вы можете добавить этих агентов в другую группу.

1. Перейдите на вкладку **Управление → Группы агентов**.
2. В списке групп нажмите значок ![Меню группы](/img/menu-dots-horizontal.svg) у группы и выберите **Удалить**.
3. Подтвердите удаление.
