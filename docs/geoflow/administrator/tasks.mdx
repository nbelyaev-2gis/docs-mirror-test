# Задачи

Вы можете создавать **задачи** для [агентов](/geoflow/administrator/managing/agents), например: доставить груз, провести инспекцию и т. д. Задача автоматически назначается агенту в смене, если он обладает всеми навыками, которые требуются для её выполнения. Агент может посмотреть список задач в [мобильном приложении](/geoflow/operator/tasks).

Перед созданием задачи [создайте шаблон задачи](/geoflow/administrator/managing/templates#creating-task-template). Шаблон содержит набор атрибутов, которые указываются при создании задачи. Импортировать задачи без шаблона нельзя.

В роли **Главный администратор**, **Администратор компании**, **Диспетчер** и **Руководитель группы агентов** вы можете работать с задачами на вкладке **Задачи**:

* [импортировать задачи](#importing-tasks),
* [просматривать информацию о задачах](#viewing-task),
* [отменять задачи](#canceling-task).

## Импорт задач {#importing-tasks}

![Импорт задачи](/img/geoflow/import-task.png)

Если вам нужно создать несколько задач с похожими атрибутами из одного [шаблона задачи](/geoflow/administrator/managing/templates#creating-task-template), вы можете скачать шаблон, заполнить его и импортировать задачи из файла в формате CSV.

1. Перейдите на вкладку **Задачи**.
2. Нажмите **Импорт из CSV**.
3. Если вы создали несколько шаблонов задачи, выберите один из них.
4. Нажмите **Скачать шаблон**.
5. Заполните шаблон данными о задачах. В шаблоне будет указано, какие данные нужно добавить для каждой задачи, и пример заполнения.

   Один файл может содержать до 1000 задач.

6. Вернитесь на вкладку **Задачи**, перетащите файл в область загрузки или нажмите **Выберите файл на компьютере**.

   Ограничение на размер файла — 5 МБ.

7. Если в файле есть ошибки, появится уведомление о строках с ошибками. Чтобы загрузить задачи, исправьте ошибки в файле и загрузите файл заново.

Задачи отобразятся в списке задач после обработки файла.

Чтобы отредактировать задачу, измените её в файле и загрузите файл заново.

## Просмотр информации о задаче {#viewing-task}

1. Перейдите на вкладку **Задачи**.
2. Вы можете использовать поиск по имени задачи, сортировку по статусу задачи (`Активные` и `Завершённые`) и типу (шаблону) задачи.

<br />

В строке задачи отображается:

* **Название**: уникальное название задачи, указанное при создании. Поле может иметь другое название в зависимости от шаблона задачи, например, `Наименование`.
* **Статус**: статус задачи.
* **Основные данные**: данные, которые были указаны при создании задачи. Зависят от шаблона задачи.
* **Обновлено**: дата и время последнего обновления информации о задаче.

## Отмена задачи {#canceling-task}

1. Перейдите на вкладку **Задачи**.
2. В строке задачи нажмите значок ![Меню задачи](/img/menu-dots-horizontal.svg) и выберите **Отменить задачу**.
3. Подтвердите отмену.
