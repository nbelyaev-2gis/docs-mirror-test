# Обзор

[2ГИС ГеоПоток](https://dev.2gis.ru/geoflow) — это решение для мониторинга работ агентов (например, мобильных сотрудников или транспорта). Решение состоит из следующих инструментов:

* Веб-панель для администраторов, которая позволяет отслеживать агентов на карте в режиме реального времени, просматривать их историю перемещений и планировать смены.
* Мобильное приложение для сотрудников, которое помогает ориентироваться на карте и просматривать список задач. Вместо мобильного приложения сотрудники могут использовать трекеры, но без возможности просмотра карты и других интерактивных опций.

## Основные особенности {#main-features}

* **Мониторинг в реальном времени**

    Позволяет просматривать актуальное, историческое местоположение и активность агентов на карте и быстро реагировать на инциденты.

* **Планирование и аналитика**

    Фиксирует отклонения в запланированных сменах и помогает выявить причины простоев и снижения эффективности.

* **Работа офлайн и в условиях слабого сигнала GPS**

    Собирает и обрабатывает GPS-данные о местоположении агентов, а также устраняет шумы для получения более точных траекторий перемещения. Карта 2ГИС в мобильном приложении для сотрудников также доступна офлайн.

## Технические требования {#technical-requirements}

Для эффективной работы системы мониторинга устройства, на которых используются инструменты 2ГИС ГеоПоток, должны удовлетворять следующим требованиям:

* **Веб-панель администратора**. Рекомендуемые требования к устройству для работы с веб-панелью:

   * Процессор: минимум 4 ядра с частотой 2 ГГц.
   * Оперативная память: минимум 8 ГБ.
   * Операционная система: Windows 11, macOS 13 Ventura или Linux последней версии.
   * Браузер: Google Chrome, Mozilla Firefox, Microsoft Edge или Safari.
   * Разрешение экрана: FullHD (1920×1080 пикселей).
   * Стабильное интернет-соединение.

* **Мобильное приложение сотрудника**. Для установки мобильного приложения требуется мобильное устройство на базе операционной системы Android версии Android 10 (API level 29) и выше.
* **Трекер сотрудника**. Трекер должен содержать ГЛОНАСС/GPS-приёмник с протоколом передачи данных EGTS.
