# Задачи

На вкладке **Задачи** мобильного приложения отображается список задач для агентов на смене.

В карточках задач отображаются данные, которые указал администратор при создании задачи (например, время выполнения, необходимые навыки и т. д.). Подробнее см. в [Руководстве администратора](/geoflow/administrator/tasks).

<img src="/img/geoflow/operator-tasks-609.png" alt="Вкладка Задачи" width="300"/>

<br />
<br />

Чтобы начать работать с задачей:

1. Перейдите на вкладку **Задачи** и откройте текущую смену.
2. Откройте вкладку **Активные**.
3. Выберите задачу и нажмите **Взять в работу**.

   <img src="/img/geoflow/operator-tasks-start.png" alt="Взять задачу в работу" width="300"/>

4. Чтобы завершить работу над задачей, откройте задачу и нажмите **Завершить задачу**.
5. Выберите статус результата задачи: **Выполнена** или **Не выполнена**.

   <img src="/img/geoflow/operator-tasks-finish-609.png" alt="Завершить задачу" width="300"/>

Завершённые задачи находятся на вкладке **Завершённые**.
