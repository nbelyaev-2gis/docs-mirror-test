# Профиль

<img src="/img/geoflow/operator-profile-609.png" alt="Вкладка Профиль" width="300"/>

На вкладке **Профиль** мобильного приложения доступны следующие функции:

* **Офлайн-режим**: загрузка офлайн-карт для работы в автономном режиме.

   <img src="/img/geoflow/operator-offline-609.png" alt="Загрузка офлайн-карт" width="300"/>

* **Выгрузить логи**: создание архива с логами в формате `.zip` для отчёта о проблеме. Вы можете скачать или отправить этот файл.
* **Выгрузить трек**: создание файла с треками. Вы можете скачать или отправить этот файл.
