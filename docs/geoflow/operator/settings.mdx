# Настройки приложения

На вкладке **Профиль** доступны следующие функции:

<img src="/img/geoflow/operator-profile.png" alt="Вкладка Профиль" width="300"/>

* **Офлайн-режим**: загрузка офлайн-карт для работы в автономном режиме.

   <img src="/img/geoflow/operator-offline.png" alt="Загрузка офлайн-карт" width="300"/>

* **Выгрузить логи**: создание архива с логами в формате `.zip` для отчёта о проблеме. Вы можете скачать или отправить этот файл.
