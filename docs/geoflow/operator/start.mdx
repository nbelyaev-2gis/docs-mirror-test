# Начало работы

Мобильное приложение 2ГИС ГеоПоток позволяет сотрудникам просматривать карту и информацию об объектах инфраструктуры.

Мобильное приложение записывает данные о перемещениях агента (сотрудника или транспорта) в течение смены. Эти данные позволяют администраторам и диспетчерам отслеживать перемещения агентов в реальном времени и анализировать их историю в веб-панели администратора: см. [Руководство администратора](/geoflow/administrator/overview).

Чтобы начать работу с мобильным приложением:

1. Авторизуйтесь в приложении одним из способов:

   * Получите приглашение на электронную почту от администратора. После прохождения регистрации авторизуйтесь в приложении по почте и паролю.
   * Получите от администратора логин и временный пароль. При повторном входе вы сможете изменить пароль на постоянный. При необходимости администратор сможет сбросить постоянный пароль в дальнейшем.

      <img src="/img/geoflow/operator-authorization.png" alt="Авторизация в приложении" width="300"/>

2. При запросе разрешите приложению доступ к функциям, необходимым для работы:

   * получение данных о точном местоположении устройства (в том числе в фоновом режиме);
   * работа в фоновом режиме (для записи данных о перемещениях в течение смены);
   * поиск устройств поблизости;
   * отправка уведомлений.

3. Для стабильной работы приложения и передачи местоположения отключите VPN.

4. (Рекомендуется) Загрузите офлайн-карту территории, на которой планируется работа, чтобы не зависеть от качества связи:

   1. Перейдите на вкладку **Профиль** и нажмите **Офлайн-режим**.
   2. Выберите и загрузите территории.

Интерфейс мобильного приложения содержит две вкладки:

<img src="/img/geoflow/operator-map.png" alt="Вкладка Карта" width="300"/>&nbsp;&nbsp;<img src="/img/geoflow/operator-profile.png" alt="Вкладка Профиль" width="300"/>

* **Карта** для просмотра [карты](/geoflow/operator/map).
* **Профиль** для просмотра личных данных и работы с [настройками](/geoflow/operator/settings).
