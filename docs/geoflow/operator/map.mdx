# Работа с картой

На вкладке **Карта** отображается карта с агентами: мобильными сотрудниками, транспортом и т. д.

<img src="/img/geoflow/operator-map.png" alt="Вкладка Карта" width="300"/>

На карте отображаются все агенты, добавленные в текущую смену:

* `Онлайн`: агенты с доступной информацией о текущем местоположении. Отображаются на карте в виде цветной иконки.
* `Офлайн`: агенты с недоступной информацией о текущем местоположении (например, трекер или мобильное приложение отключены или не передают информацию о местоположении). Отображаются на карте в виде серой иконки.
