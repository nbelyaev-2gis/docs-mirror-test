# Начало работы

Следуйте инструкциям ниже в зависимости от вашей роли в системе.

## Для администраторов {#for-administrators}

Чтобы начать работу с 2ГИС ГеоПоток:

1. Обратитесь в вашу организацию, чтобы получить доступ к веб-панели администратора 2ГИС ГеоПоток.
2. Перейдите к [Руководству администратора](/geoflow/administrator/overview).

## Для сотрудников {#for-operators}

Чтобы начать работу с 2ГИС ГеоПоток:

1. Обратитесь в вашу организацию, чтобы получить доступ к мобильному приложению 2ГИС ГеоПоток.
2. Установите приложение на вашем мобильном устройстве.
3. Перейдите к [Руководству сотрудника](/geoflow/operator/start).
