# Тарифы

Ниже указана стоимость покупки и докупки [сервисов](/platform-manager/subscription/services) для количества запросов или ответов (в зависимости от сервиса) за платёжный месяц. Цены указаны в российских рублях (RUB).

Докупка сервисов позволяет в рамках текущей подписки приобрести доступ к сервисам, которые не были подключены при оформлении подписки, или увеличить лимиты для подключённых сервисов. Подробнее см. в разделе [Покупка дополнительных лимитов и сервисов](/platform-manager/subscription/purchase#upgrading).
