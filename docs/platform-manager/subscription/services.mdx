# Доступные сервисы в подписке

В Менеджере Платформы вы можете оформить подписку для доступа к сервисам ниже. Информацию о ценах на покупку и докупку сервисов см. в инструкции [Тарифы](/platform-manager/subscription/pricing).

## Карты {#maps}

| <div style={{"width":"170px"}}>API</div> | Описание |
| ---------------------------------------- | -------- |
| [Map Tiles API](/maps/others/maptiles/overview) | Предоставляет доступ к векторным тайлам и JavaScript-библиотеке [MapGL JS API](/mapgl/overview/features), которая позволяет интегрировать на сайт или в веб-приложение трёхмерную интерактивную карту <Brand /> на WebGL-движке. |
| [Raster Tiles API](/maps/others/rastertiles/overview) | Сервис со встроенной библиотекой [RasterJS API](/maps/others/rasterjs/overview), который позволяет добавить двумерную карту на сайт или в веб-приложение. |
| [Static API](/maps/others/static/overview) | Позволяет получить статичное изображение карты без использования JavaScript-библиотек. |

## Поиск {#search}

| <div style={{"width":"170px"}}>API</div> | Описание |
| ---------------------------------------- | -------- |
| [Places API](/api/search/places/overview) | Выполняет поиск организаций, зданий и мест по различным критериям.<br />Получение некоторой информации об объектах доступно только за дополнительную плату (см. детали ниже). |
| [Geocoder API](/api/search/geocoder/overview) | Определяет координаты объекта на карте по его адресу (прямой геокодинг) и наоборот (обратный геокодинг). Также предоставляет сведения об объекте.<br />Получение некоторой информации об объектах доступно только за дополнительную плату (см. детали ниже). |
| [Suggest API](/api/search/suggest/overview) | Предлагает при вводе текста в поле поиска подсказки в виде названий и категорий объектов, подходящие под заданные критерии.<br />Получение некоторой информации об объектах доступно только за дополнительную плату (см. детали ниже). |
| [Categories API](/api/search/categories/overview) | Предоставляет информацию о категориях: группах, объединяющих компании по сферам деятельности. Используется совместно с Places API. |
| [Regions API](/api/search/regions/overview) | Позволяет выбрать территорию (города и ближайшие населённые пункты). Полученный идентификатор территории затем можно использовать в других API, чтобы ограничить область, в границах которой будет выполняться поиск организаций, зданий и мест. |
| [Markers API](/api/search/markers/overview) | Выполняет поиск организаций, зданий и мест для отображения маркеров на карте. |

Часть информации об объектах доступна только по отдельному запросу и за дополнительную плату. Чтобы приобрести доступ к методам и полям ниже, свяжитесь с [отделом продаж <Brand />](https://dev.2gis.ru/api#form):

* Методы Places API:

   * [/3.0/items/bysite](/api/search/places/reference/3.0/items/bysite) — поиск компаний по сайту;
   * [/3.0/items/byphone](/api/search/places/reference/3.0/items/byphone) — поиск компаний по номеру телефона;
   * [/3.0/items/byitin](/api/search/places/reference/3.0/items/byitin) — поиск компаний по ИНН;
   * [/3.0/items/bytradelicense](/api/search/places/reference/3.0/items/bytradelicense) — поиск компаний по торговой лицензии;
   * [/3.0/items/byfias](/api/search/places/reference/3.0/items/byfias) — поиск компаний по ФИАС-коду.

* Поля в Places API, Geocoder API и Suggest API (указываются с помощью параметра `fields`):

   * `items.contact_groups` — контакты компании;
   * `items.floors` — количество этажей;
   * `items.floor_plans` — планы этажей;
   * `items.links.database_entrances.apartments_info` — информация о квартирах в доме;
   * `items.employees_org_count` — численность сотрудников организации;
   * `items.itin` — индивидуальный номер налогоплательщика;
   * `items.trade_license` — лицензия филиала;
   * `items.fias_code` — код ФИАС улиц и административных территорий;
   * `items.address.components.fias_code` — код ФИАС зданий;
   * `items.fns_code` — код ФНС административных территорий;
   * `items.okato` — код ОКАТО улиц и административных территорий;
   * `items.address.components.okato` — код ОКАТО зданий;
   * `items.oktmo` — код ОКТМО улиц и административных территорий;
   * `items.address.components.oktmo` — код ОКТМО зданий;
   * `items.structure_info.material` — данные о материале здания;
   * `items.structure_info.apartments_count` — данные о количестве квартир;
   * `items.structure_info.porch_count` — данные о количестве подъездов;
   * `items.structure_info.floor_type` — тип перекрытий в здании;
   * `items.structure_info.gas_type` — тип газоснабжения здания;
   * `items.structure_info.year_of_construction` — год постройки здания;
   * `items.structure_info.elevators_count` — количество лифтов в здании;
   * `items.structure_info.is_in_emergency_state` — факт признания дома аварийным;
   * `items.structure_info.project_type` — серия или проект постройки здания;
   * `items.structure_info.chs_name` — название объекта культурного наследия;
   * `items.structure_info.chs_category` — категория объекта культурного наследия.

## Навигация {#navigation}

| <div style={{"width":"170px"}}>API</div> | Описание |
| ---------------------------------------- | -------- |
| [Routing API](/api/navigation/routing/overview) | Строит маршруты на автомобиле, такси, велосипеде, самокате, мотоцикле, для экстренных служб, пешеходные и на общественном транспорте. При работе с мобильным SDK используется только для построения маршрутов на общественном транспорте. |
| [Distance Matrix API](/api/navigation/distance-matrix/overview) | Рассчитывает расстояние и время в пути для множества точек. Это позволяет найти точки с необходимым временем достижения. |
| [TSP API](/api/navigation/tsp/overview) | Решает задачу коммивояжёра (TSP/VRP): строит кратчайший по времени или расстоянию маршрут обхода указанных точек. |
| [Isochrone API](/api/navigation/isochrone/overview) | Определяет область, достижимую за указанное время из заданной точки для разных типов маршрутов: автомобильного, велосипедного или пешеходного. Можно определять область доставки в зависимости от заданного времени доставки и доступных курьерам способов передвижения. |
| [Map Matching API](/api/navigation/map-matching/overview) | Строит автомобильный маршрут по набору точек, привязанных к автомобильным дорогам. Такой набор точек записывается транспортным средством. |
| [Radar API](/api/navigation/radar/overview) | Определяет приблизительное местоположение устройства по точкам доступа Wi-Fi и сотовым вышкам. |
| [Directions API](/api/navigation/directions/overview) | Строит маршруты на автомобиле, такси, велосипеде, самокате, мотоцикле, грузовике, а также пешеходные. Доступен только для работы с мобильным SDK. |
| [Truck Directions API](/api/navigation/truck-directions/overview) | Предоставляет список зон, для проезда в которые грузовому транспорту требуется пропуск. Доступен только для работы с мобильным SDK. |
