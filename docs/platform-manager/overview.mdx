# Обзор

Менеджер Платформы — личный кабинет для управления доступом к API <Brand /> для работы с картами, навигацией и поиском. Полный список API, которыми вы можете управлять через личный кабинет, см. в разделе [Сервисы](/platform-manager/subscription/services).

<CtaButton href="https://platform.2gis.ru/ru/?utm_source=docs&utm_medium=referral&utm_campaign=account" target="_blank">
   В кабинет
</CtaButton>

<br />
<br />

В Менеджере Платформы вы можете:

* Оформить [подписку](/platform-manager/subscription/purchase) на API, [создать ключи доступа](/platform-manager/subscription/keys) и [управлять ключами](/platform-manager/subscription/managing-keys).
* Создать бесплатный [демо-ключ](/platform-manager/subscription/keys#demo).
* Следить за [расходом лимитов](/platform-manager/subscription/statistics#subscription-limits) в подписке и просматривать [статистику использования](/platform-manager/subscription/statistics#view-the-statistics) ключей доступа.
* Настроить ролевую модель, чтобы [контролировать доступ сотрудников вашей компании](/platform-manager/users) к продуктам <Brand />.
* Изучить работу сервисов <Brand /> для бизнеса на практике в песочницах.

:::tip

Менеджер Платформы также доступен внутри комплекса On-Premise: см. [отличия версий On-Cloud и On-Premise](/on-premise-api-platform/api-platform/user-guide/platform-manager#on-cloud-vs-on-premise).

:::
