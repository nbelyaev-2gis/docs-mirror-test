# Релизы GIS-платформы

Чтобы установить GIS-платформу версии 2.0.0 и выше, установите [утилиту 2GIS CLI версии 3](/on-premise-gis-platform/overview/2gis-cli/releases#v3) и следуйте [инструкциям по обновлению](/on-premise-gis-platform/update).

## 2.0.0

Дата релиза: 2025-09-30

<h3 id="dependencies_v2.0.0">Зависимости</h3>

Перед обновлением GIS-платформы до версии 2.0.0 убедитесь, что установлены компоненты-зависимости нужных версий:

| <div style={{"width":"170px"}}>Компонент</div> | Минимальная версия |
| ---------------------------------------- | ------ |
| Утилита 2GIS CLI                         | 3      |
| Базовые сервисы                          | 2.0.0  |
| API-платформа                            | 2.44.0 |

Подробнее см. в разделе [Обновление GIS-платформы](/on-premise-gis-platform/update).
