# Обзор решения 

GIS-платформа для сервера — веб-приложение для работы с ГИС, которое может работать в вашей инфраструктуре.

import PartialExample from '../../on-premise-overview/_solution-overview.mdx';

<PartialExample />

GIS-платформа позволяет:

* выполнять задачи, связанные с работой над растровыми и векторными геопространственными слоями: управление и обмен слоями, их импорт и экспорт, создание и редактирование слоев, настройка визуализации;
* анализировать геоданные, используя слои и встроенные инструменты для геоаналитики.
