# Хилшейд

Хилшейд (также hillshade, отмывка рельефа, теневой рельеф, shaded relief) – это метод представления рельефа на картах, при котором кажется, что на элементы рельефа сбоку падает свет. Карты с хилшейдом подчеркивают красоту рельефа и помогают показать формы ландшафта в деталях.

![Хилшейд](/img/mapgl/hillshade/introduction.png)

Хилшейд представляет собой иллюзию трехмерности и не изменяет геометрическое расположение объектов на карте (для этого нужен [3D-рельеф](/mapgl/map-style/immersive/terrain)). Это наиболее легкий и производительный способ показать рельеф на карте.

По умолчанию хилшейд отключен. Для включения настройте его в [Редакторе стилей](https://styles.2gis.com/).

## Настройка в редакторе стилей {#setup-in-style-editor}

1. Создайте новый стилевой слой с типом `полигон` и выберите для него данные **Хилшейд светлый** или **Хилшейд темный**:

   ![Создать слой с хилшейдом](/img/mapgl/hillshade/source.png)

2. Во вкладке **Стиль** настройте внешний вид полигонов. Рекомендуемые настройки представлены ниже:

   <details>
   <summary>Светлый слой</summary>

   * Цвет заливки:
      * Точка 1: цвет `#FFFFFF1A`, масштаб `10`.
      * Точка 2: цвет `#FFFFFF00`, масштаб `14,5`.
   * Цвет обводки:
      * Точка 1: цвет `#FFFFFF1A`, масштаб `10`.
      * Точка 2: цвет `#FFFFFF00`, масштаб `14,5`.
   * Ширина обводки: `0 px`.

   ![Пример настройки стиля для светлого слоя](/img/mapgl/hillshade/light_settings.png)

   </details>

   <details>
   <summary>Тёмный слой</summary>

   * Цвет заливки:
      * Точка 1: цвет `#00000008`, масштаб `10`.
      * Точка 2: цвет `#00000000`, масштаб `14,5`.
   * Цвет обводки:
      * Точка 1: цвет `#00000008`, масштаб `10`.
      * Точка 2: цвет `#00000000`, масштаб `14,5`.
   * Ширина обводки: `0 px`.

   ![Пример настройки стиля для темного слоя](/img/mapgl/hillshade/dark_settings.png)

   </details>

### Приоритет слоев с хилшейдом

Располагать эти слои следует после всех природных площадных объектов: воды, суши, растительности.

![Пример расположения слоя с хилшейдом](/img/mapgl/hillshade/ordering.png)

## Динамическое включение и отключение

Хилшейд, как и любые другие стилевые слои, можно включать/отключать при помощи [глобальных переменных стиля](/mapgl/map-style/global-style-variables).

1. Добавьте к фильтрам стилевых слоев хилшейда зависимость от глобальной переменной. Например, для переменной `hillshade` это будет выглядеть следующим образом:

   ![Глобальная переменная hillshade](/img/mapgl/hillshade/globalvar.png)

2. Включите или отключите отображение слоев хилшейда при помощи метода [map.patchStyleState()](/mapgl/reference/class#map/patch-style-state):

   ```ts
   // Включить слои хилшейда
   map.patchStyleState({ hillshade: true });
   ```

   <br  />

   ```ts
   // Отключить слои хилшейда
   map.patchStyleState({ hillshade: false });
   ```

## Добавление к существующему стилю

Слои хилшейда можно добавить к существующему стилю при помощи метода [map.addLayer()](/mapgl/reference/class#map/add-layer). Пример:

```ts
map.on('styleload', function () {
    map.addLayer({
        type: 'polygon',
        id: 'hillshade-dark',
        filter: ['match', ['get', 'db_sublayer'], ['Hillshade_dark'], true, false],
        style: {
            color: ['interpolate', ['linear'], ['zoom'], 10, '#00000008', 14.5, '#00000000'],
            strokeWidth: 0,
        },
    });
    map.addLayer({
        type: 'polygon',
        id: 'hillshade-light',
        filter: ['match', ['get', 'db_sublayer'], ['Hillshade_light'], true, false],
        style: {
            color: ['interpolate', ['linear'], ['zoom'], 10, '#FFFFFF1A', 14.5, '#FFFFFF00'],
            strokeWidth: 0,
        },
    });
});
```

:::warning

Событие [StyleLoadEvent](/mapgl/reference/interface#style-load-event) должно вызываться перед методами [map.addLayer()](/mapgl/reference/class#map/add-layer) и [map.removeLayer()](/mapgl/reference/class#map/remove-layer). Карта не может добавлять и удалять стилевые слои, пока не загрузится ее основной стиль.

:::
