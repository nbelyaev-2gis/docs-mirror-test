# 3D-рельеф

3D-рельеф — это полноценная трёхмерная модель местности (в отличие от [хилшейда](/mapgl/map-style/immersive/hillshade)). В этом режиме поверхность Земли имеет наиболее реалистичную геометрию и внешний вид.

![3D-рельеф](/img/mapgl/relief/introduction.png)

## Включение и отключение

По умолчанию 3D-рельеф отключен. Чтобы включить его, измените [глобальную переменную стиля](/mapgl/map-style/global-style-variables) `terrainEnabled` при помощи метода [map.patchStyleState()](/mapgl/reference/class#map/patch-style-state):

```ts
// Включить рельеф
map.patchStyleState({ terrainEnabled: true });
```

<br  />

```ts
// Отключить рельеф
map.patchStyleState({ terrainEnabled: false });
```

## Настройки 3D-рельефа {#3d-terrain-settings}

Рельеф настраивается при помощи [Редактора стилей](https://styles.2gis.com/). Включите режим 3D-рельефа соответствующим переключателем в разделе **Настройки**.

![styleeditor](/img/mapgl/relief/styleeditor.png)

Параметры отображения рельефа настраиваются в левой панели со слоями, во вкладке **Другое**. Ниже приведены описания и примеры значений основных параметров.

### Множитель высоты

Регулирует степень выраженности рельефа. Правильно подобранный множитель высоты позволяет лучше отобразить ландшафт, как в примере ниже:

![Сравнение разных множителей высоты](/img/mapgl/relief/vscale_comparison.png)

### Направление освещения склонов

Направление, с которого рельеф освещается источником света. Рекомендуется использовать традиционное в картографии значение в 315°. Человеческий глаз привык воспринимать карты именно под этим углом освещения, а изменение этого значения может исказить восприятие рельефа (например, горы могут показаться впадинами). Сравните разные направления освещения в примере ниже:

![Сравнение разных направлений освещения](/img/mapgl/relief/direction_comparison.png)

### Степень затемнения или осветления склонов

Регулирует контрастность затенения и освещения склонов.

![Сравнение разных степеней затемнения](/img/mapgl/relief/intensity_comparison.png)

### Палитра рельефа

Палитра рельефа позволяет менять характер затенения/осветления рельефа, что может быть полезно для дополнительной стилизации его внешнего вида.

![palette](/img/mapgl/relief/palette.png)

## Использование вместе с хилшейдом

Вы можете использовать 3D-рельеф в сочетании с векторыми слоями [хилшейда](/mapgl/map-style/immersive/hillshade):

1. В [настройках рельефа](/mapgl/map-style/immersive/terrain#3d-terrain-settings) отключите встроенный механизм затенения склонов, выставив параметр **Степень затемнения или осветления склонов** в `0`.
2. [Добавьте](/mapgl/map-style/immersive/hillshade#setup-in-style-editor) на карту слои `Хилшейд светлый` и `Хилшейд темный`.

Векторные слои хилшейда дают более четкие очертания гор, однако они не могут отображаться при параметре `zoom` больше 14. Рекомендуемое решение: использовать векторные слои хилшейда для параметра `zoom` меньше 14, а для остальных значений `zoom` использовать встроенный в рельеф хилшейд.

Чтобы обеспечить объемное отображение рельефа на всех уровнях приближения, можно настроить плавное подключение встроенного хилшейда при значении `zoom` от 13 и выше с помощью `interpolate`-выражения:

![Задание интенсивности при помощи interpolate-выражения](/img/mapgl/relief/interpolate.png)
