# Иммерсивные возможности

MapGL JS API позволяет создать в картах эффект погружения в окружающую среду с помощью трёхмерных объектов. Вы можете добавить иммерсивности в свою карту следующими способами:

* Добавить на карту множество [3D-моделей](/mapgl/map-style/immersive/models) (например, городских достопримечательностей).
* Добавить детализированную модель здания с этажными планами и дополнительной информацией с помощью [плагина mapgl-gltf](/mapgl/objects/gltf2-plugin).
* Представить рельеф на карте с помощью [хилшейда](/mapgl/map-style/immersive/hillshade) или [3D-рельефа](/mapgl/map-style/immersive/terrain).
* Украсить карту [трёхмерными деревьями](/mapgl/map-style/immersive/trees).
* Настроить уникальное [освещение](/mapgl/map-style/immersive/lighting) карты, а также [цвет неба и тумана](/mapgl/map-style/immersive/sky).
* Настроить поведение 3D-объектов при [пересечении с камерой](/mapgl/map-style/immersive/camerafade).
* Настроить превращение карты в [глобус](/mapgl/map-style/immersive/globe) при достаточном отдалении.
* Использовать [иммерсивную дорожную инфраструктуру](/mapgl/map-style/immersive/road-infrastructure).
* Настроить [графические режимы](/mapgl/map/configuration/graphics-presets) и включить или отключить для них отображение иммерсивных дорог, теней, тумана, неба, глобуса и 3D-моделей.
