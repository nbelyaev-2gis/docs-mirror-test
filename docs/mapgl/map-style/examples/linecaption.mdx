# Подпись линии

С помощью [источника данных GeoJSON](/mapgl/geodata/geojson) и [Редактора стилей](https://styles.2gis.com/) вы можете добавить собственную подпись к линии на карте.

Чтобы добавить подпись, подключите источник данных и создайте стилевой слой c типом **Подпись линии**:

1. Подключите источник данных с помощью класса [GeoJsonSource](/mapgl/reference/class#geo-json-source). В качестве `geometry` в данных укажите объект типа `LineString` с координатами линии:

   ```js
   const data = {
       type: 'Feature',
       properties: {
           foo: 'bar',
           text: 'Текст над линией',
       },
       geometry: {
           type: 'LineString',
           coordinates: [
               [37.598234, 55.758432],
               [37.631876, 55.778654]
           ],
       },
   };

   const source = new mapgl.GeoJsonSource(map, {
       data,
   });
   ```

   Укажите параметры:

   * `foo` — уникальное свойство линии. Используйте для связи источника данных GeoJSON со стилевым слоем в Редакторе стилей, в котором будут задаваться настройки подписи.
   * `text` — текст подписи над линией.
   * `coordinates` — координаты линии.

2. Откройте [Редактор стилей](https://styles.2gis.com/).

3. В блоке **Мои стили** откройте стиль, который используется для карты.

4. В разделе **Слои** нажмите значок ![Добавить слой](/img/styles/add-layer-icon.svg).

5. Выберите тип слоя **Подпись линии**.

6. Укажите нужные настройки подписи линии: шрифт, цвет текста и другие.

7. Свяжите созданный источник данных GeoJSON со слоем. Для этого откройте блок **Данные**, нажмите **JSON — добавить вручную** и укажите уникальное свойство линии `foo`, которое вы указали в данных для линии:

    {/* prettier-ignore */}

   ```json
   [
       "==",
       [
           "get",
           "foo"
       ],
       "bar"
   ]
   ```

    <br/>
    {/* prettier-ignore */}
    <img src="/img/mapgl/linecaption/data.png" alt="Блок Данные" width="650" />

8. Свяжите атрибут `text` из источника данных GeoJSON со слоем. Для этого откройте блок **Стиль** и в поле **Текст** укажите:

   * **Тип**: Get-выражение.
   * **Атрибут, содержащий текстовую подпись**: `text`.

    {/* prettier-ignore */}

   <img src="/img/mapgl/linecaption/text.png" alt="Поле Текст" width="650"  />

9. Чтобы подписи карты не перекрывали подпись линии, добавьте её в отдельную группу лейблинга:

   1. Нажмите **Настройки** в верхнем меню и нажмите **Открыть группы лейблинга**.

   2. Создайте новую группу, например `line`. Добавьте эту группу и группу по умолчанию (`default`) в поле **Создание перекрытия групп лейблинга**:

       {/* prettier-ignore */}

      <img src="/img/mapgl/linecaption/label-groups.png" alt="Группы лейблинга" width="350"  />

   3. Перейдите в настройки слоя. Откройте блок **Стиль** и в поле **Группа лейблинга** выберите созданную группу лейблинга:

       {/* prettier-ignore */}

      <img src="/img/mapgl/linecaption/select-group.png" alt="Выбор группы лейблинга" width="650"  />

<br />

:::note[Проверка отображения подписей]

Данные, которые вы добавляете на карту через GeoJSON, не отображаются в Редакторе стилей. Чтобы проверить отображение подписи на карте, после создания и настройки слоя с линией [опубликуйте стиль](/maps/styles/connect#publishing-style-changes) и посмотрите изменения в онлайн-редакторе, например CodePen.

:::
