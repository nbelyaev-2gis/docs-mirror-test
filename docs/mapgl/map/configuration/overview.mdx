# Настройки карты

В этом разделе описаны основные инструменты, с помощью которых вы можете настроить поведение карты следующими способами:

* [Добавить элементы управления картой](/mapgl/map/configuration/controls): кнопки масштабирования, показа пробок и другие.
* [Масштабировать карту по координатам](/mapgl/map/configuration/fitbounds), чтобы сфокусировать внимание на нужных объектах.
* [Настроить поведение скролла](/mapgl/map/configuration/scroll), чтобы управлять реакцией карты на действия пользователя.
* [Получить текущее местоположение пользователя](/mapgl/map/configuration/geolocation), чтобы изменить центр карты.
* [Настроить отображение этажных планов](/mapgl/map/configuration/floors) для некоторых зданий (например, торговых центров).
* [Сделать карту зацикленной](/mapgl/map/configuration/loopedmap), чтобы бесконечно перемещать карту вдоль экватора.
* [Настроить графический режим](/mapgl/map/configuration/graphics-presets), чтобы управлять уровнем иммерсивности на карте.
