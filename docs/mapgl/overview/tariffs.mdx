# Тарифы и размещение

## Тарифы {#tariffs}

* Стоимость сервиса рассчитывается исходя из количества успешных запросов в месяц к сервису [Map Tiles API](/maps/others/maptiles/overview), который предоставляет доступ к векторным тайлам карты. Один запрос соответствует загрузке одного векторного тайла размером 256×256 пикселей.
* Чтобы определить точное количество тайлов для отображения карты:

    1. Определите размер карты для отображения в пикселях (ширина и высота).
    2. Разделите ширину и высоту карты на 256 (размер одного тайла) и округлите полученные значения до ближайшего целого числа в большую сторону.
    3. Умножьте количество тайлов по горизонтали на количество тайлов по вертикали.

    Полученное число — минимальное количество тайлов, необходимое для отображения видимой области карты. Рекомендуется добавить по одному тайлу с каждой стороны, чтобы обеспечить плавную прокрутку и масштабирование.

    Например, чтобы отобразить область карты размером 800×600 пикселей, нужно минимум 12 тайлов (4 по горизонтали и 3 по вертикали), а оптимально — 30 тайлов (6 по горизонтали и 5 по вертикали).

* Безлимитный доступ к библиотеке MapGL JS API предоставляется бесплатно.
* Актуальную стоимость и лимиты см. в разделе [Тарифы](/platform-manager/subscription/pricing).

## Использование с Map Tiles API {#distribution}

Вы можете использовать библиотеку MapGL JS API с сервисом Map Tiles API в зависимости от варианта его размещения:

* **Облако**: MapGL JS API доступна для использования с облачной версией [Map Tiles API](/maps/others/maptiles/overview).
* **On-Premise**: MapGL JS API доступна для использования с Tiles API, установленным в закрытом контуре в качестве компонента [API-платформы](/on-premise-api-platform/overview/summary) <Brand />.
