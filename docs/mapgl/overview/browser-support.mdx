# Поддержка браузеров

MapGL JS API совместима со следующим веб-браузерами:

## Desktop

* Браузеры на базе Chromium: Chrome, Microsoft Edge и другие (Windows, macOS, Linux) — две последние мажорные стабильные версии (за исключением режима IE для Microsoft Edge).
* Firefox (Windows, macOS, Linux) — две последние мажорные стабильные версии.
* Safari (macOS) — две последние мажорные стабильные версии.

## Android 7+

* Chrome 119+ — последняя версия с поддержкой Android 7+.
* Chrome WebView 119+ — последняя версия с поддержкой Android 7+.

## iOS 12+

* Safari 12+ версии.
* WebView 12+ версии.

:::note[Проверка совместимости]

Присутствие браузера в списке не гарантирует корректную работу API: проблемы могут возникать на уровне аппаратного обеспечения, операционной системы или драйверов. Чтобы убедиться в поддержке API для вашей платформы, используйте [метод `isSupported`](/mapgl/reference/method#is-supported).

:::

Чтобы использовать карты <Brand /> в неподдерживаемом браузере, рассмотрите растровую [Raster Tiles API](/maps/others/rastertiles/overview) или статичную [Static API](/maps/others/static/overview) карту или [напишите нам письмо](mailto:api@2gis.ru).
