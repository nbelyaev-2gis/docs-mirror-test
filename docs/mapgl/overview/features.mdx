# MapGL JS API

MapGL JS API — это JavaScript-библиотека, при помощи которой вы можете добавить карту <Brand /> в ваше веб-приложение или сайт.

Библиотека позволяет:

* Добавлять трехмерную карту, использующую технологию WebGL и картографические данные <Brand />, в веб-приложения и сайты.
* [Вращать, наклонять и масштабировать](https://2gis.github.io/demo/fly/) карту при помощи мыши или сенсорных жестов.
* Отображать [планы этажей](/mapgl/map/configuration/floors) торговых центров и аэропортов.
* Настраивать внешний вид карты с помощью [редактора стилей](/maps/styles/overview), включая [3D-модели](/mapgl/map-style/immersive/models), [рельеф](/mapgl/map-style/immersive/terrain) и другие [иммерсивные возможности](/mapgl/map-style/immersive/overview).
* Добавлять на карту [одиночные маркеры](/mapgl/objects/markers) и [группы маркеров](/mapgl/objects/clustering).
* Добавлять [текстовые метки](/mapgl/objects/labels) и рисовать [геометрические фигуры](/mapgl/objects/geometries).
* Строить [автомобильные](/mapgl/map/tools/directions#car-route) и [пешеходные](/mapgl/map/tools/directions#pedestrian-route) маршруты.
* Добавлять [попапы](/mapgl/objects/popups) и [подсказки](/mapgl/objects/tooltips) к объектам на карте.
* Настраивать [элементы управления](/mapgl/map/configuration/controls) картой.
* Работать с [GeoJSON](/mapgl/geodata/geojson) и [растровыми источниками WMS/WMTS](/mapgl/geodata/raster).

Картографические данные поддерживают [стандарты OGC](https://ru.wikipedia.org/wiki/Open_Geospatial_Consortium).

:::warning

Используйте только описанные в документации поля и методы MapGL JS API. Обратная совместимость для недокументированных возможностей не гарантирована.

:::
