# Растровый объект

Чтобы добавить [растр](/mapgl/reference/class#raster) на карту, нужно указать географические границы, в пределах которых он будет отображен.

```js
const raster = new mapgl.Raster(map, {
    bounds: {
        southWest: [37.594255, 55.752130],
        northEast: [37.649709, 55.787238],
    },
    image: {
        url: 'https://docs.2gis.com/img/mapgl/cloud.png',
    },
});
```
