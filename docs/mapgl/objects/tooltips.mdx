# Всплывающие подсказки

Чтобы показать всплывающую подсказку при наведении курсора на объект, создайте HTML-элемент и показывайте или скрывайте его в обработчиках событий `mouseover` и `mouseout`.

## Подсказка для маркера

Например, чтобы добавить подсказку для маркера, добавьте обработчики для событий `mouseover` и `mouseout` при помощи метода [on()](/mapgl/reference/class#marker/on):

```js
marker.on('mouseover', (e) => {
    console.log('Курсор перешёл на маркер');
});
marker.on('mouseout', (e) => {
    console.log('Курсор покинул маркер');
});
```

Основной смысл в том, чтобы показать HTML-элемент в обработчике `mouseover` и скрыть его в обработчике `mouseout`. Координаты для показа элемента можно получить через объект события (см. [MapPointerEvent](/mapgl/reference/interface#map-pointer-event)).

Перед показом элемент нужно скрыть и указать абсолютное позиционирование (`position: absolute;` или `position: fixed;`). Кроме этого, нужно указать для свойства `pointer-events` значение `none` или добавить небольшой отступ для элемента. Если этого не сделать, подсказка будет показана прямо под курсором, что приведет к событию `mouseout`, обработчик которого скроет подсказку.

Пример разметки для простой подсказки:

```html
<style>
    #tooltip {
        padding: 20px 40px;
        background: #fff;
        display: none;
        position: fixed;
        pointer-events: none;
    }
</style>
<div id="tooltip">Подсказка</div>
```

Пример кода, который покажет подсказку при наведении курсора на маркер:

```js
const tooltipEl = document.getElementById('tooltip');

marker.on('mouseover', (event) => {
    // Отступ в пикселях
    const offset = 5;

    tooltipEl.style.top = (event.point[1] + offset) + 'px';
    tooltipEl.style.left = (event.point[0] + offset) + 'px';
    tooltipEl.style.display = 'block';
});

marker.on('mouseout', (e) => {
    tooltipEl.style.display = 'none';
});
```

## Подсказка для геометрической фигуры

Подсказки можно добавлять и для других типов объектов. Например, чтобы добавить подсказку для [линии](/mapgl/objects/geometries#polyline), нужно добавить обработчики для тех же двух событий `mouseover` и `mouseout`, которые были использованы в примере выше.

Далее приведён полный пример кода. Чтобы увидеть подсказку, наведите курсор на линию.

<HtmlExampleWithSrcCode iframeStyles={{ height: '300px' }}>

    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>MapGL JS API</title>
        <meta name="description" content="Подсказка для линии" />
        <style>
            html,
            body,
            #container {
                margin: 0;
                width: 100%;
                height: 100%;
                overflow: hidden;
            }

                padding: 20px 40px;
                background: #fff;
                box-shadow: 1px 1px 2px rgba(0, 0, 0, 0.2);
                border-radius: 4px;
                display: none;
                position: fixed;
                pointer-events: none;
            }
        </style>
    </head>
    <body>
        <div id="container"></div>
        <div id="tooltip">Подсказка для линии</div>
        <script src="__MAPGL_API_URL__"></script>
        <script>
            const map = new mapgl.Map('container', {
                center: [37.615655, 55.768005],

                key: 'Your API access key',

            polyline.on('mouseout', (e) => {
