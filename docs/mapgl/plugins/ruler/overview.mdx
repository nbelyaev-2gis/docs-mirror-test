# Линейка

Плагин используется для добавления линейки на карту и измерения расстояний между точками, а также площадей и периметров фигур.

Чтобы подключить плагин, нужно добавить следующую строку после подключения основного скрипта MapGL:

```html
<script src="__MAPGL_PLUGIN_RULER_URL__"></script>
```

Или установить нужный пакет npm:

```bash
npm install @2gis/mapgl-ruler
```

Чтобы начать использовать плагин, изучите следующие ресурсы:

* [Примеры использования](/mapgl/map/tools/ruler)
* [Справочник API](/mapgl/plugins/ruler/reference/class#control)
