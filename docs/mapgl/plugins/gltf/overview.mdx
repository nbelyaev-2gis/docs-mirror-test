# Плагин для отображения glTF-моделей

Плагин `mapgl-gltf` используется для отображения glTF-моделей на карте MapGL.

Чтобы воспользоваться плагином, его нужно подключить к себе на страницу:

```html
<script src="__MAPGL_PLUGIN_GLTF_URL__"></script>
```

Также можно установить плагин с помощью npm:

```bash
npm install @2gis/mapgl-gltf@^1
```

Чтобы начать использовать плагин, изучите следующие ресурсы:

* [Примеры использования](/mapgl/objects/gltf-plugin)
* [Справочник API](/mapgl/plugins/gltf/reference/class#gltf-plugin)
