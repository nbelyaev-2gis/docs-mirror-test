# Directions

Плагин Directions позволяет построить оптимальный маршрут между несколькими точками (до десяти).

Плагин работает на основе [Routing API](/api/navigation/routing/overview). Чтобы работать с API сервиса, получите ключ доступа: см. инструкцию [Создание ключей доступа](/platform-manager/subscription/purchase).

Чтобы подключить плагин, нужно добавить следующую строку после подключения основного скрипта MapGL:

```html
<script src="__MAPGL_PLUGIN_DIRECTIONS_URL__"></script>
```

Или установить нужный пакет npm:

```bash
npm install @2gis/mapgl-directions
```

Чтобы начать использовать плагин, изучите следующие ресурсы:

* [Примеры использования](/mapgl/map/tools/directions)
* [Справочник API](/mapgl/plugins/directions/reference/class#directions)
