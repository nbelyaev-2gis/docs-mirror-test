# Релизы

## 23 марта 2026 г.

* Добавлена возможность настраивать обработку некорректных точек в запросе: см. параметр `bad_point_tolerance`.

## 1 декабря 2025 г.

* Обновлена сетка тарифов на API-сервисы. Подробнее об изменениях см. в [личном кабинете](https://platform.2gis.ru/ru/tariffs?utm_source=docs&utm_medium=referral&utm_campaign=api_mapmatching).

## 12 июля 2021 г.

* Выпущен Map Matching API — алгоритм восстановления маршрута из набора точек, записанных транспортным средством.
