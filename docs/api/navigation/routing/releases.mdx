# Релизы

## 27 мая 2026 г.

* Добавлена возможность [сохранять построенный маршрут](/api/navigation/routing/examples/special#save-route) для дальнейшего использования в мобильном SDK.

## 17 марта 2026 г.

* Добавлена возможность [строить маршруты для общественного транспорта](/api/navigation/routing/examples/types#public-transport). Если для построения маршрутов вы ранее пользовались Public Transport API, [статистика](/platform-manager/subscription/statistics) расходования лимитов и распределения запросов для Public Transport API сохранится.

## 17 декабря 2025 г.

* Добавлена возможность [получить информацию о стоимости проезда по платным дорогам](/api/navigation/routing/examples/special#toll-roads).

## 1 декабря 2025 г.

* Обновлена сетка тарифов на API-сервисы. Подробнее об изменениях см. в [личном кабинете](https://platform.2gis.ru/ru/tariffs?utm_source=docs&utm_medium=referral&utm_campaign=api_routing).

## 1 июля 2025 г.

* Добавлена возможность [строить маршруты для мотоциклов](/api/navigation/routing/examples/types#motorcycle-route).

## 25 ноября 2024 г.

* Для параметра запроса `transport`, который указывает способ передвижения, значения `car` и `pedestrian` заменены на `driving` и `walking` соответственно.

## 28 июля 2023 г.

* Первая версия Routing API с полной поддержкой всех функций [Directions API](/api/navigation/directions/overview), [Pairs Directions API](/api/navigation/pairs/overview) и [Truck Directions API](/api/navigation/truck-directions/overview).
