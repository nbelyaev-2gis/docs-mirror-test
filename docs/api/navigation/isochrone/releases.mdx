# Релизы

## 2 марта 2026 г.

* Добавлена возможность получить в ответе API статус построения каждой зоны доступности отдельно с помощью параметра `detailed_response`: см. [Статус построения зон доступности](/api/navigation/isochrone/examples#calculation-status).

## 1 декабря 2025 г.

* Обновлена сетка тарифов на API-сервисы. Подробнее об изменениях см. в [личном кабинете](https://platform.2gis.ru/ru/tariffs?utm_source=docs&utm_medium=referral&utm_campaign=api_isochrone).

## 28 апреля 2024 г.

* Добавлена возможность построения зон доступности для общественного транспорта (параметр запроса `transport: "public_transport"`). Полный список видов общественного транспорта см. в параметре [public\_transport\_types](/api/navigation/isochrone/reference/isochrone_200#:~:text=public_transport_types).

## 25 ноября 2024 г.

* Параметр запроса `mode`, который указывает тип движения, заменён на параметр `transport`.

## 30 июня 2021 г.

* Добавлена возможность указать велосипед в качестве средства передвижения (параметр запроса `mode: "bicycle"`).

## 30 марта 2021 г.

* Добавлена возможность изменить направление движения, чтобы двигаться *к* указанной точке, а не *от*. Для этого используется параметр запроса [reverse](/api/navigation/isochrone/reference/get_hull#:~:text=reverse).
* Добавлена возможность указать детализированность многоугольника для полученной области (см. параметр [simplify](/api/navigation/isochrone/reference/get_hull#:~:text=simplify)).
