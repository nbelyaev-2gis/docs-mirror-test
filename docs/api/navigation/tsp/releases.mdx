# Релизы

## 9 июня 2026 г.

* Выпущена версия TSP API 2.0. Расширены параметры [точек](/api/navigation/tsp/examples#point-parameters), [маршрутов](/api/navigation/tsp/examples#route-types) и [курьеров](/api/navigation/tsp/examples#courier-parameters) и добавлена возможность [настраивать параметры складов](/api/navigation/tsp/examples#depots). Поддержка версии API `1.1.0` прекращена.

## 1 декабря 2025 г.

* Обновлена сетка тарифов на API-сервисы. Подробнее об изменениях см. в [личном кабинете](https://platform.2gis.ru/ru/tariffs?utm_source=docs&utm_medium=referral&utm_campaign=api_tsp).

## 10 ноября 2025 г.

* Добавлена возможность задавать дополнительные ограничения для курьеров:

  * [Максимальное количество точек](/api/navigation/tsp/examples#max-points), которое курьер может посетить за маршрут.
  * [Максимальное время в пути и дистанцию](/api/navigation/tsp/examples#max-time-distance) для маршрута курьера.

* Добавлена возможность указывать [навыки](/api/navigation/tsp/examples#skills), которые необходимы курьеру для работы в точке.

## 3 февраля 2025 г.

* Добавлена возможность строить [пешеходные](/api/navigation/tsp/examples#walking), [велосипедные](/api/navigation/tsp/examples#bicycle) и [самокатные](/api/navigation/tsp/examples#scooter) маршруты.
* Добавлен параметр запроса `need_altitudes`, с помощью которого можно [получить информацию о высоте](/api/navigation/tsp/examples#route-altitude) пешеходного и велосипедного маршрута при построении.

## 25 ноября 2024 г.

* Параметр запроса `mode`, который указывает тип движения, заменён на параметр `transport`.

## 27 сентября 2024 г.

* Добавлена возможность настраивать мягкие и жёсткие [временные окна](/api/navigation/tsp/examples#time-windows) посещения точек маршрута.

## 5 июля 2024 г.

* Прекращена поддержка версии API `1.0`. Используйте методы версии API `1.1.0`.

## 6 апреля 2023 г.

* Добавлена [новая версия API 1.1.0](/api/navigation/tsp/reference/create_110).
  * Поддержка версии API `1.0` будет прекращена к концу октября 2023 года. Настоятельно рекомендуется использовать методы версии API `1.1.0`.
* Добавлены новые опции в блок маршрутизации `routing_options`: `exclude` — набор исключаемых областей, которые следует избегать при построении маршрута; `allow_locked_roads` — можно ли использовать при построении маршрута перекрытые дороги.
* Поля `filters`, `type`, `mode` перенесены в блок маршрутизации `routing_options`.
* Добавлена возможность добавлять приоритет точек на маршруте.

## 1 октября 2021 г.

Выпущен TSP API — алгоритм, который рассчитывает кратчайший маршрут обхода указанных точек для одного или нескольких курьеров.
