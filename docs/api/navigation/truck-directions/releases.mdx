# Релизы

## 28 июля 2023 г.

* Опубликована первая версия [Routing API](/api/navigation/routing/overview) с полной поддержкой всех функций [Directions API](/api/navigation/directions/overview), [Pairs Directions API](/api/navigation/pairs/overview) и [Truck Directions API](/api/navigation/truck-directions/overview).

## 16 июня 2022 г.

* Добавлена возможность получения информации о высоте маршрута и её изменении на протяжении маршрута. См. раздел [Информация о высоте](/api/navigation/truck-directions/overview#информация-о-высоте).
* Ответ на запрос теперь содержит информацию о [специальных точках на маршруте](/api/navigation/truck-directions/overview#специальные-точки) (например, начало платного участка дороги).

## 1 октября 2021 г.

* Добавлена возможность указать наличие пропуска на въезд в Москву и Санкт-Петербург в запросе на построение маршрута.
* Добавлен учет ограничений скорости для грузовых автомобилей
