## Простой способ

Этот способ подходит для простых статических страниц. Виджет появится на странице в том месте, куда был вставлен код.

Вставьте на страницу следующий код:

```html
<script
   charset="utf-8"
   src="https://floors-widget.api.2gis.ru/loader.js"
   id="dg-floors-widget-loader"
></script>
<script charset="utf-8">
   DG.FloorsWidget.init({
      width: '960px',
      height: '600px',
      initData: {
         complexId: '70000001007418793',
         options: {
            locale: 'ru_RU'
         }
      },
   });
</script>
```

## Продвинутый способ

Подключите на страницу JS-файл виджета:

```html
<script
   charset="utf-8"
   src="https://floors-widget.api.2gis.ru/loader.js"
   id="dg-floors-widget-loader"
></script>
```

Добавьте на страницу div-элемент, внутри которого хотите разместить виджет:

```html
<div id="mywidget"></div>
```

Проинициализируйте виджет:

```js
DG.FloorsWidget.init({
   container: 'mywidget',
   width: '960px',
   height: '600px',
   initData: {
      complexId: '70000001007418793',
      options: {
         locale: 'ru_RU'
      }
   },
});
```

В параметр `container` передаётся ID DOM-элемента или сам DOM-элемент. Этот способ подходит для более сложных случаев, когда требуется динамически создать виджет после открытия страницы (по нажатию на кнопку и т. п.).
