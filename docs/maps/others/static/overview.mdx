# Static API

Сервис Static API создаёт и показывает изображение статической карты <Brand /> на веб-страницах и в приложениях без использования JavaScript.

Изображение создаётся на основе параметров, указанных в URL: размера изображения, координат центра карты, масштаба карты, а также параметров дополнительных объектов (маркеров, линий и многоугольников).

Используйте Static API в следующих случаях:

* на сайтах для отображения превью карты;
* в приложениях, где не нужна интерактивность;
* на страницах, предназначенных для печати (например, листах доставки).

## Начало работы

### 1. Получите ключ {#get-access-key}

1. Зарегистрируйтесь в личном кабинете [Менеджер Платформы](https://platform.2gis.ru/ru/?utm_source=docs&utm_medium=referral&utm_campaign=api_static).
2. Создайте [демо-ключ](/platform-manager/subscription/keys#demo) или купите [подписку](/platform-manager/subscription/purchase) для доступа к API. Подробнее о стоимости сервиса см. в разделе [Тарифы](#tariffs).

Работать с ключами можно в Менеджере Платформы: подробнее см. в документации [личного кабинета](/platform-manager/overview).

### 2. Разместите карту {#add-map}

Создайте ссылку (URL), добавьте в неё ваш API-ключ и разместите её в теге `<img>` на вашем сайте или в приложении. Вы можете разместить изображение карты в любом месте на сайте.

Например, следующая ссылка генерирует изображение карты c набором дополнительных объектов:

```bash
https://static.maps.2gis.com/2.0?s=880x450&z=14&pt=55.03652,82.91871&ls=55.03652,82.91871,55.02711,82.92131,55.02470,82.92698~w:5&pt=55.02470,82.92698~k:c~n:1&ls=55.03652,82.91871,55.03758,82.93058,55.04042,82.92996~c:ff0000&pt=55.04042,82.92996~k:c~n:2&pn=55.03687,82.89827,55.04067,82.90925,55.03636,82.91355,55.03264,82.91062,55.03489,82.90021~c:03ee10~f:03ee10a0&pt=55.03484,82.92290~k:c~c:pe&key=API_KEY
```

## Тарифы {#tariffs}

* Тариф для сервиса включает фиксированное количество тарифицируемых событий за период.
* Тарифицируемое событие для Static API — успешный запрос, который возвращает одно статическое изображение карты.
* Актуальные тарифы можно посмотреть в [Менеджере Платформы](https://platform.2gis.ru/ru/tariffs?utm_source=docs&utm_medium=referral&utm_campaign=api_static).

## Варианты размещения {#distribution}

* **Облако**: Static API доступен через публичные endpoint-ы <Brand />.
* **On-Premise**: Static API недоступен при установке API-платформы <Brand /> в закрытом контуре.
