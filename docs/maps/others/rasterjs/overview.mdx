# RasterJS API

RasterJS API — это JavaScript-библиотека, с помощью которой вы можете добавить двумерную карту <Brand /> на ваш сайт.

Библиотека позволяет:

* создавать интерактивные карты на веб-странице;
* показывать на карте маркеры, попапы и геометрические объекты;
* определять координаты геообъектов по их названиям и названия по координатам.

Вы можете посмотреть примеры использования библиотеки в документации [RasterJS API](/maps/others/rasterjs/examples).

Исходный код RasterJS API доступен на [GitHub](https://github.com/2gis/mapsapi), проект открыт к предложениям и пулл-реквестам.

## Тарифы {#tariffs}

* Стоимость сервиса рассчитывается исходя из количества тарифицируемых событий сервиса [Raster Tiles API](/maps/others/rastertiles/overview), который предоставляет доступ к растровым тайлам карты. Одно событие соответствует загрузке одного растрового тайла в формате PNG. Безлимитный доступ к библиотеке RasterJS API предоставляется бесплатно.
* Актуальные тарифы можно посмотреть в [Менеджере Платформы](https://platform.2gis.ru/ru/tariffs?utm_source=docs&utm_medium=referral&utm_campaign=api_raster).

## Использование с Raster Tiles API {#distribution}

Вы можете использовать библиотеку RasterJS API с сервисом Raster Tiles API в зависимости от варианта его размещения:

* **Облако**: RasterJS API доступна для использования с облачной версией [Raster Tiles API](/maps/others/rastertiles/overview).
* **On-Premise**: RasterJS API недоступна для использования с Tiles API, установленным в закрытом контуре в качестве компонента [API-платформы](/on-premise-api-platform/overview/summary) <Brand />.
