# Начало работы

Чтобы начать работу с RasterJS API, выполните следующие шаги:

1. [Получите ключ доступа](#get-access-key).
2. [Подключите библиотеку RasterJS API к вашему проекту](#install-library).
3. [Инициализируйте карту](#initialize-map).
4. (Опционально) [Добавьте маркер на карту](#add-marker).

## 1. Получите ключ доступа {#get-access-key}

1. Зарегистрируйтесь в личном кабинете Менеджер Платформы.

   <CtaButton href="https://platform.2gis.ru/ru/?utm_source=docs&utm_medium=referral&utm_campaign=api_raster" target="_blank">
      В кабинет
   </CtaButton>

2. Создайте [демо-ключ](/platform-manager/subscription/keys#demo) или купите [подписку](/platform-manager/subscription/purchase) для доступа к сервису [Raster Tiles API](/maps/others/rastertiles/overview), который предоставляет доступ к растровым тайлам карты. Подробнее о стоимости сервиса см. в разделе [Тарифы](/maps/others/rasterjs/overview#tariffs).

   :::tip

   Если у вас уже есть действующий API-ключ, полученный для RasterJS API, вы можете использовать его с Raster Tiles API до истечения срока действия подписки. Срок действия можно посмотреть в личном кабинете, на вкладке **Главная**.

   :::

Подробнее о работе с ключами и подписками см. в [документации личного кабинета](/platform-manager/overview).

## 2. Подключите библиотеку {#install-library}

Добавьте следующую строку в код HTML-страницы:

```html
<script src="https://maps.api.2gis.ru/2.0/loader.js?pkg=full&key=YOUR_KEY"></script>
```

Если вы используете npm, см. раздел [Подключение API](/maps/others/rasterjs/reference/dg-loading).

## 3. Инициализируйте карту {#initialize-map}

Чтобы отобразить карту на странице, создайте HTML-элемент, который будет использоваться в роли контейнера, и укажите его размеры:

```html
<body>
    <div id="map" style="width:500px; height:400px"></div>
</body>
```

Чтобы инициализировать карту, вызовите метод [DG.map()](/maps/others/rasterjs/reference/map), укажите идентификатор контейнера и ваш ключ доступа. Вы также можете указать начальные координаты карты и нужный масштаб:

```html
<script type="text/javascript">
    var map;

    DG.then(function () {
        map = DG.map('map', {
            center: [54.98, 82.89],
            zoom: 13,
            key: 'Your API access key',
        });
    });
</script>
```

Здесь:

* `center` — координаты центра карты в формате `[широта, долгота]`;
* `zoom` — уровень масштабирования карты в диапазоне от 1 до 18.

Полный список параметров карты см. в [Справочнике API](/maps/others/rasterjs/reference/map).

## 4. Добавьте маркер {#add-marker}

Чтобы добавить маркер на карту, вызовите метод [DG.marker()](/maps/others/rasterjs/reference/markers), укажите инстанс карты и координаты маркера:

```html
DG.marker([54.98, 82.89]).addTo(map);
```

Чтобы добавить на карту попап (всплывающий блок) с необходимой информацией, который будет отображаться при клике на маркер:

```html
DG.marker([54.98, 82.89]).addTo(map).bindPopup('Вы кликнули по мне!');
```

Полный список параметров маркера см. в [Справочнике API](/maps/others/rasterjs/reference/markers).

## Пример кода {#code-example}

```html
<!DOCTYPE html>
<html>
    <head>
        <title>Raster Tiles API</title>
        <script src="https://maps.api.2gis.ru/2.0/loader.js?pkg=full&key=<YOUR_KEY>"></script>
        <script type="text/javascript">
            var map;

            DG.then(function () {
                map = DG.map('map', {
                    center: [54.98, 82.89],
                    zoom: 13,
                    key: 'Your API access key',
                });

                DG.marker([54.98, 82.89]).addTo(map).bindPopup('Вы кликнули на маркер');
            });
        </script>
    </head>
    <body>
        <div id="map" style="width:500px; height:400px"></div>
    </body>
</html>
```
