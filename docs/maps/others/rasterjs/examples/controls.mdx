# Элементы управления

## Описание

Ниже приведены примеры добавления, размещения и работы с элементами управления. Для получения подробной информации о работе с элементами управления см. раздел [Элементы управления](/maps/others/rasterjs/reference/controls).

## Скрытие элементов управления по умолчанию

Чтобы получить «чистую» карту без лишних элементов управления:

<HtmlExampleWithSrcCode iframeStyles={{ height: '300px' }}>
{`
<html>
    <head>
        <title>RasterJS API</title>
        <meta name="description" content="Скрытие элементов управления по умолчанию" />
        <script src="https://maps.api.2gis.ru/2.0/loader.js"></script>
    </head>
    <body>
        <div id="map" style="width: 100%; height: 400px"></div>
        <script>
            DG.then(function () {
                var map = DG.map('map', {
                    center: [54.98, 82.89],
                    zoom: 13,
                    fullscreenControl: false,
                    zoomControl: false,
                    key: 'Your API access key',
                });
            });
        </script>
    </body>
</html>
`}
</HtmlExampleWithSrcCode>
