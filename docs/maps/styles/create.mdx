# Создание стилей и настройка стилевых слоёв

Чтобы настроить внешний вид карты в Редакторе стилей, [создайте стиль](#creating-a-style).

Для изменения стиля объектов на карте [настройте существующие слои](#configuring-a-layer). Например, если вам нужно настроить иконки метро, используйте соответствующий слой, чтобы не настраивать отображение с нуля. См. список готовых слоёв для [MapGL JS API](/maps/styles/layers/configured/mapgl) и [мобильного SDK](/maps/styles/layers/configured/msdk).

Если вам нужно добавить на карту объекты, аналогов которых нет в готовых слоях, [добавьте новый слой](#adding-a-layer).

## Создание стиля {#creating-a-style}

Вы можете создать стиль для карты несколькими способами: из базового шаблона стиля, из копий стилей или с нуля.

### Из базового стиля

Для создания собственного стиля вы можете использовать готовые шаблоны с базовыми стилями и редактировать их.

Доступны стили для веб-приложений (MapGL JS API) и мобильных приложений (мобильный SDK) в дневной и ночной теме. В каждом стиле есть более 200 готовых слоёв.

Чтобы создать стиль из шаблона:

1. Откройте [Редактор стилей](https://styles.2gis.com/).

2. В списке шаблонов стилей выберите готовый шаблон.

    {/*  prettier-ignore  */}

   ![Создание стиля из шаблона](/img/styles/create-style-from-template.png)

3. Внесите изменения в базовый стиль.

Стиль автоматически сохранится как новый и отобразится в блоке **Мои стили**.

### Из копии собственного стиля

Чтобы создать новый стиль из копии созданного вами стиля:

1. Откройте [Редактор стилей](https://styles.2gis.com/).
2. В блоке **Мои стили** в карточке нужного стиля нажмите значок ![Меню стиля](/img/menu-dots.svg) и выберите **Дублировать**.

    {/*  prettier-ignore  */}

   ![Создание стиля из копии собственного стиля](/img/styles/create-style-from-copy.png)

Стиль автоматически сохранится как новый и отобразится в блоке **Мои стили**.

### Из копии стиля другого пользователя {#from-a-style-copy-of-another-user}

Чтобы скопировать стиль другого пользователя:

1. Запросите у владельца проекта ссылку для копирования стиля: см. инструкцию [Поделиться ссылкой на стиль](/maps/styles/connect#share-a-link-to-a-style).
2. Перейдите по ссылке.
3. Нажмите **Создать копию** в верхнем меню.

    {/*  prettier-ignore  */}

   ![Создание стиля из копии стиля другого пользователя](/img/styles/create-style-from-external.png)

Стиль автоматически сохранится как новый и отобразится в блоке **Мои стили**.

### С нуля

Вы можете настроить собственный стиль с нуля, создав в нём нужные слои.

Чтобы создать стиль с нуля:

1. Откройте [Редактор стилей](https://styles.2gis.com/).

2. В списке стилей выберите **Новый**.

    {/*  prettier-ignore  */}

   ![Создание стиля с нуля](/img/styles/create-style-from-scratch.png)

3. Выберите тип карты, для которой создаётся стиль:

   * **MapGL JS API** — для веб-приложений;
   * **Мобильный SDK** — для мобильных приложений.

4. Нажмите **Создать**.

5. Добавьте стилевые слои: см. инструкцию [Добавление слоя](#adding-a-layer).

Стиль автоматически сохранится как новый и отобразится в блоке **Мои стили**.

## Настройка существующего слоя {#configuring-a-layer}

:::note[Отображение слоёв]

Отображение слоёв на карте зависит от их порядка в стиле: слой, который находится в списке слоёв выше, перекрывает содержимое слоя ниже его. Например, чтобы на зданиях не перекрывались подписи, расположите слой с настройками подписей выше слоя с 3D-зданиями.

:::

1. Откройте [Редактор стилей](https://styles.2gis.com/).

2. В блоке **Мои стили** выберите нужный стиль. См. список готовых слоёв для [MapGL JS API](/maps/styles/layers/configured/mapgl) и [мобильного SDK](/maps/styles/layers/configured/msdk).

3. В разделе **Слои** выберите нужный слой.

    {/*  prettier-ignore  */}

   <img src="/img/styles/select-layer.png" alt="Выбор слоя" width="720"  />

4. Укажите нужные настройки слоя (например масштаб, видимость, стиль отображения). Доступные настройки зависят от слоя и его [типа](/maps/styles/layers/types/mapgl).

## Добавление слоя {#adding-a-layer}

Добавьте новый слой, если вам нужно добавить на карту объекты, аналогов которых нет в готовых слоях. В остальных случаях рекомендуется [настраивать существующие слои](#configuring-a-layer).

:::note[Отображение слоёв]

Отображение слоёв на карте зависит от их порядка в стиле: слой, который находится в списке слоёв выше, перекрывает содержимое слоя ниже его. Например, чтобы на зданиях не перекрывались подписи, расположите слой с настройками подписей выше слоя с 3D-зданиями.

:::

### Стандартные слои

1. Откройте [Редактор стилей](https://styles.2gis.com/).

2. В блоке **Мои стили** выберите нужный стиль.

3. В разделе **Слои** нажмите значок ![Добавить слой](/img/styles/add-layer-icon.svg).

    {/*  prettier-ignore  */}

   <img src="/img/styles/add-new-layer.png" alt="Добавление слоя" width="720"  />

4. Выберите тип слоя (например Полигон, Линия, Точка и другие). Тип слоя определяет внешний вид объектов слоя и данные для настройки. См. список типов слоёв для [MapGL JS API](/maps/styles/layers/types/mapgl) и [мобильного SDK](/maps/styles/layers/types/msdk).

5. Выберите данные для слоя. В каждом типе доступны разные наборы данных карты, стилем которых вы можете управлять. Например, для слоя с типом Линия доступны разные классы дорог и улиц, а для слоя с типом Полигон — области, кварталы, здания и другие.

6. Укажите остальные настройки слоя (например масштаб, видимость, стиль отображения). Доступные настройки зависят от типа слоя.

Новый слой автоматически добавится в стиль.

### Динамические слои

Динамический слой позволяет настроить внешний вид элементов, которые рисуются поверх карты, например градиентные линии для визуализации пробок на дорогах или маркеры местоположения пользователя. Вы можете добавить динамический слой только для карт с типом **Мобильный SDK**.

:::note[Просмотр динамических слоёв]

Динамические слои доступны для настройки, но не отображаются на карте в Редакторе стилей. Динамические слои можно посмотреть только в собранном мобильном приложении или в [демо-приложении](https://github.com/2gis/mobile-sdk-android-demo).

:::

1. Откройте [Редактор стилей](https://styles.2gis.com/).

2. В блоке **Мои стили** выберите нужный стиль.

3. В разделе **Слои** нажмите значок ![Добавить слой](/img/styles/add-layer-icon.svg).

    {/*  prettier-ignore  */}

   <img src="/img/styles/add-dynamic-layer.png" alt="Добавление динамического слоя" width="720"  />

4. Включите опцию **Динамические слои**.

5. Выберите тип слоя (например Градиентная линия, Круг в плоскости карты и другие). Тип слоя определяет внешний вид объектов слоя и данные для настройки. См. список типов слоёв для [мобильного SDK](/maps/styles/layers/types/msdk).

6. Выберите данные для слоя. В каждом типе доступны разные наборы данных карты, стилем которых вы можете управлять.

7. Укажите остальные настройки слоя (например масштаб, видимость, стиль отображения). Доступные настройки зависят от типа слоя.

Новый динамический слой автоматически добавится в стиль.
