# Готовые слои для мобильного SDK

Чтобы не настраивать отображение объектов на карте с нуля в Редакторе стилей, используйте существующие стилевые слои. Например, если вам нужно настроить иконки метро, используйте слои из папки **Metro stations icons**. Подробный список слоёв см. ниже.

## Городские объекты

Для обозначения городских объектов в виде иконок вы можете использовать слои с типом **Точка**. С помощью этих слоёв вы можете настраивать отображение иконок объектов в обычном и выделенном состоянии, а также их подписи: размер, цвет текста и другие параметры.

Подробнее о типах слоёв см. в [списке типов слоёв для мобильного SDK](/maps/styles/layers/types/msdk).

## 3D-объекты

Для отображения трёхмерных объектов вы можете использовать слои с разными типами: **Полигон с высотой**, **3D-модель** и другие. С помощью этих слоёв вы можете настраивать цвета заливки граней 3D-объекта, его высоту, расстояние до камеры, при котором у объекта появляется прозрачность, и другие параметры.

Настройки подписей для 3D-объектов (размер, цвет текста и другие параметры) указываются в отдельных слоях с типом **Точка**.

Подробнее о типах слоёв см. в [списке типов слоёв для мобильного SDK](/maps/styles/layers/types/msdk).

## Дороги и транспортные объекты

Для отображения пробок вы можете использовать слой с типом **Линия со смещением**. Для отображения дорог и улиц в зависимости от их класса и типа вы можете использовать слои с типами **Линия**, **Пунктирная линия** и **Линия из стрелок**. С помощью этих слоёв вы можете настраивать цвет заливки, ширину линии и другие параметры.

Настройки подписей для линий (размер, цвет текста и другие параметры) указываются в отдельных слоях с типом **Подпись линии**.

Для некоторых транспортных объектов (железнодорожные платформы, светофоры и другие) используются слои с другими типами, например **Точка** и **Полигон**.

## Географические объекты

Для настройки подписей (размера, цвета текста и других параметров) географических объектов на мировой карте вы можете использовать слои с типом **Точка**. С помощью этих слоёв при необходимости вы можете настраивать отображение иконок объектов в обычном и выделенном состоянии

Для настройки цвета водных объектов, природных зон и хилшейда используются слои с типом **Полигон**.

Подробнее о типах слоёв см. в [списке типов слоёв для мобильного SDK](/maps/styles/layers/types/msdk).

| ------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Подписи стран, населённых пунктов, районов | <ul><li>Папка **Global map**:<ul><li>**Medium countries**: средние страны.</li><li>**Large countries**: большие страны.</li><li>**Capitals**: столицы стран.</li><li>**Small cities**: малые города.</li><li>**Medium cities**: средние города.</li><li>**Major cities**: большие города.</li><li>**Regional centers**: административные центры районов и муниципальных округов в регионах.</li></ul></li><li>Слой **Settlement label**: сельские поселения.</li><li>Слой **District label**: административные округа и районы внутри населённого пункта.</li></ul> |
