# Обзор

<Brand /> Ситискан — это эффективное решение для фиксации и обработки данных реального мира. С помощью <Brand /> Ситискан вы можете оцифровать городскую среду для последующего анализа.

<Brand /> Ситискан предоставляет возможность:

* увидеть реальное состояние городской инфраструктуры;
* оцифровать городскую среду для дальнейшего контроля;
* инспектировать объекты реального мира с помощью искусственного интеллекта.

## Основные особенности {#main-features}

* **Простой сбор данных**

  Установите мобильное приложение, чтобы автоматически делать кадры городских объектов во время вождения автомобиля.

* **Распознавание объектов**

  Используйте искусственный интеллект, чтобы распознавать объекты на кадрах, определять их тип и связывать с географическими координатами. Информацию о городских адресах предоставляет [карта <Brand />](https://2gis.ru/).

* **Управление водителями**

  Создавайте маршруты для сбора данных и назначайте задачи водителям, чтобы обеспечить эффективное покрытие карты.

* **Аналитика**

  Отображайте на карте и анализируйте собранные данные с помощью [<Brand /> Про](/pro/start).
