# CityLens Web

Просматривайте детальную информацию об отдельных водителях и записанных треках в панели управления CityLens Web. Вы также можете отслеживать статусы загрузки кадров и распознавания объектов и контролировать, какие именно элементы отправить в [<Brand /> Про](/citylens/administrator/pro/overview).

## Начало работы {#getting-started}

1. Откройте панель CityLens Web.
2. Работайте с основными вкладками:

   * [Drivers](/citylens/administrator/web/drivers) для просмотра треков, сгруппированных по автору (водителю) и дате записи.
   * [Tracks](/citylens/administrator/web/tracks) для просмотра треков без группировки.
   * [Audit](/citylens/administrator/web/audit) для просмотра статистики распознавания дорожных знаков на треках.

## Загрузка данных в <Brand /> Про {#uploading-data-to-pro}

По умолчанию в [<Brand /> Про](/citylens/administrator/pro/overview) автоматически загружаются все данные (о водителях, записанных кадрах и распознанных объектах). Однако вы можете отфильтровать и удалить выбранные элементы из <Brand /> Про или загрузить элементы заново в случае ошибки.
