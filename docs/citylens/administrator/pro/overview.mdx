# <Brand /> Про для <Brand /> Ситискан

Анализируйте собранные данные и отображайте их на карте с помощью [<Brand /> Про](https://pro.2gis.ru/).

![Пример слоев с данными <Brand /> Ситискан](/img/pro/citylens-pro-overview.png)

## Начало работы {#getting-started}

1. Перейдите на страницу [pro.2gis.ru](https://pro.2gis.ru/).
2. Зарегистрируйтесь удобным для вас способом.
3. Откройте [специальные наборы данных для <Brand /> Ситискан](#data).
4. Узнайте, как выполнять основные задачи в <Brand /> Про:

   * как [создавать дашборды, сцены, слои и виджеты](/pro/dashboards/overview);
   * как [отображать данные на карте](/pro/data/visualization/overview);
   * как [настраивать стиль карты](/pro/styles/management).

Подробнее см. в [полной документации <Brand /> Про](/pro/start).

## Данные {#data}

<Brand /> Про предоставляет особые категории для просмотра данных <Brand /> Ситискан:

* [Кадры](/citylens/administrator/pro/frames) для просмотра кадров (изображений с привязкой к географическим координатам), которые водители записали через мобильное приложение.
* [Объекты](/citylens/administrator/pro/objects) для просмотра на карте объектов, распознанных на кадрах.
* [Водители](/citylens/administrator/pro/drivers) для просмотра информации о водителях и пройденных ими маршрутах.
* [Задачи (План)](/citylens/administrator/pro/tasks-planned) для просмотра запланированных маршрутов на карте.
* [Объезды (Факт)](/citylens/administrator/pro/tasks-actual) для просмотра фактических объездов водителей на карте.

Вы также можете использовать [системные наборы данных <Brand /> Про](/pro/data/overview), чтобы обогатить результаты вашего анализа.
