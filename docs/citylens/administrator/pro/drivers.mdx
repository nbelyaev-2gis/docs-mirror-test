# Водители

В наборе данных **Водители** представлена информация о зарегистрированных водителях <Brand /> Ситискан и пройденных ими маршрутах.

## Начало работы {#getting-started}

![Водители](/img/pro/data-drivers.png)

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите набор данных **Водители CityLens Россия**.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. На вкладке **Фильтры** настройте остальные [фильтры](#filters) при необходимости.

<br />

Данные будут отображены на карте в виде линий (пройденных маршрутов). Результаты фильтрации доступны в двух вкладках:

* **Статистика**: общие статистические данные о выборке:

   * Общее количество маршрутов.
   * Общее пройденное расстояние.
   * Общее количество часов в пути.
   * Среднее расстояние на один маршрут.
   * Среднее время в пути на один маршрут.
   * Диаграмма с топ-10 водителей по общему пройденному расстоянию.
   * Диаграмма с топ-10 водителей по среднему расстоянию.
   * Диаграмма с топ-10 водителей по среднему времени в пути.

* **Результаты**: список email-ов водителей в алфавитном порядке. При выборе водителя из списка:

   * Откроется карточка с информацией о водителе.
   * На карте отобразится последнее записанное местоположение водителя (чтобы приблизить, нажмите **Показать на карте**).

   Вы можете изменить вручную список результатов фильтрации, чтобы оставить в выборке только определённых водителей. Для этого нажмите **Изменить список**, выберите водителей и нажмите **Применить**.

Чтобы сохранить данную выборку, нажмите **Создать слой** и настройте [визуализацию данных](/pro/data/visualization/overview).

## Фильтры {#filters}

Для набора данных **Водители** доступны следующие фильтры:

* **Дата**: когда водитель завершил маршрут. Вы можете установить временной интервал, чтобы ограничить выборку.
* **Часовой пояс**: часовой пояс водителя.
* **E-mail водителя**: введите email, чтобы показать данные только для одного водителя.
* **Статус**: статус обработки данных, которые загрузил водитель.
* **Расстояние**: расстояние, которое водитель проехал при работе над задачами за сутки.
* **Длительность**: время, которое водитель провёл в работе над задачами за сутки.

Чтобы сбросить фильтр по параметру, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с именем фильтра. Чтобы сбросить все фильтры по параметрам, нажмите **Очистить фильтры** внизу вкладки.

Чтобы скрыть фильтр по территории, в окне **Настройки геофильтров** нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.
