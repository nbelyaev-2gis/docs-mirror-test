# Объезды (Факт)

В наборе данных **Объезды (Факт)** представлена информация о реальных траекториях объезда водителей в рамках задач, созданных в [Планировщике задач](/citylens/administrator/planner/overview). Набор данных содержит данные только о завершённых задачах.

## Начало работы {#getting-started}

![Объезды (Факт)](/img/pro/data-tasks-actual.png)

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите набор данных **Объезды (Факт)**.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. На вкладке **Фильтры** настройте остальные [фильтры](#filters) при необходимости.

<br />

Данные будут отображены на карте в виде линий (траекторий объезда). Результаты фильтрации доступны в двух вкладках:

* **Статистика**: общие статистические данные о выборке:

   * Общее количество задач.
   * Общая протяжённость дорог.
   * Общее время объездов.

* **Результаты**: список названий задач. При выборе задачи из списка:

   * Откроется карточка с информацией о задаче.
   * На карте отобразится маршрут (чтобы приблизить, нажмите **Показать на карте**).

   Вы можете изменить вручную список результатов фильтрации, чтобы оставить в выборке только определённые задачи. Для этого нажмите **Изменить список**, выберите задачи и нажмите **Применить**.

Чтобы сохранить данную выборку, нажмите **Создать слой** и настройте [визуализацию данных](/pro/data/visualization/overview).

## Фильтры {#filters}

Для набора данных **Объезды (Факт)** доступны следующие фильтры:

* **Идентификатор задачи**: ID задачи из Планировщика задач.
* **Название проекта**: [проект](/citylens/administrator/planner/projects), к которому относится задача.
* **Email**: email водителя.
* **ФИО водителя**: полное имя водителя.
* **Версия приложения**: версия мобильного приложения на устройстве водителя.
* **Назначена**: дата назначения водителя на задачу. Вы можете установить временной интервал, чтобы ограничить выборку.
* **Завершена**: дата завершения задачи. Вы можете установить временной интервал, чтобы ограничить выборку.
* **Фактическое расстояние**: длина фактического проезда в рамках задачи в километрах.
* **Фактическое время**: фактическое время проезда в рамках задачи в часах.
* **Выполнение**: процент выполнения задачи.

Чтобы сбросить фильтр по параметру, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с именем фильтра. Чтобы сбросить все фильтры по параметрам, нажмите **Очистить фильтры** внизу вкладки.

Чтобы скрыть фильтр по территории, в окне **Настройки геофильтров** нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.
