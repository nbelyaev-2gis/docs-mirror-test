# Объекты

В наборе данных **Объекты** представлена информация об объектах городской среды, которые были распознаны на [кадрах](/citylens/administrator/pro/frames).

## Начало работы {#getting-started}

![Объекты](/img/pro/data-objects.png)

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите набор данных **Объекты CityLens Россия**.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. На вкладке **Фильтры** настройте остальные [фильтры](#filters) при необходимости.

<br />

Данные будут отображены на карте в виде точек-центроидов: центров геометрий объектов. Результаты фильтрации доступны в двух вкладках:

* **Статистика**: общие статистические данные о выборке:

   * Общее количество объектов.
   * Диаграмма с распределением объектов по категориям детекций.

* **Результаты**: список распознанных объектов. При выборе объекта из списка:

   * Откроется карточка с информацией об объекте.
   * На карте отобразится местоположение объекта (чтобы приблизить, нажмите **Показать на карте**).

   Вы можете изменить вручную список результатов фильтрации, чтобы оставить в выборке только определённые объекты. Для этого нажмите **Изменить список**, выберите объекты и нажмите **Применить**.

Чтобы сохранить данную выборку, нажмите **Создать слой** и настройте [визуализацию данных](/pro/data/visualization/overview).

## Фильтры {#filters}

Для набора данных **Объекты** доступны следующие фильтры:

* **Категории детекций**: категории объектов городской среды, которые могут быть распознаны.
* **Типы детекций**: объекты городской среды, которые могут быть распознаны.
* **Статус объекта**: статус обработки инцидента, связанного с объектом.
* **В последний раз обнаруживался**: дата последней записи объекта. Вы можете установить временной интервал, чтобы ограничить выборку.

Чтобы сбросить фильтр по параметру, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с именем фильтра. Чтобы сбросить все фильтры по параметрам, нажмите **Очистить фильтры** внизу вкладки.

Чтобы скрыть фильтр по территории, в окне **Настройки геофильтров** нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.
