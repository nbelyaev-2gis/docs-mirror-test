# Управление стилем карты

По умолчанию в Планировщике задач для всех территорий, проектов и задач используется общий стиль, заданный в настройках карты. Вы можете изменить общий стиль карты или подключить карту из внешнего источника.

## Изменение общего стиля {#changing-style}

![Общий стиль карты](/img/citylens/style-management-project-529.png)

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории и проекта.
3. Нажмите значок ![Менеджер стилей](/img/pro/style-manager-icon.svg) в правом нижнем углу карты.
4. В окне **Настройки карты** выберите стиль карты:

   * **Встроенные темы**: варианты цветового оформления карты <Brand /> (дневная, ночная тема и другие).
   * **Карты из внешних источников**: карты с дополнительными данными (например, спутниковыми снимками). Чтобы наложить карту поверх базовой темы, в карточке нужной карты нажмите значок ![Меню карты](/img/menu-dots.svg) и выберите **Показывать поверх карты**.

   Вы также можете [подключить другие карты из внешних источников](#adding-external-map) и использовать их в качестве стилей.

## Подключение внешней карты {#adding-external-map}

Вы можете подключить карту из внешнего источника (например, спутниковые снимки) в форматах WMS и WMTS. Новые данные накладываются на основную карту <Brand /> и отображаются как отдельный стиль карты.

Вы можете добавить не более 15 источников внешних карт.

![Внешняя карта](/img/citylens/external-map-529.png)

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории и проекта.
3. Нажмите значок ![Менеджер стилей](/img/pro/style-manager-icon.svg) в правом нижнем углу карты.
4. Нажмите **Добавить источник**.
5. Укажите данные внешнего источника:

   <img src="/img/citylens/external-map-data-529.png" alt="Настройки данных карты" width="550" />

   * **Ссылка** (обязательный параметр): URL-адрес источника для получения карты.

   * **Название стиля** (обязательный параметр): имя нового стиля.

   * **Тип источника** (обязательный параметр): как представлены данные в источнике (по ссылке выше):

      * **WMS**: стандарт динамического получения географических данных в виде изображений с привязкой к местности.
      * **WMTS**: стандарт получения кешированных изображений.

   * **Копирайт**: имя правообладателя. Добавьте, если внешний источник требует его указания. Текст будет отображаться в левом нижнем углу карты.

6. Нажмите **Далее**.

7. Выберите один из вложенных слоёв для отображения. Список слоёв зависит от внешнего источника.

   <img src="/img/citylens/external-map-layer-529.png" alt="Выбор вложенного слоя" width="550" />

8. Нажмите **Добавить**.

## Операции с внешними картами {#operations-with-external-maps}

### Редактирование карты {#editing-map}

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории и проекта.
3. Нажмите значок ![Менеджер стилей](/img/pro/style-manager-icon.svg) в правом нижнем углу карты.
4. В карточке внешней карты нажмите значок ![Меню карты](/img/menu-dots.svg) и выберите **Редактировать**.
5. Внесите изменения в данные.
6. Нажмите **Сохранить**.

### Удаление карты {#deleting-map}

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории и проекта.
3. Нажмите значок ![Менеджер стилей](/img/pro/style-manager-icon.svg) в правом нижнем углу карты.
4. В карточке внешней карты нажмите значок ![Меню карты](/img/menu-dots.svg) и выберите **Удалить**.
