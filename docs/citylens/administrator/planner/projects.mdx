# Проекты

**Проект** — это набор задач с общей целью и территорией. Если необходимо выполнять одинаковые объезды регулярно, вместо создания задач с нуля вы можете копировать существующие проекты.

Вы можете работать с проектами на вкладке **Задачи**:

* [создавать проекты](#creating-project);
* [просматривать информацию о проектах](#viewing-project);
* [просматривать статистику по проектам](#viewing-statistics);
* [редактировать проекты](#editing-project);
* [дублировать проекты](#duplicating-project).

![Работа с проектами](/img/citylens/planner-projects-overview-633.png)

## Создание проекта {#creating-project}

![Создание проекта](/img/citylens/planner-projects-create-633.png)

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории.
3. Нажмите **Новые проекты**.
4. Укажите параметры проекта:

   * **Название проекта** (обязательный параметр).
   * **Описание**: описание проекта.
   * **Начало**: дата начала проекта.
   * **Конец**: дата завершения проекта.

5. Нажмите **Сохранить**.

## Просмотр информации о проекте {#viewing-project}

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории.
3. В строке проекта нажмите значок ![Меню проекта](/img/menu-dots-horizontal.svg) у проекта и выберите **Информация о проекте**.

Проекты распределены в зависимости от даты начала и даты завершения по трём вкладкам: **Текущие**, **Будущие** и **Завершённые**. Вы можете использовать поиск по списку проектов и фильтрацию по дате начала и завершения проекта.

В карточке проекта отображается:

* **Общая информация**:
  * **Территория**: территория, в которой создан проект.
  * **ID**: уникальный идентификатор проекта.
  * **Начало**: запланированная дата начала проекта.
  * **Конец**: запланированная дата завершения проекта.
  * **Отредактировано**: дата последнего редактирования проекта.
* **Задачи**:
  * **Все задачи**: общее количество задач проекта.
  * **Назначена**: количество задач проекта, назначенных на водителей (со статусом **Назначена**).
  * **В работе**: количество задач проекта в работе (со статусом **В работе**).
  * **Завершена**: количество выполненных задач проекта (со статусом **Завершена**).
  * **Отменена**: количество отменённых задач проекта (со статусом **Отменена**).
* **Водители**:
  * **Назначенные водители**: количество водителей, назначенных на задачи проекта.

## Просмотр статистики по проекту {#viewing-statistics}

Вы можете просматривать дашборды со статистикой выполнения задач для каждого проекта. Дашборды отображают прогресс по статусам задач, процент выполнения, время, расстояние, активность водителей и другие параметры:

![Статистика по проекту](/img/citylens/planner-projects-statistics-633.png)

Чтобы открыть дашборд:

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории.
3. В списке проектов нажмите значок ![Статистика проекта](/img/citylens/statistics.svg) в превью проекта. Вы можете фильтровать данные на дашборде по дате.

## Редактирование проекта {#editing-project}

Вы можете редактировать проекты: изменить название, описание, запланированные даты начала и завершения проекта.

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории.
3. В строке проекта нажмите значок ![Меню проекта](/img/menu-dots-horizontal.svg) и выберите **Редактировать**.
4. Внесите изменения в проект.
5. Нажмите **Сохранить**.

## Дублирование проекта {#duplicating-project}

При дублировании проекта создаётся новый проект для той же территории. Все задачи из исходного проекта (кроме задач со статусом **Отменена**) автоматически пересоздаются в новом проекте со статусом **Новая**.

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории.
3. В строке проекта нажмите значок ![Меню проекта](/img/menu-dots-horizontal.svg) и выберите **Скопировать проект**.
4. (Опционально) Укажите новые даты начала и завершения проекта.
5. (Опционально) Чтобы скопировать список водителей, назначенных на задачи проекта, отметьте чекбокс **Копировать водителей**.
6. Нажмите **Сохранить**.

При необходимости вы можете [отредактировать задачи](/citylens/administrator/planner/tasks#editing-task) в новом проекте.
