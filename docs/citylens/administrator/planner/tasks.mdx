# Задачи

**Задача** — это запланированный маршрут, по которому должен проехать водитель для выполнения съёмки и сбора данных.

Для каждой задачи автоматически рассчитываются общая длина дорог и примерное время объезда на автомобиле. Для задач с полигонами также рассчитывается площадь полигона.

Вы можете работать с задачами для объезда территорий на вкладке **Задачи**:

* [создавать задачи](#creating-task) и маршруты для них [разными способами](#route-options);
* [автоматически разделять полигоны на несколько задач](#splitting-polygons);
* [просматривать информацию о задачах](#viewing-task);
* [отслеживать статус созданных задач](#task-status);
* [редактировать задачи](#editing-task);
* [выгружать полигоны и маршруты из задачи](#exporting-task);
* [завершать задачи](#completing-task);
* [отменять задачи](#canceling-task).

![Работа с задачами](/img/citylens/planner-tasks-overview-633.png)

## Создание задачи {#creating-task}

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории и проекта.
3. Нажмите **Новая задача**.
4. Создайте маршрут одним из [способов](#route-options).
5. Укажите параметры задачи:

   * **Данные задачи**:
      * **Название задачи** (обязательный параметр).
      * **Водитель**: водитель, которому будет назначена задача. Вы можете использовать поиск по списку водителей и фильтрацию по группам с помощью тегов. В списке отображаются только водители, [добавленные в территорию](/citylens/administrator/planner/drivers#adding-driver-to-territory). Вы можете пропустить выбор водителя и назначить его позже.
   * **Дополнительные атрибуты**:
      * **Проезд в одном направлении**: отметьте чекбокс, если водитель должен проехать маршрут только в одном направлении (например, при съёмке вывесок с одной стороны дороги).
      * **Анализ вывесок**: отметьте чекбокс, если вы планируете съёмку вывесок. Чтобы получить доступ к этой функции, обратитесь к менеджеру.
      * **Направление камеры**: стекло автомобиля, на которое направлена камера мобильного устройства. По умолчанию используется значение **Вперёд**.
      * **Срок исполнения**: дата, до которой водитель должен выполнить задачу.
   * **Исключение дорог**: все дороги внутри полигона автоматически включаются в маршрут. Вы можете исключить некоторые дороги, например, чтобы вынести часть из них в другую задачу позже:
      * **Класс дороги**: магистральные дороги, федеральные трассы и другие.
      * **Дорожный атрибут**: тип покрытия, платные дороги, переправы.
      * **Длина дороги**: длина улицы в метрах.
      * **Сохранить для следующих задач**: отметьте чекбокс, чтобы при создании следующих задач были применены такие же фильтры исключения дорог.

      Также вы можете [исключить дороги из полигона](#excluding-roads) вручную, кликнув на них на карте.

   ![Параметры задачи](/img/citylens/planner-tasks-parameters-633.png)

6. Нажмите **Создать задачу**.

## Автоматическое разделение полигона {#splitting-polygons}

Вы можете ускорить планирование задач, автоматически разделив полигон на несколько задач. Количество задач зависит от количества водителей и одного из выбранных параметров:

* количества часов, которое водитель может потратить на объезд в рамках одной задачи;
* максимальной длины маршрута для каждой задачи.

Каждому водителю назначается одна задача на один рабочий день. По умолчанию рабочий день длится 8 часов, это значение изменить нельзя.

При разделении для каждого полигона рассчитывается протяжённость дорог и примерное время объезда на автомобиле, а также общее количество рабочих дней, которое потребуется для объезда всех полигонов.

<br />

Чтобы автоматически разделить полигон на несколько задач:

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории и проекта.
3. Нажмите **Планировщик задач**.
4. [Создайте полигон](#polygons).
5. В диалоговом окне в правом нижнем углу нажмите **Использовать**:

   ![Использование полигона](/img/citylens/planner-tasks-auto-633.png)

6. Укажите параметры планирования:

   * Чтобы разделить полигон на несколько задач, укажите один из параметров:

      * **Часы**: количество часов, за которое можно проехать каждый полигон. При выборе значения вы можете ориентироваться на максимальное количество часов, которое водитель может потратить в рабочий день на объезд. Доступные значения — от 1 до 8 часов.
      * **Расстояние**: длина маршрута в километрах для каждого полигона. Доступные значения — от 1 до 100 км.

   * **Проезд в одном направлении**: отметьте чекбокс, если водители должны проехать маршрут только в одном направлении (например, при съёмке вывесок с одной стороны дороги). Время и расстояние будут рассчитаны для одного прохода в одном направлении. Если чекбокс не отмечен, время и расстояние примерно удваиваются.
   * **Исключение дорог**: все дороги внутри полигона автоматически включаются в маршрут. Вы можете исключить некоторые дороги, например, чтобы вынести часть из них в другую задачу позже:
      * **Класс дороги**: магистральные дороги, федеральные трассы и другие.
      * **Дорожный атрибут**: тип покрытия, платные дороги, переправы.
      * **Длина дороги**: длина улицы в метрах.
      * **Сохранить для следующих задач**: отметьте чекбокс, чтобы при создании следующих задач были применены такие же фильтры исключения дорог.

   Нажмите **Далее**.

   <img src="/img/citylens/planner-tasks-auto-settings-633.png" alt="Параметры планирования" width="400"  />

7. Укажите параметры задач:

   * **Имя для всех задач** (обязательный параметр): название каждой задачи. К указанному имени будет добавляться порядковый номер (например, `Задача 1`, `Задача 2` и т. д.).
   * **Водитель** (обязательный параметр): один или несколько водителей, которым будут назначены задачи. Вы можете использовать поиск по списку водителей и фильтрацию по группам с помощью тегов. В списке отображаются только водители, [добавленные в территорию](/citylens/administrator/planner/drivers#adding-driver-to-territory).
   * **Срок исполнения**: дата, до которой водители должны выполнить задачи.
   * **Дополнительные атрибуты**:
      * **Анализ вывесок**: отметьте чекбокс, если вы планируете съёмку вывесок. Чтобы получить доступ к этой функции, обратитесь к менеджеру.
      * **Направление камеры**: стекло автомобиля, на которое направлена камера мобильного устройства. По умолчанию используется значение **Вперёд**.

   Нажмите **Далее**.

   ![Параметры задач](/img/citylens/planner-tasks-auto-parameters-633.png)

8. Проверьте результат планирования задач. Нажмите **Создать задачи**.

   <img src="/img/citylens/planner-tasks-auto-create-633.png" alt="Создание задач" width="350"  />

Все созданные задачи появятся в списке:

![Список задач](/img/citylens/planner-tasks-auto-list-633.png)

## Способы создания маршрута {#route-options}

Для построения маршрута вы можете использовать:

* [Полигоны](#polygons): рисовать полигоны на интерактивной карте <Brand /> или загружать собственные полигоны с уникальными зонами, областями, районами и т. д. Все дороги внутри полигона автоматически включаются в маршрут, при необходимости вы можете [исключить их](#excluding-roads).

  Также вы можете [автоматически разделить полигон](#splitting-polygons) на несколько задач в зависимости от количества водителей, времени на выполнение задачи и длины маршрута.

  Пример задачи с полигоном на карте:

  <img src="/img/citylens/planner-polygon-633.png" alt="Пример полигона на карте" width="700" />

* [Дороги](#roads): выделять дороги маршрута на карте. При необходимости вы можете выбрать только нужные сегменты дорог.

  Пример задачи с выделенными на карте дорогами:

  <img src="/img/citylens/planner-roads-633.png" alt="Пример дорог на карте" width="700" />

* [Точки](#points): отмечать точки на карте, которые нужно посетить в рамках задачи, или загружать точки из файла. Рассчитывается кратчайший по времени маршрут, проходящий через точки.

  Пример задачи с точками маршрута:

  <img src="/img/citylens/planner-points-633.png" alt="Пример точек на карте" width="700" />

### Полигоны {#polygons}

Вы можете [нарисовать полигон на карте](#creating-polygon) или [загрузить собственный полигон из файла](#uploading-polygon). При необходимости из полигонов можно [исключить дороги](#excluding-roads).

#### Создание полигона на карте {#creating-polygon}

Чтобы создать полигон:

1. Нажмите значок ![Рисовать полигон](/img/citylens/planner-draw-polygon.svg).
2. Отметьте не менее трёх точек на карте. Точки соединяются автоматически, линии полигона не должны пересекаться.
3. Чтобы переставить точку, зажмите её и переместите в нужное место на карте. Чтобы удалить точку, наведите на неё курсор и нажмите значок ![Крестик](/img/pro/cross-icon.svg).
4. Чтобы замкнуть полигон, соедините начальную и конечную точку или нажмите *Enter*.

![Рисование полигона](/img/citylens/planner-tasks-draw-633.png)

При создании маршрута вы можете использовать поиск по адресам, местам и организациям на карте. Также вы можете использовать в качестве полигонов стандартные территории (области, районы, населённые пункты и другие), выбранные в поиске. Полигон построится на карте автоматически:

![Поиск полигонов на карте](/img/citylens/planner-tasks-search-633.png)

#### Загрузка полигона из файла {#uploading-polygon}

Чтобы загрузить собственный полигон из файла:

1. Нажмите значок ![Рисовать полигон](/img/citylens/planner-draw-polygon.svg).
2. Нажмите ![Загрузить полигон](/img/citylens/planner-upload-polygon.svg).
3. Перетащите файл в область загрузки или нажмите **Выберите файл на компьютере**. Файл должен соответствовать [требованиям](#polygon-requirements).

   <img src="/img/citylens/planner-tasks-file.png" alt="Загрузка полигона из файла" width="370" />

##### Требования к файлу с полигоном {#polygon-requirements}

* Геометрия: Полигон (Polygon).
* Форматы: `.kml`, `.kmz`, `.shp`, `.geojson`.
* Размер: до 100 МБ.
* Кодировка: UTF-8.
* Система координат: [WGS84(4326)](https://epsg.io/4326).

Пример файла:

```json
{
    "type": "FeatureCollection",
    "features": [
        {
            "type": "Feature",
            "properties": {},

                "coordinates": [
                    [
                        [37.60556107462642, 55.76462040952444],
                        [37.614942029534205, 55.75679538816905],

                        [37.61356440678534, 55.76799723318385],
                        [37.60556107462642, 55.76462040952444]
                    ]
                ],
                "type": "Polygon"
            }
        }
    ]
}
```

</details>

#### Исключение дорог из полигона {#excluding-roads}

Все дороги внутри полигона автоматически включаются в маршрут. Вы можете исключить некоторые дороги, например, чтобы вынести часть из них в другую задачу позже.

Чтобы исключить дороги, нажмите значок ![Исключить дороги](/img/citylens/planner-exclude-roads.svg) и кликните на дороги, которые нужно исключить. В режиме редактирования исключённые дороги выделяются красным цветом.

![Исключение дорог](/img/citylens/planner-tasks-exclude-633.png)

### Дороги {#roads}

Чтобы выделить дороги на карте, нажмите значок ![Выделить дороги](/img/citylens/planner-roads.svg) и выделите дороги. Чтобы удалить дорогу из маршрута, кликните на неё на карте.

![Выделение дорог на карте](/img/citylens/planner-tasks-roads-633.png)

Чтобы включить в маршрут только часть дороги, разделите её на сегменты. Для этого нажмите значок ![Выделить сегменты дорог](/img/citylens/planner-segments.svg) и отметьте точки, по которым дорога разделится на сегменты. Чтобы исключить сегмент из маршрута, нажмите значок ![Корзина](/img/pro/trash-icon.svg) в списке сегментов.

![Выделение сегментов дорог на карте](/img/citylens/planner-tasks-segments-633.png)

### Точки {#points}

Вы можете [отметить точки на карте](#selecting-points) или [загрузить точки из файла](#uploading-points).

#### Выбор точек на карте {#selecting-points}

Чтобы выбрать точки маршрута, нажмите значок ![Выбрать точки](/img/citylens/planner-click-points.svg) и отметьте точки на карте.

Чтобы удалить точку из маршрута, наведите на неё курсор и нажмите значок ![Крестик](/img/pro/cross-icon.svg).

![Выбор точек на карте](/img/citylens/planner-tasks-points-633.png)

#### Загрузка точек из файла {#uploading-points}

Чтобы загрузить точки из файла:

1. Нажмите значок ![Менеджер стилей](/img/pro/style-manager-icon.svg) в правом нижнем углу карты.
2. Нажмите **Добавить приоритетные точки**.
3. Перетащите файл в область загрузки или нажмите **Выберите файл на компьютере**.
4. Нажмите на карточку с точками, чтобы отобразить их на карте. Файл должен соответствовать [требованиям](#points-requirements).

![Загрузка точек из файла](/img/citylens/planner-tasks-points-file-633.png)

Чтобы включить точки в маршрут, используйте один из способов:

* Нажмите значок ![Выбрать точки](/img/citylens/planner-click-points.svg) и нажмите на точки на карте.
* Создайте [полигон](#polygons), который покроет точки.
* Выделите [дороги](#roads), которые проходят через точки.

## Просмотр информации о задаче {#viewing-task}

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории и проекта.
3. Выберите задачу в списке или нажмите на соответствующий полигон или дороги маршрута на карте. Вы можете использовать поиск по задачам в списке, сортировку по параметрам задачи, а также фильтрацию по тегу, водителю, статусу и типу задачи.

<br />

* В зависимости от статуса задачи на карте отображается:

   * **Новая** и **Назначена**: запланированный маршрут.
   * **В работе**:
      * Последняя геопозиция водителя.
      * Пройденная часть запланированного маршрута (жёлтый цвет).
      * Оставшаяся часть запланированного маршрута (синий цвет).
      * Если водитель выполняет задачу в данный момент, отобразится текущая геопозиция водителя. Также вы сможете наблюдать за его перемещением на карте в режиме реального времени.
      * Сообщения о событиях на дороге от водителя.
   * **Завершена**: пройденный маршрут и сообщения о событиях на дороге от водителя. Маршруты завершённых задач также доступны в наборах данных <Brand /> Про: [Задачи](/citylens/administrator/pro/tasks-planned) (запланированные маршруты) и [Объезды](/citylens/administrator/pro/tasks-actual) (фактические маршруты).

* В карточке задачи отображается:

   * **ID**: уникальный идентификатор задачи.
   * **Дополнительные атрибуты**: дополнительные атрибуты задачи (например, **Анализ вывесок** и **Внешняя задача**).
   * **Направление камеры**: стекло автомобиля, на которое направлена камера мобильного устройства.
   * **Процент выполнения**: процент выполнения задачи. Отображается только для задач со статусом **В работе** или **Завершена**.
   * **Дата завершения**: запланированная дата выполнения задачи. Отображается, если это поле было заполнено при создании или редактировании задачи.
   * **Дата создания**: дата создания задачи.
   * **Площадь**: площадь территории (полигона) в км². Отображается только для задач с полигонами.
   * **Длина дорог**: общая протяжённость дорог маршрута.
   * **Время объезда**: расчётное время объезда на автомобиле с учётом пробок.
   * **Водитель**: водитель и время его последней активности. Отображается, если водитель был назначен при создании или редактировании задачи.

## Просмотр статуса задачи {#task-status}

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории и проекта.
3. Посмотрите статус задачи в её строке. От статуса задачи зависит цвет соответствующего полигона или дорог маршрута на карте.

<br />

Возможные статусы задачи:

* **Новая** (синий цвет): задача создана, но не назначена на водителя. Вы можете [редактировать задачу](#editing-task) или [отменить задачу](#canceling-task) с этим статусом.
* **Назначена** (жёлтый цвет): задача назначена на водителя, но водитель пока не взял её в работу. Вы можете [редактировать задачу](#editing-task) или [отменить задачу](#canceling-task) с этим статусом.
* **В работе** (жёлтый цвет): водитель взял задачу в работу в мобильном приложении. В статусе отображается прогресс выполнения задачи в процентах. Вы можете [завершить задачу](#completing-task) или [отменить задачу](#canceling-task) с этим статусом.
* **Завершена** (зелёный цвет): задача помечена как завершённая.
* **Отменена**: задача отменена. Отменённые задачи не отображаются на карте, но по ним сохраняется вся информация в карточке задачи. Отменённые задачи не удаляются из списка.

## Редактирование задачи {#editing-task}

Если задача в статусе **Новая** или **Назначена**, вы можете её отредактировать:

* отредактировать маршрут: изменить нарисованный полигон, загрузить новый, выделить или исключить дороги, выбрать или удалить точки маршрута;
* переименовать задачу;
* назначить на другого водителя или удалить водителя из задачи;
* указать дополнительные атрибуты, например сделать задачу внешней и наоборот;
* изменить направление камеры;
* изменить запланированный срок исполнения задачи.

<br />

Чтобы отредактировать задачу:

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории и проекта.
3. Выберите задачу и в открывшемся окне нажмите **Редактировать задачу**.
4. Внесите изменения в задачу.
5. Нажмите **Сохранить задачу**.

## Выгрузка геометрии из задачи {#exporting-task}

Вы можете выгружать в формате `.kml` маршруты (полигоны и выделенные дороги), созданные в Планировщике задач. Например, если водители просматривают маршрут не в [мобильном приложении](/citylens/driver/start), а в другой системе, вы можете экспортировать геометрии маршрутов из задач и передавать их водителям.

1. Чтобы выгрузить задачу, отметьте её как внешнюю. Для этого:
   1. Перейдите на вкладку **Задачи**.
   2. Откройте страницу территории и проекта.
   3. Выберите задачу и в открывшемся окне нажмите **Редактировать задачу**.
   4. Отметьте чекбокс **Внешняя задача**.
   5. Нажмите **Сохранить задачу**.
2. В списке задач выберите задачу, из которой нужно выгрузить геометрию.
3. В открывшемся окне нажмите значок ![Меню задачи](/img/menu-dots-horizontal.svg) и выберите **Экспорт (kml)**. Будет скачан файл с геометрией задачи.

Чтобы посмотреть список внешних задач, в списке задач отметьте в фильтрах чекбокс **Тип задачи → Внешняя задача**.

## Завершение задачи {#completing-task}

Вы можете пометить задачу с любым процентом выполнения как завершённую и перевести её в статус **Завершена**. Вы можете завершать любые задачи в статусе **В работе**, а также [внешние задачи](#exporting-task) в статусах **Новая** и **Назначена**.

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории и проекта.
3. Выберите задачу.
4. В открывшемся окне нажмите значок ![Меню задачи](/img/menu-dots-horizontal.svg) и выберите **Завершить задачу**.

## Отмена задачи {#canceling-task}

Если задача в статусе **Новая**, **Назначена** или **В работе**, вы можете отменить задачу и перевести её в статус **Отменена**.

1. Перейдите на вкладку **Задачи**.
2. Откройте страницу территории и проекта.
3. Выберите задачу.
4. В открывшемся окне нажмите значок ![Меню задачи](/img/menu-dots-horizontal.svg) и выберите **Отменить задачу**.
