# Обзор инструментов администратора

Администратор может использовать следующие инструменты:

* [Планировщик задач](/citylens/administrator/planner/overview) — это панель управления задачами водителей, которая позволяет:

   * Создавать маршруты объездов разными способами: рисовать полигоны, выделять дороги или отмечать точки на интерактивной карте, а также загружать собственные полигоны.
   * Планировать задачи для объезда территорий, автоматически разделять полигоны на несколько задач в зависимости от количества водителей, целевого времени или расстояния, а также назначать их водителям.
   * Просматривать информацию о водителях и статистике их работы.

* [<Brand /> Про](/citylens/administrator/pro/overview) — это геоаналитическая платформа, которая предоставляет продвинутые инструменты для пространственного анализа и специальные наборы данных для <Brand /> Ситискан. Позволяет:

   * Отображать собранные данные <Brand /> Ситискан на карте в удобном виде.
   * Просматривать статистику по выполненным задачам.
   * Использовать встроенные геопространственные данные или загружать дополнительные данные, чтобы обогатить результаты анализа.
   * Вычислять метрики, чтобы получить новые выводы о городской среде.

* [CityLens Web](/citylens/administrator/web/overview) — это панель инструментов администратора, которая позволяет:

   * Просматривать детальную информацию о треках водителей, статистике загрузки кадров и распознанных объектах по трекам.
   * Отслеживать статус обработки кадров в реальном времени.
   * Выгружать данные в формате CSV и KML.

* [Export API](/citylens/administrator/export/overview) — это сервис для автоматизированного доступа к кадрам и результатам распознавания, который позволяет:

   * Автоматически выгружать исходные кадры и кадры с распознанными объектами в формате JPEG.
   * Настраивать параметры выгрузки данных в соответствии с вашими потребностями.

:::note[Доступность данных]

Собранные данные станут доступны в <Brand /> Про, CityLens Web и через Export API только после того, как водитель [отправит записанные кадры](/citylens/driver/record#upload-tracks) со своего мобильного устройства.

:::
