# Обзор

Export API помогает администраторам <Brand /> Ситискан экспортировать результаты распознавания для анализа.

## Основные возможности {#main-features}

С помощью Export API вы можете получить:

* Список исходных кадров и их метаданных с фильтрацией по дате и времени записи.
* Исходные кадры в формате JPEG.
* Кадры с отмеченными распознанными объектами.
* Список доступных детекторов: категорий объектов городской среды, которые могут быть распознаны на кадрах.
* Список объектов, распознанных на кадрах, с фильтрацией по дате и времени записи и типу детектора.

Подробнее см. в [примерах](/citylens/administrator/export/examples) и в [Справочнике API](/citylens/administrator/export/reference).

## Начало работы {#getting-started}

### 1. Получите ключ доступа {#get-access-key}

Чтобы получить ключ доступа и URL для обращения к API сервиса, обратитесь к администратору вашей организации.

Количество выпущенных ключей для одной организации не ограничено.

### 2. Отправьте запрос {#send-request}

Чтобы получить список объектов, распознанных на кадрах, отправьте GET-запрос на `/export-api/v1/detections/objects`, указав значение API-ключа в заголовке запроса `X-API-Key` и необходимые параметры в строке запроса.

#### Пример запроса {#request-example}

```curl
curl --request GET \
  --url 'API_URL/export-api/v1/detections/objects?from_dt=2025-09-01T00:00:00&to_dt=2025-12-30T00:00:00&detector_name=signs&locale=ru_RU&limit=100' \
  --header 'X-API-Key: API_KEY'
```

Где:

* `API_URL` — URL для обращения к API, который вы получили от администратора вашей организации.
* `from_dt` (обязательный параметр) — начальная дата и время записи кадра в формате `YYYY-MM-DDTHH:MM:SS`.
* `to_dt` (обязательный параметр) — конечная дата и время записи кадра в формате `YYYY-MM-DDTHH:MM:SS`.
* `detector_name` — название детектора (категории распознаваемых объектов) для фильтрации. Например, `signs` — дорожные знаки.
* `locale` — язык названий распознанных объектов в ответе.
* `limit` — максимальное количество объектов в ответе.
* `API_KEY` — ключ доступа, который вы получили от администратора вашей организации.
