# Начало работы

Следуйте инструкциям ниже в зависимости от вашей роли в системе.

## Для администраторов {#for-administrators}

Чтобы начать работу с <Brand /> Ситискан:

1. Обратитесь в вашу организацию, чтобы получить доступ к Планировщику задач, платформе <Brand /> Про, панели CityLens Web и cервису Export API.
2. Перейдите к [Руководству администратора](/citylens/administrator/overview).

## Для водителей {#for-drivers}

Чтобы начать работу с <Brand /> Ситискан:

1. Обратитесь в вашу организацию, чтобы получить доступ к мобильному приложению <Brand /> Ситискан.
2. Установите приложение на вашем мобильном устройстве.
3. Перейдите к [Руководству водителя](/citylens/driver/start).
