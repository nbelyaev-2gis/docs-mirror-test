# Релизы

:::warning

Прекращена поддержка версий SDK, описанных ниже. Используйте [версию 12.0.0 и выше](/android/sdk/releases/latest). Cм. инструкции по [миграции с предыдущих версий](/android/sdk/releases/migration).

:::

## 10.8.0 (2024-11-02)

### 🛠 Исправления

* Исправлено потенциальное зависание в [MapObjectManager](/android/sdk/reference/10.8/class#ru.dgis.sdk.map.map-object-manager) при частом добавлении и удалении объектов через метод [MapObjectManager.removeAndAddObjects](/android/sdk/reference/10.8/class#ru.dgis.sdk.map.map-object-manager/remove-and-add-objects).
* Исправлена ошибка, при которой приложение падало при логировании сетевых запросов.
* Исправлена ошибка, при которой для некоторых регионов не работало отображение дорожных событий на карте.

## 10.7.0 (2024-08-13)

### 🌟 Новые возможности

* [CustomDashboardButton](/android/sdk/reference/10.7/class#ru.dgis.sdk.navigation.custom-dashboard-button) и [DashboardButtonAbsent](/android/sdk/reference/10.7/class#ru.dgis.sdk.navigation.dashboard-button-absent) для настройки кнопки добавления дорожных событий в дашборде навигатора.

## 10.6.0 (2024-08-07)

### 🌟 Новые возможности

* [PackageInfo.finalSizeOnDisk](/android/sdk/reference/10.6/class#ru.dgis.sdk.update.package-info/val-final-size-on-disk) — размер места на диске, которое территория занимает после установки, в байтах.
* [PackageInfo.currentSizeOnDisk](/android/sdk/reference/10.6/class#ru.dgis.sdk.update.package-info/val-current-size-on-disk) — размер места на диске, которое занимают установленные файлы территории на данный момент, в байтах.

### 🛠 Исправления

* Исправлено обновление позиции в дереве кластеризации через [MapObjectManager.withClustering](/android/sdk/reference/10.6/class#ru.dgis.sdk.map.map-object-manager) при изменении позиции маркера.
* Обновлён дизайн копирайта снимка карты, полученного методом [MapView.takeSnapshot](/android/sdk/reference/10.6/class#ru.dgis.sdk.map.map-view/take-snapshot).
* В [NavigationView](/android/sdk/reference/10.6/class#ru.dgis.sdk.navigation.navigation-view) UI-элемент спидометра теперь отображается при навигации по велосипедным маршрутам.
* В [NavigationView](/android/sdk/reference/10.6/class#ru.dgis.sdk.navigation.navigation-view) UI-элемент слежения за позицией теперь не скрывается при пешей навигации.

## 10.5.0 (2024-06-05)

### 🌟 Новые возможности

* В настройках ключа можно задавать каскадный `appId`. Благодаря этой настройке вы можете привязать ключ не к конкретному приложению, а к группе приложений с одинаковой шаблонной частью в идентификаторе.
* Обновлён дизайн копирайта.
* Добавлен [TiltSettings.maxParallelsDeviationDeg](/android/sdk/reference/10.5/class#ru.dgis.sdk.map.tilt-settings) для настройки жеста наклона карты.

### 🛠 Исправления

* Исправлена работа канала [MapView.fpsChannel](/android/sdk/reference/10.5/class#ru.dgis.sdk.map.map-view).
* При отображении маркеров с помощью [Marker](/android/sdk/reference/10.5/class#ru.dgis.sdk.map.marker) текстовые подписи не перекрывают друг друга.
* Исправлено отображение маркера геопозиции, когда перестают поступать данные от датчика направления.

## 10.4.0 (2024-05-20)

### 🌟 Новые возможности

* Добавлен [DGis.initialize](/android/sdk/reference/10.4/class#ru.dgis.sdk.dgis) с параметром для переопределения аудиодрайвера для воспроизведения звуков SDK. Добавлен интерфейс [AudioDriver](/android/sdk/reference/10.4/interface#ru.dgis.sdk.platform.audio-driver) для задания платформенной реализации аудиодрайвера.
* [ZoomFollowSettings.speedRangeToStyleZoomFreeRoamSequence](/android/sdk/reference/10.4/class#ru.dgis.sdk.navigation.zoom-follow-settings) стал сеттером.

### 🛠 Исправления

* Исправлена ошибка, при которой изменение этажа через [IndoorBuilding.activeLevelIndex](/android/sdk/reference/10.4/class#ru.dgis.sdk.map.indoor-building) не влияло на отображение [IndoorControl](/android/sdk/reference/10.4/class#ru.dgis.sdk.map.indoor-control).
* Исправлена обработка жестов, когда карта работает в фоновом режиме.
* В [NavigationView](/android/sdk/reference/10.4/class#ru.dgis.sdk.navigation.navigation-view) источник геопозиции заменяется на источник с контроллером навигатора только при старте навигации.

### ❗ Изменения, влияющие на обратную совместимость

* В объект карты [Map](/android/sdk/reference/10.4/class#ru.dgis.sdk.map.map) теперь можно добавить только один источник текущего местоположения [MyLocationMapObjectSource](/android/sdk/reference/10.4/class#ru.dgis.sdk.map.my-location-map-object-source).

## 10.3.0 (2024-05-06)

### 🌟 Новые возможности

* Теперь информация о пробках в регионах обновляется независимо от обновления конфигурационного файла SDK.
* Теперь при zoom-уровне выше 2 карту нельзя сдвинуть для отображения пустых полос сверху и снизу.
* Добавлен голосовой пакет с арабским языком для навигатора.

### 🛠 Исправления

* Исправлена работа кластеризации через [MapObjectManager.withClustering](/android/sdk/reference/10.3/class#ru.dgis.sdk.map.map-object-manager). Теперь иерархия кластеров перестраивается, если позиция одного из маркеров изменилась через [Marker.position](/android/sdk/reference/10.3/class#ru.dgis.sdk.map.marker).
* Исправлена некорректная обработка тапа в [Marker](/android/sdk/reference/10.3/class#ru.dgis.sdk.map.marker) с Lottie-анимацией.
* [Map.dataLoadingStateChannel](/android/sdk/reference/10.3/class#ru.dgis.sdk.map.map) больше не зависает в состоянии `loading` при режиме слежения за геопозицией.
* Исправлена ошибка, при которой `sources.jar` в релизном пакете генерировался пустым.
* Исправлено отображение дашборда в [NavigationView](/android/sdk/reference/10.3/class#ru.dgis.sdk.navigation.navigation-view) в режиме `FreeRoam`.

## 10.2.0 (2024-03-07)

### 🌟 Новые возможности

* Опции [CircleOptions.dashedStrokeOptions](/android/sdk/reference/10.2/class#ru.dgis.sdk.map.circle-options) и [Circle.dashedStrokeOptions](/android/sdk/reference/10.2/class#ru.dgis.sdk.map.circle) для создания пунктирной окружности или добавления пунктира для уже созданной окружности.
* Метод [RoadEventSource.setHighlighted](/android/sdk/reference/10.2/class#ru.dgis.sdk.map.road-event-source) для выделения дорожных событий с идентификаторами из [RoadEventMapObject.id](/android/sdk/reference/10.2/class#ru.dgis.sdk.map.road-event-map-object).
* Функция [calculateClosestPoint](/android/sdk/reference/10.2/method#ru.dgis.sdk.routing.calculate-closest-point-route-geometry-point-) для получения ближайшей точки на маршруте относительно переданной точки.

### 🛠 Исправления

* Исправлена ошибка в работе `layedId`, который задается в [MapObjectManager](/android/sdk/reference/10.2/class#ru.dgis.sdk.map.map-object-manager): теперь он влияет на отображение кластеризованных маркеров.

### 🗺 Изменения в стилях карты

* Для корректной работы пунктирных окружностей нужно заново выгрузить стили.

## 10.1.0 (2024-01-16)

### 🌟 Новые возможности

* Добавлена структура [DirectoryObject.buildingInfo](/android/sdk/reference/10.1/class#ru.dgis.sdk.directory.directory-object) со справочной информацией о здании: `buildingName`, `purposeName`, `purposeCode` и `buildingLevels`. [DirectoryObject.buildingLevels](/android/sdk/reference/10.1/class#ru.dgis.sdk.directory.directory-object) помечено `deprecated` и будет удалено в будущих мажорных релизах.
* Добавлен [DirectoryObject.address.fiasCode](/android/sdk/reference/10.1/class#ru.dgis.sdk.directory.directory-object) — ФИАС-код объекта справочника.
* В [DirectoryObject.address.components](/android/sdk/reference/10.1/class#ru.dgis.sdk.directory.directory-object) в структурах [AddressComponent.streetAddress](/android/sdk/reference/10.1/class#ru.dgis.sdk.directory.address-component) и [AddressComponent.number](/android/sdk/reference/10.1/class#ru.dgis.sdk.directory.address-component) добавлено поле `fiasCode` — ФИАС-код улицы.

## 10.0.0 (2023-12-29)

:::note[Обновление с версии 7.x]

Перед обновлением см. [инструкцию по миграции](/android/sdk/releases/migration#from-7-to-10).

:::

### 🌟 Новые возможности

* Поддержан Vulkan. Если Vulkan недоступен на девайсе, то используется OpenGL версии 3. По умолчанию включен OpenGL, так как поддержка Vulkan находится в процессе доработки и является экспериментальной фичей.
* Иммерсивные деревья на карте отображаются по умолчанию.
* Поддержана новая модель для отображения маркера текущего местоположения. Если необходимо переопределить иконку текущего местоположения, нужно передать [MyLocationMapObjectMarkerType.SVG\_ICON](/android/sdk/reference/10.0/enumeration#ru.dgis.sdk.map.my-location-map-object-marker-type) в конструктор [MyLocationMapObjectSource](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.my-location-map-object-source), а в стилях отредактировать соответствующий слой.
* Добавлен метод [BaseCamera.changePosition](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.base-camera) для изменения только части параметров позиции камеры. Метод принимает новую структуру [CameraPositionChange](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.camera-position-change).
* Добавлен метод [Camera.move](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.camera) для запуска анимированного перемещения карты с возможностью сброса слежения за наклоном.
* Добавлен параметр `styleZoomToTiltRelation` для функций [calcPosition](/android/sdk/reference/10.0/method#ru.dgis.sdk.map.calc-position-camera-geometry-style-zoom-to-tilt-relation-screen-area-tilt-bearing-size-) и [zoomOutToFit](/android/sdk/reference/10.0/method#ru.dgis.sdk.map.zoom-out-to-fit-camera-geometry-style-zoom-to-tilt-relation-size-): функция зависимости угла наклона камеры от стилевого zoom-уровня.
* Добавлена новая функция [createDefaultStyleZoomToTiltRelation](/android/sdk/reference/10.0/method#ru.dgis.sdk.map.create-default-style-zoom-to-tilt-relation--) для получения зависимости угла наклона камеры от стилевого zoom-уровня для стандартного контроллера слежения за наклоном.
* Добавлен enum [BearingSource](/android/sdk/reference/10.0/enumeration#ru.dgis.sdk.map.bearing-source) — источник направления.
* В [DGis.initialize](/android/sdk/reference/10.0/class#ru.dgis.sdk.DGis) добавлена возможность переопределения сетевого клиента с помощью интерфейса [HttpClient](/android/sdk/reference/10.0/interface#ru.dgis.sdk.platform.http-client).
* Добавлены [MarkerOptions.iconAnimationMode](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.marker-options) и [Marker.iconAnimationMode](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.marker) для возможности задания режима анимации в маркерах. Режим может быть обычным или зацикленным.
* Добавлены [SearchResult.actionWidgets](/android/sdk/reference/10.0/class#ru.dgis.sdk.directory.search-result) и [SearchResult.mainWidgets](/android/sdk/reference/10.0/class#ru.dgis.sdk.directory.search-result) для динамической фильтрации уже полученной поисковой выдачи. Добавлен базовый класс [Widget](/android/sdk/reference/10.0/class#ru.dgis.sdk.directory.widget), представляющий виджет для фильтрации поисковой выдачи.
* В [LocaleManager](/android/sdk/reference/10.0/class#ru.dgis.sdk.platform.locale-manager) добавлены методы `locales` и `systemLocales` и соответствующие каналы для получения всех локалей приложения и системных локалей.
* Добавлена функция [createDefaultMaxTiltRestriction](/android/sdk/reference/10.0/method#ru.dgis.sdk.map.create-default-max-tilt-restriction--) для получения дефолтной зависимости ограничения угла наклона от масштаба. По умолчанию эта зависимость устанавливается при создании карты — [BaseCamera.maxTiltRestriction](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.base-camera).
* Добавлен метод [IndoorManager.setIndoorState](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.indoor-manager) для управления отображением этажных планов на карте.
* Добавлен интерфейс [LocationService](/android/sdk/reference/10.0/interface#ru.dgis.sdk.positioning.location-service) для удобного получения последнего значения геопозиции пользователя.
* Добавлен новый UI-элемент [SearchLayout](/android/sdk/reference/10.0/class#ru.dgis.sdk.directory.search-layout), реализующий логику поисковой строки: поиск, предложения, отображение поисковой выдачи.
* Добавлено поле [PackedMapState.cameraBehaviour](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.packed-map-state) для сериализации режима слежения камеры.
* В [PackageInfo](/android/sdk/reference/10.0/class#ru.dgis.sdk.update.package-info) добавлено поле `error` — ошибка, произошедшая при попытке загрузки и/или установки пакета.
* Добавлено поле [NavigationManager.alternativeRouteSelector](/android/sdk/reference/10.0/class#ru.dgis.sdk.navigation.navigation-manager) для выбора альтернативного маршрута.
* Добавлено поле [Route.bicycleLanes](/android/sdk/reference/10.0/class#ru.dgis.sdk.routing.route) — признак наличия выделенной велополосы.
* Добавлено значение [RoutePointKind.BICYCLE\_LANE](/android/sdk/reference/10.0/enumeration#ru.dgis.sdk.map.route-point-kind) — начало и конец велополосы.
* Добавлен флаг [PedestrianRouteSearchOptions.avoidUnderpassesAndOverpasses](/android/sdk/reference/10.0/class#ru.dgis.sdk.routing.pedestrian-route-search-options) — избегать подземных и надземных переходов.
* Добавлен метод [RouteMapObjectSource.replaceObject](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.route-map-object-source) для замены одного объекта маршрута в источнике на другой.
* Добавлено поле [NavigationFollowController.mapLocationController](/android/sdk/reference/10.0/class#ru.dgis.sdk.navigation.navigation-follow-controller). [MapLocationController](/android/sdk/reference/10.0/class#ru.dgis.sdk.navigation.map-location-controller) — контроллер для управления позиционированием карты в навигаторе.
* Добавлено поле [NavigationFollowController.cameraBehaviour](/android/sdk/reference/10.0/class#ru.dgis.sdk.navigation.navigation-follow-controller) для возможности менять режим слежения карты за маркером геопозиции.
* Добавлено поле [ZoomFollowSettings.speedRangeToStyleZoomFreeRoamSequence](/android/sdk/reference/10.0/class#ru.dgis.sdk.navigation.zoom-follow-settings) — последовательность интервалов скоростей и соответствующих им масштабов при ведении в режиме FreeRoam.

### ❗ Изменения, влияющие на обратную совместимость

* Изменена работа с [FollowController](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.follow-controller):

  * Удалены методы `createRawMyLocationController()` и `createSmoothMyLocationController()`. Добавлен конструктор [MyLocationController](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.my-location-controller) для создания контроллера перемещения маркера геопозиции.
  * Удалены значения `FollowBearing.SATELLITE` и `FollowBearing.MAGNETIC`, т. к. теперь не нужно указывать источник направления. Добавлено значение [FollowBearing.ON](/android/sdk/reference/10.0/enumeration#ru.dgis.sdk.map.follow-bearing) для включения слежения — источник выбирается автоматически.
  * Удалено поле `FollowPosition.tilt`. Слежение за наклоном теперь нужно включать отдельно. Добавлен параметр `tilt` в конструктор [CameraBehaviour](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.camera-behaviour) для слежения за наклоном.
  * Удалены значения `FollowValue.SATELLITE_BEARING`, `FollowValue.MAGNETIC_BEARING` и `FollowValue.ACCURACY`. Вместо них добавлено значение [FollowValue.BEARING](/android/sdk/reference/10.0/enumeration#ru.dgis.sdk.map.follow-value) с автоматическим выбором источника.
  * Удалены методы `satelliteBearing()`, `magneticBearing()` и `accuracy()` из интерфейса [CustomFollowController](/android/sdk/reference/10.0/interface#ru.dgis.sdk.map.custom-follow-controller). Вместо них добавлен метод [CustomFollowController.bearing()](/android/sdk/reference/10.0/interface#ru.dgis.sdk.map.custom-follow-controller).
  * Удалены методы `setClock()`, `satelliteBearing()`, `magneticBearing()` и `accuracy()` из [FollowController](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.follow-controller). Вместо них добавлены методы `onAddedToMap()`, `onRemovedFromMap()` и `bearing()`.
  * Удалён канал `FollowController.nextTimestampChannel`. Вместо него добавлен [FollowController.changedChannel](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.follow-controller).
  * Изменена сигнатура метода [FollowController.updateValues()](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.follow-controller).
  * Удалён контроллер `PlatformLocationFollowController`. Вместо него добавлены контроллеры [BearingFollowController](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.bearing-follow-controller) и [CoordinatesFollowController](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.coordinates-follow-controller) для слежения за направлением карты и за координатами позиции карты, соответственно.
  * Изменён конструктор контроллера [StyleZoomFollowController](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.style-zoom-follow-controller). Удалён метод `StyleZoomFollowController.setAnimationDuration()`.
  * Изменён конструктор контроллера [TiltFollowController](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.tilt-follow-controller). Удалены методы `TiltFollowController.setTilt()` и `TiltFollowController.setAnimationDuration()`.
  * Изменены параметры метода [FollowController.setThresholds()](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.follow-controller) на типизированные.
  * Удалён класс `FollowControllerClock`. Тип параметра в методе [FollowController.setClock()](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.follow-controller) заменён на timestamp.
  * Удалён `TransitionType`. Конструктор [MyLocationControlModel](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.my-location-control-model) теперь не принимает `transitionType`. Удалено поле `CameraBehaviour.transitionType`. Удалён параметр `transitionType` из конструктора [CameraBehaviour](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.camera-behaviour).
  * Удалён `MyLocationDirectionBehaviour`. Конструктор источника геопозиции [MyLocationMapObjectSource](/android/sdk/reference/10.0/class#ru.dgis.sdk.map.my-location-map-object-source) теперь не принимает `directionBehaviour`. Удалён метод `MyLocationMapObjectSource.setDirectionBehaviour()`.

* Удалено поле `BaseCamera.styleZoomToTiltRelation`.

* Класс `MagneticHeadingProvider` переименован в `HeadingProvider`.

* Метод [LocaleManager.overrideLocales()](/android/sdk/reference/10.0/class#ru.dgis.sdk.platform.locale-manager) теперь принимает список структур `Locale` с отдельными полями `language` и `region`.

* Из Map-сборки удалены [PackageManager](/android/sdk/reference/10.0/class#ru.dgis.sdk.update.package-manager) и [Package](/android/sdk/reference/10.0/class#ru.dgis.sdk.update.package), что уменьшило размер продукта.

* Метод [Voice.playWelcome()](/android/sdk/reference/10.0/class#ru.dgis.sdk.navigation.voice) теперь возвращает [Future](/android/sdk/reference/10.0/interface#ru.dgis.sdk.future), который становится готовым, когда проигрывание сэмпла завершается.

* Удалено поле `RoadSubtype.STAIRWAY`. Вместо него добавлены поля [RoadSubtype.STAIRWAY_UP](/android/sdk/reference/10.0/enumeration#ru.dgis.sdk.routing.road-subtype) и [RoadSubtype.STAIRWAY_DOWN](/android/sdk/reference/10.0/enumeration#ru.dgis.sdk.routing.road-subtype) для использования лестниц с явным направлением.

* Удалено поле `NavigationFollowController.mapTiltController`.
