# Релизы

## 13.6.0 (2026-06-22)

### 🌟 Новые возможности

* Новый класс [MapController](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-controller):

  * [MapController](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-controller) – класс для управления картой, объединяющий [Map](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map), [MapRenderer](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-renderer) и [MapGestureRecognizer](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-gesture-recognizer).
  * [MapControllerOptions](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-controller-options) – класс с параметрами создания [MapController](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-controller).
  * [MapController.create](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-controller/create) – статический метод для асинхронного создания [MapController](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-controller) на основе [Context](/android/sdk/reference/stable/class#ru.dgis.sdk.context) и [MapControllerOptions](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-controller-options).
  * [MapController.map](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-controller/map) – свойство для получения объекта [Map](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map) из [MapController](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-controller).
  * [MapController.renderer](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-controller/renderer) – свойство для получения объекта [MapRenderer](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-renderer) из [MapController](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-controller).
  * [MapController.gestureRecognizer](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-controller/gesture-recognizer) – свойство для получения объекта [MapGestureRecognizer](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-gesture-recognizer) из [MapController](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map-controller).

* [CameraZoomRestrictions.minZoom](/android/sdk/reference/stable/class#ru.dgis.sdk.map.camera-zoom-restrictions/val-min-zoom) – значение по умолчанию изменено с `0` на `2`.

### 🛠 Исправления

* Механизм восстановления автонаклона доработан так, чтобы он работал независимо от других перелётов.
* Исправлен [WorkStatus](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.work-status) для расписаний, работающих почти круглосуточно.
* Исправлена ошибка, вызывавшая сбой приложения при увеличении масштаба карты на Samsung Tab A8 с Android 14.
* Исправлена ошибка, из-за которой не работали повторные добавления и удаления дорожных событий через [RoadEventManager](/android/sdk/reference/stable/class#ru.dgis.sdk.road-events.road-event-manager).
* Добавлена корректная логика расчёта и заполнения полей `ageMs` и `lastSeenMs` для сотовых вышек в модуле Radar с учётом текущего времени и версии Android API; также исправлена очистка устаревших значений.

* [PackedSearchQuery.geometryRestriction](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.packed-search-query/val-geometry-restriction) – свойство для получения геометрии, ограничивающей область поиска.
* [PackedSearchQuery.territoryOfInterest](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.packed-search-query/val-territory-of-interest) – свойство для нестрогого ограничения области поиска объектов: если объект не попадает в область, он не отбрасывается, но его ранг в результатах поиска понижается.
* [PolygonGeometry.elevation](/android/sdk/reference/stable/class#ru.dgis.sdk.geometry.polygon-geometry/val-elevation) – свойство для задания высоты полигона над поверхностью.

* [Polyline.elevation](/android/sdk/reference/stable/class#ru.dgis.sdk.map.polyline/var-elevation) – свойство для задания высоты полилинии над поверхностью.
* [PolylineOptions.elevation](/android/sdk/reference/stable/class#ru.dgis.sdk.map.polyline-options/val-elevation) – свойство для задания высоты полилинии над поверхностью в параметрах полилинии.
* [Polygon.elevation](/android/sdk/reference/stable/class#ru.dgis.sdk.map.polygon/var-elevation) – свойство для задания высоты полигона над поверхностью.
* [PolygonOptions.elevation](/android/sdk/reference/stable/class#ru.dgis.sdk.map.polygon-options/val-elevation) – свойство для задания высоты полигона над поверхностью в параметрах полигона.
* [DashedPolygonOptions](/android/sdk/reference/stable/class#ru.dgis.sdk.map.dashed-polygon-options) – класс для параметров пунктирного контура полигона.
* [Polygon.dashedPolygonOptions](/android/sdk/reference/stable/class#ru.dgis.sdk.map.polygon/var-dashed-polygon-options) – свойство для параметров пунктирного контура полигона.

* [SharedRouteData](/android/sdk/reference/stable/class#ru.dgis.sdk.routing.shared-route-data) – класс с информацией о заранее построенном маршруте и параметрах его поиска.

## 13.3.1 (2026-03-20)

### 🌟 Новые возможности

* Теперь вы можете создать и скачать файл ключа в Менеджере Платформы: см. [Получение ключей доступа](/android/sdk/start#get-api-key).
* В манифесте ключа, передаваемого в метод [DGis.initialize()](/android/sdk/reference/stable/class#ru.dgis.sdk.dgis/initialize), теперь необязательно наличие заполненного `restrictions/app_id`, если не планируется работа с офлайн-данными (`offlineRouting`, `offlineMap`, `offlineDirectory`). В противном случае наличие `restrictions/app_id` обязательно.
* При передаче пустого идентификатора этажа в параметр [IndoorControlModel.activeLevelIndex](/android/sdk/reference/stable/class#ru.dgis.sdk.map.indoor-control-model/var-active-level-index) этажный план здания будет скрываться.
* [Closed](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.closed) — класс, представляющий статус `Закрыто` у объекта справочника с информацией о причине закрытия и времени до открытия.
* [ClosedReason](/android/sdk/reference/stable/enumeration#ru.dgis.sdk.directory.closed-reason) — причины установки статуса `Закрыто` у объекта справочника.
* [ClosingHours](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.closing-hours) — класс с информацией о нерабочих интервалах и причине закрытия объекта справочника.
* [ClosingSoon](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.closing-soon) — класс, представляющий статус `Скоро закроется` у объекта справочника с информацией о причине и времени закрытия.
* [OpenStatus](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.open-status) — класс, описывающий детализированный статус работы объекта справочника в данный момент (`Открыто`, `Скоро откроется`, `Скоро закроется`, `Закрыто`).
* [Opened](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.opened) — класс, представляющий статус `Открыто` у объекта справочника с информацией о времени до закрытия.
* [OpeningSoon](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.opening-soon) — класс, представляющий статус `Скоро откроется` у объекта справочника с информацией о времени до открытия.
* [OpeningHours.weekClosingHours](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.opening-hours/val-week-closing-hours) — свойство для хранения нерабочих интервалов объекта справочника по каждому дню недели.
* [OpeningHours.isOpen24x7](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.opening-hours/val-is-open24x7) — свойство для признака круглосуточной работы объекта справочника.
* [WorkStatus.openStatus](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.work-status/val-open-status) — свойство с детальным статусом `Открыто` у объекта справочника.
* [WorkStatus.openStatusHint](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.work-status/val-open-status-hint) — свойство текстовой подсказки о статусе `Открыто` у объекта справочника.
* [WorkStatus.scheduleHint](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.work-status/val-schedule-hint) — свойство текстовой подсказки о расписании работы объекта справочника.
* [WorkStatus.breakHint](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.work-status/val-break-hint) — свойство текстовой подсказки о текущем или ближайшем перерыве в работе объекта справочника.
* [Reviews.countWithStars](/android/sdk/reference/stable/class#ru.dgis.sdk.directory.reviews/val-count-with-stars) — свойство для общего количества оценок объекта справочника, включая отзывы без текста.

* Увеличен уровень масштабирования для растровых тайлов, добавляемых через источник [RasterTileSource](/android/sdk/reference/stable/class#ru.dgis.sdk.map.raster-tile-source), до 16.

* Исправлена работа канала готовности карты [Map.dataLoadingStateChannel](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map/val-data-loading-state-channel) в иммерсивном режиме (когда для [Map.graphicsPreset](/android/sdk/reference/stable/class#ru.dgis.sdk.map.map/var-graphics-preset) установлено значение [GraphicsPreset.IMMERSIVE](/android/sdk/reference/stable/enumeration#ru.dgis.sdk.map.graphics-preset/immersive)).

* Исправлены рывки и рассинхронизация отображения маркера геопозиции и линии маршрута при навигации по длительным маршрутам между регионами.
* Исправлено отображение километров в баблах маршрутов в арабской локали.
* Обновлены значения по умолчанию настроек [ZoomFollowSettings](/android/sdk/reference/stable/class#ru.dgis.sdk.navigation.zoom-follow-settings) в навигаторе для корректного отображения видимого пространства на карте.
* Исправлены UX-ошибки в [RouteEditorComposable](https://2gis.github.io/android-compose-docs/compose-navigation/ru.dgis.sdk.compose.routeeditor/-route-editor-composable.html): например мигание UI-элемента для выбора типа маршрута.
* Теперь для получения маршрутов на общественном транспорте через [PublicTransportRouteSearchOptions](/android/sdk/reference/stable/class#ru.dgis.sdk.routing.public-transport-route-search-options) по умолчанию используется [Routing API](/api/navigation/routing/overview).

* Не озвучиваются пешеходные препятствия и подтипы дорог для непешеходных маршрутов.
