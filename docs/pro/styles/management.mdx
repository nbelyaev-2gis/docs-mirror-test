# Управление стилем карты

По умолчанию для всех дашбордов и сцен используется общий стиль, заданный в настройках карты. Вы можете как изменить общий стиль карты, так и установить разные стили для сцен в рамках одного дашборда.

## Изменение общего стиля {#common-style}

![Общий стиль карты](/img/pro/style-management-project.png)

При создании новых дашбордов и сцен используется общий стиль карты. Если вы измените общий стиль карты, то он будет использоваться только при создании новых дашбордов и сцен. В созданных сценах стили останутся прежними.

1. Перейдите на вкладку **Главная**.
2. Нажмите значок ![Менеджер стилей](/img/pro/style-manager-icon.svg) в правом нижнем углу карты.
3. В окне **Настройки карты** выберите стиль карты:

   * **Встроенные темы**: варианты цветового оформления карты <Brand />. Доступные темы:
     * **Автоматически** (по умолчанию): тема карты будет меняться в зависимости от темы, установленной в браузере.
     * **Дневная**.
     * **Ночная**.
     * **Приглушённые цвета**.
     * **Оттенки серого**.
   * **Карты из внешних источников**: карты с дополнительными данными (например, спутниковые снимки), которые вы можете наложить поверх базовой темы.

     Вы также можете [подключить другие карты из внешних источников](/pro/styles/external) и использовать их в качестве стилей.

## Изменение стиля для сцены {#scene-style}

Вы можете использовать разные стили карты в сценах одного дашборда:

![Управление стилем карты для первой сцены](/img/pro/style-management-first-scene.png)

![Управление стилем карты для второй сцены](/img/pro/style-management-second-scene.png)

Чтобы изменить стиль карты для сцены:

1. Перейдите на вкладку **Дашборды** и откройте дашборд.
2. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.
3. Нажмите значок ![Менеджер стилей](/img/pro/style-manager-icon.svg) в правом нижнем углу карты.
4. В окне **Настройки карты для сцены** выберите стиль карты:

   * **Встроенные темы**: варианты цветового оформления карты <Brand />. Доступные темы:
     * **Автоматически** (по умолчанию): тема карты будет меняться в зависимости от темы, установленной в браузере.
     * **Дневная**.
     * **Ночная**.
     * **Приглушённые цвета**.
     * **Оттенки серого**.
   * **Карты из внешних источников**: карты с дополнительными данными (например, спутниковые снимки), которые вы можете наложить поверх базовой темы.

     Вы также можете [подключить другие карты из внешних источников](/pro/styles/external) и использовать их в качестве стилей.
