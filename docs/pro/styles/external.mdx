# Подключение внешних карт

Вы можете подключить карту из внешнего источника (например, спутниковые снимки) в форматах Mapbox, TMS, WMS и WMTS. Новые данные накладываются на основную карту <Brand /> и отображаются как отдельный стиль карты.

## Подключение внешней карты {#adding-an-external-map}

Вы можете добавить не более 15 источников внешних карт.

![Внешняя карта](/img/pro/external-map.png)

1. Перейдите на вкладку **Главная**.

2. Нажмите значок ![Менеджер стилей](/img/pro/style-manager-icon.svg) в правом нижнем углу карты.

3. Нажмите **Добавить источник**.

4. Укажите данные внешнего источника:

   <img src="/img/pro/external-map-data.png" alt="Настройки данных карты" width="600" />

   * **Ссылка** (обязательный параметр): URL источника для получения карты.

   * **Обложка**: изображение, которое будет отображаться для подключенной карты в окне **Настройки карты**. Можно загрузить новое изображение или оставить стандартное.

   * **Тип источника** (обязательный параметр): характеристика источника данных по ссылке. Доступные варианты:

     * **WMS**: стандарт динамического получения географических данных в виде изображений с привязкой к местности.
     * **WMTS**: стандарт получения кешированных изображений.
     * **TMS**: формат для отображения предварительно отрисованных тайлов карты через интернет.
     * **Mapbox**: поставщик подложек для карт. Вместо URL для этого типа требуется указать токен доступа из аккаунта Mapbox.

   * **Название стиля** (обязательный параметр): имя нового стиля.

   * **Правообладатель**: имя правообладателя. Добавьте, если внешний источник требует его указать. Текст будет отображаться в левом нижнем углу карты.

5. Нажмите **Далее**.

6. Выберите один из вложенных слоёв для отображения. Список слоёв зависит от внешнего источника.

   <img src="/img/pro/external-map-layer.png" alt="Выбор вложенного слоя" width="550" />

7. Нажмите **Добавить**.

## Операции с внешними картами {#operations-with-external-maps}

### Редактирование карты

1. Перейдите на вкладку **Главная**.
2. Нажмите значок ![Менеджер стилей](/img/pro/style-manager-icon.svg) в правом нижнем углу карты.
3. Наведите курсор на обложку внешней карты, нажмите значок ![Настройки карты](/img/pro/style-manager-wheel-icon.svg) и выберите **Редактировать**.
4. Внесите изменения в данные.

### Наложение карты

![Наложенная карта](/img/pro/external-map-overlay.png)

Вы можете отобразить две карты из внешних источников одновременно, наложив их друг на друга.

1. Перейдите на вкладку **Главная**.
2. Нажмите значок ![Менеджер стилей](/img/pro/style-manager-icon.svg) в правом нижнем углу карты.
3. Наведите курсор на обложку внешней карты, которая будет отображаться сверху, нажмите значок ![Настройки карты](/img/pro/style-manager-wheel-icon.svg) и выберите **Перекрыть карту**.
4. Нажмите на обложку другой внешней карты, которая будет отображаться снизу.

Обложки обеих выбранных карт получат зелёную обводку, а на обложке верхней карты появится значок ![Перекрыть карту](/img/pro/external-map-overlay-icon.svg).

### Снятие наложения карты

1. Перейдите на вкладку **Главная**.
2. Нажмите значок ![Менеджер стилей](/img/pro/style-manager-icon.svg) в правом нижнем углу карты.
3. Нажмите на обложку внешней карты.

### Удаление карты

1. Перейдите на вкладку **Главная**.
2. Нажмите значок ![Менеджер стилей](/img/pro/style-manager-icon.svg) в правом нижнем углу карты.
3. Наведите курсор на обложку внешней карты, нажмите значок ![Настройки карты](/img/pro/style-manager-wheel-icon.svg) и выберите **Удалить**.
4. Чтобы подтвердить удаление, нажмите **Удалить**.
