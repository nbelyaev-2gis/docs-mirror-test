# Линейка

Вы можете измерить расстояние между произвольными точками на карте при помощи встроенной линейки.

1. Чтобы включить режим линейки, нажмите значок ![Линейка](/img/pro/ruler-icon.svg) в правом нижнем углу карты.
2. Чтобы поставить точки, кликайте по карте.

   Вы можете строить ломаные линии из нескольких точек, а также добавлять точки на уже нарисованную линию.

Рядом с каждой точкой отображается расстояние от стартовой точки.

![Пример полилинии](/img/pro/ruler-example.png)

<br />

Чтобы удалить точку:

1. Наведите курсор на нужную точку.
2. Нажмите значок ![Крестик](/img/pro/cross-icon.svg).

Чтобы стереть все нарисованные линии и выключить режим линейки, нажмите значок ![Линейка](/img/pro/ruler-icon.svg).
