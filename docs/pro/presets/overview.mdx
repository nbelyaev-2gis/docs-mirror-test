# Обзор

**Сценарии аналитики** — это готовые сцены с набором слоёв на основе системных данных с настроенной визуализацией и виджетами.

## Доступные сценарии {#scenarios}

:::note[Доступность в демо-тарифе]

Если у вас [демо-тариф](/pro/prices), вы можете использовать сценарии **Конкуренты и спрос** и **Инфраструктура города**. Некоторые категории и наборы данных могут быть недоступны.

:::

* [Конкуренты и спрос](#competitors): позволяет находить зоны с высоким спросом и низкой конкуренцией и определять лучшие локации для открытия бизнеса.
* [Инфраструктура города](#infrastructure): помогает оценить плотность населения, транспортную доступность и типы застройки в разных районах города.
* [Демографический портрет населения](#population): позволяет исследовать половозрастную структуру населения и определить концентрацию целевой аудитории в разных частях города.
* [Центры притяжения коммерции](#commercial): показывает ключевые точки притяжения деловой и торговой активности в городе.
* [Транспортная доступность](#transport): помогает анализировать транспортную и пешеходную доступность территорий.

:::tip[Аналитика локаций]

Вы можете проанализировать потенциал мест для открытия малого и среднего бизнеса с помощью интерактивного дашборда [Аналитика локаций](/pro/presets/insights), не регистрируясь в сервисе <Brand /> Про.

:::

## Данные в сценах {#data}

Сценарий можно использовать, только если у вас есть доступ ко всем наборам данных, которые в нём используются (кроме сценариев **Конкуренты и спрос** и **Инфраструктура города** для демо-тарифа). Созданные сцены будут содержать слои и виджеты для выбранных территорий и категорий данных.

### Конкуренты и спрос {#competitors}

* Слой **Открытие фирм** (недоступен в демо-тарифе): выборка из набора данных [Динамика организаций](/pro/data/built-in/dynamics). Способ визуализации — [Точка](/pro/data/visualization/point).
* Слой **Фирмы**: выборка из набора данных [Фирмы](/pro/data/built-in/firms) по указанным категориям. Способ визуализации — [Точка](/pro/data/visualization/point).
* Слой **Закрытие фирм** (недоступен в демо-тарифе): выборка из набора данных [Динамика организаций](/pro/data/built-in/dynamics). Способ визуализации — [Точка](/pro/data/visualization/point).
* Слой **Карта спроса**: выборка из набора данных [Карта спроса](/pro/data/built-in/demand) по указанным категориям. Способ визуализации — [Соты](/pro/data/visualization/hexbin).
* Виджет **Рейтинг конкурентов**: [столбчатая диаграмма](/pro/dashboards/widgets/column-chart) с рейтингом конкурентов из справочника <Brand />.
* Виджет **Отзывы конкурентов**: [гистограмма](/pro/dashboards/widgets/histogram) с распределением количества отзывов у конкурентов.
* Виджет **Спрос за год**: [график с областями](/pro/dashboards/widgets/area-chart) с распределением спроса за последние 12 месяцев.

![Сценарий Конкуренты и спрос](/img/pro/presets-competitors.png)

### Инфраструктура города {#infrastructure}

* Слой **Здания**: выборка из набора данных [Здания](/pro/data/built-in/buildings) (в демо-тарифе доступны только нежилые здания). Способ визуализации — [Здания](/pro/data/visualization/buildings).
* Слой **Проживающее население** (недоступен в демо-тарифе): выборка из набора данных [Здания](/pro/data/built-in/buildings). Способ визуализации — [H3](/pro/data/visualization/h3).
* Слой **Автомобильный трафик (обобщённый)** (недоступен в демо-тарифе): выборка из набора данных [Автомобильный трафик (обобщённый)](/pro/data/built-in/cars). Способ визуализации — [H3](/pro/data/visualization/h3).
* Слой **Работающее население** (недоступен в демо-тарифе): выборка из набора данных [Здания](/pro/data/built-in/buildings). Способ визуализации — [H3](/pro/data/visualization/h3).
* Слой **Остановки общественного транспорта**: выборка из набора данных [Остановки общественного транспорта](/pro/data/built-in/stops). Способ визуализации — [Точка](/pro/data/visualization/point).
* Слой **Парковки** (недоступен в демо-тарифе): выборка из набора данных [Парковки](/pro/data/built-in/parkings). Способ визуализации — [Точка](/pro/data/visualization/point).
* Виджет **Топ-10 назначений зданий**: [круговая диаграмма](/pro/dashboards/widgets/pie-chart) с распределением зданий по назначению.
* Виджет **Этажность зданий**: [столбчатая диаграмма](/pro/dashboards/widgets/column-chart) с распределением зданий по количеству этажей.

## Начало работы {#getting-started}

Чтобы создать сцену для сценария аналитики:

1. Перейдите на вкладку **Главная**.
2. В блоке **Сценарии аналитики** выберите [сценарий](#scenarios).
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. Если вы выбрали сценарий **Конкуренты и спрос**, выберите категории организаций. Можно выбрать несколько категорий и подкатегорий.
5. Нажмите **Продолжить**.
6. Добавьте сцену в дашборд. Если у вас нет созданных дашбордов, будет создан дашборд с одной сценой. Если создан хотя бы один дашборд, в диалоговом окне укажите, куда добавить сцену:

   * Выберите в списке существующий дашборд и нажмите **Добавить**.
   * Чтобы создать новый дашборд и добавить в него сцену, нажмите **Новый дашборд**, введите его имя и нажмите **Добавить**.

После создания сцены вы можете [изменить выборку данных](/pro/data/manage/modify), настроить параметры [визуализации](/pro/data/visualization/overview) для любого слоя и создать другие [виджеты](/pro/dashboards/widgets/overview).
