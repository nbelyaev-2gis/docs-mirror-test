# Аналитика локаций для малого и среднего бизнеса

Вы можете проанализировать локацию для своего бизнеса с помощью интерактивного дашборда, купив доступ на [отдельной странице](https://pro.2gis.ru/l/insights?utm_source=2gis_documentation\&utm_medium=post\&utm_campaign=validation).

Интерактивный дашборд **Аналитика локаций** позволяет оценить потенциал мест для открытия малого и среднего бизнеса, не регистрируясь в сервисе <Brand /> Про. Дашборд помогает найти место для открытия бизнеса или нового филиала в сфере общественного питания для выбранной территории, а также проанализировать конкурентов, оценить трафик и целевую аудиторию.

Пример готового дашборда:

![Аналитика локаций](/img/pro/presets-insights-competitors.png)

## Сценарии использования {#usage-scenarios}

Дашборд подходит для владельцев и управляющих кафе, ресторанов, баров, кофеен и других заведений общественного питания. Помогает решать различные задачи:

* **Анализ конкурентной среды**. Посчитать, сколько аналогичных заведений находится на выбранной территории и какая у них специализация.
* **Потенциал локации**. Определить плотность населения и покупательскую способность.
* **Трафик**. Оценить поток пешеходов и автомобилей рядом с потенциальной точкой.
* **Демография и интересы аудитории**. Определить потенциальных клиентов на выбранной территории.

## Данные на дашборде {#data-on-the-dashboard}

Дашборд содержит сцены — наборы слоёв на основе системных данных с настроенной визуализацией и виджетами.

Перед покупкой дашборда вы можете протестировать готовый демо-дашборд. Откройте дашборд на [странице для заказа дашборда](https://pro.2gis.ru/l/insights) (ссылка `демо-дашборд`) и [переключайте сцены](/pro/dashboards/scenes/manage), чтобы посмотреть, как можно работать с данными.

Дашборд, который вы приобретёте, будет содержать такие же компоненты (сцены, слои, виджеты), как в демо-дашборде, но только для выбранной территории и выбранных категорий общественного питания:

* Сцена **Конкуренция и спрос**:

  * Слой **Конкуренты**: выборка из набора данных [Фирмы](/pro/data/built-in/firms) по указанным категориям. Способ визуализации — [Точка](/pro/data/visualization/point).
  * Слой **Плотность конкурентов**: выборка из набора данных [Фирмы](/pro/data/built-in/firms) по указанным категориям. Способ визуализации — [H3](/pro/data/visualization/h3).
  * Слой **Бизнес-активность**: выборка из набора данных [Фирмы](/pro/data/built-in/firms) по всем фирмам. Способ визуализации — [H3](/pro/data/visualization/h3).
  * Слой **Спрос на услуги**: выборка из набора данных [Карта спроса](/pro/data/built-in/demand) по указанным категориям. Способ визуализации — [H3](/pro/data/visualization/h3).
  * Виджет **Рейтинг конкурентов**: [столбчатая диаграмма](/pro/dashboards/widgets/column-chart) с рейтингом конкурентов из справочника <Brand />.
  * Виджет **Спрос по категориям**: [линейчатая диаграмма](/pro/dashboards/widgets/bar-chart) со спросом на заведения в разных категориях общественного питания.
  * Виджет **Динамика спроса**: [столбчатая диаграмма](/pro/dashboards/widgets/column-chart) с изменением спроса на заведения общественного питания за последние 18 месяцев.

    ![Сцена Конкуренция и спрос](/img/pro/presets-insights-competitors.png)

* Сцена **Население**:

  * Слои **Проживающее население**: несколько слоёв с выборками из набора данных [Половозрастной состав населения](/pro/data/built-in/sex-age-composition) с разными возрастными категориями. Способ визуализации — [H3](/pro/data/visualization/h3).
  * Слой **Работающее население**: выборка из набора данных [Население](/pro/data/built-in/population) с общим количеством работающего населения. Способ визуализации — [H3](/pro/data/visualization/h3).
  * Слой **Количество квартир в жилых зданиях**: выборка из набора данных [Здания](/pro/data/built-in/buildings). Способ визуализации — [Здания](/pro/data/visualization/buildings).
  * Виджет **Половой состав населения**: [круговая диаграмма](/pro/dashboards/widgets/pie-chart) с распределением населения по полам.
  * Виджет **Возрастной состав населения**: [столбчатая диаграмма](/pro/dashboards/widgets/column-chart) с распределением проживающего населения по возрастным группам.
  * Виджет **Количество квартир**: [гистограмма](/pro/dashboards/widgets/histogram) с распределением жилых зданий по количеству квартир.

    ![Сцена Население](/img/pro/presets-insights-population.png)

* Сцена **Трафик**:

  * Слои **Станции метро** и **Остановки общественного транспорта**: выборки из набора данных [Остановки общественного транспорта](/pro/data/built-in/stops). Способ визуализации — [Точка](/pro/data/visualization/point).
  * Слои **Пешеходный трафик**: выборки из набора данных [Пешеходный трафик](/pro/data/built-in/pedestrians), отфильтрованные по будним и выходным дням. Способ визуализации — [Линия](/pro/data/visualization/line).
  * Слои **Автомобильный трафик**: выборки из набора данных [Автомобильный трафик (детальный)](/pro/data/built-in/cars-detailed), отфильтрованные по будним и выходным дням. Способ визуализации — [Линия](/pro/data/visualization/line).
  * Слой **Автомобильный трафик (обобщённый)**: выборка из набора данных [Автомобильный трафик (обобщённый)](/pro/data/built-in/cars). Способ визуализации — [Линия](/pro/data/visualization/line).
  * Виджет **Автотрафик по месяцам**: [столбчатая диаграмма](/pro/dashboards/widgets/column-chart) с распределением среднего автомобильного трафика по месяцам.
  * Виджет **Автотрафик по дням недели**: [столбчатая диаграмма](/pro/dashboards/widgets/column-chart) с распределением среднего автомобильного трафика по дням недели за последние 12 месяцев.
  * Виджет **Автотрафик по часам**: [гистограмма](/pro/dashboards/widgets/histogram) с распределением среднего часового автомобильного трафика за последние 12 месяцев.

    ![Сцена Трафик](/img/pro/presets-insights-traffic.png)

* Сцена **Инфраструктура**:

  * Слой **Новостройки**: выборка из набора данных [Здания](/pro/data/built-in/buildings). Способ визуализации — [Точка](/pro/data/visualization/point).
  * Слой **Строящиеся здания**: выборка из набора данных [Здания](/pro/data/built-in/buildings). Способ визуализации — [Точка](/pro/data/visualization/point).
  * Слои **Бизнес-центры**, **ТЦ и ТРЦ**, **Супермаркеты**, **Школы**, **Красота и здоровье** и многие другие: выборки из набора данных [Фирмы](/pro/data/built-in/firms). Способ визуализации — [Точка](/pro/data/visualization/point).
  * Виджет **Строящиеся здания**: [круговая диаграмма](/pro/dashboards/widgets/pie-chart) с распределением строящихся зданий по назначению.
  * Виджет **Новостройки**: [столбчатая диаграмма](/pro/dashboards/widgets/column-chart) с распределением новостроек по годам.

    ![Сцена Инфраструктура](/img/pro/presets-insights-buildings.png)

Вы можете просматривать и переключать сцены дашборда, показывать и скрывать слои, перемещать виджеты на сцене, а также [фильтровать данные через виджеты](/pro/data/manage/filtering#via-widgets).

## Получение доступа {#getting-access}

Чтобы получить доступ к дашборду:

1. Перейдите на [страницу для заказа дашборда](https://pro.2gis.ru/l/insights).
2. Укажите город или район, для которого построится дашборд.
3. Выберите до 20 категорий общественного питания.
4. Введите электронную почту.
5. Нажмите **Купить** и оплатите дашборд картой.

Уникальная ссылка на интерактивный дашборд будет отправлена на указанную электронную почту. Дашборд доступен в течение 7 дней.
