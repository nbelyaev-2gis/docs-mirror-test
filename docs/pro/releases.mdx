# Релизы

## Июнь 2026 г.

* Добавлена возможность выбирать координаты с помощью клика по карте при [создании или редактировании точек](/pro/data/manage/datasets#adding-objects).
* Добавлена возможность [создавать слои с точками](/pro/dashboards/layers/create#creating-points-layer) на карте, не загружая набор данных. Слой с точками доступен только в рамках дашборда, в котором он создан.
* Добавлена возможность выбирать в виджетах даты после текущего дня.
* Добавлен [глоссарий](/pro/glossary) с терминами, используемыми в документации и интерфейсе <Brand /> Про.
* Добавлено предупреждение об отсутствии данных пешеходного трафика для некоторых городов в форме настройки тарифа **Базовый**. Полный список доступных территорий см. в инструкции [Пешеходный трафик](/pro/data/built-in/pedestrians#available-territories).
* Теперь при отсутствии данных для слоя с визуализацией **Контур** отображается тултип с пояснением.
* Исправлена ошибка, возникавшая при редактировании атрибутов в пользовательских наборах данных с точками.
* Исправлена проблема со скоростью получения легенды при экспорте дашборда.

## Май 2026 г.

* Добавлена возможность [экспортировать все сцены дашборда](/pro/dashboards/dashboards/manage#exporting-dashboard) в один файл в формате PDF.

## Апрель 2026 г.

* Добавлена возможность рассчитывать удельные значения атрибутов на единицу площади в виджете [Сравнение](/pro/dashboards/widgets/comparison).
* Добавлен новый набор данных [Привлекательность локаций](/pro/data/built-in/attractiveness).
* Добавлена возможность самостоятельно подключить [тариф Базовый](/pro/prices) с расширенными возможностями работы и доступом к данным.
* Добавлен новый тип виджета [Произвольный виджет](/pro/dashboards/widgets/custom-widget) для встраивания внешних ресурсов или произвольных элементов на сцены дашборда.
* Добавлена возможность указывать группу с целевыми значениями в виджете [Сравнение](/pro/dashboards/widgets/comparison) и отображать разницу между ними и значениями в других группах.

## Март 2026 г.

* Добавлена возможность [редактировать точки](/pro/data/manage/datasets#editing-points) в загруженных вручную наборах данных.

## Февраль 2026 г.

* Добавлена возможность указывать единицы измерения для удельных показателей при [агрегации данных](/pro/geospatial/aggregation#parameters).
* Добавлена возможность использовать сквозные атрибуты из разных наборов данных при фильтрации данных на сцене [через срезы](/pro/data/manage/filtering#via-slices).
* Добавлена поддержка форматирования значений в легенде в виджетах [Линейчатая диаграмма](/pro/dashboards/widgets/bar-chart), [Столбчатая диаграмма](/pro/dashboards/widgets/column-chart), [Гистограмма](/pro/dashboards/widgets/histogram) и [Круговая диаграмма](/pro/dashboards/widgets/pie-chart).
* Добавлена возможность переименовывать группы для сравнения на вкладке **На карте** в виджете [Сравнение](/pro/dashboards/widgets/comparison).

## Январь 2026 г.

* Добавлены новые возможности [публикации дашборда по ссылке](/pro/dashboards/dashboards/manage#publishing-dashboard). Теперь владельцы дашборда могут:

   * ограничить территории, которые будут доступны по ссылке;
   * отменить публикацию ссылки;
   * выбрать дату, до которой будет действовать ссылка.

* Добавлена возможность настраивать цвета для диапазонов данных в легенде во всех [способах визуализации](/pro/data/visualization/overview).

## Декабрь 2025 г.

* Добавлен новый набор данных [Благосостояние населения](/pro/data/built-in/welfare).
* Добавлена возможность использовать сквозные атрибуты из разных наборов данных при фильтрации данных на сцене [через виджеты](/pro/data/manage/filtering#via-widgets).
* Добавлена возможность рассчитывать удельные показатели (отношение двух значений атрибутов) при [агрегации данных](/pro/geospatial/aggregation).
* Добавлена возможность использовать [Умный ассистент](/pro/assistant) — инструмент для работы с <Brand /> Про на основе искусственного интеллекта.

## Ноябрь 2025 г.

* Добавлена возможность настраивать тултипы для способа визуализации [Контур](/pro/data/visualization/contour). Теперь вы можете настроить список атрибутов данных и выбирать тип агрегации числовых атрибутов.
* Добавлена возможность [фильтровать данные на сцене с помощью срезов](/pro/data/manage/filtering#via-slices) и применять фильтры сразу для нескольких слоёв или атрибутов в одном виджете.
* Добавлена возможность настраивать тултипы для способов визуализации [Сетка](/pro/data/visualization/grid), [Соты](/pro/data/visualization/hexbin), [Кластер](/pro/data/visualization/cluster) и [H3](/pro/data/visualization/h3). Теперь вы можете настроить список атрибутов данных и выбирать тип агрегации числовых атрибутов.

## Октябрь 2025 г.

* Добавлена возможность выбирать период до текущей даты (режим **Последние**) для группировки значений в виджете [Сравнение](/pro/dashboards/widgets/comparison).
* Добавлена возможность [загружать данные](/pro/data/files-upload) в виде ячеек H3.
* Добавлена возможность группировать значения по атрибутам с типом `Дата` для просмотра изменения данных с течением времени в виджете [Сравнение](/pro/dashboards/widgets/comparison).
* Добавлена возможность настраивать ширину линий на основе числового атрибута данных для способа визуализации [Линия](/pro/data/visualization/line).
* Добавлена возможность настраивать цвета столбцов и секторов для отдельных значений в виджетах [Линейчатая диаграмма](/pro/dashboards/widgets/bar-chart), [Столбчатая диаграмма](/pro/dashboards/widgets/column-chart), [Гистограмма](/pro/dashboards/widgets/histogram) и [Круговая диаграмма](/pro/dashboards/widgets/pie-chart).
* Добавлена возможность группировать значения по атрибутам с типом `Дата` для просмотра изменения данных с течением времени в виджетах [Линейчатая диаграмма](/pro/dashboards/widgets/bar-chart), [Столбчатая диаграмма](/pro/dashboards/widgets/column-chart), [Гистограмма](/pro/dashboards/widgets/histogram), [Линейный график](/pro/dashboards/widgets/line-graph), [Круговая диаграмма](/pro/dashboards/widgets/pie-chart) и [График с областями](/pro/dashboards/widgets/area-chart).
* Добавлена возможность настраивать прозрачность заливки и обводки полигонов для способа визуализации [Полигон](/pro/data/visualization/polygon).
* Добавлена возможность настраивать радиус точки на основе числового атрибута данных для способа визуализации [Точка](/pro/data/visualization/point).
* Добавлена возможность форматировать значения числовых атрибутов данных во всех [виджетах](/pro/dashboards/widgets/overview), кроме текстового.
* Добавлена возможность [обновлять загруженный набор данных](/pro/data/manage/datasets#updating-dataset).
* Добавлены новый формат даты и времени для [загрузки и работы](/pro/data/files-upload#dates-and-time) с собственными геоданными: `YYYY-MM-DDTHH:MM:SS.sss±HH:MM`.
* Добавлена возможность создавать новые [сценарии аналитики](/pro/presets/overview) (готовые сцены со слоями и виджетами): **Конкуренты и спрос**, **Инфраструктура города**, **Демографический портрет населения**, **Центры притяжения коммерции**, **Транспортная доступность**.

## Сентябрь 2025 г.

* Добавлен новый набор данных [Территориальное деление](/pro/data/built-in/territorial).
* Добавлена возможность [копировать слой](/pro/dashboards/layers/manage#copying-a-layer) из одной сцены дашборда в другую.
* Добавлена возможность использовать новые типы агрегации цвета (параметр **Тип агрегации цвета**):
  * По медиане, минимуму и максимуму для способов визуализации [Сетка](/pro/data/visualization/grid), [Соты](/pro/data/visualization/hexbin), [Здания](/pro/data/visualization/buildings) и [H3](/pro/data/visualization/h3).
  * По минимуму и максимуму для способов визуализации [Контур](/pro/data/visualization/contour) и [Кластер](/pro/data/visualization/cluster).
* Добавлена возможность использовать новые типы агрегации высоты (параметр **Тип агрегации высоты**):
  * По медиане, минимуму и максимуму для способов визуализации [Сетка](/pro/data/visualization/grid), [Соты](/pro/data/visualization/hexbin) и [H3](/pro/data/visualization/h3).
  * По минимуму и максимуму для способа визуализации [Контур](/pro/data/visualization/contour).

## Август 2025 г.

* Добавлена возможность [экспортировать сцены дашборда](/pro/dashboards/scenes/manage) в файл в формате PDF.
* Добавлена возможность экспортировать данные из пользовательских наборов: [загруженных как файл](/pro/data/manage/datasets#exporting-data) и [загруженных по API](/pro/data/manage/datasets#exporting-data-api).
* Добавлена возможность использовать единый цвет столбцов для [линейчатой диаграммы](/pro/dashboards/widgets/bar-chart#parameters) и [столбчатой диаграммы](/pro/dashboards/widgets/column-chart#parameters) при выборе текстового атрибута группировки: см. параметр **Однотонный график**.
* Добавлена возможность строить на карте полигоны и использовать их в качестве [геофильтров](/pro/data/overview#geofilters).
* Добавлен новый набор данных [Динамика организаций](/pro/data/built-in/dynamics).
* Добавлена возможность вручную [обновлять содержимое дашборда](/pro/dashboards/dashboards/manage#updating-a-dashboard), к которому выдан доступ на просмотр.

## Июль 2025 г.

* Добавлена возможность показывать в виджете [Индикатор](/pro/dashboards/widgets/counter) целевое значение и разницу между текущим и целевым значениями.
* Добавлена возможность группировать значения в виджете [Сравнение](/pro/dashboards/widgets/comparison) по элементам визуализации на карте. Для этого кликните на элемент визуализации с типом H3, Контур или Полигон.
* Добавлена возможность строить [зоны доступности](/pro/geospatial/proximity) для общественного транспорта.
* Добавлена возможность выдавать совместный доступ другим пользователям к наборам данных, [загруженным как файл](/pro/data/manage/datasets#access-management) и [загруженным по API](/pro/data/manage/datasets#access-management-api).
* Теперь, если на сцене дашборда отсутствуют данные из выборки из-за [применённых фильтров](/pro/data/manage/filtering), в списке слоёв рядом с соответствующим слоем появляется предупреждающий значок.
* Теперь пользователи с [доступом на просмотр дашборда](/pro/dashboards/dashboards/manage#granting-access-to-a-dashboard) могут скрывать виджеты или включать отображение, если виджеты ранее были скрыты. Настройки сохраняются до тех пор, пока владелец дашборда не отредактирует виджеты (например, добавит или удалит их).
* Теперь при построении [зон доступности](/pro/geospatial/proximity) вы можете отключать вырезание внутренних зон из внешних. При отключении опции создаются зоны с накоплением: каждая следующая зона включает данные из предыдущих зон.
* Описаны [системные требования](/pro/overview#system-requirements) для использования <Brand /> Про.
* Добавлена возможность [скрывать тип агрегации](/pro/dashboards/widgets/statistics#parameters) в виджете **Статистика**.
* Добавлена возможность выбирать логический агрегирующий атрибут в виджетах [Сравнение](/pro/dashboards/widgets/comparison), [Статистика](/pro/dashboards/widgets/statistics) и [Индикатор](/pro/dashboards/widgets/counter).
* Добавлена возможность указывать тип распределения для [линейчатой диаграммы](/pro/dashboards/widgets/bar-chart#parameters), [столбчатой диаграммы](/pro/dashboards/widgets/column-chart#parameters), [гистограммы](/pro/dashboards/widgets/histogram#parameters) и [линейного графика](/pro/dashboards/widgets/line-graph#parameters).
* Теперь у пользователей с [доступом на просмотр дашборда](/pro/dashboards/dashboards/manage#granting-access-to-a-dashboard) сохраняется настроенное положение виджетов на сценах дашборда после обновления страницы.
* Добавлен новый тип виджета [Индикатор](/pro/dashboards/widgets/counter) для отслеживания значения атрибута данных из слоя.
* Добавлены новые функции для совместной работы над дашбордами:
  * Возможность [выдавать совместный доступ](/pro/dashboards/dashboards/manage#granting-access-to-a-dashboard) на просмотр или редактирование дашборда другим пользователям.
  * Возможность [отправлять копию дашборда](/pro/dashboards/dashboards/manage#sending-a-dashboard-copy).
* Добавлен новый тип виджета [Сравнение](/pro/dashboards/widgets/comparison) для сравнения значений данных из разных групп в виде таблицы.
* Добавлен новый тип виджета [Статистика](/pro/dashboards/widgets/statistics) для просмотра краткой статистики по данным.

## Июнь 2025 г.

* Добавлена возможность использовать [линейчатые диаграммы](/pro/dashboards/widgets/bar-chart) и [столбчатые диаграммы](/pro/dashboards/widgets/column-chart) с накоплением: объединять столбцы из одной группы в один столбец из нескольких сегментов.
* Добавлена возможность использовать автоматическую [тему карты](/pro/styles/management): тема карты будет меняться в зависимости от темы в браузере. Теперь используется как тема по умолчанию.
* Добавлена возможность указывать собственные названия для осей для [линейчатой диаграммы](/pro/dashboards/widgets/bar-chart#parameters), [столбчатой диаграммы](/pro/dashboards/widgets/column-chart#parameters), [гистограммы](/pro/dashboards/widgets/histogram#parameters), [линейного графика](/pro/dashboards/widgets/line-graph#parameters) и [графика с областями](/pro/dashboards/widgets/area-chart#parameters).

## Май 2025 г.

* Добавлена возможность использовать второй атрибут группировки для [линейчатой диаграммы](/pro/dashboards/widgets/bar-chart) и [столбчатой диаграммы](/pro/dashboards/widgets/column-chart), чтобы сравнивать значения сразу по двум параметрам.
* Добавлен график распределения населения по полу и возрасту в виде пирамиды в результатах фильтрации для набора данных [Половозрастной состав населения](/pro/data/built-in/sex-age-composition).
* Добавлена возможность выбирать на карте с помощью клика стандартные территории и использовать их в качестве [геофильтров](/pro/data/overview#geofilters).
* Добавлена возможность показывать и скрывать сразу все слои сцены: см. инструкцию [Операции со слоями](/pro/dashboards/layers/manage).
* Добавлен новый тип виджета [График с областями](/pro/dashboards/widgets/area-chart): аналог линейного графика с заливкой области между линией и осями.
* Добавлен новый тип виджета [Текстовый виджет](/pro/dashboards/widgets/text) для отображения блоков с текстом.
* Добавлены новые функции для работы с данными, [загруженными как файл](/pro/data/files-upload):
  * Теперь появляется предупреждение при попытке загрузить файл, если он содержит два столбца с одинаковыми названиями.
  * Добавлена возможность редактировать выборку (например, добавлять объекты в виде точек), если из выборки удалены все объекты.
* Добавлена поддержка отображения только даты (без времени) для поля с типом `date` в тултипе и карточке объекта.

## Апрель 2025 г.

* Добавлена поддержка тёмной темы в интерфейсе Про. Чтобы переключиться на тёмную тему, нажмите значок ![Тёмная тема](/img/pro/dark-theme-icon.svg) в левом нижнем углу интерфейса.
* Добавлена возможность настраивать порядок слоёв в виджете [Легенда](/pro/dashboards/widgets/legend).
* Добавлены новые функции для всех способов [визуализации](/pro/data/visualization/overview) данных (кроме способа **Тепловая карта**):
  * Возможность использовать экспоненциальный тип шкалы для отображения элементов на карте и в легенде.
  * Возможность настраивать диапазоны данных в легенде вручную.
* Добавлены новые функции [круговой диаграммы](/pro/dashboards/widgets/pie-chart#parameters):
  * Возможность разделять секторы круга промежутками.
  * Возможность указывать внутренний радиус круга.
* Добавлена возможность использовать типы агрегации по медиане, 80-ому, 95-ому и 99-ому процентилю для [линейчатой диаграммы](/pro/dashboards/widgets/bar-chart#parameters), [столбчатой диаграммы](/pro/dashboards/widgets/column-chart#parameters), [линейного графика](/pro/dashboards/widgets/line-graph#parameters) и [круговой диаграммы](/pro/dashboards/widgets/pie-chart#parameters).
* Добавлена возможность [открывать виджеты на весь экран](/pro/dashboards/widgets/manage#opening-a-widget-in-full-screen-mode).
* Добавлены новые функции [временной шкалы](/pro/dashboards/widgets/timeline):
  * Возможность строить график с распределением количества объектов в зависимости от периода времени.
  * Возможность указывать размер шага в месяцах и годах.
* Добавлена возможность [фильтровать данные на сцене с помощью нажатия на виджет](/pro/data/manage/filtering#via-widgets).
* Добавлен новый набор данных [Туристический потенциал](/pro/data/built-in/tourists).

## Март 2025 г.

* Добавлена возможность [создавать сцены](/pro/dashboards/scenes/create) для работы с разными сценариями аналитики данных в одном дашборде.
