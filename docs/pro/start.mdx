# Начало работы

Инструкция поможет вам начать работу с сервисом <Brand /> Про и познакомиться с его возможностями.

Пройдя все шаги, вы создадите дашборд для анализа спроса на кафе-кондитерские. В дашборде будет создана сцена, которая содержит:

* два слоя с выборками данных и разными способами визуализации;
* виджет с легендой;
* виджет с линейчатой диаграммой, которая отображает рейтинг кафе-кондитерских из выборки;
* виджет с гистограммой, которая отображает средний чек в кафе-кондитерских из выборки.

![Результат](/img/pro/start-result.png)

:::tip[Готовые дашборды]

Для новых пользователей <Brand /> Про доступен демо-дашборд на вкладке **Дашборды**. Также вы можете проанализировать потенциал мест для открытия малого и среднего бизнеса с помощью интерактивного дашборда [Аналитика локаций](/pro/presets/insights), не регистрируясь в сервисе <Brand /> Про.

:::

## 1. Авторизуйтесь в <Brand /> Про {#authenticating}

1. Перейдите на страницу <Brand /> Про.

   <CtaButton href="https://pro.2gis.ru/" target="_blank">
      Открыть <Brand /> Про
   </CtaButton>

2. Зарегистрируйтесь удобным для вас способом.

Для новых пользователей <Brand /> Про по умолчанию доступен демо-тариф: см. подробнее в инструкции [Тарифы](/pro/prices).

:::note[Доступность территорий в тарифах]

В примере используется территория, доступная в демо-тарифе. Если у вас подключен другой тариф и вы не приобретали к ней доступ, вы можете выполнить шаги инструкции на любой доступной территории.

:::

## 2. Создайте первый слой с данными и дашборд со сценой {#create-first-layer}

Чтобы создать слой на основе системных данных и добавить его на сцену дашборда:

1. Перейдите на вкладку **Главная**.

2. Выберите набор данных **Фирмы**.

3. Выберите территорию, для которой будет создана выборка данных. Для этого в окне **Настройки геофильтров** выберите `Пресненский район, Москва` и нажмите **Готово**.

4. На вкладке **Фильтры** укажите следующие данные:

   1. Нажмите **Выбрать категории**. Далее выберите **Общественное питание** → **Кафе-кондитерские** и нажмите **Применить**.
   2. В поле **Рейтинг** укажите значение от `3` до `5`.

5. Нажмите **Создать слой**.

Будет создан новый дашборд с одной сценой, на которую автоматически добавится новый слой. На слое будут отображены все кафе-кондитерские Пресненского района Москвы с рейтингом от 3 до 5. Чтобы переименовать слой, нажмите на него два раза, введите имя `Кафе-кондитерские с рейтингом от 3 и выше` и нажмите *Enter*.

![Создание слоя Кафе-кондитерские с рейтингом от 3 и выше](/img/pro/start-create-layer.png)

## 3. Создайте второй слой с данными {#create-second-layer}

Вы можете добавлять слои на сцены дашборда и удалять их из сцен.

Чтобы создать новый слой и добавить его на сцену дашборда:

1. Перейдите на вкладку **Дашборды**.
2. Откройте созданный дашборд. Автоматически откроется сцена дашборда.
3. В верхнем меню дашборда нажмите значок ![Слои](/img/pro/layers-icon.svg).
4. Нажмите **Создать слой**.
5. Выберите набор данных **Карта спроса**.
6. Выберите территорию, для которой будет создана выборка данных. Для этого в окне **Настройки геофильтров** выберите `Пресненский район, Москва` и нажмите **Готово**.
7. На вкладке **Фильтры** нажмите **Выбрать категории**. Далее выберите **Общественное питание** → **Кафе-кондитерские** и нажмите **Применить**.
8. Нажмите **Создать слой**.

Будет создан слой, на котором отображена карта спроса (количество поисковых запросов) на все кафе-кондитерские Пресненского района Москвы в виде сот. Слой добавится на сцену дашборда. Чтобы переименовать слой, нажмите на него два раза, введите имя `Спрос на кафе-кондитерские` и нажмите *Enter*.

![Добавление на дашборд нового слоя Спрос на кафе-кондитерские](/img/pro/start-add-layer.png)

## 4. Измените способ визуализации слоя {#change-visualization}

Сервис <Brand /> Про позволяет по-разному визуализировать данные, например в виде тепловых карт: см. другие [способы визуализации](/pro/data/visualization/overview). Для каждого слоя можно настроить свои параметры визуализации.

Чтобы изменить способ визуализации для слоя `Спрос на кафе-кондитерские`:

1. В верхнем меню дашборда нажмите значок ![Слои](/img/pro/layers-icon.svg).
2. Выберите слой `Спрос на кафе-кондитерские`.
3. В блоке **Базовые настройки** укажите следующие данные:

   * **Способ отображения**: выберите **H3**.
   * **Цвет на основе**: выберите цветовую палитру (набор готовых цветовых образцов), в соответствии с которой будут раскрашены элементы на карте.
   * **Шаги**: выберите количество цветов в палитре, например `5`.

Для слоя `Спрос на кафе-кондитерские` способ визуализации будет изменён на H3 — систему с фиксированным расположением сетки, которая не зависит от данных. Этот способ полезен при анализе пространственных данных из-за единого формата расположения ячеек. Также будет изменена цветовая палитра.

![Изменение способа визуализации данных на слое](/img/pro/start-visualization.png)

## 5. Добавьте виджет с линейчатой диаграммой на сцену дашборда {#add-bar-chart-widget}

На сцены дашборда можно добавлять виджеты с различными диаграммами на основе данных.

Чтобы добавить виджет на сцену дашборда:

1. В верхнем меню дашборда нажмите значок ![Виджеты](/img/pro/widgets-icon.svg).

2. Выберите тип виджета **Линейчатая диаграмма**.

3. Настройте параметры виджета:

   * **Название виджета**: введите имя виджета `Рейтинг заведений`.
   * **Слой**: выберите слой `Кафе-кондитерские с рейтингом от 3 и выше` (с набором данных **Фирмы**).
   * **Атрибут группировки**: выберите значение `Рейтинг`.
   * **Количество диапазонов**: добавьте три диапазона с данными и укажите значения. Чтобы добавить диапазон, нажмите значок ![Плюс](/img/pro/plus-icon.svg).
   * **Агрегирующий атрибут**: выберите значение `Количество объектов`.

      <img src="/img/pro/start-create-first-widget.png" alt="Создание первого виджета" width="700"  />

4. Нажмите **Создать**.

На сцену дашборда будет добавлен виджет с линейчатой диаграммой, которая отображает рейтинг кафе-кондитерских из выборки и количество заведений.

![Результат создания первого виджета](/img/pro/start-result-first-widget.png)

## 6. Добавьте виджет с гистограммой на сцену дашборда {#add-histogram-widget}

Чтобы добавить виджет на сцену дашборда:

1. В верхнем меню дашборда нажмите значок ![Виджеты](/img/pro/widgets-icon.svg).

2. Выберите тип виджета **Гистограмма**.

3. Настройте параметры виджета:

   * **Название виджета**: введите имя виджета `Средний чек`.
   * **Слой**: выберите слой `Кафе-кондитерские с рейтингом от 3 и выше` (с набором данных **Фирмы**).
   * **Атрибут группировки**: выберите значение `Средний чек (Ресторан / Кафе)`.
   * **Размер шага**: введите размер диапазона с данными, например `50`.

      <img src="/img/pro/start-create-second-widget.png" alt="Создание второго виджета" width="700"  />

4. Нажмите **Создать**.

На сцену дашборда будет добавлен виджет с гистограммой, которая отображает распределение среднего чека в кафе-кондитерских из выборки и количество заведений.

![Результат](/img/pro/start-result.png)
