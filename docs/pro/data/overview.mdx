# Обзор

В сервисе [<Brand /> Про](https://pro.2gis.ru/) можно работать с наборами подготовленных [системных данных](#built-in-data) и собственных данных: [загруженных как файл](/pro/data/files-upload) и [загруженных по API](/pro/data/api-upload/dynamic-upload).

Системные и загруженные наборы данных доступны на вкладках **Главная** и **Данные**.

* [Благосостояние населения](/pro/data/built-in/welfare),
* [Городская среда](/pro/data/built-in/urban),

* [Жилая недвижимость](/pro/data/built-in/residential),
* [Здания](/pro/data/built-in/buildings),
* [Карта спроса](/pro/data/built-in/demand),
* [Коммерческая недвижимость](/pro/data/built-in/commercial),
* [Население](/pro/data/built-in/population),
* [Остановки общественного транспорта](/pro/data/built-in/stops),
* [Парковки](/pro/data/built-in/parkings),
* [Пешеходный трафик](/pro/data/built-in/pedestrians),
* [Половозрастной состав населения](/pro/data/built-in/sex-age-composition),

## Доступные территории {#available-territories}

Системные данные доступны для следующих стран (могут быть исключения для определённых наборов данных):

* Азербайджан,
* Армения,
* Беларусь,
* Грузия,
* Казахстан,
* Кыргызстан,
* Россия,
* Узбекистан.

:::note[Ограниченная доступность на территории]

Данные могут быть доступны не на всей территории страны, а только в населённых пунктах, в которых присутствует компания <Brand />. Чтобы уточнить доступность данных, используйте [геофильтры](#geofilters) или обратитесь в [техническую поддержку](mailto:support.pro@2gis.ru).

:::

## Сценарии использования {#usage-scenarios}

Для решения аналитических задач из каждого набора данных можно создавать различные выборки. Это может быть полезно, если из набора данных нужно оставить только определённые данные для работы.

Выборка содержит данные из набора, которые отфильтрованы по территориям и другим параметрам. На основе выборок можно [создавать слои](/pro/dashboards/layers/create) для визуализации данных.

### Типовой сценарий работы с данными

1. Выберите набор системных данных или загрузите собственные данные.

2. Отфильтруйте данные по территориям, указав геофильтры. Применение геофильтров обязательно для создания выборки данных и последующего создания слоя.

   В качестве геофильтров вы можете использовать стандартные территории, собственные загруженные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](#geofilters).

3. Отфильтруйте данные по другим доступным параметрам, которые зависят от набора данных.

4. Сохраните полученную выборку данных, создав на её основе слой для дальнейшей визуализации данных и создания виджетов.

См. подробнее в разделе [Начало работы](#getting-started).

### Изменение выборки данных

После создания слоя вы можете изменить выборку данных:

* Применить к исходным данным геофильтры (фильтры по территориям).
* Применить к исходным данным фильтры по параметрам.
* Изменить результаты фильтрации вручную, убрав определённые объекты.

Во всех слоях и виджетах, которые были созданы из этой выборки данных, автоматически отразятся изменения. При любых изменениях выборки данные из набора не удаляются.

См. подробнее в инструкции [Изменение выборки данных](/pro/data/manage/modify).

### Фильтрация при работе с дашбордом

При работе с дашбордом вы можете применять к выборке данных дополнительную фильтрацию. Чтобы отобразить на сценах дашборда только часть данных, вы можете фильтровать данные из выборки:

* По территориям: через геофильтры.
* По атрибутам данных: через диаграммы в виджетах и через виджет **Фильтр-срез**.
* По дате и времени: через виджет **Временная шкала**.

Выборка данных при этом не изменяется.

См. подробнее в инструкции [Фильтрация данных на дашборде](/pro/data/manage/filtering).

## Начало работы {#getting-started}

![Данные](/img/pro/data-overview.png)

1. Перейдите на вкладку **Главная** или **Данные**.

2. Выберите нужную категорию системных данных или набор загруженных данных.

   Вы можете использовать поиск, чтобы конкретизировать выдачу данных. Результаты поиска могут включать информацию об организациях, а также данные из других категорий.

3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](#geofilters). После выбора территорий нажмите **Готово**.

4. Если вы выбрали системный набор данных **Пешеходный трафик** или загрузили данные с геометриями в виде линий или полигонов, вверху вкладки **Фильтры** выберите способ отображения данных:

   * **Точки**: в виде точек-центроидов (центров геометрии объектов).
   * **Геометрии**: в виде реальных геометрий данных (линий или полигонов).

   В остальных наборах данных доступен только способ отображения **Точки**.

5. На вкладке **Фильтры** настройте остальные фильтры при необходимости. Доступные фильтры зависят от исходных данных. Список фильтров для системных данных см. в описании каждого набора, например [Фирмы](/pro/data/built-in/firms#filters).

<br />

Данные будут отображены на карте. Результаты фильтрации отображаются справа от вкладки **Фильтры**.

Чтобы сохранить данную выборку и настроить её [визуализацию](/pro/data/visualization/overview) (например, в виде тепловой карты), нажмите **Создать слой**.

## Геофильтры {#geofilters}

Для создания выборок данных вы можете фильтровать данные по территориям с помощью разных видов геофильтров в окне **Настройки геофильтров**. Для каждой выборки вы можете выбрать до 50 территорий.

:::note[Доступность в демо-тарифе]

Если у вас [демо-тариф](/pro/prices), вы можете выбрать не более трёх геофильтров в рамках одной сцены дашборда.

:::

### Виды геофильтров

* **Стандартные территории**: области, районы, населённые пункты и другие. По умолчанию для выбора предлагаются территории, которые находятся в текущей области видимости карты. В зависимости от [тарифа](/pro/prices) некоторые территории могут быть недоступны. Вы можете как использовать поиск по стандартным территориям, так и выбирать их на карте по клику.

  Чтобы найти стандартные территории, перейдите на вкладку **Поиск** и используйте поиск по названиям. Вы можете указывать поисковые фильтры (например, искать только области). Пример создания геофильтра по стандартной территории через поиск:

  <img src="/img/pro/geofilters-standard.png" alt="Поиск по стандартным территориям" width="900"  />

  <br />

  Чтобы выбрать территорию на карте, перейдите на вкладку **Выбрать на карте**, укажите деление территорий на карте (например, показывать только административные районы) и кликните на территорию. Пример создания геофильтра по стандартной территории через клик на карте:

  <img src="/img/pro/geofilters-select.png" alt="Выбор стандартных территорий на карте" width="900"  />

* **Собственные территории**: предварительно загрузите собственный набор данных с полигонами и [создайте фильтр по территориям](/pro/data/files-upload#create-territory-filter). Чтобы выбрать собственную территорию, перейдите на вкладку **Поиск** и используйте поиск по названиям территорий, которые вы указали в `properties` при загрузке набора данных:

  <img src="/img/pro/geofilters-search.png" alt="Поиск по территориям" width="600"  />

  Пример создания геофильтра по собственной территории:

  <img src="/img/pro/geofilters-custom.png" alt="Собственные территории" width="900"  />

* **Полигоны**: произвольные площади на карте. Чтобы построить полигон, перейдите на вкладку **Нарисовать** и нажмите **Полигон**. Поставьте не менее трёх точек на карте. Точки соединяются автоматически, линии полигона не должны пересекаться. Чтобы замкнуть полигон, соедините начальную и конечную точку. Чтобы переставить точку, зажмите её и переместите в нужное место на карте.

  Пример создания геофильтра по полигону:

  ![Полигоны](/img/pro/geofilters-polygons.png)

* **Зоны доступности**: площади на карте, до границ которых можно добраться пешком или на автомобиле за определённый промежуток времени. Чтобы построить зону доступности, перейдите на вкладку **Нарисовать** и нажмите **Доступность**. Поставьте на карте точку, вокруг которой будет построена область, укажите способ перемещения и время от `1` до `60` мин.

  Пример создания геофильтра по зоне доступности:

  ![Зоны доступности](/img/pro/geofilters-proximity.png)

* **Окружности**: окружности на карте. Чтобы построить окружность, перейдите на вкладку **Нарисовать** и нажмите **Радиус**. Поставьте точку на карте, вокруг которой будет очерчена окружность, и укажите радиус от `10` до `100 000` м.

  Пример создания геофильтра по окружности:

  ![Радиус](/img/pro/geofilters-radius.png)

### Операции с геофильтрами

#### Отображение границ и названий

Чтобы на сцене дашборда отображались границы и названия территорий, нажмите **Геофильтры** и в окне **Настройки геофильтров** отметьте чекбокс **Всегда показывать границы и названия**:

![Исходная выборка](/img/pro/modify-data-geo-before.png)

#### Переименование геофильтра

1. Откройте окно **Настройки геофильтров**.
2. Нажмите два раза на нужный геофильтр.
3. Введите новое имя геофильтра и нажмите *Enter*.

#### Скрытие и показ геофильтра

1. Откройте окно **Настройки геофильтров**.
2. Чтобы скрыть геофильтр, нажмите значок ![Скрыть геофильтр](/img/pro/hide-eye-icon.svg) рядом с его именем.
3. Чтобы показать геофильтр, нажмите значок ![Показать геофильтр](/img/pro/open-eye-icon.svg) рядом с его именем.

Если вы [изменяете выборку данных](/pro/data/manage/modify), при скрытии геофильтра в выборку не попадают данные из этой территории. При этом геофильтр не удаляется, и вы сможете включить данные в выборку позже.

Если вы [фильтруете данные на дашборде через геофильтры](/pro/data/manage/filtering#via-geofilters), при скрытии геофильтра выборка не меняется, но на сценах отображаются данные только для выбранных территорий.

#### Удаление геофильтра

1. Откройте окно **Настройки геофильтров**.
2. Чтобы удалить один геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.
