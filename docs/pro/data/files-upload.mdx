# Загрузка файлов с данными

Вы можете загружать собственные геоданные в сервис [<Brand /> Про](https://pro.2gis.ru/).

Загрузить данные можно для любой территории, но детальная карта будет доступна только в странах, для которых в <Brand /> есть системные данные: см. [список стран](/pro/data/overview).

:::note[Доступность в демо-тарифе]

Если у вас [демо-тариф](/pro/prices), вы не можете загружать файлы с данными.

:::

Чтобы загрузить файл с набором данных и работать с ним:

1. Подготовьте файл с данными.
2. Загрузите данные в сервис.
3. Создайте фильтр по территориям из полигонов (опционально).
4. Создайте слой с выборкой данных.

После загрузки данных вы можете [обновить данные в наборе](/pro/data/manage/datasets#updating-dataset).

Если вы загрузили набор данных с объектами в виде точек, вы можете вручную добавить новые точки в набор, отредактировать или удалить существующие. См. подробнее в разделе [Изменение набора данных с точками](/pro/data/manage/datasets#managing-points).

## 1. Подготовьте файл с данными {#prepare-a-data-file}

### Общие требования

* Форматы:
  * `.csv`, `.csv.gz`;
  * `.geojson`, `.geojson.gz`;
  * `.shp` (дополнительно нужно загрузить файлы `.shx` и `.dbf`);
  * `.shp.zip` (архив должен содержать файлы `.shp`, `.shx`, `.dbf` и `.prj`);
  * `.kml`;
  * `.kmz`;
  * `.xlsx` (поддерживается загрузка только первого листа файла);
  * `.gpkg`.
* Размер: зависит от вашего [тарифа](/pro/prices).
* Кодировка: UTF-8.
* Система координат: [WGS84(4326)](https://epsg.io/4326).

### Геометрия

* Доступные типы геометрии:

  * Точка (Point);
  * Линия (LineString);
  * Полигон (Polygon) — можно использовать для [создания новых фильтров по территории](#create-territory-filter);
  * Мультиточка (MultiPoint);
  * Мультилиния (MultiLineString);
  * Мультиполигон (MultiPolygon).

  FeatureCollection и GeometryCollection не поддерживаются. Файл должен содержать только один тип геометрии.

* Данные в форматах `.csv` и `.csv.gz` должны содержать минимум один столбец с геометрией. Данные в форматах `.geojson` и `.geojson.gz` — минимум одно поле с геометрией. Поддерживаемые форматы геометрии:

  * WKT (Well-Known Text);
  * широта и долгота в виде чисел с плавающей запятой (только для объектов в иде точек).

### Даты и время {#dates-and-time}

Поддерживаемые форматы:

* Только дата (по умолчанию применяется часовой пояс UTC±00:00):

  * `YYYY-MM-DD` (2023-10-05)
  * `DD/MM/YYYY` (05/10/2023)
  * `DD.MM.YYYY` (05.10.2023)
  * `DD Mon YYYY` (05 Oct 2023)

* Дата и время (по умолчанию применяется часовой пояс UTC±00:00):

  * `YYYY-MM-DD HH:MM` (2023-10-05 14:30)
  * `YYYY-MM-DD HH:MM:SS` (2023-10-05 14:30:00)
  * `DD/MM/YYYY HH:MM` (05/10/2023 14:30)
  * `DD/MM/YYYY HH:MM:SS` (05/10/2023 14:30:00)
  * `DD.MM.YYYY HH:MM` (05.10.2023 14:30)
  * `DD.MM.YYYY HH:MM:SS` (05.10.2023 14:30:00)

* Дата и время с указанием часового пояса:

  * `YYYY-MM-DDTHH:MM:SS±HH:MM` (2023-10-05T14:30:00+03:00)
  * `YYYY-MM-DDTHH:MM:SSZ` (2023-10-05T11:30:00Z)
  * `YYYY-MM-DD HH:MM:SSZ` (2023-10-05 11:30:00Z)
  * `YYYY-MM-DDTHH:MM:SS.sss±HH:MM` (2023-10-05T14:30:00.000+03:00)
  * `Ddd, DD Mon YYYY HH:MM:SS TIMEZONE` (Thu, 05 Oct 2023 13:30:00 GMT)

### Данные

* Допустимы любые дополнительные столбцы с информацией об объектах.
* Дополнительные столбцы могут использоваться в качестве фильтров. Максимальное количество фильтров — 40.
* Максимальное количество уникальных значений в фильтре — 30.
* При настройке псевдонимов поля (field aliases) можно использовать цифры, буквы, пробел, символы `.`, `_`, `$` и круглые скобки `()`. Не используйте запятые.

Вы можете подготовить данные в любом геоинформационном ПО, с помощью языков программирования или в Excel (только объекты в виде точек).

## 2. Загрузите данные {#upload-the-data}

1. Перейдите на вкладку **Главная** или **Данные**.
2. Нажмите **Загрузить данные**.

   <img src="/img/pro/data-upload-button.png" alt="Кнопка Загрузить данные" width="600"  />

3. Перетащите файл в область загрузки или нажмите **Выберите файл на компьютере**.
4. Нажмите **Загрузить**.
5. Введите название набора данных. Максимальное количество символов — 50. Название может содержать буквы, цифры, пробелы и специальные символы.
6. Выберите [группу](/pro/data/manage/groups), в которую будет добавлен набор данных.
7. (Опционально) Введите описание набора данных.
8. Сервис автоматически формирует структуру атрибутов для набора данных. Чтобы включить атрибут в итоговый набор, отметьте его чекбокс и при необходимости измените [параметры](#attributes) атрибута.

   <img src="/img/pro/data-uploaded.png" alt="Настройка атрибутов" width="600" />

9. В списке атрибутов отметьте чекбокс минимум у одного атрибута, содержащего информацию о геометрии. Выберите в поле **Тип** тип атрибута:

   * `Широта` или `Долгота` для координат;
   * `WKT` для геометрии в формате WKT;
   * `H3 ID` для данных в виде H3 (гексагональной сетки пространственных индексов [Hexagonal Hierarchical Spatial Index](https://h3geo.org/)).

   Если вы отмените настройку атрибутов или закроете диалоговое окно, загруженный набор данных останется на вкладке **Данные**. В карточке набора появится пометка о необходимости настройки атрибутов. Вы сможете продолжить работу с набором данных только после настройки.

10. Нажмите **Создать набор данных**.

Данные появятся в выбранной группе наборов данных в виде карточки (по умолчанию это группа **Загруженные данные**).

### Параметры атрибутов данных {#attributes}

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Заголовок** | Имя поля. |
| **Тип** | Тип атрибута:<ul><li>`Широта`,</li><li>`Долгота`,</li><li>`WKT`,</li><li>`H3 ID`,</li><li>`Название`,</li><li>`Атрибут`,</li><li>`Дата`.</li></ul> |
| **Тип фильтра** | Тип фильтра для атрибута:<ul><li>`Селектбокс` — выбор одного значения из списка;</li><li>`Мультивыбор из списка` — выбор нескольких значений из списка;</li><li>`Диапазон числовых значений`;</li><li>`Текстовый фильтр`;</li><li>`Диапазон дат с временем`.</li></ul> |

## 3. (Опционально) Создайте фильтр по территориям из полигонов {#create-territory-filter}

Если загруженные данные содержат геометрии в виде полигонов, вы можете использовать их для создания собственного территориального деления. Это позволит создать новые фильтры по территориям (геофильтры) и фильтровать данные как по стандартным видам территорий (областям, районам, населённым пунктам и другим), так и по собственным зонам.

Например, вы можете загрузить полигоны с границами интересующих районов и проанализировать информацию о [фирмах](/pro/data/built-in/firms) и [городской среде](/pro/data/built-in/urban) внутри них.

Чтобы создать фильтр по территориям:

1. Убедитесь, что загружаемые данные содержат:

   * геометрии в виде полигонов;
   * атрибут с типом **Название** (при этом атрибут может иметь любое название).

2. При загрузке данных отметьте чекбокс **Создать фильтр по территории** и нажмите **Продолжить**.

   <img src="/img/pro/data-territory-filter.png" alt="Создание фильтра по полигонам" width="400"  />

В дальнейшем при фильтрации данных по территориям вы сможете выбирать территории из загруженного набора данных в окне **Настройки геофильтров**:

![Геофильтры с полигонами](/img/pro/data-territory-filter-use.png)

## 4. Создайте слой {#create-a-layer}

Чтобы сохранить отфильтрованную выборку и настроить её [визуализацию](/pro/data/visualization/overview) (например, в виде тепловой карты), создайте слой:

1. Перейдите на вкладку **Главная** или **Данные**.

2. Выберите набор загруженных данных.

3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.

4. Если загруженные данные содержат геометрии в виде линий или полигонов, вверху вкладки **Фильтры** выберите способ отображения данных:

   * **Точки**: в виде точек-центроидов (центров геометрии объектов);
   * **Геометрии**: в виде реальных геометрий данных (линий или полигонов).

5. На вкладке **Фильтры** настройте остальные фильтры при необходимости.

6. Нажмите **Создать слой**.

   Если у вас нет созданных дашбордов, будет создан дашборд с одной сценой, в которую автоматически добавится слой с данными.

7. Если создан хотя бы один дашборд, в диалоговом окне укажите, куда добавить слой:

   * Выберите в списке существующий дашборд и сцену, в которую добавится слой. Нажмите **Создать**.
   * Нажмите **Новый дашборд**, введите название нового дашборда и нажмите **Создать**. Будет создан новый дашборд с одной сценой, в которую автоматически добавится слой с данными.
