# Операции с наборами данных

После загрузки собственных наборов данных вы можете выполнять с ними различные операции. Доступные операции зависят от способа загрузки данных: [в виде файла](#uploaded-as-file) или [по API](#uploaded-via-api).

Вы можете объединять загруженные наборы данных в группы и перемещать их между группами: см. инструкцию [Операции с группами наборов данных](/pro/data/manage/groups).

## Наборы, загруженные как файл {#uploaded-as-file}

В интерфейсе [<Brand /> Про](https://pro.2gis.ru/) вы можете управлять только наборами данных, которые вы [загрузили как файл](/pro/data/files-upload):

* [обновлять наборы данных](#updating-dataset);
* [переименовывать наборы данных](#renaming-dataset);
* [добавлять и изменять описания наборов данных](#editing-description);
* [создавать фильтры по территории](#creating-territory-filter) и [удалять их](#deleting-territory-filter);
* [добавлять объекты в виде точек в набор данных, редактировать или удалять их](#adding-objects);
* [управлять доступом к наборам данных](#access-management);
* [экспортировать данные из наборов](#exporting-data);
* [удалять наборы данных](#deleting-dataset) (доступно также для наборов данных, загруженных по API).

<br />

Для работы с набором данных перейдите на вкладку **Главная** и нажмите **Все данные** или перейдите на вкладку **Данные**.

![Вкладка Данные](/img/pro/data-tab.png)

### Обновление набора данных {#updating-dataset}

:::warning

Загружаемый новый файл полностью заменит исходный.

:::

1. Перейдите на вкладку **Данные**.
2. В карточке нужного набора данных нажмите значок ![Меню набора данных](/img/menu-dots.svg) и выберите **Обновить набор данных**.
3. Перетащите файл в область загрузки или нажмите **Выберите файл на компьютере**. Формат и структура данных (атрибуты, типы и названия полей) загружаемого файла должны соответствовать исходному файлу.
4. Нажмите **Обновить**.
5. После завершения загрузки нажмите **Далее**.
6. (Опционально) Если загруженные данные содержат геометрии в виде полигонов, отметьте чекбокс **Создать фильтр по территории**, чтобы создать фильтр по территориям.
7. Нажмите **Продолжить**.

Во всех слоях и виджетах, которые были созданы из этого набора данных, автоматически отразятся изменения.

### Переименование набора данных {#renaming-dataset}

1. Перейдите на вкладку **Данные**.
2. В карточке нужного набора данных нажмите значок ![Меню набора данных](/img/menu-dots.svg) и выберите **Редактировать**.
3. Введите новое имя набора данных (не более 100 символов).
4. Нажмите **Применить**.

### Изменение описания набора данных {#editing-description}

1. Перейдите на вкладку **Данные**.
2. В карточке нужного набора данных нажмите значок ![Меню набора данных](/img/menu-dots.svg) и выберите **Редактировать**.
3. В поле **Описание** введите текст (не более 200 символов).
4. Нажмите **Применить**.

Если описание слишком длинное, оно не отображается полностью. Чтобы увидеть всплывающую подсказку с полным описанием, наведите курсор на карточку набора данных:

<img src="/img/pro/data-tab-tooltip.png" alt="Всплывающая подсказка с полным описанием" width="300" />

### Создание фильтра по территории {#creating-territory-filter}

Если загруженные данные содержат геометрии в виде полигонов, вы можете использовать их для создания собственного территориального деления. Таким образом вы создадите новые фильтры по территориям (геофильтры) и сможете фильтровать любые данные в <Brand /> Про не только по стандартным видам территорий (области, районы, населённые пункты и другие), но и по вашим уникальным зонам.

1. Перейдите на вкладку **Данные**.
2. В карточке нужного набора данных нажмите значок ![Меню набора данных](/img/menu-dots.svg) и выберите **Создать фильтр по территории**.

В дальнейшем при фильтрации данных по территориям вы сможете выбирать территории из загруженного набора данных в окне **Настройки геофильтров**:

![Геофильтры с полигонами](/img/pro/data-territory-filter-use.png)

### Удаление фильтра по территории {#deleting-territory-filter}

1. Перейдите на вкладку **Данные**.
2. В карточке нужного набора данных нажмите значок ![Меню набора данных](/img/menu-dots.svg) и выберите **Удалить фильтр по территории**.
3. Чтобы подтвердить удаление, нажмите **Удалить**.

### Добавление и редактирование точечных объектов {#adding-objects}

Если вы загрузили набор данных с объектами в виде точек, при создании слоя или после можно добавить новые объекты в набор, изменить или удалить существующие. При этом изменятся данные в наборе. Изменения отразятся во всех выборках, созданных из набора данных, а также в слоях и виджетах.

Линии и полигоны добавить нельзя.

1. Чтобы изменить набор данных:

   <Tabs>
   <TabItem value="during-creation" label="При создании слоя" default>

   1. Перейдите на вкладку **Главная** или **Данные** и выберите набор загруженных данных.
   2. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.

   </TabItem>
   <TabItem value="after-creation" label="После создания слоя">

   2. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.
   3. В верхнем меню дашборда нажмите значок ![Слои](/img/pro/layers-icon.svg) и выберите слой.
   4. Нажмите **Редактировать выборку**.

      <img src="/img/pro/add-object-dashboard.png" alt="Добавление объектов в выборку при работе с созданным слоем" width="600"  />

   </TabItem>
   </Tabs>

2. Чтобы добавить объект:

   1. Перейдите в раздел **Результаты** и нажмите **Добавить объект**.

      ![Добавление объектов в выборку при создании слоя](/img/pro/add-object-layer.png)

   2. Укажите атрибуты данных.

      ![Форма заполнения атрибутов](/img/pro/add-object-dashboard-attributes.png)

      Обязательные поля:

      * **Название:** укажите, если загруженные данные содержат атрибуты такого типа.
      * **Местоположение:** укажите координаты вручную или переставьте значок ![Новый объект](/img/pro/add-object.svg) в нужное место на карте.

      Если при загрузке данных для атрибута был установлен тип фильтра `Мультивыбор из списка`, вы можете ввести его значение вручную или выбрать существующее из списка.

   3. Нажмите **Сохранить**.

3. Чтобы изменить объект:

   1. Перейдите в раздел **Результаты** и выберите объект.
   2. Нажмите **Изменить**. Вы можете отредактировать:

      * Название объекта.
      * Координаты объекта (широту и долготу): введите новые значения или перетащите точку в другое место на карте.
      * Атрибуты данных. Если при загрузке данных для атрибута был установлен тип фильтра `Мультивыбор из списка`, вы можете добавить своё значение.

   3. Сохраните изменения.

4. Чтобы удалить объект:

   1. Перейдите в раздел **Результаты** и выберите нужный объект.
   2. Нажмите **Удалить** и подтвердите удаление.

5. Примените изменения:

   <Tabs>
   <TabItem value="save-during-creation" label="При создании слоя" default>

   Нажмите **Создать слой**.

   </TabItem>
   <TabItem value="save-after-creation" label="При работе с созданным слоем">

   Нажмите одну из кнопок:

   * **Обновить слой**: все изменения будут сохранены, и слой будет обновлен.
   * **Создать слой**: будет создан новый слой, который автоматически добавится на текущую сцену дашборда.

### Управление доступом к набору данных {#access-management}

Вы можете выдать доступ на просмотр или редактирование набора данных другим зарегистрированным пользователям.

В зависимости от уровня доступа, пользователь сможет выполнять разные операции:

* **Просмотр**: просматривать набор данных и создавать из него выборки.
* **Редактирование**: просматривать набор данных, создавать из него выборки, а также выполнять любые операции, доступные владельцу набора: обновлять и переименовывать набор; добавлять и изменять описание набора; создавать фильтры по территории и удалять их; добавлять объекты в виде точек в набор или удалять их; настраивать тултип с атрибутами данных; выдавать, редактировать и удалять доступ к набору для других пользователей; экспортировать набор. При редактировании набора изменения будут отображаться как у владельца, так и у всех пользователей, которым выдан доступ.

  Если пользователь удалит набор данных, к которому ему выдан доступ, он потеряет доступ к этому набору и связанным слоям. Исходный набор данных и доступы других пользователей останутся без изменений. См. подробнее в разделе [Удаление набора данных](#deleting-dataset).

<img src="/img/pro/dataset-access.png" alt="Выдача доступа к набору" width="450" />

Чтобы выдать доступ к набору данных:

1. Перейдите на вкладку **Данные**.
2. В карточке нужного набора данных нажмите значок ![Меню набора данных](/img/menu-dots.svg) и выберите **Поделиться**.
3. Введите адрес электронной почты пользователя, которому нужно выдать доступ, и нажмите **Добавить**.
4. Рядом с адресом пользователя выберите уровень доступа в списке: `Просмотр` или `Редактирование`.
5. Нажмите **Сохранить**.

Чтобы отозвать доступ у пользователя, рядом с его адресом выберите в списке `Удалить`.

Наборы данных, к которым выдан доступ, находятся в профиле получателя на вкладке **Данные** в группе наборов **Мои данные**. У таких наборов в карточке указан тег **В общем доступе** и электронная почта пользователя, который выдал доступ. При необходимости вы можете [перенести набор в другую группу](/pro/data/manage/groups#moving-dataset).

### Экспорт данных {#exporting-data}

Если вы владелец набора данных (вы ранее [загрузили файл с данными](/pro/data/files-upload)) или вам [выдан доступ на редактирование набора данных](#access-management), вы можете экспортировать данные из набора в формате `.csv`. В выгрузку попадают только данные из [выборки](/pro/data/manage/modify).

Если для набора данных с объектами в виде точек вы [добавили новые объекты или удалили существующие](#adding-objects), экспортируются только данные из последней версии набора.

Чтобы экспортировать данные:

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите нужный набор загруженных данных.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. При необходимости примените к данным фильтры по параметрам или измените результаты фильтрации вручную: см. инструкцию [Изменение выборки данных](/pro/data/manage/modify).
5. Нажмите значок ![Экспортировать набор данных](/img/pro/export-dataset.svg).
6. Отметьте чекбоксы напротив нужных строк, чтобы атрибуты попали в итоговую выгрузку.
7. Проверьте результат в режиме предпросмотра. В таблице будет показано только 10 первых строк.
8. Нажмите **Экспорт**.

<img src="/img/pro/dataset-export.png" alt="Экспорт данных" width="1000" />

### Удаление набора данных {#deleting-dataset}

:::warning

Все слои и фильтры по территориям, связанные с набором данных, удаляются без возможности восстановления.

:::

Если вы владелец набора данных и выдали к набору доступ другим пользователям, то после удаления они также потеряют эти данные. Если вы удаляете набор, к которому вам выдали доступ, исходный набор данных и доступы других пользователей не изменятся.

1. Перейдите на вкладку **Данные**.
2. В карточке нужного набора данных нажмите значок ![Меню набора данных](/img/menu-dots.svg) и выберите **Удалить**.
3. Чтобы подтвердить удаление, нажмите **Удалить**.

* [получать параметры наборов данных](#getting-parameters);

<br />

Для отправки запросов из примеров ниже используйте данные, которые вы получили при [создании набора](/pro/data/api-upload/dynamic-upload#create-a-dataset-for-uploading-data):

* ID набора данных (`assetId`). Также вы можете скопировать его в [<Brand /> Про](https://pro.2gis.ru/): перейдите на вкладку **Данные**, в карточке нужного набора данных нажмите значок ![Меню набора данных](/img/menu-dots.svg) и выберите **Скопировать ID**.
* Токен доступа для авторизации (`access_token`). Также вы можете скопировать его в [<Brand /> Про](https://pro.2gis.ru/): нажмите значок ![Профиль](/img/account-icon.svg) в левом нижнем углу, перейдите в раздел **Токены** и скопируйте значение в поле **Токен для работы с динамическими наборами данных**.

### Получение параметров набора данных {#getting-parameters}

Отправьте GET-запрос к `/dynamic_asset` с указанием ID набора данных (параметр `assetId`):

```bash
https://pro-api.2gis.ru/dynamic_asset/{assetId}
```

Подробную информацию о структуре тела ответа см. в [Справочнике API](/pro/data/api-upload/api/reference#/paths/~1dynamic_asset~1{assetId}/get).

### Изменение параметров набора данных {#changing-parameters}

Если вы добавите новый фильтр к существующему атрибуту, фильтрация будет применяться только к новым данным, загруженным после добавления фильтра. Чтобы учесть все данные, добавьте атрибут вместе с новым фильтром.

1. Отправьте PUT-запрос к `/dynamic_asset`:

   ```bash
   https://pro-api.2gis.ru/dynamic_asset
   ```

2. В теле запроса передайте ID набора данных (параметр `assetId`), название набора данных, а также параметры, которые необходимо изменить, в формате JSON.

   Полный список параметров см. в [Справочнике API](/pro/data/api-upload/api/reference#/paths/~1dynamic_asset/put).

### Управление доступом к набору данных {#access-management-api}

Вы можете выдать доступ на просмотр или редактирование набора данных другим зарегистрированным пользователям.

В зависимости от уровня доступа, пользователь сможет выполнять разные операции:

* **Просмотр**: просматривать набор данных, создавать из него выборки и получать параметры набора.
* **Редактирование**: просматривать набор данных; создавать из него выборки; получать и изменять параметры набора; удалять объекты из набора (все или несколько); выдавать, редактировать и удалять доступ к набору для других пользователей; экспортировать набор. При редактировании набора изменения будут отображаться как у владельца, так и у всех пользователей, которым выдан доступ.

  Если пользователь удалит набор данных, к которому ему выдан доступ, он потеряет доступ к этому набору и связанным слоям. Исходный набор данных и доступы других пользователей останутся без изменений. См. подробнее в разделе [Удаление набора данных](#deleting-dataset-api).

<img src="/img/pro/dataset-access.png" alt="Выдача доступа к набору" width="450" />

Чтобы выдать доступ к набору данных:

1. Перейдите на вкладку **Данные**.
2. В карточке нужного набора данных нажмите значок ![Меню набора данных](/img/menu-dots.svg) и выберите **Поделиться**.
3. Введите адрес электронной почты пользователя, которому нужно выдать доступ, и нажмите **Добавить**.
4. Рядом с адресом пользователя выберите уровень доступа в списке: `Просмотр` или `Редактирование`.
5. Нажмите **Сохранить**.
6. Если необходимо отозвать доступ у пользователя, рядом с его адресом выберите в списке `Удалить`.

Наборы данных, к которым выдан доступ, находятся в профиле получателя на вкладке **Данные** в группе наборов **Мои данные**. У таких наборов в карточке указан тег **В общем доступе** и электронная почта пользователя, который выдал доступ. При необходимости вы можете [перенести набор в другую группу](/pro/data/manage/groups#moving-dataset).

### Экспорт данных {#exporting-data-api}

Если вы владелец набора данных (вы ранее [загрузили данные по API](/pro/data/api-upload/dynamic-upload)) или вам [выдан доступ на редактирование набора данных](#access-management-api), вы можете экспортировать данные из набора в формате `.csv`. В выгрузку попадают только данные [выборки](/pro/data/manage/modify).

Если вы [загрузили новые порции данных](/pro/data/api-upload/dynamic-upload#upload-data-to-the-dataset) в набор после его создания или [удалили из него объекты](#deleting-objects), экспортируются только данные из последней версии набора.

Чтобы экспортировать данные:

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите нужный набор загруженных данных.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. При необходимости примените к данным фильтры по параметрам или измените результаты фильтрации вручную: см. инструкцию [Изменение выборки данных](/pro/data/manage/modify).
5. Нажмите значок ![Экспортировать набор данных](/img/pro/export-dataset.svg).
6. Отметьте чекбоксы напротив нужных строк, чтобы атрибуты попали в итоговую выгрузку.
7. Проверьте результат в режиме предпросмотра. В таблице будет показано только 10 первых строк.
8. Нажмите **Экспорт**.

<img src="/img/pro/dataset-export.png" alt="Экспорт данных" width="1000" />

### Удаление нескольких объектов из набора данных {#deleting-objects}

1. Отправьте DELETE-запрос к `/dynamic_asset`:

   ```bash
   https://pro-api.2gis.ru/dynamic_asset/{assetId}/data
   ```

2. В теле запроса передайте ID набора данных (параметр `assetId`) и список ID объектов для удаления в формате JSON.

   Более подробную информацию см. в [Справочнике API](/pro/data/api-upload/api/reference#/paths/~1dynamic_asset~1{assetId}~1data/delete).

### Удаление всех объектов из набора данных {#deleting-all-objects}

Отправьте DELETE-запрос к `/dynamic_asset` с указанием ID набора данных (параметр `assetId`):

```bash
https://pro-api.2gis.ru/dynamic_asset/{assetId}/data/all
```

Более подробную информацию см. в [Справочнике API](/pro/data/api-upload/api/reference#/paths/~1dynamic_asset~1{assetId}~1data~1all/delete).

### Удаление набора данных {#deleting-dataset-api}

:::warning

Все слои, связанные с набором данных, удаляются без возможности восстановления.

:::

Если вы владелец набора данных и выдали к набору доступ другим пользователям, то после удаления они также потеряют эти данные. Если вы удаляете набор, к которому вам выдали доступ, исходный набор данных и доступы других пользователей не изменятся.

1. Перейдите на вкладку **Данные**.
2. В карточке нужного набора данных нажмите значок ![Меню набора данных](/img/menu-dots.svg) и выберите **Удалить**.
3. Чтобы подтвердить удаление, нажмите **Удалить**.
