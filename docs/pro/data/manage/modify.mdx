# Изменение выборки данных

После [создания выборки данных](/pro/data/overview#usage-scenarios) вы можете её изменить:

* Применить к исходным данным геофильтры (фильтры по территориям).
* Применить к исходным данным фильтры по параметрам.
* Изменить результаты фильтрации вручную, убрав определённые объекты.

Изменить выборку можно для наборов [системных данных](/pro/data/overview#built-in-data) и собственных данных: [загруженных как файл](/pro/data/files-upload) и [загруженных по API](/pro/data/api-upload/dynamic-upload).

Если на основе выборки данных созданы слои и виджеты, при изменении выборки в них автоматически отразятся эти изменения. При любых изменениях выборки данные из набора не удаляются.

Также вы можете выполнять с выборками данных другие операции:

* Применить дополнительную фильтрацию к выборкам данных при работе с дашбордом, не изменяя при этом выборку. См. подробнее в инструкции [Фильтрация данных на сцене](/pro/data/manage/filtering).
* Экспортировать выборки из пользовательских наборов данных. См. подробнее в разделе [Экспорт данных](/pro/data/manage/datasets#exporting-data).
* Если вы [загрузили набор данных](/pro/data/files-upload) с объектами в виде точек, вы можете вручную добавить новые точки в набор, отредактировать или удалить существующие. См. подробнее в разделе [Изменение набора данных с точками](/pro/data/manage/datasets#managing-points).

## Изменение выборки {#modifying-sample}

1. Перейдите на вкладку **Дашборды**.

2. Откройте дашборд.

3. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.

4. В верхнем меню дашборда нажмите значок ![Слои](/img/pro/layers-icon.svg) и выберите нужный слой.

5. Нажмите **Редактировать выборку**.

    {/*  prettier-ignore  */}

   <img src="/img/pro/modify-data-dashboard.png" alt="Кнопка Редактировать выборку" width="600" />

6. Чтобы отфильтровать данные по территориям, в окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.

7. Чтобы отфильтровать данные по параметрам, на вкладке **Фильтры** настройте нужные фильтры.

8. Чтобы изменить результаты фильтрации:

   1. Перейдите в раздел **Результаты** и нажмите **Изменить список**.

       {/*  prettier-ignore  */}

      ![Изменение результатов выборки для созданного слоя](/img/pro/modify-data-existing-layer.png)

   2. Выберите объекты, которые нужно оставить в выборке.

   3. Нажмите **Применить**.

9. Чтобы закончить редактирование выборки данных, нажмите одну из кнопок:

   * **Обновить слой**: все изменения будут сохранены, и слой будет обновлен.
   * **Создать слой**: будет создан новый слой, который автоматически добавится на текущую сцену дашборда.
   * **Назад к слою**: изменения не будут сохранены, откроется текущая сцена дашборда.
