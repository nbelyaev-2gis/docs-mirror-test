# Операции с группами наборов данных

Загруженные наборы данных можно объединять в группы и перемещать между ними. В одну группу можно объединить наборы данных, [загруженные как файл](/pro/data/files-upload) и [загруженные по API](/pro/data/api-upload/dynamic-upload).

Наборы [системных данных](/pro/data/overview#built-in-data) нельзя добавить в группу или переместить.

Для работы с группами наборов данных перейдите на вкладку **Главная** и нажмите **Все данные** или перейдите на вкладку **Данные**.

![Вкладка Данные](/img/pro/data-tab.png)

## Создание группы {#creating-a-group}

1. Перейдите на вкладку **Данные**.
2. Нажмите **Создать группу**.
3. Введите имя группы.
4. Нажмите **Создать группу наборов данных**.

Созданная пустая группа появится на вкладке **Данные**.

## Переименование группы {#renaming-a-group}

Вы можете переименовать только созданные вами группы.

1. Перейдите на вкладку **Данные**.
2. В карточке нужной группы наборов данных нажмите значок ![Меню набора данных](/img/menu-dots.svg) и выберите **Переименовать**.
3. Введите новое имя группы.
4. Нажмите ![Чекбокс](/img/pro/checkbox-icon.svg).

## Перемещение набора данных в другую группу {#moving-dataset}

1. Перейдите на вкладку **Данные**.
2. В карточке нужного загруженного набора данных нажмите значок ![Меню набора данных](/img/menu-dots.svg) и выберите **Переместить**.
3. Выберите группу из выпадающего списка **Разместить в группе**.
4. Нажмите **Переместить**.

Набор данных будет отображаться в выбранной группе.

## Удаление группы {#deleting-a-group}

:::warning

Все наборы данных, содержащиеся в группе, удаляются без возможности восстановления.

:::

Вы можете удалить только созданные вами группы.

1. Перейдите на вкладку **Данные**.
2. В карточке нужной группы наборов данных нажмите значок ![Меню набора данных](/img/menu-dots.svg) и выберите **Удалить**.
3. Чтобы подтвердить удаление группы и всех содержащихся в ней наборов данных, нажмите **Удалить**.
