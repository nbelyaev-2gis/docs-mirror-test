# Фильтрация данных на сцене

По умолчанию на сцене дашборда отображаются все данные из выборок, на основе которых созданы слои сцены. Вы можете применить дополнительную фильтрацию данных при работе со сценой и временно скрыть часть данных, не изменяя исходную выборку. Данные можно фильтровать:

* по территориям: [через геофильтры](#via-geofilters);
* по атрибутам данных:

   * [через виджеты](#via-widgets), нажимая на компоненты диаграмм;
   * [через срезы](#via-slices), выбирая несколько фильтров для разных слоёв;

* по дате и времени: [через виджет Временная шкала](#via-timeline).

Когда вы фильтруете данные на сцене, выборки в слоях не изменяются. Обновляется только количество объектов на сцене: на карте, в виджете `Легенда` и других [виджетах](/pro/dashboards/widgets/overview) (кроме текстового). Чтобы изменить выборку данных, см. инструкцию [Изменение выборки данных](/pro/data/manage/modify).

## Фильтрация через геофильтры {#via-geofilters}

Чтобы оставить на сцене дашборда данные только для определённых территорий, примените геофильтры:

1. Перейдите на вкладку **Дашборды** и откройте дашборд.
2. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.
3. Выберите территории одним из способов:

   * В правом верхнем углу сцены нажмите на названия территорий, данные из которых нужно скрыть:

     ![Фильтрация через геофильтры](/img/pro/filtering-geo.png)

   * Нажмите **Геофильтры** и в окне **Настройки геофильтров** укажите территории. Чтобы скрыть фильтр по территории, нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**:

     ![Фильтрация через геофильтры](/img/pro/filtering-geo-select.png)

В результате на сцене останутся данные только для выбранных территорий:

   * Пример сцены до применения геофильтра:

     ![Геофильтры не применены](/img/pro/filtering-geo-widget.png)

   * Пример сцены после применения геофильтра:

     ![Геофильтры применены](/img/pro/filtering-geo-widget-after.png)

## Фильтрация через виджеты {#via-widgets}

Чтобы отфильтровать данные по значениям атрибутов, нажмите на компоненты диаграмм (столбцы или секторы) или элементы легенды в виджетах.

<img src="/img/pro/widgets-filter-example.png" alt="Пример виджета с применённым фильтром" width="350"  />

Вы можете применить фильтрацию в [одном слое](#for-one-layer) или в [нескольких слоях](#for-multiple-layers) сразу. Также можно выбрать несколько значений и виджетов одновременно: подробнее см. в разделе [Сочетание условий фильтрации](#combining-filter-conditions).

Фильтрация доступна через виджеты:

* [линейчатая диаграмма](/pro/dashboards/widgets/bar-chart),
* [столбчатая диаграмма](/pro/dashboards/widgets/column-chart),
* [гистограмма](/pro/dashboards/widgets/histogram),
* [круговая диаграмма](/pro/dashboards/widgets/pie-chart).

:::warning

Если вы примените к слою  [фильтрацию через срезы](#via-slices), в нём сбросится фильтрация через виджеты.

:::

### Фильтрация в одном слое {#for-one-layer}

При фильтрации данных в одном слое:

* Данные отфильтруются только в этом слое: на карте, в `Легенде` и других виджетах из этого слоя.
* Другие слои и виджеты из других слоёв не изменятся.
* В виджете с применённым фильтром появится значок ![Фильтр](/img/pro/filter-icon.svg).
* В списке слоёв рядом со слоем с применённым фильтром появится значок ![Применён фильтр](/img/pro/filter-icon-fade.svg).

**Пример фильтрации**

На сцене отображаются все данные слоя `Кафе-кондитерские`:

   ![Фильтрация через виджеты не применена](/img/pro/widgets-filter-one-layer.png)

Чтобы оставить только кафе-кондитерские c рейтингом от 4 до 5, в виджете `Рейтинг`, созданном из слоя `Кафе-кондитерские`, нажмите на диапазон оценок 4–5 (столбец на диаграмме).

В результате данные отфильтруются по атрибуту `Рейтинг`. Останутся кафе-кондитерские только с рейтингом от 4 до 5:

   * на карте; 
   * в виджетах `Рейтинг` и `Количество отзывов`, созданных из слоя `Кафе-кондитерские`;
   * в виджете `Легенда`.

   ![Применён фильтр по рейтингу](/img/pro/widgets-filter-one-layer-result.png)

Чтобы удалить фильтр, в виджете нажмите значок ![Фильтр](/img/pro/filter-icon.svg) или нажмите на выбранный компонент диаграммы.

### Фильтрация в нескольких слоях {#for-multiple-layers}

Вы можете применить фильтр к нескольким слоям на сцене дашборда одновременно. Фильтр применится:

* К слоям, созданным [из этого набора данных](#from-same-dataset).
* К слоям, созданным из других наборов данных, если эти наборы содержат выбранный [сквозной атрибут](#global-attributes). Например, системные наборы данных [Динамика организаций](/pro/data/built-in/dynamics), [Карта спроса](/pro/data/built-in/demand) и [Фирмы](/pro/data/built-in/firms) содержат сквозной атрибут `Категория`.

#### Из одного набора данных {#from-same-dataset}

При фильтрации данных в нескольких слоях из одного набора данных:

* Данные отфильтруются только в слоях из этого набора данных: на карте, в `Легенде` и других виджетах из этих слоёв.
* Слои, созданные из других наборов данных, и виджеты из этих слоёв не изменятся (только если не выбран [сквозной атрибут](#global-attributes)).
* В виджете с применённым фильтром появится значок ![Фильтр ко всем слоям](/img/pro/filter-all-icon.svg).
* В списке слоёв рядом со слоями с применённым фильтром появится значок ![Применён фильтр](/img/pro/filter-icon-fade.svg).

**Пример фильтрации**

На сцене отображаются все данные двух слоёв `Гостиницы` и `Кафе-кондитерские` из одного набора данных **Фирмы**:

   ![Дашборд с двумя слоями из одного набора данных](/img/pro/widgets-filter-two-layers.png)

Чтобы оставить только кафе-кондитерские и гостиницы с рейтингом от 4 до 5:

1. В виджете `Рейтинг кафе-кондитерских`, созданном из слоя `Кафе-кондитерские`, нажмите на диапазон оценок 4–5 (столбец на диаграмме). По умолчанию фильтр применится только к этому слою:

   ![Фильтр применён только к одному слою](/img/pro/widgets-filter-two-layers-applied.png)

2. В виджете `Рейтинг кафе-кондитерских` откройте выпадающий список рядом со значком ![Фильтр](/img/pro/filter-icon.svg) и выберите **Применить ко всем слоям**. В списке отобразятся все слои, к которым применится фильтр:

   <img src="/img/pro/widgets-filter-two-layers-apply-all-520.png" alt="Применить фильтр ко всем слоям" width="300"  />

В результате фильтр `Рейтинг` применится к обоим слоям, так как они созданы из одного набора данных. Останутся только кафе-кондитерские и гостиницы с рейтингом от 4 до 5:

   * на карте;
   * в виджетах `Рейтинг кафе-кондитерских` и `Количество отзывов`, созданных из слоя `Кафе-кондитерские`;
   * в виджете `Рейтинг гостиниц`, созданном из слоя `Гостиницы`;
   * в виджете `Легенда`.

   ![Фильтр применён ко всем слоям из одного набора данных](/img/pro/widgets-filter-two-layers-result.png)

Чтобы применить фильтр только к исходному слою, в виджете откройте выпадающий список рядом со значком ![Фильтр](/img/pro/filter-all-icon.svg) и выберите **Только к исходному слою**. Чтобы удалить фильтр, выберите **Сбросить фильтр** или нажмите на выбранный компонент диаграммы.

#### По сквозным атрибутам {#global-attributes}

При фильтрации данных в нескольких слоях со сквозными атрибутами:

* Данные отфильтруются только в слоях из наборов данных, содержащих сквозной атрибут: на карте, в `Легенде` и других виджетах из этих слоёв.
* Слои, созданные из наборов данных без сквозного атрибута, и виджеты из этих слоёв не изменятся.
* В виджете с применённым фильтром появится значок ![Фильтр ко всем слоям](/img/pro/filter-all-icon.svg).
* В списке слоёв рядом со слоями с применённым фильтром появится значок ![Применён фильтр](/img/pro/filter-icon-fade.svg).

**Пример фильтрации**

На сцене отображаются все данные слоя `Спрос на заведения` из набора данных **Карта спроса** и слоя `Заведения` из набора **Фирмы**. Эти наборы содержат сквозной атрибут `Категории`:

   ![Дашборд с двумя слоями из разных наборов данных](/img/pro/widgets-filter-two-layers-global.png)

Чтобы оставить только кафе-кондитерские в обоих слоях:

1. В виджете `Спрос`, созданном из слоя `Спрос на заведения`, нажмите на значение `Кафе-кондитерские` (столбец на диаграмме). По умолчанию фильтр применится только к этому слою:

   ![Фильтр применён только к одному слою](/img/pro/widgets-filter-two-layers-applied-global.png)

2. В виджете `Спрос` откройте выпадающий список рядом со значком ![Фильтр](/img/pro/filter-icon.svg) и выберите **Применить ко всем слоям**. В списке отобразятся все слои, к которым применится фильтр:

   <img src="/img/pro/widgets-filter-two-layers-apply-all-global-520.png" alt="Применить фильтр ко всем слоям" width="300"  />

В результате фильтр `Категории` со значением `Кафе-кондитерские` применится к обоим слоям, так как это их общий сквозной атрибут. Останутся только кафе-кондитерские:

   * на карте;
   * в виджетах `Рейтинг заведений` и `Количество отзывов`, созданных из слоя `Заведения`;
   * в виджете `Спрос`, созданном из слоя `Спрос на заведения`;
   * в виджете `Легенда`.

   ![Фильтр применён ко всем слоям из разных наборов данных](/img/pro/widgets-filter-two-layers-result-global.png)

Чтобы применить фильтр только к исходному слою, в виджете откройте выпадающий список рядом со значком ![Фильтр](/img/pro/filter-all-icon.svg) и выберите **Только к исходному слою**. Чтобы удалить фильтр, выберите **Сбросить фильтр** или нажмите на выбранный компонент диаграммы.

### Сочетание условий фильтрации {#combining-filter-conditions}

* Вы можете выбрать более одного компонента на диаграмме. В этом случае условия фильтрации объединяются логическим оператором `ИЛИ`. Например, если выбрать два диапазона среднего чека (600 и 650) в виджете **Средний чек**, то на карте и в других виджетах останутся кафе-кондитерские, которые отвечают любому из условий (со средним чеком 600 *или* 650 рублей).

  ![Применён фильтр по двум диапазонам одного виджета](/img/pro/widgets-filter-two-ranges.png)

* Вы можете фильтровать данные через несколько виджетов одновременно. В этом случае условия фильтрации объединяются логическим оператором `И`. Например, если выбрать размер среднего чека 600 в виджете **Средний чек** и диапазон оценок ≥5 в виджете **Рейтинг заведений**, на карте и в других виджетах останутся только кафе-кондитерские, которые отвечают обоим условиям (со средним чеком 600 *и* рейтингом ≥5).

  Если виджеты относятся к разным слоям, см. детали в разделе [Фильтрация в нескольких слоях](#for-multiple-layers).

  ![Применён фильтр по диапазонам из двух виджетов](/img/pro/widgets-filter-two-widgets.png)

## Фильтрация через срезы {#via-slices}

Чтобы отфильтровать данные по значениям атрибутов, используйте виджет **Фильтр-срез**.

<img src="/img/pro/widgets-slicer-example.png" alt="Пример виджета Фильтр-срез" width="700" />

### Основные особенности {#main-features}

* Вы можете применить дополнительные фильтры как в одном слое, так и в нескольких слоях сразу с помощью сквозных атрибутов. Например, системные наборы данных [Динамика организаций](/pro/data/built-in/dynamics), [Карта спроса](/pro/data/built-in/demand) и [Фирмы](/pro/data/built-in/firms) содержат сквозной атрибут `Категория`.
* В фильтр-срезе доступны только значения атрибутов из исходной выборки. Например, у фильтра **Материал** есть значения `Кирпич`, `Бетон`, `Металл` и `Дерево`. При создании или изменении выборки данных вы применили фильтр и выбрали значения `Кирпич` и `Дерево`. С помощью виджета вы сможете фильтровать здания только по этим двум значениям. Например, применить срез `Кирпич`, чтобы оставить только кирпичные здания.
* В срезах могут отображаться значения фильтров, которые вы не указывали при создании слоя или изменении выборки. Например, вы выбрали для фильтра `Способ оплаты` в исходной выборке слоя только значение `Оплата картой`. В виджете **Фильтр-срез** для этого фильтра могут появиться значения `Наличный расчёт`, `Оплата через приложение` и другие, т. к. одна организация может поддерживать несколько способов оплаты.
* В фильтр-срезе доступен поиск по названию слоёв и атрибутов.
* Если фильтр относится к одному слою, при фильтрации данных через срезы:

  * Данные отфильтруются только в этом слое: на карте, в `Легенде` и других виджетах из этого слоя.
  * Другие слои и виджеты из других слоёв не изменятся.

* Если выбран сквозной фильтр (общий для нескольких слоёв):

  * Данные отфильтруются только в слоях из наборов данных, содержащих сквозной атрибут: на карте, в `Легенде` и других виджетах из этих слоёв.
  * Слои, созданные из наборов данных без сквозного атрибута, и виджеты из таких слоёв не изменятся.

* В списке слоёв рядом со слоями с применённым фильтром появится значок ![Применён фильтр](/img/pro/filter-icon-fade.svg).

:::warning

Если изменить значения фильтра в исходной выборке слоя, к которому применены срезы, фильтр на сцене сбросится. Если вы примените к слою  фильтрацию через срезы, в нём сбросится [фильтрация через виджеты](#via-widgets).

:::

### Пример фильтрации

На сцене отображаются все данные слоя `Спрос на заведения` из набора данных **Карта спроса** и слоя `Заведения` из набора **Фирмы**. Эти наборы содержат сквозной атрибут `Категории`:

   ![Дашборд с двумя слоями из разных наборов данных](/img/pro/widgets-filter-two-layers-global.png)

Чтобы оставить только кафе-кондитерские в обоих слоях:

1. Нажмите значок ![Фильтр-срез](/img/pro/slicer-icon.svg) в левом нижнем углу сцены.

2. Нажмите **Добавить фильтр**.

3. Выберите фильтры:

   * `Категории`: укажите значение `Кафе-кондитерские`.
   * `Число сотрудников`: укажите значение `до 15`.

     <img src="/img/pro/widgets-filter-slices-apply.png" alt="Применить срезы" width="600"  />

В результате:

   * Фильтр `Категории` со значением `Кафе-кондитерские` применится к обоим слоям, так как это их общий сквозной атрибут. Останутся только кафе-кондитерские:

     * на карте;
     * в виджетах `Рейтинг заведений` и `Количество отзывов`, созданных из слоя `Заведения`;
     * в виджете `Спрос`, созданном из слоя `Спрос на заведения`;
     * в виджете `Легенда`.

   * Фильтр `Число сотрудников` применится только к слою `Заведения` и не повлияет на слой `Спрос на заведения`, так как это не сквозной атрибут. Останутся только кафе-кондитерские с числом сотрудников до 15:
   
      * на карте;
      * в виджетах `Рейтинг заведений` и `Количество отзывов`, созданных из слоя `Заведения`;
      * в виджете `Легенда`.

   ![Срезы применены](/img/pro/widgets-filter-slices-result.png)

Чтобы сбросить фильтр, откройте виджет **Фильтр-срез** и нажмите значок ![Фильтр](/img/pro/filter-all-icon.svg) рядом с именем фильтра. Чтобы удалить фильтр, нажмите значок ![Меню фильтра](/img/menu-dots.svg) рядом с его именем и выберите **Удалить** или снимите чекбокс у фильтра в списке.

## Фильтрация через временную шкалу {#via-timeline}

Если вы [загрузили файл с данными](/pro/data/files-upload), которые содержат атрибут с типом `Дата`, вы можете фильтровать данные из выборки по этому атрибуту с помощью виджета **Временная шкала**. На сцене дашборда отобразятся данные за выбранный период. Подробнее см. в инструкции [Временная шкала](/pro/dashboards/widgets/timeline).
