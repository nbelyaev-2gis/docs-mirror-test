# Парковки

В наборе данных **Парковки** представлена информация о парковках и их расположении.

![Парковки](/img/pro/data-parkings-overview.png)

## Сценарии использования {#usage-scenarios}

Набор данных **Парковки** помогает решать задачи:

* **Выбор локации для открытия бизнеса**. Помогает по данным о наличии и типах парковок выбирать места для открытия бизнеса, ориентированного на водителей: автосервисов, автокафе, АЗС и т. д.
* **Оценка доступности парковки для клиентов**. Позволяет медицинским учреждениям, коворкингам, банкам и другим организациям фильтровать парковки с доступом `Общедоступная`, чтобы оценить, насколько удобно добираться до места на личном транспорте.
* **Поиск зон с малым количеством парковок**. Помогает органам власти и застройщикам выявлять районы с малым количеством парковок и использовать данные для планирования новых объектов.

## Как были получены данные? {#data-collection}

Специалисты <Brand /> регулярно собирают и проверяют данные: см. подробнее на странице [О данных <Brand />](https://dev.2gis.ru/company).

## Обновление данных и историчность {#data-update-and-history}

* **Частота обновления**: данные обновляются один раз в неделю.
* **Историчность**: исторические данные недоступны. Вы можете просматривать только последние актуальные данные.

## Доступные территории {#available-territories}

Данные доступны для следующих стран:

* Азербайджан,
* Армения,
* Беларусь,
* Грузия,
* Казахстан,
* Кыргызстан,
* Россия,
* Узбекистан.

:::note[Ограниченная доступность на территории]

Данные могут быть доступны не на всей территории страны, а только в населённых пунктах, в которых присутствует компания <Brand />. Чтобы уточнить доступность данных, используйте [геофильтры](/pro/data/overview#geofilters) или обратитесь в [техническую поддержку](mailto:support.pro@2gis.ru).

:::

## Доступность данных {#data-availability}

* **Облако** (сервис [<Brand /> Про](https://pro.2gis.ru/)):

  * **Демо-тариф**: данные недоступны. Чтобы начать работать с данными, [смените тариф](/pro/prices) и запросите доступ к набору данных.
  * **Другие тарифы**: данные доступны по запросу.

* **On-Premise** (установка сервиса в [закрытом контуре](/on-premise-pro/architecture/pro-architecture)): данные доступны.

## Начало работы {#getting-started}

![Парковки](/img/pro/data-parkings.png)

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите набор данных **Парковки**.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. На вкладке **Фильтры** настройте остальные [фильтры](#filters) при необходимости.

<br />

Данные будут отображены на карте в виде точек. Результаты фильтрации доступны в двух вкладках:

* **Статистика**: общие статистические данные о выборке:

  * Общее количество парковок.
  * Диаграмма с распределением парковок по уровню расположения. Отображается, если выбрано более одного типа.

* **Результаты**: общая информация о парковках и их список. При выборе парковки из списка:

  * Откроется карточка с информацией о парковке.
  * На карте будет указана точка, в которой находится парковка.

  Вы можете изменить вручную список результатов фильтрации, чтобы оставить в выборке только выбранные парковки. Для этого нажмите **Изменить список**, выберите парковки и нажмите **Применить**.

Чтобы сохранить выборку, нажмите **Создать слой** и настройте [визуализацию данных](#data-visualization).

## Фильтры {#filters}

Для набора данных **Парковки** доступны следующие фильтры:

* **Назначение**: тип транспорта, для которого предназначены парковочные места (например, `Для автомобилей`, `Для мотоциклов/мопедов` и т. д.).
* **Доступ**: доступность парковочных мест для разных групп населения (например, `Общедоступная`, `Для резидентов`, `Для такси` и т. д.).
* **Вместимость**: количество парковочных мест.
* **Платная**: выберите, чтобы отобразить только платные парковки.
* **Бесплатная**: выберите, чтобы отобразить только бесплатные парковки.
* **Тёплая**: выберите, чтобы отобразить только отапливаемые парковки.
* **Перехватывающая**: выберите, чтобы отобразить только перехватывающие парковки.
* **Охраняемая**: выберите, чтобы отобразить только охраняемые парковки.
* **Освещённая**: выберите, чтобы отобразить только парковки с освещением.
* **Для грузовиков**: выберите, чтобы отобразить только парковки, предназначенные для грузового транспорта.
* **Есть трансфер**: выберите, чтобы отобразить только парковки с услугой трансфера.
* **Уровень**: уровень парковки относительно земли (например, `Наземная`, `Подземная`, `Многоуровневая`).
* **Число уровней**: количество уровней на парковке. Для парковок с уровнем `Подземная` значение равно `1`.
* **Тип покрытия**: тип покрытия на парковке (например, `Асфальт`, `Без покрытия` и т. д.).

Чтобы сбросить фильтр по параметру, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с именем фильтра. Чтобы сбросить все фильтры по параметрам, нажмите **Очистить фильтры** внизу вкладки.

Чтобы скрыть фильтр по территории, в окне **Настройки геофильтров** нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.

Для набора данных доступны следующие способы визуализации:

* [Точка](/pro/data/visualization/point),

После [создания слоя](#getting-started) настройте базовую визуализацию данных:

1. Выберите способ визуализации **Точка**:

   <img src="/img/pro/data-parkings-visualization.png" alt="Выбор способа визуализации" width="650" />

2. В поле **Иконка** выберите иконку для отображения точек на карте:

3. Чтобы проанализировать расположение парковок, измените цвет точек на карте в зависимости от вместимости парковки. В поле **Цвет на основе** выберите атрибут `Вместимость` и укажите цветовую палитру:

   ![Настройка цвета](/img/pro/data-parkings-color.png)

## Покупка данных {#data-purchasing}

Вы можете купить данные отдельно в виде выгрузки в следующих форматах:

* Табличные форматы:
   * `.csv`;
   * `.txt`;
   * `.json`;
* Форматы для хранения географических данных:
   * `.shp` (Shape);
   * `.gpkg` (GeoPackage);
   * `.geojson` (GeoJSON).

Чтобы узнать стоимость, доступные территории и приобрести данные, заполните форму на [dev.2gis.ru](https://dev.2gis.ru/data).
