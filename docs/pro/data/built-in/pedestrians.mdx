# Пешеходный трафик

В наборе данных **Пешеходный трафик** представлена информация о пешеходном трафике на улицах крупных городов.

![Пешеходный трафик](/img/pro/data-pedestrians-overview.png)

## Сценарии использования {#usage-scenarios}

Набор данных **Пешеходный трафик** помогает решать задачи:

* **Выбор локации для открытия бизнеса**. Помогает выбрать оптимальное место с высоким потоком пешеходов для открытия бизнеса, ориентированного на спонтанные покупки: кафе или кофейню с напитками на вынос, киоск, павильон и т. д.
* **Наружная реклама**. Помогает выбрать места с большой проходимостью для установки рекламных конструкций и экранов и охватить максимальное количество пешеходов.
* **Транспортное планирование и строительство дорог**. Позволяет определить загруженность тротуаров и пешеходных переходов, а также принять решение о расширении инфраструктуры, капитальном ремонте, установке навесов и освещения.

## Как были получены данные? {#data-collection}

Пешеходный трафик моделируется на основе повседневной активности жителей городов: перемещений от дома к месту работы и от дома до мест ежедневного пользования (точек притяжения) и обратно. На улично-дорожном графе строятся маршруты от зданий до точек притяжения с учётом численности населения в зданиях. Также учитываются активность и модели поведения разных возрастных групп.

Алгоритм вычисления пешеходного трафика состоит из следующих шагов:

1. Группировка населения в зависимости от ежедневной активности на основе данных Росстата: моложе трудоспособного возраста, трудоспособного возраста, старше трудоспособного возраста.
2. Группировка точек притяжения, до которых жители чаще всего добираются пешком, по дням недели и времени суток.
3. Построение 15-минутных пешеходных зон доступности от зданий с жилым и работающим населением и определение объектов в рамках этих зон.
4. Построение маршрутов от зданий до ближайших точек притяжения для каждой группы в пределах зон доступности.
5. Подсчёт трафика по рёбрам улично-дорожного графа.

## Обновление данных и историчность {#data-update-and-history}

* **Частота обновления**: данные обновляются регулярно. Для большинства городов данные обновлялись в 2025 году.
* **Историчность**: исторические данные недоступны.

## Доступные территории {#available-territories}

Данные доступны для следующих стран и городов:

* Казахстан: Актобе, Актау, Атырау, Караганда, Кокшетау, Костанай, Кызылорда, Павлодар, Петропавловск, Талдыкорган, Тараз, Уральск, Усть-Каменогорск, Шымкент.
* Кыргызстан: Бишкек.
* Россия: города с населением более 500 000 человек.
* Узбекистан: Ташкент.

Чтобы уточнить доступность данных, используйте [геофильтры](/pro/data/overview#geofilters) или обратитесь в [техническую поддержку](mailto:support.pro@2gis.ru).

## Доступность данных {#data-availability}

* **Облако** (сервис [<Brand /> Про](https://pro.2gis.ru/)):

  * **Демо-тариф**: данные недоступны. Чтобы начать работать с данными, [смените тариф](/pro/prices) и запросите доступ к набору данных.
  * **Другие тарифы**: данные доступны по запросу.

* **On-Premise** (установка сервиса в [закрытом контуре](/on-premise-pro/architecture/pro-architecture)): данные недоступны.

## Начало работы {#getting-started}

![Пешеходный трафик](/img/pro/data-pedestrians.png)

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите набор данных **Пешеходный трафик (детальный)**.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. На вкладке **Фильтры** выберите базовые фильтры для анализа данных:

   * Чтобы посмотреть данные о трафике в будние дни, выберите фильтр **Трафик за будний день**.
   * Чтобы посмотреть данные о трафике в выходные дни, выберите фильтр **Трафик за выходной день**.

5. Настройте остальные [фильтры](#filters) при необходимости.

<br />

Данные будут отображены на карте. В результатах фильтрации доступны:

* Общее количество объектов.
* Диаграмма с распределением пешеходного трафика.

Чтобы сохранить выборку, нажмите **Создать слой** и настройте [визуализацию данных](#data-visualization).

## Фильтры {#filters}

Для набора данных **Пешеходный трафик** доступны следующие фильтры:

* **Трафик за будний день**: количество пешеходов за типовой будний день.
* **Трафик за выходной день**: количество пешеходов за типовой выходной день.
* **Трафик за 1 час буднего дня**: количество пешеходов за один час типового буднего дня.
* **Трафик за 1 час выходного дня**: количество пешеходов за один час типового выходного дня.

Чтобы сбросить фильтр по параметру, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с именем фильтра. Чтобы сбросить все фильтры по параметрам, нажмите **Очистить фильтры** внизу вкладки.

Чтобы скрыть фильтр по территории, в окне **Настройки геофильтров** нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.

## Визуализация данных {#data-visualization}

Для набора данных доступен только способ визуализации [Линия](/pro/data/visualization/line), так как информация о трафике привязана к сегментам дорог.

После [создания слоя](#getting-started) настройте базовую визуализацию данных:

1. Выберите способ визуализации **Линия**:

   <img src="/img/pro/data-pedestrians-visualization.png" alt="Выбор способа визуализации" width="650" />

2. В поле **Тип шкалы** выберите значение **Логарифмическая**:

   <img src="/img/pro/data-pedestrians-scale.png" alt="Выбор типа шкалы" width="650" />

3. Чтобы проанализировать дорожную сеть, измените цвет линий на карте в зависимости от трафика. В поле **Цвет на основе** выберите атрибут `Трафик за будний день` и укажите цветовую палитру:

   ![Настройка цвета](/img/pro/data-pedestrians-color.png)

## Покупка данных {#data-purchasing}

Вы можете купить данные отдельно в виде выгрузки в следующих форматах:

* Табличные форматы:
   * `.csv`;
   * `.txt`;
   * `.json`;
* Форматы для хранения географических данных:
   * `.shp` (Shape);
   * `.gpkg` (GeoPackage);
   * `.geojson` (GeoJSON).

Чтобы узнать стоимость, доступные территории и приобрести данные, заполните форму на [dev.2gis.ru](https://dev.2gis.ru/data).
