# Терминалы и банкоматы

В наборе данных **Терминалы и банкоматы** представлена информация о местах оплаты услуг, не требующих присутствия обслуживающего персонала (банкоматах, платёжных терминалах и т. д.).

![Терминалы и банкоматы](/img/pro/terminals-overview.png)

## Сценарии использования {#usage-scenarios}

Набор данных **Терминалы и банкоматы** помогает решать задачи:

* **Оптимизация сети банкоматов и терминалов**. Помогает банкам анализировать плотность и географическое распределение банкоматов и терминалов, чтобы сокращать малоиспользуемые и устанавливать новые в районах с высоким спросом.

* **Выбор локаций для новых платёжных терминалов**. Помогает операторам платёжной инфраструктуры определять оптимальные места для установки терминалов, учитывая спрос на оплату услуг в зонах с высокой проходимостью и низкой конкуренцией, например, в торговых центрах, на вокзалах и рынках.

* **Рыночный анализ конкурентов для финансовых услуг**. Помогает банкам и финтех-компаниям отслеживать активность конкурентов с помощью фильтров по организациям. Например, выявлять новые точки установки терминалов и анализировать наличие устройств в отделениях банков.

## Как были получены данные? {#data-collection}

Специалисты <Brand /> регулярно собирают и проверяют данные: см. подробнее на странице [О данных <Brand />](https://dev.2gis.ru/company).

## Обновление данных и историчность {#data-update-and-history}

* **Частота обновления**: данные обновляются один раз в неделю.
* **Историчность**: исторические данные недоступны. Вы можете просматривать только последние актуальные данные.

## Доступные территории {#available-territories}

Данные доступны для следующих стран:

* Азербайджан,
* Армения,
* Беларусь,
* Казахстан,
* Кыргызстан,
* Россия,
* Узбекистан.

:::note[Ограниченная доступность на территории]

Данные могут быть доступны не на всей территории страны, а только в населённых пунктах, в которых присутствует компания <Brand />. Чтобы уточнить доступность данных, используйте [геофильтры](/pro/data/overview#geofilters) или обратитесь в [техническую поддержку](mailto:support.pro@2gis.ru).

:::

## Доступность данных {#data-availability}

* **Облако** (сервис [<Brand /> Про](https://pro.2gis.ru/)):

  * **Демо-тариф**: данные недоступны. Чтобы начать работать с данными, [смените тариф](/pro/prices) и запросите доступ к набору данных.
  * **Другие тарифы**: данные доступны по запросу.

* **On-Premise** (установка сервиса в [закрытом контуре](/on-premise-pro/architecture/pro-architecture)): данные доступны.

## Начало работы {#getting-started}

![Терминалы и банкоматы](/img/pro/data-terminals.png)

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите набор данных **Терминалы и банкоматы**.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. На вкладке **Фильтры** настройте остальные [фильтры](#filters) при необходимости.

<br />

Данные будут отображены на карте в виде точек. Результаты фильтрации доступны в двух вкладках:

* **Статистика**: общие статистические данные о выборке:

  * Общее количество объектов.
  * Диаграмма с распределением объектов по категориям. Отображается, если выбрано несколько категорий.
  * Максимальное и среднее количество филиалов.

* **Результаты**: общая информация об объектах и их список. При выборе объекта из списка:

  * Откроется карточка с информацией об объекте.
  * На карте будет указана точка, в которой находится объект.

  Вы можете изменить вручную список результатов фильтрации, чтобы оставить в выборке только выбранные объекты. Для этого нажмите **Изменить список**, выберите объекты и нажмите **Применить**.

Чтобы сохранить выборку, нажмите **Создать слой** и настройте [визуализацию данных](#data-visualization).

## Фильтры {#filters}

Для набора данных **Терминалы и банкоматы** доступны следующие фильтры:

* **Категории**: категории объектов. Вы можете выбрать несколько категорий.
* **Фирма**: поиск организаций, которые владеют терминалами или банкоматами, по названию.
* **Правовая форма**: организационно-правовая форма организации.
* **Способы оплаты**: способы оплаты на объекте.
* **Язык обслуживания**: язык обслуживания на объекте.
* **Круглосуточно**: выберите, чтобы отобразить объекты только с круглосуточным графиком работы.
* **Рейтинг**: рейтинг объекта от `1` до `5`.
* **Количество отзывов**: количество отзывов об объекте.
* **Бонусы Спасибо**: возможность накопить или списать бонусы Спасибо.
* **Дополнительные фильтры**: при выборе некоторых категорий будут доступны дополнительные фильтры. Например, для подкатегории `Банкоматы` доступны фильтры:
  * **Доступный вход для людей с инвалидностью**: организации с доступным входом в здание.
  * **Доступная среда**: тип приспособления для доступа в здание.

Чтобы сбросить фильтр по параметру, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с именем фильтра. Чтобы сбросить все фильтры по параметрам, нажмите **Очистить фильтры** внизу вкладки.

Чтобы скрыть фильтр по территории, в окне **Настройки геофильтров** нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.

## Покупка данных {#data-purchasing}

Вы можете купить данные отдельно в виде выгрузки в следующих форматах:

* Табличные форматы:
   * `.csv`;
   * `.txt`;
   * `.json`;
* Форматы для хранения географических данных:
   * `.shp` (Shape);
   * `.gpkg` (GeoPackage);
   * `.geojson` (GeoJSON).

Чтобы узнать стоимость, доступные территории и приобрести данные, заполните форму на [dev.2gis.ru](https://dev.2gis.ru/data).
