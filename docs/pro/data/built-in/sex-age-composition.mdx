# Половозрастной состав населения

В наборе данных **Половозрастной состав населения** представлена информация о распределении населения, проживающего в жилых домах, по полу и возрасту.

![Половозрастной состав населения](/img/pro/data-sex-age-overview.png)

## Сценарии использования {#usage-scenarios}

Набор данных **Половозрастной состав населения** помогает решать задачи:

* **Открытие бизнеса для целевой аудитории**. Помогает оценить численность целевой группы в районе и выбрать локацию. Например, узнать количество женщин трудоспособного возраста для открытия салона красоты или выбрать место с высокой плотностью пожилого населения для аптеки.
* **Оценка обеспеченности социальной инфраструктурой**. Позволяет анализировать количество детских садов, школ и поликлиник по возрастным группам.
* **Планирование жилого строительства и социальной инфраструктуры**. Помогает оценить структуру населения по полу и возрасту в зоне застройки и планировать строительство дворов и учебных учреждений.

## Как были получены данные? {#data-collection}

Алгоритм сбора данных состоит из следующих шагов:

1. Получение и обработка данных Росстата о половозрастной структуре населения по муниципальным образованиям и регионам.
2. Сопоставление муниципальных образований Росстата с данными <Brand />.
3. Расчёт распределения населения по полу и возрасту для каждого муниципального образования.
4. Расчёт распределения населения по каждому жилому зданию с учётом ограничений:

   * Должно сохраняться соотношение по полу и общая численность населения, как в данных Росстата по муниципальным образованиям.
   * В каждом доме обязательно должен быть житель старше 18 лет.
   * Разница в возрасте между проживающим взрослым и проживающим ребёнком должна быть не менее 16 лет.
   * Если нет данных по муниципальному образованию, должно применяться распределение по региону.

## Обновление данных и историчность {#data-update-and-history}

* **Частота обновления**: данные обновляются один раз в месяц.
* **Историчность**: исторические данные недоступны.

## Доступные территории {#available-territories}

Данные доступны только для России.

:::note[Ограниченная доступность на территории]

Данные могут быть доступны не на всей территории страны, а только в населённых пунктах, в которых присутствует компания <Brand />. Чтобы уточнить доступность данных, используйте [геофильтры](/pro/data/overview#geofilters) или обратитесь в [техническую поддержку](mailto:support.pro@2gis.ru).

:::

## Доступность данных {#data-availability}

* **Облако** (сервис [<Brand /> Про](https://pro.2gis.ru/)):

  * **Демо-тариф**: данные недоступны. Чтобы начать работать с данными, [смените тариф](/pro/prices) и запросите доступ к набору данных.
  * **Другие тарифы**: данные доступны по запросу.

* **On-Premise** (установка сервиса в [закрытом контуре](/on-premise-pro/architecture/pro-architecture)): данные недоступны.

## Начало работы {#getting-started}

![Половозрастной состав населения](/img/pro/data-sex-age-composition.png)

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите набор данных **Половозрастной состав населения**.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. На вкладке **Фильтры** настройте остальные [фильтры](#filters) при необходимости.

<br />

Данные будут отображены на карте в виде точек-центроидов: центров геометрий объектов. В результатах фильтрации доступны:

* Общее количество проживающего населения.
* Диаграмма с распределением населения по полу. Отображается, если выбраны оба пола.
* Половозрастная пирамида с распределением населения по полу и возрасту. На диаграмме в виде горизонтальных столбцов показана численность населения в зависимости от возраста. В левой части показаны данные для мужчин, а в правой — для женщин. Отображается, если выбраны оба пола.

Чтобы сохранить выборку, нажмите **Создать слой** и настройте [визуализацию данных](#data-visualization).

## Фильтры {#filters}

Для набора данных **Половозрастной состав населения** доступны следующие фильтры:

* **Пол**: `Мужчины` и `Женщины`.
* **Возраст**: возраст населения от 0 до 70 лет. Значение `70 лет` включает в себя данные обо всех жителях от 70 лет и старше.
* **Проживающее население**: количество зарегистрированных жителей в зданиях.

Чтобы сбросить фильтр по параметру, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с именем фильтра. Чтобы сбросить все фильтры по параметрам, нажмите **Очистить фильтры** внизу вкладки.

Чтобы скрыть фильтр по территории, в окне **Настройки геофильтров** нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.

## Покупка данных {#data-purchasing}

Вы можете купить данные отдельно в виде выгрузки в следующих форматах:

* Табличные форматы:
   * `.csv`;
   * `.txt`;
   * `.json`;
* Форматы для хранения географических данных:
   * `.shp` (Shape);
   * `.gpkg` (GeoPackage);
   * `.geojson` (GeoJSON).

Чтобы узнать стоимость, доступные территории и приобрести данные, заполните форму на [dev.2gis.ru](https://dev.2gis.ru/data).
