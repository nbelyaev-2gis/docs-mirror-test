# Территориальное деление

В наборе данных **Территориальное деление** представлена информация о географических границах территорий (единиц территориального деления) разного уровня: областях, районах, населённых пунктах и т. д. Для каждой территории представлены базовые статистические показатели: площадь, количество населения и другие.

![Территориальное деление](/img/pro/data-territorial-overview.png)

## Сценарии использования {#usage-scenarios}

Набор данных **Территориальное деление** помогает решать задачи:

* **Развитие сети розничной торговли и общественного питания**. Помогает сфере ритейла, гостинично-ресторанного бизнеса и сервисным компаниям принимать решения об открытии локаций в новых районах с помощью статистических показателей на разных территориях.
* **Поиск перспективных жилмассивов**. Позволяет с помощью данных о количестве компаний и населения выявлять жилмассивы с высокой плотностью проживающего населения и низкой насыщенностью организациями и принимать решения об открытии магазина или кафе.

## Как были получены данные? {#data-collection}

Специалисты <Brand /> регулярно собирают и проверяют данные: см. подробнее на странице [О данных <Brand />](https://dev.2gis.ru/company).

## Обновление данных и историчность {#data-update-and-history}

* **Частота обновления**: данные обновляются один раз в неделю.
* **Историчность**: исторические данные недоступны. Вы можете просматривать только последние актуальные данные.

## Доступные территории {#available-territories}

Данные доступны для следующих стран:

* Азербайджан,
* Армения,
* Беларусь,
* Грузия,
* Казахстан,
* Кыргызстан,
* Россия,
* Узбекистан.

:::note[Ограниченная доступность на территории]

Данные могут быть доступны не на всей территории страны, а только в населённых пунктах, в которых присутствует компания <Brand />. Чтобы уточнить доступность данных, используйте [геофильтры](/pro/data/overview#geofilters) или обратитесь в [техническую поддержку](mailto:support.pro@2gis.ru).

:::

## Доступность данных {#data-availability}

* **Облако** (сервис [<Brand /> Про](https://pro.2gis.ru/)):

  * **Демо-тариф**: данные недоступны. Чтобы начать работать с данными, [смените тариф](/pro/prices) и запросите доступ к набору данных.
  * **Другие тарифы**: данные доступны по запросу.

* **On-Premise** (установка сервиса в [закрытом контуре](/on-premise-pro/architecture/pro-architecture)): данные доступны.

## Начало работы {#getting-started}

![Территориальное деление](/img/pro/data-territorial.png)

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите набор данных **Территориальное деление**.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. На вкладке **Фильтры** выберите базовый фильтр для анализа данных: **Тип территориального деления**. Для одного слоя рекомендуется выбирать только один тип территориального деления, иначе полигоны на карте будут накладываться друг на друга.
5. Настройте остальные [фильтры](#filters) при необходимости.

<br />

Данные будут отображены на карте в виде полигонов. Результаты фильтрации доступны в двух вкладках:

* **Статистика**: общее количество территорий.
* **Результаты**: общая информация о территориях и их список. При выборе территории из списка:

  * На карте будет указана точка, в которой находится центр территории.
  * Откроется карточка с информацией о территории:

    * Площадь территории в км².
    * Общее количество проживающего населения.
    * Плотность проживающего населения в чел./км².
    * Общее количество работающего населения.
    * Общее количество организаций во всех категориях.
    * Количество организаций на 1000 жителей.

    <img src="/img/pro/data-territorial-results.png" alt="Карточка территории" width="600" />

  Вы можете изменить вручную список результатов фильтрации, чтобы оставить в выборке только выбранные территории. Для этого нажмите **Изменить список**, выберите территории и нажмите **Применить**.

Чтобы сохранить выборку, нажмите **Создать слой** и настройте [визуализацию данных](#data-visualization).

## Фильтры {#filters}

Для набора данных **Территориальное деление** доступны следующие фильтры:

* **Название**: поиск территории по названию (например, `Москва` или `Пресненский`).

* **Тип территориального деления**: тип территории. Доступные типы:

  * Страны.
  * Области.
  * Районы: муниципальные районы. Пример: Жуковский городской округ.
  * Населённые пункты.
  * Административные округа.
  * Сельские поселения.
  * Административные районы: районы города. Пример: Пресненский район Москвы.
  * Жилмассивы: СНТ, микрорайоны и другие. Для Москвы и Санкт-Петербурга при применении фильтра отображаются границы муниципальных образований.
  * Места: жилые комплексы и другие.

* **Площадь**: площадь территории в км².

* **Количество фирм**: количество организаций, расположенных на территории (см. набор данных [Фирмы](/pro/data/built-in/firms)).

* **Проживающее население**: количество жителей на территории (см. набор данных [Население](/pro/data/built-in/population)).

* **Плотность проживающего населения**: отношение количества проживающего населения к площади территории.

* **Работающее население**: количество работников организаций, расположенных на территории (см. набор данных [Население](/pro/data/built-in/population)).

* **Количество фирм на 1000 чел.**: количество организаций, расположенных на территории, на 1000 жителей.

Чтобы сбросить фильтр по параметру, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с именем фильтра. Чтобы сбросить все фильтры по параметрам, нажмите **Очистить фильтры** внизу вкладки.

Чтобы скрыть фильтр по территории, в окне **Настройки геофильтров** нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.

## Визуализация данных {#data-visualization}

Для набора данных доступен только способ визуализации [Полигон](/pro/data/visualization/polygon).

После [создания слоя](#getting-started) настройте базовую визуализацию данных:

1. Выберите способ визуализации **Полигон**:

   <img src="/img/pro/data-territorial-visualization.png" alt="Выбор способа визуализации" width="650" />

2. Чтобы проанализировать население в разных районах, измените цвет полигонов на карте в зависимости от количества жителей. В поле **Цвет на основе** выберите атрибут `Проживающее население` и укажите цветовую палитру:

   ![Настройка цвета](/img/pro/data-territorial-color.png)

## Покупка данных {#data-purchasing}

Вы можете купить данные отдельно в виде выгрузки в следующих форматах:

* Табличные форматы:
   * `.csv`;
   * `.txt`;
   * `.json`;
* Форматы для хранения географических данных:
   * `.shp` (Shape);
   * `.gpkg` (GeoPackage);
   * `.geojson` (GeoJSON).

Чтобы узнать стоимость, доступные территории и приобрести данные, заполните форму на [dev.2gis.ru](https://dev.2gis.ru/data).
