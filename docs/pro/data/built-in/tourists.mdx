# Туристический потенциал

В наборе данных **Туристический потенциал** представлена информация о количестве туристов: уникальных пользователей <Brand /> из других регионов.

![Туристический потенциал](/img/pro/data-tourists-overview.png)

## Сценарии использования {#usage-scenarios}

Набор данных **Туристический потенциал** помогает решать задачи:

* **Развитие туристической инфраструктуры и маршрутов**. Помогает городским властям оценить популярность и загруженность туристических зон. Данные позволяют определять потенциальные туристические места, развивать новые маршруты, разгружать популярные точки и привлекать людей в другие районы.
* **Анализ туристических потоков**. Показывает, из каких регионов приезжает основное количество туристов. Информацию можно использовать для настройки таргетированной рекламы и акций для целевых регионов.
* **Выбор локаций для открытия бизнеса**. Помогает выбрать оптимальное место для открытия отеля, ресторана, сувенирного магазина. С помощью данных можно определить районы с высоким пешеходным туристическим трафиком.

## Как были получены данные? {#data-collection}

Данные полностью обезличены. Алгоритм определяет основной город пребывания пользователя <Brand /> и классифицирует его как туриста в другом городе, если он активно пользуется в нём приложением.

Алгоритм состоит из следующих шагов:

1. Анализ активности пользователя: в каких городах за последние три месяца пользователь открывал приложение <Brand />.
2. Определение основного города пользователя:

   * Подсчёт общего количества дней, проведённых в каждом городе. Считается, что между запусками приложения пользователь остаётся в городе, в котором он запустил приложение в предыдущий раз.
   * Применение весовых коэффициентов. Чем активность ближе к текущей дате, тем больший вес ей присваивается. Это позволяет определить актуальный основной город пользователя, даже если он недавно переехал.
   * Выбор города с наибольшим взвешенным количеством дней пребывания в качестве основного города.

3. Классификация пользователя как туриста, если он активно использует приложение <Brand /> в городе, который не является для него основным.
4. Агрегация данных. Информация о количестве уникальных туристов из разных регионов суммируется и представляется в виде обобщённых показателей.

## Обновление данных и историчность {#data-update-and-history}

* **Частота обновления**: данные обновляются один раз в месяц.
* **Историчность**: исторические данные доступны за последние 12 месяцев.

## Доступные территории {#available-territories}

Данные доступны для следующих стран:

* Азербайджан,
* Армения,
* Беларусь,
* Грузия,
* Казахстан,
* Кыргызстан,
* Россия,
* Узбекистан.

:::note[Ограниченная доступность на территории]

Данные могут быть доступны не на всей территории страны, а только в населённых пунктах, в которых присутствует компания <Brand />. Чтобы уточнить доступность данных, используйте [геофильтры](/pro/data/overview#geofilters) или обратитесь в [техническую поддержку](mailto:support.pro@2gis.ru).

:::

## Доступность данных {#data-availability}

* **Облако** (сервис [<Brand /> Про](https://pro.2gis.ru/)):

  * **Демо-тариф**: данные недоступны. Чтобы начать работать с данными, [смените тариф](/pro/prices) и запросите доступ к набору данных.
  * **Другие тарифы**: данные доступны по запросу.

* **On-Premise** (установка сервиса в [закрытом контуре](/on-premise-pro/architecture/pro-architecture)): данные недоступны.

## Начало работы {#getting-started}

![Туристический потенциал](/img/pro/data-tourists.png)

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите набор данных **Туристический потенциал**.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. На вкладке **Фильтры** настройте остальные [фильтры](#filters) при необходимости.

<br />

Данные будут отображены на карте в виде точек-центроидов: центров геометрий объектов. В результатах фильтрации доступны:

* Среднее количество туристов на выбранной территории.
* Максимальное количество туристов на выбранной территории.
* Диаграмма с распределением среднего количества туристов на выбранной территории по месяцам. Отображается, если выбраны данные для более чем одного месяца.
* Диаграмма с распределением максимального количества туристов на выбранной территории по месяцам. Отображается, если выбраны данные для более чем одного месяца.

Чтобы сохранить выборку, нажмите **Создать слой** и настройте [визуализацию данных](#data-visualization).

## Фильтры {#filters}

Для набора данных **Туристический потенциал** доступны следующие фильтры:

* **Год – месяц**: год и месяц для отображения данных. Вы можете выбрать несколько значений.
* **Количество туристов**: количество туристов на выбранной территории за выбранный период времени.

Чтобы сбросить фильтр по параметру, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с именем фильтра. Чтобы сбросить все фильтры по параметрам, нажмите **Очистить фильтры** внизу вкладки.

Чтобы скрыть фильтр по территории, в окне **Настройки геофильтров** нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.

## Покупка данных {#data-purchasing}

Вы можете купить данные отдельно в виде выгрузки в следующих форматах:

* Табличные форматы:
   * `.csv`;
   * `.txt`;
   * `.json`;
* Форматы для хранения географических данных:
   * `.shp` (Shape);
   * `.gpkg` (GeoPackage);
   * `.geojson` (GeoJSON).

Чтобы узнать стоимость, доступные территории и приобрести данные, заполните форму на [dev.2gis.ru](https://dev.2gis.ru/data).
