# Привлекательность локаций

В наборе данных **Привлекательность локаций** представлена оценка потенциала мест для открытия бизнеса из разных категорий на основе модели машинного обучения. Модель анализирует данные <Brand />: население, трафик, инфраструктуру, конкурентную среду и историческую успешность заведений.

Оценка рассчитывается отдельно для определённой категории организации и города в виде шкалы от 1 до 10:

* `10` — лучшие локации города (топ-10%);
* `1` — наименее перспективные, рискованные локации.

:::tip

Оценка не является фактическим показателем, не учитывает индивидуальные особенности бизнеса (бренд, формат и т. д.) и не заменяет детальный анализ данных. Рекомендуется использовать в дополнение другие наборы данных, например: [Автомобильный трафик (детальный)](/pro/data/built-in/cars-detailed), [Население](/pro/data/built-in/population) и другие.

:::

## Сценарии использования {#usage-scenarios}

Набор данных **Привлекательность локаций** помогает решать задачи:

* **Выбор локации для открытия бизнеса**. Позволяет определить, где лучше открыть кафе, ресторан, магазин или сервис без ручного анализа нескольких источников данных.
* **Сравнение нескольких локаций**. Упрощает сравнение нескольких мест между собой, чтобы определить, какое из них имеет более высокий потенциал.
* **Поиск подходящих зон в городе**. Показывает районы с наибольшей концентрацией перспективных локаций для определённой категории организаций. 
* **Предварительная оценка рынка**. Помогает оценить, насколько город или район подходит для выбранного типа бизнеса.

## Как были получены данные? {#data-collection}

Оценка привлекательности рассчитывается с помощью модели машинного обучения на основе данных <Brand />.

Алгоритм расчёта состоит из следующих шагов:

1. Разбивка на пространственную сетку. Город делится на ячейки H3 с уровнем детализации 9, для каждой из которых рассчитывается потенциал привлекательности. Приблизительный размер каждой ячейки — 0,1 км².
2. Определение целевой метрики. Для модели в качестве метрики выбирается историческая успешность заведений: их рейтинг и количество отзывов.
3. Сбор признаков. Модель анализирует каждую ячейку H3 по признакам, которые влияют на привлекательность локации:

   * количество проживающего и работающего населения;
   * активность пользователей <Brand />;
   * количество и рейтинг конкурентов;
   * деловая активность района;
   * транспортная доступность: метро и остановки;
   * наличие парковок;
   * типы застройки: жильё, офисы, ритейл и другие.

4. Обучение модели. Модель обучается на исторических данных по городам и категориям организаций и выявляет закономерности, характерные для успешных локаций.
5. Расчёт и нормализация оценки. Для каждой категории организаций и города рассчитывается относительная оценка (процентиль), которая позволяет корректно сравнивать локации внутри города.

## Обновление данных и историчность {#data-update-and-history}

* **Частота обновления**: данные обновляются один раз в три месяца.
* **Историчность**: исторические данные недоступны.

## Доступные территории {#available-territories}

Данные доступны только для некоторых городов России.

:::note[Ограниченная доступность на территории]

Доступность зависит от наличия модели и данных для категории организации. Чтобы уточнить доступность данных, используйте [геофильтры](/pro/data/overview#geofilters) или обратитесь в [техническую поддержку](mailto:support.pro@2gis.ru).

:::

## Доступность данных {#data-availability}

* **Облако** (сервис [<Brand /> Про](https://pro.2gis.ru/)):

  * **Демо-тариф**: данные недоступны. Чтобы начать работать с данными, [смените тариф](/pro/prices) и запросите доступ к набору данных.
  * **Другие тарифы**: данные доступны по запросу.

* **On-Premise** (установка сервиса в [закрытом контуре](/on-premise-pro/architecture/pro-architecture)): данные недоступны.

## Начало работы {#getting-started}

![Привлекательность локаций](/img/pro/data-attractiveness.png)

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите набор данных **Привлекательность локаций**.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. На вкладке **Фильтры** выберите базовый фильтр для анализа данных: **Категория бизнеса**. Чтобы посмотреть оценку для организаций из определённой категории, выберите одну категорию, например, **Быстрое питание**.

<br />

Данные будут отображены на карте в виде ячеек H3 (гексагонов) с уровнем детализации 9. В результатах фильтрации доступна диаграмма с распределением локаций по уровням привлекательности внутри одной категории организаций. Уровни рассчитываются на основе оценки локации (от 1 до 10) для определённой категории и города.

Чтобы сохранить выборку, нажмите **Создать слой** и настройте [визуализацию данных](#data-visualization).

## Фильтры {#filters}

Для набора данных **Привлекательность локаций** доступны следующие фильтры:

* **Категория бизнеса**: категории организаций в справочнике. Вы можете выбрать только одну категорию, например, **Быстрое питание**.
* **Оценка локации**: уровень привлекательности локации в категории организации. Доступные значения: от 1 до 10, где 1 — низкий уровень привлекательности, а 10 — высокий.

Чтобы сбросить фильтр по параметру, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с именем фильтра или нажмите **Очистить фильтры** внизу вкладки.

Чтобы скрыть фильтр по территории, в окне **Настройки геофильтров** нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.

## Покупка данных {#data-purchasing}

Вы можете купить данные отдельно в виде выгрузки в следующих форматах:

* Табличные форматы:
   * `.csv`;
   * `.txt`;
   * `.json`;
* Форматы для хранения географических данных:
   * `.shp` (Shape);
   * `.gpkg` (GeoPackage);
   * `.geojson` (GeoJSON).

Чтобы узнать стоимость, доступные территории и приобрести данные, заполните форму на [dev.2gis.ru](https://dev.2gis.ru/data).
