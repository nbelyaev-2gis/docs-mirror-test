# Жилая недвижимость

В наборе данных **Жилая недвижимость** представлена агрегированная информация о стоимости аренды и продажи жилой недвижимости из внешних баз данных за последний год.

![Жилая недвижимость](/img/pro/data-residential-overview.png)

## Сценарии использования {#usage-scenarios}

Набор данных **Жилая недвижимость** помогает решать задачи:

* **Выбор локации для открытия бизнеса**. Помогает оценивать покупательную способность жителей в различных районах города. Высокая стоимость покупки или аренды жилья может говорить о более обеспеченных жителях.
* **Инвестиционный анализ**. Позволяет строительным компаниям и девелоперам оценивать востребованность и целесообразность строительства в конкретных районах. Высокая средняя цена за квадратный метр может свидетельствовать о наличии платёжеспособного спроса.
* **Формирование стратегий ценообразования**. Помогает торговым сетям и сервисам доставки адаптировать ценовую политику в зависимости от стоимости покупки или аренды жилья. Например, предлагать премиальные товары в районах с дорогими квартирами и удерживать низкие цены в доступных спальных районах.

## Как были получены данные? {#data-collection}

Данные собраны из внешних баз данных.

## Обновление данных и историчность {#data-update-and-history}

* **Частота обновления**: данные обновляются один раз в месяц. Каждый месяц выгружаются данные за последний год. Например, в июне 2025 года выгружались данные по объявлениям с мая 2024 года по май 2025 года.
* **Историчность**: исторические данные недоступны. Вы можете просматривать только последние актуальные данные.

## Доступные территории {#available-territories}

Данные доступны только для России.

:::note[Ограниченная доступность на территории]

Данные могут быть доступны не на всей территории страны, а только в населённых пунктах, в которых присутствует компания <Brand />. Чтобы уточнить доступность данных, используйте [геофильтры](/pro/data/overview#geofilters) или обратитесь в [техническую поддержку](mailto:support.pro@2gis.ru).

:::

## Доступность данных {#data-availability}

* **Облако** (сервис [<Brand /> Про](https://pro.2gis.ru/)):

  * **Демо-тариф**: данные недоступны. Чтобы начать работать с данными, [смените тариф](/pro/prices) и запросите доступ к набору данных.
  * **Другие тарифы**: данные доступны по запросу.

* **On-Premise** (установка сервиса в [закрытом контуре](/on-premise-pro/architecture/pro-architecture)): данные недоступны.

## Начало работы {#getting-started}

![Жилая недвижимость](/img/pro/data-residential.png)

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите набор данных **Жилая недвижимость**.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. На вкладке **Фильтры** выберите базовые фильтры для анализа данных:

   * Чтобы проанализировать рынок аренды, выберите в фильтре **Тип сделки** значение `Аренда` и используйте фильтр **Средняя цена**.
   * Чтобы проанализировать рынок продажи, выберите в фильтре **Тип сделки** значение `Продажа` и используйте фильтры **Средняя цена за м²** и **Средняя цена**.
   * Чтобы посмотреть данные по типу помещения, выберите фильтр **Тип помещения**.

5. Настройте остальные [фильтры](#filters) при необходимости.

<br />

Данные будут отображены на карте в виде точек-центроидов: центров геометрий объектов. В результатах фильтрации доступны:

* Средняя цена за м² в рублях.
* Средняя цена предложения в рублях.
* Минимальная цена за м² в рублях.
* Максимальная цена за м² в рублях.
* Диаграмма с распределением предложений по типу сделки. Отображается, если выбраны оба типа сделки.
* Диаграмма с распределением предложений по типу помещения. Отображается, если выбраны оба типа недвижимости или только значение `Загородная недвижимость`.
* Диаграмма с распределением предложений по количеству комнат. Отображается, если выбран тип помещения `Квартира`.

Чтобы сохранить выборку, нажмите **Создать слой** и настройте [визуализацию данных](#data-visualization).

## Фильтры {#filters}

Для набора данных **Жилая недвижимость** доступны следующие фильтры:

* **Тип сделки**: `Аренда` или `Продажа`. Для корректного анализа данных рекомендуется выбрать только один вариант, т. к. стоимость аренды и продажи существенно различается.
* **Тип недвижимости**: `Жилая недвижимость` или `Загородная недвижимость`. Для корректного анализа данных рекомендуется выбрать только один вариант, т. к. стоимость квартир и загородных домов с участками различается.
* **Тип помещения**: тип помещения (например, квартира, коттедж, дом и т. д.).
* **Средняя цена за м²**: средняя цена жилой недвижимости за квадратный метр в рублях. Рассчитывается по всем предложениям внутри одной ячейки H3 (гексагона).
* **Средняя цена**: средняя цена жилой недвижимости в рублях. Рассчитывается по всем предложениям внутри одной ячейки H3 (гексагона). Для типа сделки `Аренда` — средняя стоимость аренды в месяц для всех объектов, `Продажа` — средняя стоимость всех объектов.
* **Количество комнат в квартире**: количество комнат (например, `Студия`, `1`, `2` и т. д.). Доступно только для предложений с типом помещения `Квартира`.

Чтобы сбросить фильтр по параметру, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с именем фильтра. Чтобы сбросить все фильтры по параметрам, нажмите **Очистить фильтры** внизу вкладки.

Чтобы скрыть фильтр по территории, в окне **Настройки геофильтров** нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.
