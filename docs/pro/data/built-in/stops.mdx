# Остановки общественного транспорта

В наборе данных **Остановки общественного транспорта** представлена информация о расположении остановок общественного транспорта и станций метро.

![Остановки общественного транспорта](/img/pro/data-stops-overview.png)

## Сценарии использования {#usage-scenarios}

Набор данных **Остановки общественного транспорта** помогает решать задачи:

* **Выбор локации для открытия бизнеса**. Помогает посчитать количество остановок общественного транспорта рядом с точками размещения бизнеса и выбрать место с большим количеством пассажиров.
* **Геопространственный анализ**. Вы можете [агрегировать данные](/pro/geospatial/aggregation) или строить [зоны доступности](/pro/geospatial/proximity). Например, посчитать среднюю стоимость квадратного метра жилья в 5 или 10 минутах пешком от станции метро, а также построить зоны доступности от остановок общественного транспорта для определения мест для открытия бизнеса (например, киосков).

## Как были получены данные? {#data-collection}

Специалисты <Brand /> регулярно собирают и проверяют данные: см. подробнее на странице [О данных <Brand />](https://dev.2gis.ru/company).

## Обновление данных и историчность {#data-update-and-history}

* **Частота обновления**: данные обновляются один раз в неделю.
* **Историчность**: исторические данные недоступны. Вы можете просматривать только последние актуальные данные.

## Доступные территории {#available-territories}

Данные доступны для следующих стран:

* Азербайджан,
* Армения,
* Беларусь,
* Грузия,
* Казахстан,
* Кыргызстан,
* Россия,
* Узбекистан.

:::note[Ограниченная доступность на территории]

Данные могут быть доступны не на всей территории страны, а только в населённых пунктах, в которых присутствует компания <Brand />. Чтобы уточнить доступность данных, используйте [геофильтры](/pro/data/overview#geofilters) или обратитесь в [техническую поддержку](mailto:support.pro@2gis.ru).

:::

## Доступность данных {#data-availability}

* **Облако** (сервис [<Brand /> Про](https://pro.2gis.ru/)):

  * **Демо-тариф**: данные доступны.
  * **Другие тарифы**: данные доступны по запросу.

* **On-Premise** (установка сервиса в [закрытом контуре](/on-premise-pro/architecture/pro-architecture)): данные доступны.

## Начало работы {#getting-started}

![Остановки общественного транспорта](/img/pro/data-stops.png)

1. Перейдите на вкладку **Главная** или **Данные**.

2. Выберите набор данных **Остановки общественного транспорта**.

3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.

4. На вкладке **Фильтры** выберите базовые фильтры для анализа данных:

   * Чтобы посмотреть данные об остановках, выберите фильтр **Категория** → **Остановки**.
   * Чтобы посмотреть данные о станциях метро, выберите фильтр **Категория** → **Станции метро**.

5. Настройте остальные [фильтры](#filters) при необходимости.

<br />

Данные будут отображены на карте в виде точек. Результаты фильтрации доступны в двух вкладках:

* **Статистика**: общие статистические данные о выборке:

  * Общее количество остановок.
  * Среднее количество маршрутов, проходящих через остановки.

* **Результаты**: общая информация об остановках и их список. При выборе остановки из списка:

  * Откроется карточка с информацией об остановке, включая количество маршрутов, проходящих через остановку, виды транспорта и их маршруты.
  * На карте будет указана точка, в которой находится остановка.

  <img src="/img/pro/data-stops-results.png" alt="Карточка остановки" width="600" />

  Вы можете изменить вручную список результатов фильтрации, чтобы оставить в выборке только выбранные остановки. Для этого нажмите **Изменить список**, выберите остановки и нажмите **Применить**.

Чтобы сохранить выборку, нажмите **Создать слой** и настройте [визуализацию данных](#data-visualization).

## Фильтры {#filters}

Для набора данных **Остановки общественного транспорта** доступны следующие фильтры:

* **Категория**:

  * `Остановки`: расположение остановочных (посадочных) платформ на карте: мест посадки на общественный транспорт. Для станций метро отображается расположение входов и выходов.

    <img src="/img/pro/data-stops-stations.png" alt="Фильтр Остановки" width="410" />

  * `Станции метро`: для каждой станции метро отображается только одна точка на карте.

    <img src="/img/pro/data-stops-metro.png" alt="Фильтр Станции метро" width="410" />

* **Количество маршрутов**: общее количество маршрутов, проходящих через остановку общественного транспорта или станцию метро.

* **Группа транспорта**:

  * `Наземный транспорт`: остановки наземного транспорта.
  * `Метро`: входы и выходы станций метро.
  * `Ж/д транспорт`: станции пригородного железнодорожного сообщения.

  Доступно только для категории `Остановки`.

* **Тип транспорта**: `Автобус`, `Троллейбус`, `Трамвай`, `Маршрутное такси`, `Метро`, `Электропоезд`, `Фуникулёр`, `Монорельс`, `Водный транспорт`, `Канатная дорога`, `Аэроэкспресс`, `МЦД` или `МЦК`.

  Доступно только для категории `Остановки`.

Чтобы сбросить фильтр по параметру, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с именем фильтра. Чтобы сбросить все фильтры по параметрам, нажмите **Очистить фильтры** внизу вкладки.

Чтобы скрыть фильтр по территории, в окне **Настройки геофильтров** нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.

## Визуализация данных {#data-visualization}

Для набора данных доступны следующие способы визуализации:

* [Точка](/pro/data/visualization/point) (предпочтительный способ);
* [Сетка](/pro/data/visualization/grid);
* [Соты](/pro/data/visualization/hexbin);
* [Тепловая карта](/pro/data/visualization/heatmap);
* [Контур](/pro/data/visualization/contour);
* [Кластер](/pro/data/visualization/cluster);
* [H3](/pro/data/visualization/h3).

<br />

После [создания слоя](#getting-started) настройте базовую визуализацию данных:

1. Выберите способ визуализации **Точка**:

   <img src="/img/pro/data-stops-visualization.png" alt="Выбор способа визуализации" width="650" />

2. Чтобы настроить тултип, включите опцию **Настройка тултипа** и выберите базовые атрибуты данных: `Категория` и `Количество маршрутов`:

   <img src="/img/pro/data-stops-tooltip.png" alt="Настройка тултипа" width="650" />

3. В поле **Иконка** выберите иконку для отображения точек на карте:

   <img src="/img/pro/data-stops-icons.png" alt="Настройка иконок" width="650" />

4. Чтобы проанализировать расположение остановок, измените цвет точек на карте в зависимости от категории остановки. В поле **Цвет на основе** выберите атрибут `Категория` и укажите цветовую палитру:

   ![Настройка цвета](/img/pro/data-stops-color.png)

## Покупка данных {#data-purchasing}

Вы можете купить данные отдельно в виде выгрузки в следующих форматах:

* Табличные форматы:
   * `.csv`;
   * `.txt`;
   * `.json`;
* Форматы для хранения географических данных:
   * `.shp` (Shape);
   * `.gpkg` (GeoPackage);
   * `.geojson` (GeoJSON).

Чтобы узнать стоимость, доступные территории и приобрести данные, заполните форму на [dev.2gis.ru](https://dev.2gis.ru/data).
