# Благосостояние населения

В наборе данных **Благосостояние населения** представлена информация о доходах работающего населения.

![Благосостояние населения](/img/pro/data-welfare-overview.png)

## Сценарии использования {#usage-scenarios}

Набор данных **Благосостояние населения** помогает решать задачи:

* **Выбор локации для открытия бизнеса**. Помогает оценивать покупательную способность жителей в различных районах города. Например, магазины премиального сегмента могут ориентироваться на районы с высоким средним доходом на человека.
* **Инвестиционный анализ**. Позволяет строительным компаниям и девелоперам оценивать востребованность и целесообразность строительства в конкретных районах. Например, районы с высоким средним доходом могут быть привлекательными для инвестиций в жилую недвижимость.
* **Формирование стратегий ценообразования**. Помогает торговым сетям и сервисам доставки адаптировать ценовую политику в зависимости от доходов населения. Например, предлагать недорогие товары в районах с низким доходом.

## Как были получены данные? {#data-collection}

Алгоритм расчёта средневзвешенного дохода работающего населения состоит из следующих шагов:

1. Получение и обработка данных Росстата о среднемесячной заработной плате работников организаций по разделам ОКВЭД.
2. Сопоставление данных Росстата с границами муниципальных образований из данных <Brand /> о территориальном делении.
3. Выгрузка данных о количестве работников в каждой организации.
4. Сопоставление категорий из справочника <Brand /> с разделами ОКВЭД.
5. Разбивка территорий муниципальных образований на ячейки H3 (гексагоны). Подробнее о способе визуализации H3 см. в инструкции [H3](/pro/data/visualization/h3).
6. Расчёт средней заработной платы и количества работников по каждому разделу ОКВЭД для каждой ячейки H3.
7. Расчёт средневзвешенной заработной платы для каждой ячейки H3.

## Обновление данных и историчность {#data-update-and-history}

* **Частота обновления**: данные обновляются один раз в месяц.
* **Историчность**: исторические данные недоступны.

## Доступные территории {#available-territories}

Данные доступны только для России.

:::note[Ограниченная доступность на территории]

Данные могут быть доступны не на всей территории страны, а только в населённых пунктах, в которых присутствует компания <Brand />. Чтобы уточнить доступность данных, используйте [геофильтры](/pro/data/overview#geofilters) или обратитесь в [техническую поддержку](mailto:support.pro@2gis.ru).

:::

## Доступность данных {#data-availability}

* **Облако** (сервис [<Brand /> Про](https://pro.2gis.ru/)):

  * **Демо-тариф**: данные недоступны. Чтобы начать работать с данными, [смените тариф](/pro/prices) и запросите доступ к набору данных.
  * **Другие тарифы**: данные доступны по запросу.

* **On-Premise** (установка сервиса в [закрытом контуре](/on-premise-pro/architecture/pro-architecture)): данные недоступны.

## Начало работы {#getting-started}

![Благосостояние населения](/img/pro/data-welfare.png)

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите набор данных **Благосостояние населения**.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. На вкладке **Фильтры** настройте остальные [фильтры](#filters) при необходимости.

<br />

Данные будут отображены на карте в виде точек-центроидов: центров геометрий объектов. В результатах фильтрации доступен показатель среднего дохода на человека в месяц в рублях.

Чтобы сохранить выборку, нажмите **Создать слой** и настройте [визуализацию данных](#data-visualization).

## Фильтры {#filters}

Для набора данных **Благосостояние населения** доступен фильтр **Средний доход на чел. в месяц**: средний доход на одного человека в месяц в рублях.

Чтобы сбросить фильтр по параметру, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с именем фильтра или нажмите **Очистить фильтры** внизу вкладки.

Чтобы скрыть фильтр по территории, в окне **Настройки геофильтров** нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.

## Покупка данных {#data-purchasing}

Вы можете купить данные отдельно в виде выгрузки в следующих форматах:

* Табличные форматы:
   * `.csv`;
   * `.txt`;
   * `.json`;
* Форматы для хранения географических данных:
   * `.shp` (Shape);
   * `.gpkg` (GeoPackage);
   * `.geojson` (GeoJSON).

Чтобы узнать стоимость, доступные территории и приобрести данные, заполните форму на [dev.2gis.ru](https://dev.2gis.ru/data).
