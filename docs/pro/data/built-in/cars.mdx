# Автомобильный трафик (обобщённый)

В наборе данных **Автомобильный трафик (обобщённый)** представлена информация о среднечасовом автомобильном трафике. Трафик сгруппирован по дням недели и месяцам.

Обобщённый автомобильный трафик позволяет увидеть общее распределение трафика в городе или районе. Трафик представлен в виде ячеек H3: примерный диаметр ячейки H3 с уровнем детализации `10` — 66 м, а с уровнем `9` – 175 м. 

![Автомобильный трафик (обобщённый)](/img/pro/data-cars-aggregated-overview.png)

:::tip[Детальное распределение трафика]

Если вам нужны данные о движении автомобилей с точностью до конкретной улицы и с детализацией направления движения, используйте набор данных [Автомобильный трафик (детальный)](/pro/data/built-in/cars-detailed).

:::

## Сценарии использования {#usage-scenarios}

Набор данных **Автомобильный трафик (обобщённый)** помогает решать задачи:

* **Транспортное планирование и строительство дорог**. Помогает определить загруженность дорог и принять решение о строительстве новых дорог и развязок или расширении существующих. Например, высокий автомобильный трафик в каком-либо районе может свидетельствовать о необходимости улучшения транспортной инфраструктуры.
* **Выбор локации для открытия бизнеса**. Помогает выбрать место с большим количеством проезжающих автомобилей и потоком клиентов для магазина, ресторана, заправки и т. д.
* **Городское планирование**. Помогает проектировать жилые районы, учитывая транспортные потоки. Трафик показывает, где требуется строительство новых тротуаров, перекрёстков, автобусных остановок или станций метро.
* **Наружная реклама**. Помогает выбрать места для рекламных щитов и размещать рекламу на улицах с высоким автомобильным трафиком.
* **Геопространственный анализ**. Вы можете [агрегировать данные](/pro/geospatial/aggregation) или строить [зоны доступности](/pro/geospatial/proximity). Например, узнать сколько автомобилей проезжает через определённый полигон или зону доступности, а также посчитать количество трафика в районе 1 км от магазина или 15-минутной зоне доступности на автомобиле.

## Как были получены данные? {#data-collection}

Алгоритм вычисления автомобильного трафика состоит из следующих шагов:

1. Сбор GPS-треков пользователей навигатора в приложении <Brand />.
2. Определение уникальных идентификаторов поездок в рамках ячеек H3 с уровнем детализации `10`.
3. Применение коэффициентов для коррекции данных. Этот шаг помогает учитывать не только данные приложения <Brand />, а показывать реальную загруженность дорог. Учитываются следующие показатели:
   * Общее количество пользователей приложения <Brand />.
   * Количество проживающего населения в городе.
   * Доля автомобилистов среди населения.
   * Процент трудоспособного населения.
4. Расчёт среднего количества поездок отдельно для каждого дня недели.
5. Расчёт почасового трафика отдельно для каждого дня недели.

## Обновление данных и историчность {#data-update-and-history}

* **Частота обновления**: данные обновляются один раз в месяц.
* **Историчность**: доступны исторические данные за последние 12 месяцев.

## Доступные территории {#available-territories}

Данные доступны для следующих стран:

* Азербайджан,
* Армения,
* Беларусь,
* Грузия,
* Казахстан,
* Кыргызстан,
* Россия,
* Узбекистан.

:::note[Ограниченная доступность на территории]

Данные могут быть доступны не на всей территории страны, а только в населённых пунктах, в которых присутствует компания <Brand /> и собираются данные о поездках пользователей. Чтобы уточнить доступность данных, используйте [геофильтры](/pro/data/overview#geofilters) или обратитесь в [техническую поддержку](mailto:support.pro@2gis.ru).

:::

## Доступность данных {#data-availability}

* **Облако** (сервис [<Brand /> Про](https://pro.2gis.ru/)):

  * **Демо-тариф**: данные недоступны. Чтобы начать работать с данными, [смените тариф](/pro/prices) и запросите доступ к набору данных.
  * **Другие тарифы**: данные доступны по запросу.

* **On-Premise** (установка сервиса в [закрытом контуре](/on-premise-pro/architecture/pro-architecture)): данные недоступны.

## Начало работы {#getting-started}

![Автомобильный трафик (обобщённый)](/img/pro/data-cars-aggregated.png)

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите набор данных **Автомобильный трафик (обобщённый)**.
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. На вкладке **Фильтры** настройте остальные [фильтры](#filters) при необходимости.

<br />

Данные будут отображены на карте в виде точек-центроидов: центров геометрий объектов. В результатах фильтрации доступны:

* Среднечасовое количество зафиксированных поездок.
* Распределение среднего трафика по часам.
* Распределение максимального трафика по часам.

Чтобы сохранить выборку, нажмите **Создать слой** и настройте [визуализацию данных](#data-visualization).

## Фильтры {#filters}

Для набора данных **Автомобильный трафик (обобщённый)** доступны следующие фильтры:

* **Год – месяц**: год и месяц для отображения данных. Вы можете выбрать несколько значений.
* **День недели**: день недели для отображения данных. Вы можете выбрать несколько значений.
* **Час**: час для отображения данных. Вы можете выбрать несколько значений.
* **Трафик**: количество трафика за одни сутки.

Чтобы сбросить фильтр по параметру, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с именем фильтра. Чтобы сбросить все фильтры по параметрам, нажмите **Очистить фильтры** внизу вкладки.

Чтобы скрыть фильтр по территории, в окне **Настройки геофильтров** нажмите значок ![Глаз](/img/pro/hide-eye-icon.svg) рядом с именем геофильтра. Чтобы удалить геофильтр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. Чтобы удалить все геофильтры, нажмите **Очистить фильтры**.

## Покупка данных {#data-purchasing}

Вы можете купить данные отдельно в виде выгрузки в следующих форматах:

* Табличные форматы:
   * `.csv`;
   * `.txt`;
   * `.json`;
* Форматы для хранения географических данных:
   * `.shp` (Shape);
   * `.gpkg` (GeoPackage);
   * `.geojson` (GeoJSON).

Чтобы узнать стоимость, доступные территории и приобрести данные, заполните форму на [dev.2gis.ru](https://dev.2gis.ru/data).
