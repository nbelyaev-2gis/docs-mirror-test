# Точка

Способ визуализации **Точка** полезен, когда нужно отобразить отдельные элементы и при этом сохранить их детализацию.

:::tip[Объединение элементов при масштабировании]

Чтобы визуально объединять элементы при увеличении масштаба карты, используйте способ визуализации [Кластер](/pro/data/visualization/cluster).

:::

## Обзор {#overview}

![Точка](/img/pro/visualization-point.png)

Чтобы выбрать способ визуализации:

1. Перейдите на вкладку **Дашборды** и откройте дашборд.
2. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.
3. Чтобы открыть слой, нажмите значок ![Слои](/img/pro/layers-icon.svg) и выберите нужный слой.
4. В блоке **Базовые настройки** выберите способ визуализации **Точка**.
5. Настройте остальные [параметры](#parameters) визуализации при необходимости.

Данные будут визуализированы на карте.

## Параметры {#parameters}

### Настройка тултипа

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Настройка тултипа** | Включите опцию, чтобы настроить список атрибутов, которые будут отображаться в тултипе при наведении курсора на точку на карте. Отметьте нужные атрибуты и при необходимости измените их порядок: подробнее см. в разделе [Настройка тултипа](/pro/data/visualization/overview#configuring-tooltip).<br />Если опция выключена, в тултипе будут отображаться все атрибуты из набора данных. |

### Радиус точек

Вы можете задать радиус точек на карте одним из способов:

* На основе атрибута данных: радиус каждого элемента зависит от значений атрибута.
* Настроить единый радиус для всех элементов.

На основе атрибута данных:

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Радиус на основе** | Числовой атрибут, на основе которого рассчитывается радиус точек на карте. Доступные значения зависят от данных в выборке.<br />Чтобы сбросить параметр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. |
| **Диапазон радиуса** | Диапазон возможного радиуса точек в пикселях от `1` до `50`. |

Настройка единого радиуса:

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Радиус** | Радиус точек в пикселях от `1` до `20`. |

### Иконка

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Вид точки** | Иконка для отображения точек на карте.<br />Чтобы сбросить параметр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. |
| **Цвет иконки** | Цвет иконки. Выберите цвет из палитры или укажите его в формате HEX. Доступно, если выбрана иконка. |
| **Обводка** | Включите опцию, чтобы отобразить обводку точек. |
| **Цвет обводки и иконки** | Цвет обводки и иконки. Выберите цвет из палитры или укажите его в формате HEX. Доступно, если включена обводка. |
| **Ширина линии** | Ширина обводки в пикселях от `1` до `5`. |

### Цвет

Вы можете задать цвет точек на карте одним из способов:

* На основе атрибута данных: цвет каждого элемента зависит от значений атрибута.
* Настроить единый цвет для всех элементов.

На основе атрибута данных:

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Цвет на основе** | Текстовый или числовой атрибут, который определяет цвет заливки точек на карте и в легенде в соответствии с цветовой палитрой. Доступные значения зависят от данных в выборке.<br />Чтобы сбросить параметр, нажмите значок ![Крестик](/img/pro/cross-icon.svg) рядом с его именем. |
| **Тип шкалы**<br />(для числовых атрибутов) | Тип распределения, на основе которого рассчитываются диапазоны с данными в легенде:<ul><li>`Логарифмическая` — логарифмическое распределение. Подходит для отображения значений, которые изменяются в широком диапазоне или отличаются на несколько порядков.</li><li>`Линейная` — равномерное распределение. Подходит для отображения значений одного порядка.</li><li>`Экспоненциальная` — экспоненциальное распределение. Подходит, если необходимо выделить малые значения, а большими можно пренебречь, а также для отображения данных, которые быстро меняются.</li><li>`Настраиваемая` — ручная настройка диапазонов. Во всех диапазонах, кроме последнего, указанная верхняя (правая) граница не включается в диапазон.</li></ul> |
| **Количество диапазонов** | Количество диапазонов с данными в легенде и цветов в палитре. Для текстовых атрибутов доступное количество зависит от данных. Для числовых атрибутов — от 1 до 10.<br /><br />Для текстовых атрибутов диапазоны рассчитываются автоматически в зависимости от значений данных. Если вы уменьшите количество диапазонов, последние диапазоны объединятся в диапазон **Прочее**. Вы не можете перемещать диапазоны, добавлять новые или удалять.<br /><br />Для числовых атрибутов диапазоны рассчитываются автоматически в зависимости от **Типа шкалы** (для типов `Логарифмическая`, `Линейная` и `Экспоненциальная`). Чтобы настроить диапазоны вручную, введите нужные границы или измените порядок и количество диапазонов (тип `Настраиваемая`). Во всех диапазонах, кроме последнего, указанная верхняя (правая) граница не включается в диапазон.<br /><br />Чтобы изменить порядок диапазонов, зажмите значок ![Перемещение](/img/pro/drag-n-drop-icon.svg) у диапазона и перетащите его. Чтобы добавить диапазон, нажмите значок ![Плюс](/img/pro/plus-icon.svg). Чтобы удалить диапазон, нажмите значок ![Корзина](/img/pro/trash-icon.svg). |
| **Палитра** | Цветовая палитра (набор готовых цветовых образцов). Чтобы изменить цвет для определённого значения или диапазона, нажмите на цвет в легенде и выберите новый из палитры или укажите его в формате HEX. Чтобы сбросить цвет, наведите на него курсор и нажмите значок ![Крестик](/img/pro/cross-icon.svg). Цвет также сбрасывается при изменении атрибута, но не сбрасывается при изменении границ диапазона или количества диапазонов. Пример:<br /><img src="/img/pro/visualization-custom-color.png" alt="Выбор цвета" width="300"  /> |
| **Инвертировать** | Включите опцию, чтобы инвертировать цветовую палитру. Инвертируются все цвета в палитре, кроме настроенных вручную. |
| **Прозрачность** | Прозрачность заливки точек в процентах. |

Настройка единого цвета:

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Цвет** | Цвет заливки точек на карте и в легенде. Сбросьте параметр **Цвет на основе**, выберите цвет из палитры или укажите его в формате HEX. Точки на карте будут раскрашены одинаково. |
| **Прозрачность** | Прозрачность заливки точек в процентах. |
