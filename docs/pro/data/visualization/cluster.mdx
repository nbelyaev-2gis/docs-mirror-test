# Кластер

Способ визуализации **Кластер** полезен для анализа большого количества точек на карте. В этом случае удобнее работать с группами точек: кластерами. Размер кластера зависит от количества точек, которые он объединяет.

При увеличении масштаба карты (приближении) крупные кластеры сначала распадаются на более мелкие кластеры, а затем на отдельные точки. При уменьшении масштаба карты (отдалении) кластеры группируются в более крупные.

## Обзор {#overview}

![Кластер](/img/pro/visualization-cluster.png)

Чтобы выбрать способ визуализации:

1. Перейдите на вкладку **Дашборды** и откройте дашборд.
2. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.
3. Чтобы открыть слой, нажмите значок ![Слои](/img/pro/layers-icon.svg) и выберите нужный слой.
4. В блоке **Базовые настройки** выберите способ визуализации **Кластер**.
5. Настройте остальные [параметры](#parameters) визуализации при необходимости.

Данные будут визуализированы на карте.

## Параметры {#parameters}

### Настройка тултипа

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Настройка тултипа** | Включите опцию, чтобы настроить список атрибутов, которые будут отображаться в тултипе при наведении курсора на кластер на карте. Отметьте нужные атрибуты и при необходимости измените их порядок и выберите тип агрегации атрибута: подробнее см. в разделе [Настройка тултипа](/pro/data/visualization/overview#configuring-tooltip).<br />Если опция выключена, в тултипе будут отображаться все атрибуты из набора данных. |

### Радиус

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Размер кластера** | Размер кластера в пикселях от `1` до `250`. Точка включается в кластер, если находится на указанном расстоянии от его центра. |
| **Диапазон радиуса** | Радиус кластера в пикселях от `1` до `100`. |

### Цвет

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Цвет на основе** | Числовой атрибут, который определяет цвет заливки кластеров на карте и в легенде в соответствии с цветовой палитрой. Доступные значения зависят от данных в выборке. |
| **Тип агрегации цвета** | Функция, на основе которой агрегируются значения атрибута в одном кластере:<ul><li>`Сумма` — сумма значений.</li><li>`Среднее` — среднее значение.</li><li>`Медиана` — медианное значение, которое равно или больше, чем 50% значений в группе.</li><li>`Мин.` — минимальное значение.</li><li>`Макс.` — максимальное значение.</li></ul>|
| **Тип шкалы** | Тип распределения, на основе которого рассчитываются диапазоны с данными в легенде:<ul><li>`Логарифмическая` — логарифмическое распределение. Подходит для отображения значений, которые изменяются в широком диапазоне или отличаются на несколько порядков.</li><li>`Линейная` — равномерное распределение. Подходит для отображения значений одного порядка.</li><li>`Экспоненциальная` — экспоненциальное распределение. Подходит, если необходимо выделить малые значения, а большими можно пренебречь, а также для отображения данных, которые быстро меняются.</li><li>`Настраиваемая` — ручная настройка диапазонов. Во всех диапазонах, кроме последнего, указанная верхняя (правая) граница не включается в диапазон.</li></ul> |
| **Количество диапазонов** | Количество диапазонов с данными в легенде и цветов в палитре от 1 до 10.<br /><br />Диапазоны рассчитываются автоматически в зависимости от **Типа шкалы** (для типов `Логарифмическая`, `Линейная` и `Экспоненциальная`). Чтобы настроить диапазоны вручную, введите нужные границы или измените порядок и количество диапазонов (тип `Настраиваемая`). Во всех диапазонах, кроме последнего, указанная верхняя (правая) граница не включается в диапазон.<br /><br />Чтобы изменить порядок диапазонов, зажмите значок ![Перемещение](/img/pro/drag-n-drop-icon.svg) у диапазона и перетащите его. Чтобы добавить диапазон, нажмите значок ![Плюс](/img/pro/plus-icon.svg). Чтобы удалить диапазон, нажмите значок ![Корзина](/img/pro/trash-icon.svg). |
| **Палитра** | Цветовая палитра (набор готовых цветовых образцов). Чтобы изменить цвет для диапазона, нажмите на цвет в легенде и выберите новый из палитры или укажите его в формате HEX. Чтобы сбросить цвет, наведите на него курсор и нажмите значок ![Крестик](/img/pro/cross-icon.svg). Цвет также сбрасывается при изменении атрибута, но не сбрасывается при изменении границ диапазона или количества диапазонов. Пример:<br /><img src="/img/pro/visualization-custom-color.png" alt="Выбор цвета" width="300"  />|
| **Инвертировать** | Включите опцию, чтобы инвертировать цветовую палитру. Инвертируются все цвета в палитре, кроме настроенных вручную. |
| **Прозрачность** | Прозрачность заливки кластеров в процентах. |

### Обводка

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Обводка** | Включите опцию, чтобы отобразить обводку кластера. |
| **Цвет обводки** | Цвет обводки. Выберите цвет из палитры или укажите его в формате HEX. |
| **Ширина линии** | Ширина обводки в пикселях от `1` до `25`. |
