# Обзор

Сервис [<Brand /> Про](https://pro.2gis.ru/) предоставляет несколько способов визуализации данных:

* [Точка](/pro/data/visualization/point)
* [Сетка](/pro/data/visualization/grid)
* [Соты](/pro/data/visualization/hexbin)
* [Тепловая карта](/pro/data/visualization/heatmap)
* [Здания](/pro/data/visualization/buildings)
* [Контур](/pro/data/visualization/contour)
* [Кластер](/pro/data/visualization/cluster)
* [H3](/pro/data/visualization/h3)
* [Линия](/pro/data/visualization/line)
* [Полигон](/pro/data/visualization/polygon)

Для каждого [слоя](/pro/dashboards/layers/create) вы можете настроить визуализацию разными способами, даже если слои созданы из одной выборки данных. Для способов визуализации доступны разные настройки. Вы можете изменить визуализацию слоя в любое время.

## Начало работы {#getting-started}

![Визуализация данных](/img/pro/visualization-hexbin.png)

1. Перейдите на вкладку **Дашборды** и откройте дашборд.
2. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.
3. Чтобы открыть слой, нажмите значок ![Слои](/img/pro/layers-icon.svg) и выберите нужный слой.
4. В блоке **Базовые настройки** выберите способ визуализации.
5. Настройте остальные параметры визуализации при необходимости. Списки параметров см. в описании каждого способа визуализации, например, [Точка](/pro/data/visualization/point#parameters).

Данные будут визуализированы на сцене дашборда.

## Просмотр информации об элементе {#viewing-elements}

Чтобы посмотреть информацию об элементе визуализации (точке, ячейке сетки, соте и т. д.) на карте, используйте следующие способы:

* Нажмите на элемент на карте. В открывшейся панели отобразится:

  * Вкладка **Данные слоя**: информация о слое в рамках выбранного элемента визуализации (статистика и список результатов).

    <img src="/img/pro/item-full-information.png" alt="Полная информация об элементе" width="500" />
  
  * Вкладка **На карте**: информация об отдельном элементе на карте: организации, здании, улице и т. д.

    <img src="/img/pro/item-full-information-map.png" alt="Элемент на карте" width="500" />

* Наведите курсор на элемент. Откроется всплывающая подсказка с краткой справкой об элементе — тултип. В тултипе отобразится:

  * Список атрибутов данных и их значения.
  
    Вы можете настроить список атрибутов, их порядок, а также тип агрегации (только для числовых атрибутов): подробнее см. в разделе [Настройка тултипа](#configuring-tooltip).

    <img src="/img/pro/item-tooltip.png" alt="Тултип с краткой справкой об элементе" width="500" />

  * Цвет элемента визуализации и тип агрегации цвета (функция, на основе которой агрегируются значения атрибута в одном элементе визуализации).
  
    Вы можете выбрать атрибут данных для определения цвета элемента с помощью параметра **Цвет на основе** для любых способов визуализации, а тип агрегации цвета — с помощью параметра **Тип агрегации цвета** (только для способов визуализации [Сетка](/pro/data/visualization/grid), [Соты](/pro/data/visualization/hexbin), [Здания](/pro/data/visualization/buildings), [Контур](/pro/data/visualization/contour), [Кластер](/pro/data/visualization/cluster) и [H3](/pro/data/visualization/h3)).
  
    <img src="/img/pro/item-tooltip-color.png" alt="Тултип с типом агрегации цвета" width="500" />

## Настройка тултипа {#configuring-tooltip}

При настройке визуализации слоя вы можете настроить тултип, указав:

* Список атрибутов данных, которые будут отображаться в тултипе, и их порядок. Можно настроить список для [одного слоя](#for-one-layer) или сразу для [нескольких слоёв](#for-multiple-layers) (только для наборов данных, загруженных как файл).

  Настройка списка атрибутов доступна для слоёв со способами визуализации [Точка](/pro/data/visualization/point), [Сетка](/pro/data/visualization/grid), [Соты](/pro/data/visualization/hexbin), [Здания](/pro/data/visualization/buildings), [Контур](/pro/data/visualization/contour), [Кластер](/pro/data/visualization/cluster), [H3](/pro/data/visualization/h3), [Линия](/pro/data/visualization/line) и [Полигон](/pro/data/visualization/polygon).

* Тип агрегации атрибута (только для числовых атрибутов): функцию, на основе которой агрегируются значения атрибута, расположенные в одном элементе визуализации.

  Настройка типа агрегации атрибута доступна для слоёв со способами визуализации [Сетка](/pro/data/visualization/grid), [Соты](/pro/data/visualization/hexbin), [Здания](/pro/data/visualization/buildings), [Контур](/pro/data/visualization/contour), [Кластер](/pro/data/visualization/cluster) и [H3](/pro/data/visualization/h3).

### Для одного слоя {#for-one-layer}

1. Перейдите на вкладку **Дашборды** и откройте дашборд.
3. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.
4. Чтобы открыть слой, нажмите значок ![Слои](/img/pro/layers-icon.svg) и выберите нужный слой.
5. Включите опцию **Настройка тултипа**. Если опция выключена, в тултипе будут отображаться все атрибуты из набора данных.
6. Настройте тултип:

   * Отметьте атрибуты, которые будут отображаться в тултипе элемента. Доступные значения зависят от данных в выборке.
   * Чтобы изменить порядок атрибутов в тултипе, зажмите атрибут и переместите его в нужное место в списке.
   * Чтобы изменить тип агрегации атрибута (только для числовых атрибутов), нажмите значок ![Меню атрибута](/img/menu-dots.svg) у нужного атрибута и выберите тип агрегации в списке:

     * `Сумма` — сумма значений.
     * `Среднее` — среднее значение.
     * `Медиана` — медианное значение, которое равно или больше, чем 50% значений в группе.
     * `Минимум` — минимальное значение.
     * `Максимум` — максимальное значение.

   <br />
   <img src="/img/pro/configuring-tooltip-list.png" alt="Настройка списка атрибутов" width="650" />

### Для всех слоёв из одного набора данных {#for-multiple-layers}

Вы можете настроить список атрибутов в тултипе для всех слоёв, созданных из [набора данных, загруженного как файл](/pro/data/files-upload). Все слои, созданные из набора, будут по умолчанию использовать эти настройки.

1. Перейдите на вкладку **Данные**.
2. В карточке нужного набора данных нажмите значок ![Меню набора данных](/img/menu-dots.svg) и выберите **Редактировать**.
3. Включите опцию **Настройка тултипа**. Если опция выключена, в тултипе будут отображаться все атрибуты из набора данных. При этом сохранится порядок атрибутов, указанных при загрузке набора данных.
4. Настройте тултип:

   * Отметьте атрибуты, которые будут отображаться в тултипе элемента. Доступные значения зависят от данных в выборке.
   * Чтобы изменить порядок атрибутов в тултипе, зажмите атрибут и переместите его в нужное место в списке.

   <br />
   <img src="/img/pro/configuring-tooltip.png" alt="Настройка списка атрибутов для набора данных" width="700" />

5. В поле **Тултип** справа от списка атрибутов проверьте, как будет отображаться тултип. Нажмите **Применить**.
