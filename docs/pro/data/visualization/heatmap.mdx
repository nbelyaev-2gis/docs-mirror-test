# Тепловая карта

Способ визуализации **Тепловая карта** полезен, если нужно оценить плотность распределения данных на карте, а также выявить концентрации значений: большие значения или наоборот — отсутствие данных. Визуализация адаптируется под масштаб карты.

## Обзор {#overview}

![Тепловая карта](/img/pro/visualization-heatmap.png)

Чтобы выбрать способ визуализации:

1. Перейдите на вкладку **Дашборды** и откройте дашборд.
2. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.
3. Чтобы открыть слой, нажмите значок ![Слои](/img/pro/layers-icon.svg) и выберите нужный слой.
4. В блоке **Базовые настройки** выберите способ визуализации **Тепловая карта**.
5. Настройте остальные [параметры](#parameters) визуализации при необходимости.

Данные будут визуализированы на карте.

## Параметры {#parameters}

### Радиус

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Радиус** | Радиус областей тепловой карты в пикселях от `1` до `100`. |

### Уровень детализации

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Зафиксировать уровень** | Нажмите, чтобы зафиксировать уровень детализации карты. При приближении карты уровень детализации не будет больше указанного значения, а при отдалении карты будет уменьшаться. |

### Цвет

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Цвет на основе** | Числовой атрибут, который определяет цвет заливки областей тепловой карты на карте и в легенде в соответствии с цветовой палитрой. Доступные значения зависят от данных в выборке. |
| **Количество диапазонов** | Количество диапазонов с данными в легенде и цветов в палитре от 1 до 10. |
| **Палитра** | Цветовая палитра (набор готовых цветовых образцов). |
| **Инвертировать** | Включите опцию, чтобы инвертировать цветовую палитру. |
| **Прозрачность** | Прозрачность заливки областей тепловой карты в процентах. |
