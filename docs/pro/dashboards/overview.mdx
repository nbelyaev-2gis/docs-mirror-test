# Обзор

Сервис [<Brand /> Про](https://pro.2gis.ru/) позволяет [создавать слои](/pro/dashboards/layers/create) на основе выборок данных и настраивать [визуализацию данных](/pro/data/visualization/overview) на карте для каждого слоя. Подробнее о работе со слоями см. в инструкции [Операции со слоями](/pro/dashboards/layers/manage).

Вы можете работать со слоями в рамках [сцены](/pro/dashboards/scenes/create) дашборда. Каждая сцена содержит уникальный набор слоёв и виджетов и может описывать определённый сценарий аналитики данных. Для каждой сцены вы можете настроить [стили карты](/pro/styles/management), указать [геофильтры](/pro/data/overview#geofilters) и настроить положение камеры.

В один [дашборд](/pro/dashboards/dashboards/create) вы можете добавить несколько сцен. Между сценами дашборда можно переключаться — это помогает хранить несколько сценариев в одном дашборде. При необходимости вы можете дублировать сцены, чтобы заново не настраивать слои, их визуализацию и виджеты. Подробнее о работе с дашбордами и сценами см. в инструкциях [Операции с дашбордами](/pro/dashboards/dashboards/manage) и [Операции со сценами](/pro/dashboards/scenes/manage).

Вы можете добавлять на сцены дашборда различные [виджеты](/pro/dashboards/widgets/overview) с диаграммами на основе данных слоя, а также виджеты с текстом и отдельными значениями. Подробнее о работе с виджетами см. в инструкции [Операции с виджетами](/pro/dashboards/widgets/manage).

### Сценарии использования {#usage-scenarios}

Типовой сценарий работы с дашбордами:

1. Создайте дашборд и сцену.
2. Создайте слои на основе выборок данных и добавьте их в сцену дашборда.
3. Настройте визуализацию слоёв.
4. Укажите геофильтры для слоёв.
5. Укажите настройки камеры и карты для сцены.
6. Добавьте в сцену виджеты на основе данных из слоя.
7. При необходимости создайте другие сцены в рамках дашборда, чтобы отразить другие аналитические сценарии.
