# Создание нового дашборда

**Дашборд** — это набор [сцен](/pro/dashboards/scenes/create), которые описывают разные сценарии аналитики данных. Дашборд используется для создания и хранения сцен в едином месте. В один дашборд вы можете добавить несколько сцен, между которыми можно переключаться.

Пример дашборда с несколькими сценами:

<Video src="/video/Pro.Scenes.mp4" width="900px" />

:::note[Доступность в демо-тарифе]

Если у вас [демо-тариф](/pro/prices), вы можете создать не более одного дашборда.

:::

## На вкладке Дашборды {#on-the-dashboards-tab}

<img src="/img/pro/dashboard-create.png" alt="Создание дашборда на вкладке Дашборды" width="600"  />

1. Перейдите на вкладку **Дашборды**.
2. Нажмите **Создать**.
3. Введите имя нового дашборда.
4. Введите описание нового дашборда.
5. Нажмите **Создать**.

Будет создан дашборд с одной сценой. Вы можете [создать слои](/pro/dashboards/layers/create) для работы с данными и добавить их в сцену дашборда.

## При работе с данными {#working-with-data}

![Создание дашборда при работе с данными](/img/pro/data-overview.png)

1. Перейдите на вкладку **Главная** или **Данные**.

2. Выберите нужную категорию системных данных или набор загруженных данных.

3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.

4. Если вы выбрали системные данные категории **Пешеходный трафик** или загруженные данные с геометриями в виде линий или полигонов, вверху вкладки **Фильтры** выберите способ отображения данных:

   * **Точки**: в виде точек-центроидов (центров геометрии объектов);
   * **Геометрии**: в виде реальных геометрий данных (линий или полигонов).

5. На вкладке **Фильтры** настройте остальные фильтры при необходимости.

6. Нажмите **Создать слой**.

   Если у вас нет созданных дашбордов, будет создан дашборд с одной сценой, в которую автоматически добавится слой с данными.

7. Если создан хотя бы один дашборд, в диалоговом окне укажите, куда добавить слой:

   * Выберите в списке существующий дашборд и сцену, в которую добавится слой. Нажмите **Создать**.
   * Нажмите **Новый дашборд**, введите имя нового дашборда и нажмите **Создать**. Будет создан новый дашборд с одной сценой, в которую автоматически добавится слой с данными.
