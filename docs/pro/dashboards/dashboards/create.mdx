# Создание дашборда

**Дашборд** — это набор [сцен](/pro/dashboards/scenes/create), которые описывают разные сценарии аналитики данных. Дашборд используется для создания и хранения сцен в едином месте. Вы можете добавить несколько сцен в один дашборд и переключаться между сценами.

Пример дашборда с несколькими сценами:

:::note[Доступность в демо-тарифе]

Если у вас [демо-тариф](/pro/prices), вы можете создать не более одного дашборда.

:::

## Создание дашборда {#creating-dashboard}

<Tabs>
<TabItem value="dashboard" label="На вкладке Дашборды" default>

<img src="/img/pro/dashboard-create.png" alt="Создание дашборда на вкладке Дашборды" width="670"  />

1. Перейдите на вкладку **Дашборды**.
2. Нажмите **Создать**.
3. Введите название дашборда.
4. (Опционально) Введите описание дашборда.
5. Нажмите **Создать**.

Будет создан дашборд с одной сценой. Вы можете [создать слои](/pro/dashboards/layers/create) и добавить их в сцену дашборда для работы с данными, а также [создать сцены](/pro/dashboards/scenes/create).

</TabItem>
<TabItem value="data" label="При работе с данными">

![Создание дашборда при работе с данными](/img/pro/data-overview.png)

1. Перейдите на вкладку **Главная** или **Данные**.
2. Выберите [системный](/pro/data/overview#built-in-data) или [загруженный набор данных](/pro/data/files-upload).
3. В окне **Настройки геофильтров** выберите до 50 территорий, для которых будет создана выборка данных. Вы можете использовать стандартные и собственные территории, а также строить полигоны, зоны доступности и окружности на карте. Подробнее см. в разделе [Геофильтры](/pro/data/overview#geofilters). После выбора территорий нажмите **Готово**.
4. Если вы выбрали системный набор данных **Пешеходный трафик** или загруженные данные с геометриями в виде линий или полигонов, вверху вкладки **Фильтры** выберите способ отображения данных:

   * **Точки**: в виде точек-центроидов (центров геометрии объектов).
   * **Геометрии**: в виде реальных геометрий данных (линий или полигонов).

5. На вкладке **Фильтры** настройте остальные фильтры при необходимости.
6. Нажмите **Создать слой**.

   Если у вас нет созданных дашбордов, будет создан дашборд с одной сценой, в которую автоматически добавится слой с данными.

7. Если создан хотя бы один дашборд, в диалоговом окне укажите, куда добавить слой:

   * Выберите в списке существующий дашборд и сцену, в которую добавится слой. Нажмите **Создать**.
   * Нажмите **Новый дашборд**, введите название нового дашборда и нажмите **Создать**. Будет создан новый дашборд с одной сценой, в которую автоматически добавится слой с данными.

</TabItem>
</Tabs>
