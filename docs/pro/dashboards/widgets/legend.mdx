# Легенда

С помощью виджета **Легенда** вы можете просматривать информацию о данных для слоёв, которые добавлены в сцену дашборда. Виджет создаётся автоматически при создании сцены, а все изменения данных в слоях (например, изменение выборки) отражаются в легенде.

Пример виджета **Легенда**:

<img src="/img/pro/widgets-legend.png" alt="Виджет Легенда" width="370"  />

## Основные особенности {#main-features}

* Цветовая палитра и диапазоны данных наследуются в легенду из настроек [визуализации](/pro/data/visualization/overview) слоя с данными. При необходимости вы можете [настроить цветовую палитру и диапазоны данных для легенды](#customize-color-scale), а также [изменить порядок блоков с данными слоёв](#changing-order-of-data-layers) в виджете.

* При наведении на элемент визуализации на карте появляется подсказка с легендой:

  ![Легенда в подсказках на карте для точек](/img/pro/widgets-legend-tooltip-point.png)

  <br />

  ![Легенда в подсказках на карте для сот](/img/pro/widgets-legend-tooltip-hexbin.png)

* Если на сцене дашборда применена фильтрация по территориям или атрибутам данных, в виджете отобразятся только значения, соответствующие фильтрам. Подробнее см. в инструкции [Фильтрация данных на сцене](/pro/data/manage/filtering).

## Настройка цветовой палитры и диапазонов данных для легенды {#customize-color-scale}

Вы можете настроить условные обозначения для легенды, изменив цветовую палитру и диапазоны данных в настройках [визуализации](/pro/data/visualization/overview) слоя. Настройка диапазонов недоступна для способа визуализации **Тепловая карта**.

<img src="/img/pro/widgets-legend-customize-color.png" alt="Настройки легенды через визуализацию слоя" width="600" />

1. Перейдите на вкладку **Дашборды** и откройте дашборд.
2. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.
3. Чтобы открыть слой, нажмите значок ![Слои](/img/pro/layers-icon.svg) и выберите нужный слой.
4. Настройте параметры легенды в зависимости от [способа визуалиазции](/pro/data/visualization/overview).

## Изменение порядка слоёв в легенде {#changing-order-of-data-layers}

По умолчанию блоки со слоями в виджете **Легенда** отображаются в том порядке, в котором расположены слои в сцене. Вы можете настроить собственный порядок слоёв в легенде, не изменяя порядок в списке слоёв, например, если требуется показать в начале легенды определённые слои.

1. Перейдите на вкладку **Дашборды** и откройте дашборд.
2. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.
3. Если легенда не показана на карте, в верхнем меню дашборда нажмите значок ![Виджеты](/img/pro/widgets-icon.svg). В списке виджетов отметьте чекбокс (![Чекбокс](/img/pro/checkbox-icon.svg)) у легенды или нажмите значок ![Меню виджета](/img/menu-dots.svg) и выберите **Показать на карте**.
4. Зажмите значок ![Перемещение](/img/pro/drag-n-drop-icon.svg) у нужного слоя и перетащите его.
