# Временная шкала

Временна́я шкала позволяет просматривать, как изменялись данные с течением времени: например, как менялся спрос на покупку недвижимости в разные сезоны года. С помощью временной шкалы вы можете:

* Фильтровать данные слоя по атрибуту с типом `Дата` без изменения выборки данных. Для этого выберите на временной шкале период времени, для которого будут отображаться данные слоя на сцене дашборда.
* Проигрывать анимацию изменения данных за выбранный период.
* Просматривать график с распределением количества объектов в зависимости от периода времени.

Вы можете использовать временную шкалу для слоёв с любым [способом визуализации данных](/pro/data/visualization/overview), кроме способа **Здания**.

## Настройка виджета {#configuring-widget}

1. Перейдите на вкладку **Дашборды** и откройте дашборд.
2. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.
3. Нажмите значок ![Временная шкала](/img/pro/timeline-icon-520.svg) в левом нижнем углу.
4. Настройте [параметры](#parameters) временной шкалы.
5. Управляйте просмотром данных с помощью проигрывателя любым из способов:

   * с помощью кнопок ![Кнопка Play](/img/pro/play-icon.svg) и ![Кнопка Pause](/img/pro/pause-icon.svg) (недоступны для способа визуализации [Контур](/pro/data/visualization/contour));
   * используйте стрелки ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg);
   * передвигайте ползунок вручную.

## Параметры {#parameters}

<img src="/img/pro/widgets-timeline-parameters.png" alt="Параметры временной шкалы" width="370"  />

| Параметр | Описание |
| -------- | -------- |
| **Источник данных** | Слой с данными. |
| **Атрибут времени** | Атрибут данных с типом `Дата`: только дата, дата и время или дата и время с указанием часового пояса. Убедитесь, что при [загрузке данных как файла](/pro/data/files-upload) вы указали данные в поддерживаемых [форматах даты и времени](/pro/data/files-upload#dates-and-time) и выбрали в столбце **Тип** значение `Дата`. |
| **Размер шага** | Размер шага, на который сдвигаются границы временного интервала при проигрывании анимации. На графике временной шкалы будет отображаться распределение количества объектов в зависимости от заданного размера шага.<br />Вы можете указать любое значение в секундах, минутах, часах, днях, месяцах и годах. |
| **Заданный интервал** | Границы временного интервала, за который будут отображаться данные. |
| **Режим анимации** | Как заданный временной интервал сдвигается при проигрывании анимации:<ul><li>**Изменение**: обе границы временного интервала сдвигаются на размер шага.<br/><Video src="/video/Pro.Timeline.Change.mp4" width="900px" /></li><li>**Накопление**: левая граница интервала остаётся неизменной, а правая сдвигается на размер шага. Данные накапливаются с течением времени.<br/><Video src="/video/Pro.Timeline.Accumulation.mp4" width="900px" /></li></ul> |
| **Скорость анимации** | Скорость обновления данных на сцене дашборда при проигрывании анимации. Настраивается в левом нижнем углу временной шкалы. Например, `1.0` — один раз в секунду, `2.0` — два раза в секунду и т. д. |
