# Обзор

**Виджет** — это визуальный элемент на [сцене](/pro/dashboards/scenes/create) дашборда. Вы можете добавлять виджеты с диаграммами на основе данных слоя, текстом и отдельными показателями.

Виджеты делятся на две группы:

* [Пользовательские виджеты](#custom-widgets): вы можете создавать их, настраивать содержимое, изменять размер и расположение на сцене.
* [Системные виджеты](#system-widgets) доступны по умолчанию. Такие виджеты нельзя удалить, но вы можете скрыть их на сцене.
