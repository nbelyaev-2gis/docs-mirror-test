# Создание виджета

**Виджет** — это визуальный элемент на [сцене](/pro/dashboards/scenes/create) дашборда. Вы можете добавлять виджеты с диаграммами на основе данных слоя, текстом и отдельными показателями.

## Создание виджета {#creating-widget}

<img src="/img/pro/widgets-create-594.png" alt="Создание виджета" width="500" />

1. Перейдите на вкладку **Дашборды** и откройте дашборд.
2. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.
3. В верхнем меню дашборда нажмите значок ![Виджеты](/img/pro/widgets-icon.svg).
4. Выберите тип виджета. Полный список см. в разделе [Пользовательские виджеты](/pro/dashboards/widgets/overview#custom-widgets).
5. Настройте параметры виджета в зависимости от типа. Подробное описание параметров см. в инструкциях для каждого типа виджета.
6. Нажмите **Создать**.

Виджет автоматически добавится в выбранную сцену дашборда.
