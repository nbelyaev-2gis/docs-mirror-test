# Индикатор

Виджет **Индикатор** позволяет отображать на сценах дашборда следующие показатели:

* Текущее значение: константное значение или значение выбранного атрибута данных (например, средний рейтинг или количество построенных зданий).
* Целевое значение.
* Разницу между текущим и целевым значениями.

Используйте виджет **Индикатор** для отслеживания ключевых показателей эффективности (KPI) и важных метрик, например выполнения плана продаж, среднего рейтинга или изменения численности населения.

## Основные особенности {#main-features}

* Вы можете отображать в виджете только текущее значение или включить отображение целевого значения и разницы между ними.

* В качестве текущего и целевого значения вы можете выбрать:

  * Константные значения, которые вводятся вручную и не зависят от данных слоя.
  * Значения атрибутов данных из слоя, которые будут обновляться при изменении данных в слое. В качестве атрибутов вы можете выбрать:

    * Числовые атрибуты: для подсчёта количества значений в выборке или подсчёта значений этих атрибутов с помощью функций агрегации: суммарного, среднего, минимального, максимального значения, а также медианы и процентилей.
    * Текстовые, логические атрибуты или количество объектов: для подсчёта количества значений в выборке.

* Текущее и целевое значения могут быть разных типов. Например, целевое значение может быть константным, а текущее получено из атрибутов данных. Также атрибуты для текущего и целевого значения могут принадлежать разным слоям с данными.

* Вы можете выбрать способ, на основе которого будет рассчитываться разница между текущим и целевым значениями.

* Изменения данных в слоях автоматически отражаются в виджете.

* Если на сцене дашборда применена фильтрация по территориям или атрибутам данных, в виджете отобразятся только значения, соответствующие фильтрам. Подробнее см. в инструкции [Фильтрация данных на сцене](/pro/data/manage/filtering).

## Создание виджета {#creating-a-widget}

<img src="/img/pro/widgets-counter-create.png" alt="Пример создания виджета-индикатора" width="650" />

1. Перейдите на вкладку **Дашборды**.

2. Откройте дашборд.

3. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.

4. В верхнем меню дашборда нажмите значок ![Виджеты](/img/pro/widgets-icon.svg).

5. Выберите тип виджета **Индикатор**.

6. Укажите обязательные параметры. В блоке **Значение** выберите текущее значение, которое будет отображаться в виджете:

   * **Из данных**: выбор атрибута из набора данных (значение в виджете обновится, если изменятся данные):
     * **Слой**: слой с данными, содержащий нужный атрибут. Если вы работаете с [загруженными данными](/pro/data/files-upload), убедитесь, что они содержат хотя бы один атрибут.
     * **Агрегирующий атрибут**: атрибут, на основе которого агрегируются данные и рассчитывается значение.
     * **Тип агрегации**: функция, на основе которой рассчитывается значение агрегирующего атрибута.
   * **Задать вручную**: ввод константного значения вручную.

7. Настройте остальные [параметры](#parameters) виджета при необходимости.

8. Нажмите **Создать**.

Новый виджет автоматически добавится в выбранную сцену дашборда.

### Текущее значение {#current-value}

Вы можете задать текущее значение, которое будет отображаться в виджете, одним из способов:

* [выбрать текущее значение](#current-from-datasets) из атрибутов данных слоя на вкладке **Из данных**;
* [ввести константное текущее значение](#current-set-manually) на вкладке **Задать вручную**.

### Разница {#difference}

| <div style={{"width":"180px"}}>Параметр</div> | Описание |
| --------------------------------------------- | -------- |
| **Способ вычисления** | Функция, на основе которой рассчитывается изменение текущего значения относительно целевого:<ul><li>**Разница**: показывает разность между текущим и целевым значениями с учётом знака. Пример:<ul><li>Текущее значение: `70`</li><li>Целевое значение: `100`</li><li>Результат вычисления: `70 — 100 = —30`</li></ul></li><li>**Абсолютная разница**: показывает разность между текущим и целевым значениями без учёта знака (модуль разности). Пример:<ul><li>Текущее значение: `70`</li><li>Целевое значение: `100`</li><li>Результат вычисления: `\|70 — 100\| = 30`</li></ul></li><li>**Процент от целевого значения**: показывает в процентах отношение текущего значения к целевому. Пример:<ul><li>Текущее значение: `120`</li><li>Целевое значение: `80`</li><li>Результат вычисления: `(120 ÷ 80) ⋅ 100% = 150%`</li></ul></li><li>**Разница в процентах**: показывает в процентах отношение разности между текущим и целевым значениями к целевому значению. Пример:<ul><li>Текущее значение: `120`</li><li>Целевое значение: `80`</li><li>Результат вычисления: `((120 — 80) ÷ 80) ⋅ 100% = +50%`</li></ul></li><li>**Соотношение**: показывает в виде коэффициента, во сколько раз текущее значение больше или меньше целевого. Пример:<ul><li>Текущее значение: `200`</li><li>Целевое значение: `100`</li><li>Результат вычисления: `200 ÷ 100 = 2`</li></ul></li><li>**Относительное изменение**: показывает отношение разности между текущим и целевым значениями к целевому значению. Пример:<ul><li>Текущее значение: `120`</li><li>Целевое значение: `80`</li><li>Результат вычисления: `(120 — 80) ÷ 80 = +0,50`</li></ul></li></ul> |
| **Иконки-индикаторы** | Настройки цвета иконок, которые показывают изменение текущего значения относительно целевого. Выберите цвет из палитры или укажите его в формате HEX.<br /><br />Правила отображения иконок:<ul><li>Стрелка вверх (▲) отображается, если текущее значение больше целевого. Примеры: **Разница** `> 0` или **Процент от целевого значения** `> 100%`.</li><li>Стрелка вниз (▼) отображается, если текущее значение меньше целевого. Примеры: **Разница** `< 0` или **Процент от целевого значения** `< 100%`.</li><li>Иконки не отображаются, если текущее значение равно целевому. Примеры: **Разница** `= 0` или **Процент от целевого значения** `= 100%`.</li></ul>Иконки отображаются для всех способов вычисления, кроме способа **Абсолютная разница**, так как у этой метрики нет направления. |
