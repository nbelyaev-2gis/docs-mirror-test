# Текстовый виджет

Текстовый виджет содержит блок с текстовым описанием. Используйте текстовый виджет, если необходимо добавить на сцену дашборда описание данных, решаемых задач или целей cцены.

Вы можете применить форматирование к тексту:

* настроить заголовки;
* выделить текст полужирным шрифтом, курсивом или подчёркиванием;
* добавить ссылку;
* выбрать цвет текста из палитры или указать в формате HEX.

## Создание виджета {#creating-a-widget}

<img src="/img/pro/widgets-text-widget-create.png" alt="Пример создания текстового виджета" width="650"  />

1. Перейдите на вкладку **Дашборды** и откройте дашборд.
2. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.
3. В верхнем меню дашборда нажмите значок ![Виджеты](/img/pro/widgets-icon.svg).
4. Выберите тип виджета **Текстовый виджет**.
5. Введите текст, который будет отображаться в виджете.
6. Настройте [параметры](#parameters) форматирования текста при необходимости.
7. Нажмите **Создать**.

Новый виджет автоматически добавится в выбранную сцену дашборда.

## Параметры {#parameters}

| <div style={{"width":"180px"}}>Параметр</div>           | Описание                                                                       |
| ------------------------------------------------- | ------------------------------------------------------------------------------ |
| **Название виджета**                              | Название виджета.                                                                   |
| **Текст**                                         | Текст, который отобразится в виджете. Максимальное количество символов — 5000. |
| **H1**                                            | Основной заголовок (первого уровня).                                           |
| **H2**                                            | Подзаголовок второго уровня.                                                   |
| **H3**                                            | Подзаголовок третьего уровня.                                                  |
| **B**                                             | Полужирный шрифт.                                                              |
| ***I***                                           | Курсив.                                                                        |
| **<u>U</u>**                                      | Подчёркнутый текст.                                                            |
| ![Ссылка](/img/pro/link-icon.svg) | Вставка ссылки.                                                                |
| **Цвет**                                          | Цвет текста. Выберите цвет из палитры или укажите его в формате HEX.           |
