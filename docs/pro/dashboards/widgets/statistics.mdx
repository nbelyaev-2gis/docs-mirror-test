# Статистика

Виджет **Статистика** показывает краткую статистику по выбранным слоям и атрибутам данных (например, среднему рейтингу или количеству построенных зданий).

Используйте виджет, чтобы просматривать важные агрегированные значения внутри одного виджета.

Пример задачи: посмотреть статистику по зданиям, населению и автомобильному трафику в определённом районе:

<img src="/img/pro/widgets-statistics.png" alt="Виджет Статистика" width="400"  />

## Основные особенности {#main-features}

* Для каждого выбранного слоя в виджете отображается отдельный блок со значениями выбранных атрибутов. В качестве атрибутов вы можете выбрать:

  * Числовые атрибуты: для подсчёта количества значений в выборке или подсчёта значений этих атрибутов с помощью функций агрегации: суммарного, среднего, минимального, максимального значения, а также медианы и процентилей.
  * Текстовые, логические атрибуты или количество объектов: для подсчёта количества значений в выборке.

* Максимальное количество слоёв (блоков в виджете) — 20. Максимальное количество атрибутов для каждого слоя — 20.

* При необходимости вы можете изменить для атрибутов названия, которые будут отображаться в виджете. При этом имена атрибутов в наборе данных не изменятся.

* Изменения данных в слоях автоматически отражаются в виджете.

* Если на сцене дашборда применена фильтрация по территориям или атрибутам данных, в виджете отобразятся только значения, соответствующие фильтрам. Подробнее см. в инструкции [Фильтрация данных на сцене](/pro/data/manage/filtering).

## Создание виджета {#creating-a-widget}

<img src="/img/pro/widgets-statistics-create.png" alt="Пример создания виджета Статистика" width="650"  />

1. Перейдите на вкладку **Дашборды**.

2. Откройте дашборд.

3. Откройте сцену с помощью стрелок ![Стрелка влево](/img/pro/left-arrow-icon.svg) и ![Стрелка вправо](/img/pro/right-arrow-icon.svg). Если в дашборде одна сцена, она откроется автоматически.

4. В верхнем меню дашборда нажмите значок ![Виджеты](/img/pro/widgets-icon.svg).

5. Выберите тип виджета **Статистика**.

6. Укажите обязательные параметры:

   * **Слой**: слой с данными, содержащий нужный атрибут. Если вы работаете с [загруженными данными](/pro/data/files-upload), убедитесь, что они содержат хотя бы один атрибут.
   * **Агрегирующий атрибут**: атрибут, на основе которого агрегируются данные и рассчитываются значения.
   * **Тип агрегации**: функция, на основе которой рассчитываются значения агрегирующего атрибута.

7. Настройте остальные [параметры](#parameters) виджета при необходимости.

8. Нажмите **Создать**.

Новый виджет автоматически добавится в выбранную сцену дашборда.

Только для чисел от 10.<br />Пример: `12345.6789 → 12 350`.</li><li>**1 256 → 1 300**: умное округление до сотен. Целая часть округляется до сотен, дробная отбрасывается. Только для чисел от 100.<br />Пример: `12345.6789 → 12 300`.</li><li>**1 257 → 1 000**: умное округление до тысяч. Целая часть округляется до тысяч, дробная отбрасывается. Только для чисел от 1000.<br />Пример: `12345.6789 → 12 000`.</li><li>**0,12345 → 12,345 %**: перевод в проценты. Максимальное количество знаков после запятой — 3.<br />Примеры: `0.12345 → 12,345%`, `1 → 100%`.</li><li>**1 200 000 → 1,2E6**: научный формат (экспоненциальная запись). <br />Примеры: `12345 → 1,235E4`, `1 → 1E0`.</li></ul> |
