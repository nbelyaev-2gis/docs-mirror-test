# Релизы API-платформы

Чтобы установить сервисы API-платформы версии 2.44.0 и выше, установите [утилиту 2GIS CLI версии 3](/on-premise-api-platform/overview/2gis-cli/releases#v3) и следуйте инструкциям по обновлению.

## 2.54.0

Дата релиза: 2026-05-27

<h3 id="dependencies_v2.54.0">Зависимости</h3>

Перед обновлением любого сервиса в составе API-платформы версии 2.54.0 убедитесь, что установлены компоненты-зависимости нужных версий:

| <div style={{"width":"200px"}}>Компонент</div> | Минимальная версия |
| ---------------------------------------- | ------ |
| Утилита 2GIS CLI                         | 3      |
| Базовые сервисы                          | 2.7.0  |

Подробнее см. в разделе **Руководство администратора → Обновление**.

<h3 id="new_v2.54.0">Новое</h3>

| <div style={{"width":"170px"}}>Сервис</div> | Изменения |
| --- | --- |
| API карт<br />(MapGL JS API) | См. [полный список изменений](/mapgl/releases#v1730) с версии 1.72.0 по версию 1.73.0. |
| API карт<br />(Tiles API) | Добавлена поддержка векторных тайлов версии 4 для версий мобильного SDK выше 12.10 (`types[0].kind: native-v4`). См. [пример values.yaml на GitHub](https://github.com/2gis/on-premise-helm-charts/blob/Platform-2.54.0/charts/tiles-api/values.yaml), а также см. [обновлённый конфигурационный файл](/on-premise-api-platform/api-platform/install/maps#install-tiles-api). |
| API поиска<br />(Search API v8) | <ul><li>Добавлены параметры `worker.gracefulShutdownInterval`, `head.head.gracefulShutdownInterval`, `ingressRouter.gracefulShutdownInterval` и `geodetector.gracefulShutdownInterval` для задания времени, в течение которого приложение завершает обработку текущих запросов перед остановкой.</li><li>Добавлен параметр `controller.allowedProjectKeys` для управления списком проектов, которые должны быть проиндексированы.</li></ul>См. [пример values.yaml на GitHub](https://github.com/2gis/on-premise-helm-charts/blob/Platform-2.54.0/charts/search-api-v8/values.yaml).|
| Менеджер Платформы | <ul><li>Добавлен параметр `ui.bffSecure` для включения протокола HTTPS при использовании cookies.</li><li>Добавлен параметр `ui.license.url` для указания URL сервиса лицензий.</li><li>Добавлено значение `license` параметра `ui.pages` для отображения страницы **Лицензии** в приложении. Сервис Static API будет доступен в следующих релизах.</li></ul>См. [пример values.yaml на GitHub](https://github.com/2gis/on-premise-helm-charts/blob/Platform-2.54.0/charts/platform/values.yaml), а также [обновлённый конфигурационный файл](/on-premise-api-platform/api-platform/install/platform#install-platform). |

Routing API подключится автоматически с сохранением прежних лимитов и условий, а статистика расходования лимитов и распределения запросов для Public Transport API сохранится. |
