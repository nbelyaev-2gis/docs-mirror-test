# Сервисы поиска

Сервисы поиска позволяют получать информацию об объектах: организациях, зданиях, общественных местах и т. д.

## Архитектура {#architecture}

Сервисы поиска состоят из двух сервисов:

* Catalog APIs предоставляют RESTful APIs для конечных пользователей и приложений.
* Search API — поисковый движок, который обрабатывает запросы от Catalog APIs.

## Зависимости {#dependencies}

* **Для Search API и Search API v8**

  * Общая инфраструктура:

    * Хранилище артефактов установки.

* **Для Catalog APIs**

  * Общая инфраструктура:

    * Хранилище PostgreSQL для хранения данных об объектах (с расширением PostGIS).

      :::tip[Можно использовать аналог из реестра Минцифры]

      Доступные аналоги PostgreSQL, включённые в реестр Минцифры, см. в разделе [Системные требования](/on-premise-api-platform/requirements).

      :::

  * Сервисы:

    * Сервис лицензий.
    * Сервис API-ключей.
    * Сервис сбора статистики.
    * Search API.

Подробнее о том, как проверить требования для каждого сервиса, см. в документе [Системные требования](/on-premise-api-platform/requirements).
