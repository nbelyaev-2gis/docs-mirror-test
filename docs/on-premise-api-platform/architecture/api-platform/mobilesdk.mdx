# Мобильный SDK

Мобильный SDK предоставляет набор инструментов для добавления основных сервисов On-Premise (карта, поиска и навигация) в ваше мобильное приложение.

## Архитектура {#architecture}

<NoInvert>
![Архитектура мобильного SDK](/img/on-premise/mobile-sdk-architecture.svg)
</NoInvert>

Мобильный SDK получает данные из следующих сервисов On-Premise:

* [Сервисы карт](/on-premise-api-platform/architecture/api-platform/maps) (Tiles API)

  Предоставляют мобильные тайлы (изображения участков карты) для отображения карты в приложении.

* [Сервисы поиска](/on-premise-api-platform/architecture/api-platform/search)

  Предоставляют RESTful APIs, которые приложение может использовать для реализации поискового движка.

* [Сервисы навигации](/on-premise-api-platform/architecture/api-platform/navigation)

  Предоставляют RESTful APIs, которые приложение может использовать для реализации навигатора и построения маршрутов.

* [Прокси для API пробок](/on-premise-api-platform/architecture/api-platform/trafficproxy)

  Позволяет получать данные о пробках в реальном времени с публичных серверов обновлений <Brand />. Прокси должен быть предоставлен доступ в интернет.
