# Сервис лицензий

Сервис лицензий предоставляет доступ к остальным сервисам программного комплекса <Brand />.

Для работы сервисов программного комплекса <Brand /> необходима лицензия. Для получения лицензии используется утилита [2GIS CLI](/on-premise-api-platform/overview/2gis-cli/overview) в режиме `license`.

Лицензия генерируется на серверах <Brand /> и далее записывается в S3-совместимое хранилище или файловую систему в публичной части инфраструктуры партнёра. При дальнейшей работе доступ к интернету не требуется — сервис лицензий считывает лицензию из выбранного хранилища.

Сервис лицензий проверяет:

* наличие и валидность лицензии;
* разрешение у приобретённой лицензии на работу с определёнными сервисами программного комплекса;
* срок действия лицензии.
