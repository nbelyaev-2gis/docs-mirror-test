# Релизы базовых сервисов

Чтобы установить базовые сервисы версии 2.0.0 и выше, установите [утилиту 2GIS CLI версии 3](/on-premise-api-platform/overview/2gis-cli/releases#v3) и следуйте инструкциям по обновлению.

## 2.7.0

Дата релиза: 2026-05-27

<h3 id="dependencies_v2.7.0">Зависимости</h3>

Перед обновлением любого базового сервиса до версии 2.7.0 убедитесь, что установлены компоненты-зависимости нужных версий:

| <div style={{"width":"170px"}}>Компонент</div> | Минимальная версия |
| ---------------------------------------------- | ------------------ |
| Утилита 2GIS CLI                               | 3                  |

Подробнее см. в разделе **Руководство администратора → Обновление**.

<h3 id="new_v2.7.0">Новое</h3>

| <div style={{"width":"170px"}}>Сервис</div> | Изменения |
| --- | --- |
| Сервис лицензий | Теперь, если у пользователя нет прав на получение событий об обновлениях файлов из хранилища MinIO, происходит автоматическое переключение на pull-режим для проверки обновлений файла с лицензией. |
| Сервис API-ключей | Добавлен параметр `KEYS_FEATURE_FLAGS_SINGLE_PARTNER_MODE` для получения первой компании пользователя, если отсутствует заголовок `X-Company-Id`. |

<h3 id="breaking_v2.7.0">Изменения, влияющие на обратную совместимость</h3>

| <div style={{"width":"170px"}}>Сервис</div> | Изменения |
| --- | --- |
| Сервис API-ключей | Для корректной работы Менеджера Платформы добавьте заголовок `X-Company-Id` в `Access-Control-Allow-Headers`. |
