# Обзор

Дата.2ГИС — это сервис для заказа выборок данных из справочника <Brand />.

Сервис позволяет:

* искать бизнес-клиентов в каталоге организаций и получать их контактные данные;
* создавать собственные клиентские базы для продаж и коммуникаций;
* анализировать рынок и конкурентов.

Сервис входит в Единый реестр российского программного обеспечения (реестр Минцифры).

## Основные особенности {#main-features}

* **Данные в выборке**

   Вы можете приобрести данные об организациях из справочника <Brand /> в виде выборки: название организации, адрес, телефон, количество филиалов и другие данные. Подробнее см. в разделе [Доступные данные](#available-data).

* **Частота обновления данных**

    Специалисты <Brand /> регулярно собирают и проверяют данные: см. подробнее на странице [О данных <Brand />](https://dev.2gis.ru/company).

* **Доступные территории**

   Доступны данные об организациях России и Казахстана.

## Доступные данные {#available-data}

Выборки создаются в формате `.xlsx` и содержат следующие атрибуты организаций:

* название организации,
* юридическое название,
* рубрики (категории),
* количество сотрудников,
* количество филиалов,
* способ оплаты,
* время работы,
* дата добавления,
* город,
* субъект,
* район субъекта,
* район города,
* улица,
* номер дома,
* этаж,
* офис,
* почтовый индекс,
* координаты,
* телефон,
* email,
* сайт,
* социальные сети.

Максимальный объём выборки — 2 ГБ.

Если вы не нашли нужные атрибуты, оставьте заявку на [dev.2gis.ru](https://dev.2gis.ru/data#form).
