# Тарифы и оплата

## Расчёт стоимости {#pricing}

Стоимость выборки = количество строк × стоимость строки.

* Количество строк — количество организаций, соответствующих выбранным категориям и территориям. Чтобы уменьшить количество строк, вы можете исключить пустые строки или выгрузить данные только для одного филиала.
* Стоимость строки складывается из базового набора атрибутов и дополнительных атрибутов.

   :::note

   При заказе определённого количества строк распространяется скидка на стоимость каждой строки. Вы можете узнать конечную стоимость при [создании заказа](/data/purchase#creating-order).

   :::

   Данные для любых территорий и категорий стоят одинаково.

### Пример расчёта стоимости {#pricing-example}

Например, вы заказываете выборку со следующими параметрами:

* Территория: городской округ Новосибирск.
* Категория: чайные клубы.
* Атрибуты данных: базовый набор и количество сотрудников.
* Количество строк в выборке: 10.

Общая стоимость выборки: 10 строк × (33,50 ₽ + 1,10 ₽) = 346 ₽.

## Способы оплаты {#payment-methods}

Вы можете оплатить [созданный заказ](/data/purchase#creating-order) следующими способами:

* Онлайн: банковской картой или электронным кошельком.

   Ограничения:
   
   * Принимаются банковские карты, выпущенные в Российской Федерации.
   * Оплата через электронный кошелёк доступна только из Российской Федерации.
   * Можно оплачивать заказы стоимостью до 50 000 рублей.

* По счёту. Оплата доступна для заказов любой стоимости и из стран СНГ. Обработка банковских поступлений может занимать до пяти дней. Выборка будет доступна после оплаты.

Выборки не облагаются НДС.

## Сохранение реквизитов {#adding-details}

Вы можете сохранить реквизиты, чтобы не вводить их при каждом заказе.

1. Откройте страницу [Дата.2ГИС](https://new.data.2gis.com/account/personal).
2. Нажмите на имя профиля в правом верхнем углу страницы.
3. В блоке **Реквизиты владельца лицензии** нажмите **Добавить реквизиты**.
4. Укажите реквизиты в зависимости от юридической формы:

   * Юридическое лицо: название организации, юридический адрес организации, ИНН и КПП.
   * Индивидуальный предприниматель: название, юридический адрес и ИНН.
   * Физическое лицо: ФИО, email для получения документов и ИНН.

   <img src="/img/data/adding-details.png" alt="Сохранение реквизитов" width="450"/>

5. Нажмите **Сохранить**.
