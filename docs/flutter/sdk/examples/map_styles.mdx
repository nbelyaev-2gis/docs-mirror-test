# Стили карты

Вы можете изменять внешний вид карты и её объектов с помощью стилей. Стили — это правила, которые определяют в каком порядке и в каком виде отображать данные карты.

## Подключение стиля к карте {#connect-style}

Вы можете подключить стили из [Редактора стилей](https://styles.2gis.com/) к карте, добавленной в мобильное приложение. В Редакторе стилей вы можете детально настраивать отрисовку любых компонентов карты с помощью более 200 стилевых слоёв: см. список [готовых слоёв](/maps/styles/layers/configured/msdk).

Чтобы подключить стиль из Редактора стилей:

1. Создайте стиль карты и настройте стилевые слои с помощью Редактора стилей: см. инструкцию [Создание стилей и настройка стилевых слоёв](/maps/styles/create). Для создания вы можете использовать базовые стили с готовыми слоями.
2. Экспортируйте стиль: см. инструкцию [Подключение стиля для Flutter SDK](/maps/styles/connect#flutter-sdk).
3. Переместите скачанный стиль в проект с мобильным приложением.
4. Подключите стиль с помощью метода [loadStyle()](https://pub.dev/documentation/dgis_mobile_sdk_full/latest/dgis_mobile_sdk/StyleBuilder/loadStyle.html) и укажите объект стиля в качестве параметра `style`:

   ```dart
   Future<void> _initializeMapOptions() async {
   // Асинхронная загрузка стиля карты из файла
   final style = await sdk.StyleBuilder(_sdkContext)
   .loadStyle(sdk.File('style.2gis'))
   .value;
   // Создание MapOptions с загруженным стилем
   final mapOptions = sdk.MapOptions(style: style);
   }
   // Теперь MapOptions можно передать в MapWidget
   sdk.MapWidget(
       sdkContext: sdkContext,
       mapOptions: mapOptions,
       controller: mapWidgetController,
   );
   ```

5. Если ранее у вас был подключён другой стиль, выпустите новую версию приложения.

## Изменение стиля карты {#change-style}

Чтобы изменить стиль карты, которая уже добавлена в приложение:

1. Экспортируйте обновлённый стиль из Редактора стилей: см. инструкцию [Подключение стиля для Flutter SDK](/maps/styles/connect#flutter-sdk).
2. Загрузите новый стиль с помощью [StyleBuilder](https://pub.dev/documentation/dgis_mobile_sdk_full/latest/dgis_mobile_sdk/StyleBuilder-class.html) и укажите его в свойстве [style](https://pub.dev/documentation/dgis_mobile_sdk_full/latest/dgis_mobile_sdk/Map/style.html) объекта [Map](https://pub.dev/documentation/dgis_mobile_sdk_full/latest/dgis_mobile_sdk/Map-class.html):

    ```dart
    // Загрузка стиля из ассетов приложения
    final style = await sdk.StyleBuilder(sdkContext)
        .loadStyle(
            sdk.File.fromAsset(sdkContext, 'custom_styles.2gis'),
        )
        .value;
    
    map.style = style;
    ```

3. Выпустите новую версию приложения.

## Переключение темы {#switch-theme}

Стили карты могут содержать несколько тем (например, дневную и ночную), между которыми можно переключаться «на лету» без необходимости загрузки дополнительного стиля.

### При создании карты {#switch-theme-on-map-creation}

Вы можете указать темы при создании карты с помощью параметра [appearance](https://pub.dev/documentation/dgis_mobile_sdk_full/latest/dgis_mobile_sdk/MapOptions/appearance.html) в `MapAppearance`:

```dart
final sdk.AutomaticAppearance appearance;
appearance = const sdk.AutomaticAppearance(
    sdk.MapTheme.defaultDayTheme(),
    sdk.MapTheme.defaultNightTheme(),
);
final options = sdk.MapOptions(appearance:appearance);
```

### После создания карты {#switch-theme-after-map-creation}

Изменить тему после создания карты вы можете с помощью параметра [appearance](https://pub.dev/documentation/dgis_mobile_sdk_full/latest/dgis_mobile_sdk/MapWidgetController/appearance.html) в `MapWidgetController`:

```dart
final sdk.AutomaticAppearance appearance;
appearance = const sdk.AutomaticAppearance(
    sdk.MapTheme.defaultDayTheme(),
    sdk.MapTheme.defaultNightTheme(),
);
mapWidgetController.appearance = appearance;
```

## Управление стилевыми слоями {#manage-style-layers}

Вы можете динамически включать и выключать отображение стилевого слоя на карте. Чтобы включить слой (например, слой с парковками):

1. Инициализируйте глобальную переменную:

   1. Откройте [Редактор стилей](https://styles.2gis.com/).
   2. В блоке **Мои стили** откройте карточку нужного стиля.
   3. В разделе **Слои** выберите слой, который нужно включить.
   4. В блоке **Данные (JSON)** отредактируйте JSON-файл и добавьте нужную глобальную переменную (`global`). Например, для слоя с парковками — это переменная `parkingOn`:

        {/*  prettier-ignore  */}

      <img src="/img/mobile-sdk/add-attribute.png" alt="Добавление глобальной переменной" width="650"  />

      Глобальная переменная может использоваться в нескольких слоях, но только внутри одного стиля.

2. Экспортируйте и подключите стиль: см. инструкцию [Подключение стиля для Flutter SDK](/maps/styles/connect#flutter-sdk).

3. Установите значение `true` для атрибута с помощью метода [setAttributeValue()](https://pub.dev/documentation/dgis_mobile_sdk_full/latest/dgis_mobile_sdk/Attributes/setAttributeValue.html). Например, чтобы включить отображение слоя парковок:

   ```dart
   map.attributes.setAttributeValue("parkingOn", sdk.AttributeValue.boolean(true));
   ```

4. Выпустите новую версию приложения.
