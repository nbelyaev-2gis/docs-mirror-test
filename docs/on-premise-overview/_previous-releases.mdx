:::note[Требования к версии 2GIS CLI]

С сентября 2025 г. новые версии всех сервисов внутри On-Premise могут быть установлены или обновлены только с помощью <Link to="./overview/2gis-cli/releases">последней версии утилиты 2GIS CLI</Link> и с соблюдением обновлённой процедуры установки. Вы можете продолжать пользоваться предыдущими версиями утилиты только для установки версий On-Premise 1.43.0 и ниже.

:::

После версии On-Premise 1.43.0 релизы публикуются отдельно для каждого компонента с собственным версионированием:

* [Базовые сервисы](/on-premise-api-platform/core/releases) (c версии 2.0.0) — включают сервис лицензий и сервис API-ключей;
* [API-платформа](/on-premise-api-platform/api-platform/releases) (с версии 2.44.0);
* [<Brand /> Про](/on-premise-pro/releases) (с версии 2.0.0);
* [<Brand /> Ситискан](/on-premise-citylens/releases) (с версии 2.0.0);
* [GIS-платформа](/on-premise-gis-platform/releases) (с версии 2.0.0).
